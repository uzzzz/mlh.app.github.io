获取免费Windows Store开发者账户方法

  • 柚子社区

 

本周微软宣布Windows Store应用商店正式允许所有开发人员提交应用，包括个人开发者，开发团队等，截至今日Windows Store覆盖率达到120个市场。

在"Windows Store now open to all developers in 120 markets" 一文中提及"All eligible MSDN subscribers receive a free, one-year Windows Store developer account as part of their MSDN benefits. (Eligible subscriptions include Visual Studio Professional, Test Professional, Premium, Ultimate, and BizSpark.)". 所有合法MSDN订阅账户都可以获取为期一年的免费Windows Store开发者账户。本文将介绍如何获取以及激活该免费账户提交Windows 8应用到Windows Store。

 

在阅读本文前，首先需要符合以下条件才能完成帐号申请流程：

1. MSDN订阅账户，权限为Visual Studio Professional, Test Professional, Premium, Ultimate, and BizSpark；

2. 信用卡，注册账户时需要填写；

 

申请步骤：

 

1. 首先登录MSDN订阅网站。


 

2. 点击”Get Code“获取注册码。 这里可以看到在Windows Phone Developer Account中也有"Get Code”，也可以获取一个免费的开发者账户，两者方法类似，不再赘述。


 

3. 点击“Go to the Windows Dev Center”，进入 Windows 开发中心，

 

4. 点击"Dashboard“时，会提示”Help us protect your account“，这时在你指定的邮箱中会收到一份邮件，复制其中的验证码到当前页面，

同时选中"I trust this PC. Use it to verify my account in the future".填写你的开发机器名.

如果邮箱长时间没有收到验证码，点击“I didn't get the code”，系统会重新发送验证码。


 

5. 验证完成后，即可进入账户信息页面。在该页面需要开发人员确认个人信息，另外允许开发人员创建或者链接开发者服务账户。

因为我已经具有Windows Phone开发者账户，所以这里我选择“Use this Microsoft Developer Services Account”。


 

6.  进入Agreement页面，一些合约说明，需要点击“I accept..."后，进入下一步设置。


 

7. 注册费用信息，填写步骤1申请的注册码，点击”Update Total“，即可看到价格被更新为0.00，点击进入下一步设置。

 

 

 

 

8.这一步是付费信息填写页面，这里就不在抓图了，需要准确填写个人或者团队的信用卡信息。因为注册费是0，所以不用担心会被扣钱。完成付费信息 页面，即可进入最终购买页面。

9.进入购买确认页面，在该页面将显示注册费为0和你的个人账户信息。点击”Purchase“继续下一步。

 

 

10. 完成购买后，点击”Dashboard“，即可看到My apps信息页面，由该页面可以上传Apps到Windows Store。

 

 

 

 

随后会收到购买注册帐号成功的邮件。
