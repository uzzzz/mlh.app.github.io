基础 CUP 挖矿教程

如何挖矿？

Step1:获得一个钱包地址

钱包分为两个部分讲,一个是在线钱包,一个是本地钱包.(按需选择)

1.在线钱包

在 XMR 的官方网站下载钱包: Monero:Downloads(点击downloads这几个字就是下载地址,群里也有,下不下来就去官网下)

建议下载 GUI 版本的，比较便于操作．由于我演示用的是 Mac,我就下这个 mac 版的,其他系统的 GUI都是基本相同的，直接下载就行了。

客户端支持中文，选择简体中文。

然后点击创建一个新的钱包(如果之前有钱包的可以用从密钥或者种子码恢复钱包,具体的进阶再讲吧.)

按照要求记录下必要的信息 。

给钱包设置一个密码。

点击侧边栏中的收款，红框中的就是你的地址了。

2.在线钱包

在线钱包的创建

Step1:获得一个钱包地址

在 MYMINERO 的官方网站 My Monero

Step2:点击Create an Account(创建一个账户)

Step3:保存备份好Login Key

Step4:确认Login Key

Step5:钱包已经建好了,红框内为钱包地址.

Step2:寻找一个靠谱的矿池

这里使用的是国内首家CryptoNight自动算法矿池：猫池

目前对比了一下其它的门罗矿池，收益整整翻了一倍。

猫池使用的过程中非常稳定。官方的工作人员，也都不错。有什么问题，都会第一时间帮你解决。

非常的负责！

Step3:挖矿设置

由于是CPU挖矿教程,这里显卡驱动的安装步骤跳过[以后再讲].

下载挖矿软件并设置,这里分为两个软件介绍.

XMRIG

在这里下载挖矿软件：xmrig(点击downloads这几个字就是下载地址,群里也有,下不下来就去官网下)

下载后解压，应该得到三个文件

打开config文件，把文件中的url、user、pass修改成下面的样子，记得修改user为你的钱包地址。

“url” : “mine.c3pool.com:13333”,

“user” : “这里填写刚才你在Step1中得到的钱包地址”,

“pass” : “矿工名:邮箱地址”,（记住哦，矿工名字和邮箱中间有一个:）

然后双击xmrig，如果一切顺利，你将看到类似的界面


我这里使用的是猫池的专用xmrig挖矿工具。是可以自动切换算法。让我们可以挖收益最高的币种算法。

首次打开挖矿程序，程序会自动测算每一个币的算力，然后将结果提交到矿池，再由矿池按照当时收益最高币种的算法，下发到程序，让用户挖掘。

这里解释一下箭头标注的地方，懒人翻译官．

简单将两个比较主要的设置

这个软件默认的捐赠是 5 %,最低为 0 %

“donate-level”: 5, // donate level, mininum 0%

算力打印时间,默认是 60 秒.

“print-time”: 60, // print hashrate report every N seconds

我这里使用的是猫池提供的免捐赠的xmrig程序。

有需要的朋友们，可以去猫池首页，去下载。

xmr-stak

下载软件xmr-stak(点击downloads这几个字就是下载地址,群里也有,下不下来就去官网下)

下载后解压，应该得到6个文件

首次运行xmr-stak的时候，需要配置你的钱包和矿池地址。双击打开软件，你会看到这些提示

其实在这前面还有一张图，没有截图，意思就是如果输入0就退出，其他任意数字字母回车继续

输入monero，回车继续

输入矿池地址：mine.c3pool.com:13333 回车继续

这里填写刚才你在Step1中得到的钱包地址，回车继续

Password (mostly empty or x):这是输入你要设置的矿工名称，如果不想设置，就输入X就好了，回车继续

这里我们直接回车就好了。不需要设置。

Does this pool port support TLS/SSL? Use no if unknown. (y/N) 通讯是否加密。区分大小写，请输入N，然后回车

Do you want to use nicehash on this pool? (y/n) nicehash一般只有代理才会用，请输入n，然后继续回车

Do you want to use multiple pools? (y/n) 是否要再输入一个矿池，输入n，然后再继续回车

按照以上的操作一步步来，然后就开始挖矿了。切换至英文输入法，按下h键，就可以查看到你当前的挖矿速度。

Step4:查看收益

在猫池首页的最下部，输入你的钱包地址，点击查询，就可以看你的挖矿和收益情况
