一个学习网站

链接：https://www.learnopencv.com/

这是在论坛里发现的一个学习网站，这个网站像一个博客一样，定期更新文章。下图右边是这个作者Satya Mallick博士。我第一次打开时，给我的感觉就是：哇！opencv原来能干这么多的事？为什么我之前感觉效果很差？？？一下就激发了我的兴趣

他更新的还是挺多的，看到最下边目前有24页

如果你点击这个网站右上角这个资源，他会给你发来所有的学习资源。这个在他GitHub上：https://github.com/spmallick/learnopencv

还有他的dlib项目https://github.com/spmallick/dlib

项目很吸引人，不错，还有码！

我的opencv环境安装-再次推荐python

我用opencv是在Anaconda python环境下使用的，安装很方便，不用像VS里面还要设置工程路径。最简单直接使用pip insatll opencv或者conda install opencv

或者要是开发还是用opencv contrib版本。

从清华库中下载https://www.lfd.uci.edu/~gohlke/pythonlibs/

直接pip install opencv_python‑3.4.4+contrib‑cp36‑cp36m‑win_amd64.whl
