区块链内的十大强强联手

 

 

 

2018年对于区块链是多事的一年，尽管存在熊市，但还是有许多区块链项目启动，增加了工作人员，与其他公司合作（项目），收购其他公司，根据政策改变了规定，并进一步开发区块链技术的解决方案。

从2018年第一季度和第二季度对加密市场的分析来看，该行业的整体发展速度是渐进的，尽管速度很慢。法规确实放慢了ICO的步伐，但这个行业受到监管是一件好事。这是因为机构资金在真正受到全面监管之前无法正式流入加密行业。

对一个区块链项目来说，它必定是为解决某个痛点而生，因不同的资源，技术，视角，需求，通常会有企业在某一个区块链项目上与其他企业进行合作。虽然基本目标都是为解决问题而建立一个项目，但具体目的可能有所不同，所找的合作伙伴也有所不同。

下面列出了区块链和加密货币今年的十大顶级企业合作，没有特别的顺序。

 

IBM和Stellar

 

IBM和Stellar之间的关系可以追溯到2017年10月。在最终决定使用Stellar之前，IBM 曾经在与比特币，Ripple和Stellar之间考虑。最终因为非盈利性，透明度和可扩展性，Stellar被选为候选人。

IBM为外包技术集成的机构提供分布式分类账解决方案。虽然他们为专用网络实施Hyperledger，但Stellar是他们对公共网络的选择。IBM目前在Stellar网络上运行9个验证器。

IBM目前使用的通用支付解决方案是使用 Stellar Lumen（XLM）作为跨境交易的桥梁。但并不仅限于此，对在平台上部署的其他资产也持开放态度。IBM 目前正在考虑通过加密市场的波动来稳定使用稳定币。

最近，IBM发布了区块链全球支付系统：Blockchain World Wire，其目标是使用Stellar协议在几秒钟而不是几天内完成跨境支付。此外，IBM 透露与 Veridium Labs 的合作伙伴关系，以确定碳排放额度，使公司能够追踪他们的碳足迹。

 

litecoin和tokenpay

 

 

2018年7月10日，分散支付平台TokenPay 收购了德国 WEG银行的9.9％，以及优先购买剩余90％（大约）的权利，但须遵守惯例监管部门的批准。该银行9.9％的股权已转让给Litecoin基金会。

Litecoin将帮助TokenPay开发自己的区块链，确保其稳定性和未来的成功。反过来，TokenPay会将Litecoin整合到WEG银行并进入他们的商业服务平台。

 

binance和马耳他证券交易所

 

 

2018年9月11日，Binance与马耳他证券交易所(MSX)签署了一份谅解备忘录，启动了一个新的安全令牌数字交易所。这意味着Binance可能会在马耳他推出世界上第一个安全令牌交易平台。

马耳他经常被称为区块链岛，因为它非常支持加密，Binance在今年早些时候将业务转移到该岛后，与政府保持着良好的关系。

马耳他致力于为加密行业建立一个适当的监管框架。MSX PLC是由马耳他新成立的，旨在为区块链岛开发金融技术和数字资产。

 

AWS和Qtum

 

 

Qtum于2018年10月17日宣布与亚马逊网络服务(AWS)中国区合作。该合作伙伴关系旨在为企业和开发人员开发区块链即服务解决方案。Qtum将在AWS上提供一个智能合同开发平台。

通过包含Qtum Core、solid和Qmix web IDE的Amazon Machine Image (AMI)，开发人员和用户可以高效地创建和启动智能契约。

Qtum基金会CIO Miguel Palencia表示:Qtum在AWS市场上的发布为终端用户和企业提供了一个易于使用和强大的基于云的解决方案。任何想在Qtum平台上开发和构建dapps或者将其用作节点的人都将从中受益。

 

琼斯传媒集团和Brave Browser

 

 

琼斯传媒集团和Brave Browser早在2018就有了很多的合作。2017年底，Brave Browser大约100万的活跃用户。同时，在2018还没有结束，平台上已有400万活跃用户和超过1000万的下载。

2018年4月，道琼斯传媒集团宣布与Brave合作，为用户提供优质内容，并测试区块链支付技术。可用的出版物包括Barrons.com(完全访问)和MarketWatch(高级访问)。根据合作协议，这两家出版商都是BAT平台上的认证出版商。

 

ICON和Line

 

 

ICON是一个专注于区块链互操作性的区块链项目。Line是一个月活跃用户超过2亿的社交媒体平台。

2018年5月15日，ICON宣布将与Line合作开发自己的区块链网络。这个合资企业被称为Unchain。Unchain将是Line的区块链生态系统的开发者组织。

Line和Icon的合资企业预计将把更多的dapps和用户连接到Icon网络，因为Icon将利用Line那超过2亿的用户。

 

万维链和MakerDAO

 

 

万维链是一个区块链项目，专注于区块链的互操作性(或交互和通信)。

MakerDAO是一个生产稳定货币Dai的项目，该货币与美元的汇率是1:1。MakerDAO不再像Tether那样在一个银行账户中拥有相当于所有Dai的美元，而是使用一个仅在Ethereum持有的担保债务凭证系统，以将Dai的价值维持在1美元。

2018年10月11日，万维链宣布Dai为第一个跨链ERC-20代币。通过万维链的跨链基础平台，Dai可实现与比特币之间的去中心化交易，并允许Dai将比特币和以太坊作为其抵押物（MakerDao正在推进Dai的抵押物多样化）。万维链和MakerDAO之间的这种双向关系有望为平台和各自的货币都带来巨大的价值。

 

Coinbase 和 Circle

 

 

到目前为止，你会听说Coinbase在其平台上列出了一个名为USDC 的新稳定币。USDC是着名的加密货币钱包和交易所 Coinbase 与高盛支持的科技公司 Circle 之间合作的产物。

这个合作伙伴创建了Centre，这是USDC的创举。预计这种合作关系将继续生产更多稳定的法定货币，将所有人与金融服务联系起来。

Centre计划为英镑，日元和欧元生产稳定币。它们每个都有望在Coinbase上市。

 

binance和Tron

 

 

Binance可以说是世界上最大的加密交换机，该平台在2018年一直在蓬勃发展。

2018年10月4日，Binance宣布了Binance区块链慈善基金会及其支持开发计划署的可持续发展目标。Blockchain慈善基金会由Binance创建，旨在探索区块链技术在社交方面的应用。

Tron与Binance共同捐赠并致力于非营利部门的发展。

Blockchain.com和Ledger

 

 

最后介绍一下最早的加密货币网络钱包提供商之一Blockchain.com与Ledger合作开发了一款名为Lockbox的新硬件钱包。

该设备与Blockchain.com钱包兼容。预计它将使Blockchain.com的用户能够更灵活地将一部分资金放在一个冷钱包里，同时保留网络钱包的选项，以方便访问。

Lockbox可以兼容Windows 7或更高版本，MacOS 10.9或更高版本，Linux和Chrome OS。Lockbox与其他分类产品不兼容，因为它运行在定制的Blockchain.com固件。

这款手机价值99.99美元，预计将于11月中旬开始出货。

 

最后的思考

 

这只是2018年在加密世界中发生的少数顶级合作。这些合作关系中的大多数都是实验性，因为区块链世界之外的许多组织仍在等待区块链行业的验证。

虽然有些合作关系直接影响到加密项目的价格，但另一些是进步的证明。但在任何情况下，这些合作关系都是衡量区块链项目在区分其解决方案与市场上其他方案之间的努力的一种方式。
