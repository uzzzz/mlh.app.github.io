PascalCoin是一个非常创新和特别的全新的虚拟货币，它的钱包地址是通过挖矿产生的并不是像Btc那样可以随意生成，目前每一个区块产生会奖励5个地址和100pasc，使用钱包必须要有一个地址，具体获取地址的方法请看另一篇文章：PascalCoin钱包账号(地址)获取方法

钱包安装太简单了，在此略过。

打开钱包后界面如下：



1、公钥私钥

了解虚拟货币的人应该都不陌生，通俗的说，公钥就是可以公开的对外使用的key，私钥则是私密的key相当于密码 是你拥有这个账号的唯一凭证，必须保密好备份好，丢了就没了。

首先生成一个密钥 Project -> Private keys -> Generate a new Key，Name随意，选择Key Type密钥类型，有4种长度可以选择，点ok生成密钥。

选中刚才生成的密钥就可以看到下面框里的Private Key私钥，把它备份保存好，如果觉得直接备份密钥不安全可以选择Export Private Key输入密码进行加密保存（密码一定要记好了不然就没了）。

恢复密钥时选择Import Private Key 输入你的密钥，如果是加密过密钥的软件会自动识别并提示你输入密码。

选中密钥点击Export Public Key 生成对应的公钥并且复制好，公钥与地址绑定，在获取地址时填写这个公钥。

点击Lock 设置密码以免被人盗用（密码同样需要记好 忘了密码钱包里的密钥就找不回来了），每次启动钱包交易都需要输入密码解锁。




2、使用钱包交易pasc

通过前面的教程相信已经拥有了自己的pasa（钱包地址），拥有了地址才可以使用钱包交易，在这里我们会发现地址分为两节以 - 分开，前面的112800是账号 -后面的98相当于验证码用于防止输入错误的，在转账时输入112800软件会自动追加后面的-98验证，在挖矿时候也不需要-和后面的。



2、转账

Operations -> New single Openation    Transaction 按照下面图片设置好 Execute确认交易，等待几分钟网络确认交易。



3、更改地址密钥（转移地址，交易地址）

Operations -> New single Openation      Change Key

pasc地址是与密钥绑定的，通过更改地址的密钥就可以转移或者交易地址。

勾选转移的类型选择密钥或者输入公钥，输入费用，同样是费用越高速度越快，确认交易，等待几分钟网络确认交易。
