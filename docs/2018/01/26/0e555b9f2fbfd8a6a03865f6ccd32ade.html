一、go安装

https://www.golangtc.com/download

直接下载，安装，默认安装位置/usr/local/go

 

将/usr/local/go/bin 目录添加至PATH环境变量

vi ~/.bash_profile ，有就直接打开，没有就创建一个新的。

export GOPATH=/Users/liuhanlin/GO

export GOBIN=$GOPATH/bin

export PATH=$PATH:$GOBIN

 

加入以上的环境变量控制，最后保存

source ~/.bash_profile

 

检查搭建情况

goenv

出现waring,需要把gopath 设置一下，并且不要和goroot 设置成同一个目录

 

配置gopath，也就是go项目的代码存放位置

mkdir -p /Users/liuhanlin/GO

·       其次需要vi ~/.bash_profile ，有就直接打开，没有就创建一个新的。

export GOPATH=/Users/liuhanlin/GO
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN

加入如下环境变量控制，最后保存

source ~/.bash_profile

再次检查go的搭建情况

go env

此时gopath和goroot目录不一样了，此时也不再报warning了
