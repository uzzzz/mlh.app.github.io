CesiumJS环境搭建

  • 柚子社区

一. 准备阶段

首先你要下载Cesium (当前版本1.41,下载地址：https://cesiumjs.org/downloads/)

和NodeJS安装包，（当前9.3.0, 下载地址：https://nodejs.org/en/）


二 搭建WEB服务器

这里有两种方法，一种是官网引导教程中的NodeJS,另一种是Windows自带的IIS。这里先介绍NodeJS的安装使用。

2.1 运行安装文件，注意选择安装位置最好选择一个没有中文和空格等特殊字符的路径。


2.2 测试NodeJS，在任意位置打开命令行并分别输入:

node -v
npm -v


能查看到则说明安装成功【npm是随同NodeJS一同安装的包管理工具，当Node.js安装完成后，npm理论上也已安装】。

2.3 解压你的Cesium压缩包，然后进入该文件夹根目录，按住Shift同时右键，打开命令行输入：


npm install



三. 测试，安装完成后输入

node server.js

 

 

测试，输入网址http://localhost:8080/Apps/HelloWorld.html

 

OK

【说明】

· 上面的教程中的NodeJS部分仅针对于Cesium环境搭建，NodeJS还有超级多功能用法这里不做详述。

· 如果NodeJS安装有问题(尤其是npm -v时无响应)时，请删除 C:\Users\[你的用户名]\AppData\Roaming和 C:\Users\[你的用户名] 两个路径下和npm有关的文件后重新安装NodeJS。

· 不建议为计算机添加过多环境变量。因此，每次做Cesium相关开发时，请先进入Cesium根目录，开启NodeJS服务(参照第三步)。

· 第二步中提到两种方法,另一种就是使用Windows自带的IIS进行建站，这样就不需要安装NodeJS了，参考另一篇。
