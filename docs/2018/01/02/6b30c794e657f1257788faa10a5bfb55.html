写在前面

在尝试搭建属于自己的RSS网站之前，我使用的是Feedly+Nextgen Reader 组合，自建之后，几乎就不用原来的组合了，不是说原来的组合有问题，只是想把一切都交由网络来完成

相比Tiny Tiny RSS传统的界面，Miniflux其实更对我的胃口，不论是安装和使用，都比前者方便许多
官方是这么说的：

Miniflux is a minimalist RSS reader
Miniflux is not for everyone, it’s for people who want to read their feeds efficiently


它的界面也确实像官方说的一样，简洁又高效

这里是Miniflux的官方网站
这里是官方安装指南
本次要安装的是Minimalist RSS reader (version 1.x)
Miniflux 2 是go语言的重制版本，正在开发中

由于安装方法和上一篇文章非常相似，故若对本文有疑问，烦请去上一篇文章一看

第一步 准备

完成这篇文章里做的事情
特别的，PHP版本须在5.3.3及以上，推荐7.x
下面的是官方要求

PHP >= 5.3.3 (PHP 7.x recommended)
PHP extensions: ctype, dom, hash, iconv, json, xml, mbstring, pdq_sqlite/pdo_pgsql, pcre, session, SimpleXML (curl is recommended or set allow_url_fopen=On)
Sqlite 3 or Postgres >= 9.3
libxml2 >= 2.7.x

第二步 安装

下载https://miniflux.net/miniflux-latest.zip ，解压并删除源文件

第三步 使用

访问http://你的解析好的域名/miniflux/
Username和Password都是admin

登陆进去后点击Preference
在general里更改时区语言及其他偏好设置
在user里新建管理员账户，退出当前账户并用新建的管理员账户登陆，重新回到Preference->user，删除原有的admin账户

支持https的feed，支持opml导入和导出，支持css
一切都很简洁，却能满足所有的基本需要
