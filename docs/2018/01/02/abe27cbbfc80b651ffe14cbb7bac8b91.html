英格兰银行或在2018发行加密货币

委内瑞拉推出了Petro的数字货币。俄罗斯则正在研究加密卢布。那么是否还会出现Britcoin这种属于英国的加密货币呢？还是Ukoin？英国《电讯报》（The Telegraph）首席记者Robert Mendick就对读者给出了肯定的答案。据Mendick报道称，英格兰银行行长马克·卡尼（Mark Carney）“在圣诞节前对财政部特别委员会（Treasury Select Committe）表示，他曾与其他国家的央行就发行数字货币事宜进行了商谈。”

Mark Carney

这位52岁的行长卡尼是英格兰银行有史以来的首位外籍行长。卡尼曾任加拿大银行行长，并在高盛工作超过十年。就在几天前，卡尼证实比特币“是其关注的活跃领域”，而且最感兴趣的似乎就是其底层交付机制，去中心化分布式分类账本。他表示，比特币的零售潜力并不在其关注范围之内，他告诉英国立法者“这项技术最后不会以带来这些金融稳定风险而告终，你会得到金融稳定性的有利条件。而且会节省大量计算能源的强度。”

他接着形容英格兰银行对比特币的研究“十分的积极，但也相当的遵守纪律。如果我们要在系统的核心应用这些东西的话，那就需要达到五西格玛的质量评级。”


研究加密货币

他最终的评论或许为《电讯报》的独家报道埋下了伏笔，“今天我就这一主题发表的讲话可能在6个月之后就不适用了，因为事情发展的太快了。”确实，这家英国报纸在对该主题的报道中开篇便写到：“据《电讯报》了解，英格兰银行或将在2018年就对发行类似于比特币的数字货币计划放行。”

而明年年初将继续举行会议对此进行探讨，他证实道：“我已经参与到了与多国主要央行就发行央行支持的数字货币的问题的讨论之中。”而就在今年夏天，英格兰银行使用区块链技术进行了测试，来了解这项技术是否能够用于各央行之间的结算。测试的结果是成功的。据报道，他向财政部特别委员会（Treasury Select Committe）解释道：

“实际上这项基础技术相当有趣。我们英格兰银行正在研究这项技术。”

去年春天，英格兰银行委托伦敦大学学院的George Danezis博士设计了一个由国家支持并掌管的加密货币。由此便诞生了RS Coin，而由于其涉及到银行业，这个货币确实也带来了许多与自动化有关的效率：央行在任何时候都可以毫不费力地控制货币的供应，而且无疑可以确定流通中的货币数量。

George Danezis

虽然卡尼可能认为英格兰央行数字代币的潜力尽可用于央行之间的交易，但是《电讯报》的报道却认为，“由该银行设立的研究部门正在研究推出与英镑挂钩的加密货币的可能性。”很难想象传统零售银行会同意这种可能的实现，因为这注定会将他们推向终结。传统银行业是无法在无摩擦、动动手指就能够交易房产、汽车等资产的世界中生存下来的。

本文仅代表作者个人观点，不代表区块链铅笔的立场，不构成投资建议，内容仅供参考。
