一：必备的账号

访问GitHub上的 NVIDIA GameWorks 前你需要具备一些条件，可在 NVIDIA GameWorks and UE4 中找到（如果你会浏览器的用网页翻译，或英文可直接点击如果不会则请参照下方的1.1~1.3处理。）
1 访问前需要一个订阅了GitHub 的Unrealengine账户
1.1如果没有GitHub账户的话，英文不好的请搜索： 注册GitHub账户，能搜出一大把的教程！
动手能力强些的点击 UE官方教程
英文好的直接点击 GitHub （GitHub官网） 自行创建好账户
1.2 没有Unrealengine账户的同1.1 方法搜索或点击 Unrealengine账户 创建

1.3 UE账户的GitHub订阅，订阅后 可以下载UE4的完整源代码

2.1 完成 UE账户的GitHub订阅后便可访问在GitHub上的 NVIDIA GameWorks 了

二 下载 NVIDIA Game works相关的 分支


左边的蓝色字体是NVIDIA官网介绍，右边的branch是GitHub上的分支。下载的话 点branch就行。

VXGI为例子

下载时建议用有会员的迅雷，或百度云盘离线下载，或翻墙，印象中直连速度很慢。下载后解压到一个至少60GB空余的硬盘（编译后41GB）
点击根目录的Setup.bat 下载，若下载速度慢 可以搜索 UE4 Setup.bat加速
编译的UE4源码方法可以百度（有时间再更新）。
一点经验： x5645，2.5G 6核心12线程 24g内存 VXGI 编译一次两个半小时。
