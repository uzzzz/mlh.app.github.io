根据10月23日公告，软件开发公司甲骨文发布了一套基于区块链的软件即服务（SaaS）应用程序，基于Oracle区块链云服务。新产品据称旨在改进整个供应链的可追溯性和透明度。

Oracle是一家成立于1977年的美国公司，目前是商业软件市场上最大的供应商之一。

根据2018福布斯名单，甲骨文2018年总收入398亿美元，是全球第二大软件开发公司。

这款名为Oracle Blockchain Applications Cloud的新产品包括四个应用程序：智能追踪，批次血统和来源，智能冷链以及保修和使用跟踪。

与许多基于区块链的供应链管理工具一样，Oracle区块链应用程序声称能够让客户通过供应链跟踪产品，提高透明度，加速产品交付并提高客户满意度。据报道，该产品可以让用户跟踪产品组件的真实性和温度控制，并减少纸张浪费。

最初，甲骨文宣布计划在5月推出平台即服务产品和分布式分类账本应用程序。 当时，甲骨文已经与尼日利亚政府合作，据报道，尼日利亚政府正在寻求利用区块链技术记录海关和进口关税。

Oracle Blockchain Cloud Service的普通版本已于7月得到Oracle的正式确认。 该产品侧重于交易效率和供应链认证，使用Hyperledger Fabric作为其基础。 在此次发布之前已针对银行，商业和政府客户进行了一系列试验。

本文仅代表作者个人观点，不代表区块链铅笔的立场，不构成投资建议，内容仅供参考。
