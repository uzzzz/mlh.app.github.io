一、平台介绍
Tushare Pro是在Tushare 原版基础上的一个更具特色的金融数据平台。相比于原版，Pro版数据更稳定质量更好了，但Pro依然是个开放的，免费的平台，不带任何商业性质和目的。
从数据的重新梳理规划，服务流程的规范，到全新数据平台的实现，工程师们及粉丝都付出了不少努力，可以说Tushare是所有人的Tushare，是社区所有用户共同参与的Tushare！
TusharePro版数据内容将扩大到包含股票、基金、期货、债券、外汇、行业大数据，同时包括了数字货币行情等区块链数据的全数据品类的金融大数据平台，为各类金融投资和研究人员提供适用的数据和工具。
二、下载及安装
方式1：pip install tushare，如果安装网络超时可尝试国内pip源，如pip install tushare -i https://pypi.tuna.tsinghua.edu.cn/simple（推荐方式1）
方式2：访问https://pypi.python.org/pypi/tushare/下载安装
方式3：访问https://github.com/waditu/tushare,将项目下载或者clone到本地，进入到项目的目录下，执行： python setup.py install
