Qbao Network CTO 何鹏飞：始终保持对未知世界的好奇心

  • 柚子社区

初见何鹏飞先生，只见他身着格子衬衣，脸上露出谦和的笑容，很难想象眼前这位阳光帅气的年轻男士竟然已经有十多年系统架构的工作经验，还平均每天要在四块大屏幕后工作时长达12个小时。
10月24日，Qbao Network 即将迎来创办一周年的生日。近日，金色财经采访了Qbao Network联合创始人兼CTO何鹏飞，了解这款在区块链行业广受好评却一直低调行事的多功能数字货币钱包背后的故事。

Qbao Network联合创始人兼CTO何鹏飞

10月24日，三个IT男的程序员情结
“老陈、若禹和我是大学同学，在上海华东师范大学计算机系上学时就是很好的朋友。毕业时流行去外企工作，我们三个分别去了微软、百胜和惠普从事技术开发工作。”何鹏飞回忆起当时。
他口中的老陈，陈琳，Qbao Network的创始人，有着10年互联网产品与开发经验，也是区块链技术早期的探索者和布道者。孙若禹，Qbao Network COO，区块链行业专家，有着10年互联网产品运营经验。
据何鹏飞回忆，三人在2016年接触到区块链技术，带着对未知世界的好奇，系统地研究了区块链的底层技术和可能的相关应用，最后三人得出一个结论：区块链技术尚处于早期，谈不上大规模的应用。而数字货币钱包应该是未来区块链世界的流量入口。
有了这个结论，三人决定一起做一款数字货币钱包。
“老陈第一个辞职出来创业，我和若禹先是一边工作一边兼职开发。时机成熟后，我们都辞了职。”2017年7月，三人在共享空间租了三张桌子，就这样正式加入了创业大军，开始了创业之旅。
三个月后，Qbao Network诞生，三人决定在10月24日上线1.0版本。“也算是我们作为程序员的情结吧。”何鹏飞调侃。
创业维艰，如何完成比难更难的事
Qbao Network的诞生是在区块链行业极度疯狂的时候。去年第四季度区块链第一个“小牛市”到来，Qbao Network 一上线就很快迎来了新用户注册狂潮。
然而，这种狂热背后也凸显出了Qbao Network的技术短板。“我们最初的产品架构和服务器根本没有做好应对这么多用户的准备。”何鹏飞介绍，今年1月份Qbao Network运营部策划了注册Qbao Network 送币的推广活动，导致大批羊毛党蜂拥而至。热情的用户混着羊毛党和机器人，Qbao Network 在2018年1月14号一天的注册量达到了140余万。“我们的服务器全部崩溃了，后来我们把Qbao Network 的后台改为了模块化的微服务系统，又升级为混合云服务器才解决了这个问题。”
区块链是无国界的，Qbao Network 同时支持Android 和IOS 两种系统，有7个语言版本，用户遍布很多国家。而每个国家都有自己独特的网络运营商，用户手机的型号也有上千种。
“Qbao Network App 每个月更新1-2个版本，测试部门人力有限、用于测试的手机机型也只有几十种，每次上线前加班加点测试，但还是会有些小bug测不出来，正式上线后用户用了才发现。这不仅影响客户体验，也给Qbao Network各个部门造成巨大的压力。”为此，Qbao Network开启了封闭内测机制，每个版本上线前都邀请不同国家不同型号手机的用户先测试几天，发动社区力量群策群力才解决了这个问题。

往事历历在目，这一年的创业经历让何鹏飞成熟了很多。“当初我想独立做一款深受大众喜爱的App，现在看来这个想法有些幼稚。我在真正实践过程中才意识到做一款App，尤其是应对高并发量的App 是一件多么不容易的事情，靠一个人的力量是不可能做到的。”何鹏飞坦言，“创业的过程有点像升级打怪兽，每一步都不太容易。”
创业是个很艰辛的过程。在创业过程中经历了前所未有的困难、压力甚至是痛苦。但是在这过程中，何鹏飞实现了自己想要成为全栈工程师，创造一款真正由自己主导的App的愿望。
“现在回忆起Qbao Network一路走来的经历，发现所有的困难和挑战其实都是一种考验，在攻克一个个挑战的过程中，Qbao Network 成长为一款性能稳定、功能全面的产品，这让我们团队所有成员都很欣慰。”
未来，道路虽然曲折，但终会走向光明
从技术的角度看，何鹏飞认为区块链技术还处于早期阶段。“现在公链非常多，每个公链都喜欢强调自己的TPS（Transaction per Second）有多高，但是过分强调TPS 没有任何意义。”
何鹏飞解释道，根据CAP 定理：一个分布式系统不可能同时很好的满足一致性（Consistency），可用性（Availability）和分区容错性（Partition tolerance）这三个需求，最多只能同时较好地满足两个。所谓分布式系统的TPS 高，无非是优先满足了一致性和分区容错性，同时一定程度地牺牲了可用性。
“在我看来，既然如此，那些需要高TPS 的运算不如就交给中心化的网络去处理，并不是什么都是去中心化的好。”何鹏飞说。
何鹏飞认为，区块链技术是一项足以改变人类生活的新技术，前景还是光明的，但是发展过程中难免有曲折。“我喜欢历史，一切历史其实都是当代史。纵观历史的发展，不难发现一个行业想要有突破性发展，必然是为人类社会做出一定贡献的，或者提升了效率，或者带来了财富，又或是深刻地改变了人类的日常生活。”
这就像互联网泡沫前后的公司业态。上世纪90年代，市值高的互联网公司主要是PC 端的门户网站如雅虎、搜狐等，而泡沫破裂后重新崛起的互联网巨头都是以移动端的互动式交互为主的，如Facebook、Amazon 等，这些公司在移动端和通讯网络技术的发展的大背景下极大地改变了人们的生活方式，才有了市值的大幅增长。
“区块链技术离真正发展成为从根本上改变人类生活的革命性技术还有很长的路要走，底层的问题没有解决，仅仅只是炒作概念是没有任何实质意义的。”何鹏飞强调。
在采访最后，谈及当前数字货币钱包的激烈竞争，何鹏飞依旧流露出自信的微笑。在他看来，每个行业都存在竞争，激烈的竞争才会带来更大的动力。何鹏飞表示Qbao Network 作为一款起步较早的数字货币钱包，在技术层面是积累了一定优势的，未来Qbao Network 会继续踏实地积累，同时希望能与更多优秀的竞争对手进行切磋、交流，加速自我变革和创新，努力打造一款区块链世界的平台级应用。
