       lz是基于anaconda3的情况，安装的pyecharts。

很明显的了，上来首先下载pyecharts这个包。

       这里顺便详细说明一下吧，因为有些兄dei可能不知道去哪里下载一些包，顺便扯两句：

       1. 在网站（https://pypi.org/）上，想要下载那个包，就先输入包名（在截图红框标出来的地方），然后enter搜索咯。

       2. 然后选择对应的版本，左键点开如下：


       这里lz选择第一个，点开得到如下：


       接着，点击红色框框哪里，得到如下：


       最后就是选择红框里，你所要下载的对应的版本了。（对于其他的第三方包，都可以通过类似的方法来搜索下载。）

       从这里面下载的包，一般有两种形式吧，一个是whl文件，一种是一个压缩包（解压后得到一个文件夹）。两种形式，安装的方法也不一样。

       对于whl文件，我们只需要在开始菜单里，打开Anaconda Prompt

在里面输入‘’pip install 绝对地址‘’（如下）


       至于绝对地址是什么，可能一部分朋友会问，如果不知道，那就找到你要安装的包，然后右键查看属性，如下图


复制红色方框里的字，按1、2顺序，用一个“\”连接起来即可（按上述截图即是： C:\Users\Administrator\Desktop\pyecharts-0.5.5-py2.py3-none-any.whl）

       而对于另一种形式的文件，安装就是另一种方式了，而且简单得多。解压后的文件夹里，有一个setup.py的文件，只需在当前文件夹打开命令行，可能有小伙伴问，怎么在当前文件夹打开命令行啊？只需要在这个文件夹空白地方，按住shift键，再右键即可弹出，然后输入 python setup.py  install即可安装。


       言归正传，输入好之后，只需要enter执行，就可以安装对应的包了。但是很不幸运，可能提示需要安装其他的包才可以安装pyecharts。没关系，需要什么其他的包就下载那个包，一一安装就可以。lz在安装pyecharts的时候，安装了十多个其它第三方包才安装成功啊。
