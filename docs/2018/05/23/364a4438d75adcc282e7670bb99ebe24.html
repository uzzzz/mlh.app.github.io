2018苹果开发者账户忘记密保问题，导致协议无法更新解决方法，

开发者中心提示：

1.

In order to access certain membership resources, you must accept the latest license agreement. First, you’ll need to update the mobile phone number associated with your Apple ID. Your mobile phone number must be based in your country/region. Edit this number in the Account section of appleid.apple.com, and return to your developer account to continue.

2.Program License Agreement updated

The ‘Apple Developer Program License Agreement’ has been updated. In order to access certain membership resources, you must accept the latest license agreement.

很多开发者的开发者账号不是自己注册的，并且注册的人已经消失了，导致密保忘记。

而此次开发者账户协议更新后开发者必须更新开发者账户信息，完善手机号才可以继续接受协议。

那么忘记密保了怎么办呢。

1.继续使用老账号。（此方法不一定对所有账号都可用）

在手机上登陆开发者账号，然后在手机上添加手机号。然后再次登陆苹果开发者中心，更新手机号的提示消失，直接点击接受新协议即可。

2.更改开发者账号。（推荐此方法，直接改账号比较彻底，避免苹果以后又出什么幺蛾子，因为没有密保很多事情做不了）

很多开发者使用第一种方法后发现开发者中心的提示仍未改变，还是要更新手机号。那么怎么办呢。

首先我们要准备另外一个appleID，作为新的开发者账号，请自行注册，千万要记住密保了。

老得开发者账号登陆开发者中心https://developer.apple.com/account/，点击左侧的people

新账号点击接受邀请

接受邀请后，退出登陆，登陆原账号，点击左侧membership


点击左侧membership

往下拉。点击点击transfer team agent role

新协议
