pycharm安装第三方库


首先启动pycharm


随后通过快捷键进入setting界面，快捷键为ctrl+Als+S


点击右上角的加号，添加第三方库



在安装之前，首先点击Manage Repositories


增加一些国内的镜像源，包括

豆瓣http://pypi.douban.com/simple/

阿里的，http://mirrors.aliyun.com/pypi/simple/

https://pypi.mirrors.ustc.edu.cn/simple/

清华的 https://pypi.tuna.tsinghua.edu.cn/simple/

最后如图所示



随后以安装umysql为例，搜索umysql


点击install即可。

安装成功后如图所示



如果遇到信任问题，可以在options中添加

诸如–trusted-host pypi.douban.com

pip install requests --trusted-host pypi.doubanio.com

或者本地安装 pip install -i https://pypi.tuna.tsinghua.edu.cn/simple some-package
