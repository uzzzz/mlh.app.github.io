区块链钱包新玩法，YYCoin构建糖果、钱包、应用一体化生态

数字货币钱包，顾名思义就是存储和使用数字货币的工具，如同支付宝之于移动支付，数字货币钱包一直是数字资产流通不可或缺的基础应用。区块链技术的更新迭代和蓬勃发展，持有数字货币的人越来越多，数字货币钱包市场的竞争也愈演愈烈。对于总市值稳定在千亿美元级别的数字货币，每个投身区块链的创业者都关注着被称为“未来数字资产交易的关键入口”的数字货币钱包市场，那么数字货币钱包市场现状究竟如何？又有哪些新的玩法值得关注？ 

钱包用户占持币用户不足一半  行业格局未定

由于数字货币的匿名性和钱包创建的简易性，持币人数准确度无处可查。但有数据显示，目前全球持币人数大约5000万，中国持币人数在500~600万，占总人口比例还是很少，这一情况决定了该领域还处于增量市场。

 

全球区块链钱包用户数（数据来源：Statista）

从市场研究公司Statista公布的数据来看，2015年第一季度全球区块链钱包用户数仅317.77万，2018年第一季度该数字则增长到了2395.28万，用户数净增长超2000万，约占全球持币人数约50%，覆盖率依然有待提升。

 从另一角度看，市面上的数字货币钱包种类繁多，有单资产钱包、单区块链钱包、多区块链钱包、冷热钱包、非确定性钱包、确定性钱包、全节点钱包、轻节点钱包、托管钱包等，操作难易程度不一。

虽说市面上用于管理数字货币资产的钱包种类繁多，但它们的共性是功能单一，支持的数字资产种类十分有限。用户往往要切换多种钱包产品，才能管理所有的数字资产。这不仅极大程度地破坏了用户体验，还给用户的资产安全带来风险。

此外，据相关机构不完全统计，目前已知的国内数字货币钱包已有超过20个，在全球则有上百家。然而，国内用户最熟悉的只是imToken、Blockchain等少数产品，同时近期发展正盛的Kcash、Cobo也大有弯道超车之势。

其中，作为目前数字货币钱包市场王者的imToken月活跃用户达到了200万，这个数字据称已经超过了市场全部体量的50%。后起之秀Kcash的用户数也已接近百万，而距离Kcash上线仅仅过去了一年。 

不过，仅从用户体量与产品成熟程度来说，目前还没有一款真正的区块链产品落地。从没有一家独大的态势也说明，新进入的创业者还有机会。

营销手段层出不穷  YY Coin定义新玩法

随着市场越来越成熟，互联网的玩法也逐渐被引入数字货币领域。比如钱包已经从单纯的资产存储功能发展到支持POS收益分成、引入理财产品、允许币币交易闪兑、行情信息聚合、DApp交易跳转等等。

另一方面，新的营销手段也层出不穷，比如Kcash此前推出的为1亿人免除转账手续费、imToken的闪兑大赛、征文活动等等。

 

​YY Coin

 近日，来自深圳的洞悉网络科技有限公司也宣布进军数字货币钱包市场。他们推出的全新数字资产管理平台——YY Coin结合了“撸糖果”、游戏娱乐应用等新玩法，致力于打造基于数字货币钱包的游戏娱乐撸糖果平台。

作为币圈最活跃的群体之一，撸糖果人群一直处于边缘甚至是被忽略的状态。但随着越来越多的项目开始采用发放糖果的方式进行营销推广，撸糖果人群规模也开始凸显出快速增长的趋势。而数字货币钱包用户和撸糖果人群存在极大的重合度，因为用户获得的项目糖果通常存储在各个数字货币钱包之中，而加入撸糖果大军的新用户也需在钱包应用创建地址才可领取糖果。由此看来，嵌入在钱包当中的糖果平台具有极强的用户聚集能力。

YY Coin还计划在未来接入更多的娱乐游戏应用，通过玩游戏获得糖果的方式，形成“糖果+游戏”的应用生态闭环。 

此外，基于数字货币钱包，合作的项目方还可以在YYCoin应用中建立一个社区生态体系。一方面社区内部可以自行衍生出各种社群，另一方面社区成员之间也形成了相应的动态社交网络，该网络可以同时实现信息传递和价值传递。它们相对独立又彼此关联。 

从功能上看，YY Coin与市面上常用的产品保持着一致步调，支持主流资产、多链资产一体化，可实现资产与资产、糖果之间交换，以及资产的自动化添加，省去手动添加等复杂操作，安全保护用户个人资产。

数字货币钱包的赛道竞争格局已经初现，仅靠对头部产品的复制已无入场空间，长远来看，增加使用场景、构建钱包生态、拓展业务边界才是数字货币钱包市场的切入点。因此，对数字货币钱包来说，最好的策略是找到定位，创新玩法，构建独特的竞争壁垒。
