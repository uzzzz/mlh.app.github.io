关于炒币，我是如何选到有发展前景项目的？

  • 柚子社区

我13年开始买币的，直到2018年，断断续续也投资了近百个区块链项目，站过最高位，守过最低位，破过产，爆过仓，经历了太多的炒币起伏，让我积累了很多经验，在很多韭菜眼里，让我成功步入了项目选择“大神”行列。


炒币和股票很相似，作为投资人，我们要找到未来十年有前景的项目，提前布局，并且降低参与风险，才能更好的挣钱。就像孙中山先生说的“天下大势，得知者昌”。

2018年我入手5个区块链项目，目前全抛2个，其余3个，2个短线，1个长线。

短线项目在此不做过多介绍，主要聊下我为什么对CIC币选择长线持有？

区块链作为一种先进的信息技术，解决了很多传统互联网不能解决的问题，大大降低了建立信任的成本，带来了更多和更高效的交易，成了当下的“网红”，招致大量的人和资金纷纷涌入，发自己的币，搞起了区块链项目。

这也是为什么很多人会说，90%的项目会死掉的原因。这不是危言耸听，因为在区块链技术发展初期没有正规的监管，野蛮生长的状态下必然会带来虚假的繁荣，原本没有能力推动区块链的人也要来分一杯羹，失败也必然的。

而那些未来能活下来的项目，一种是技术牛逼，能够得到资本青睐，注入资金让其推动技术发展变革的项目；另一种就是靠自己的项目能够盈利的项目。这也是我选项目买币投机的一个标准。入手的CIC币就属于后者。

对于CIC币，我是通过朋友圈发的区块链产品体验才接触到的，当时出于好奇就体验了一下，就抛之脑后了，直到朋友问起，你的CIC币到账了吗？才想起来去看了看，发现24小时就赔付了，从此开始我关注起了这个区块链项目。

发现这是一个很务实的团队，cic币上架的是全球著名的数字资产交易平台OKEx。他们专注于一件事情就是，解决传统保险行业痛点，为区块链用户提供更多风险保障。目前已先后落地了币种下架保、cic共识网络APP等产品，通过他们产品周报了解到，后续还将上线转账超时保和暴跌保等产品。越多的关注，让我发现这是一个很有意义的区项目，这也是我对于cic币选择长线的主要原因。
