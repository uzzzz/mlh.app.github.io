一、安装环境

操作系统版本：windows10专业版

VS版本：vs2010

Qt版本：Qt4.8

二、下载Qt和插件

1、下载Qt4.8（需要注意qt对应的VS版本，vs2010、vs2013、vs2015）

下载地址： http://download.qt.io/official_releases/qt/

百度网盘：https://pan.baidu.com/s/1qZVVnFTjHbShLGknizepSQ

2、下载Qt插件（需要注意下载与qt之间对应的版本，qt4还是qt5，这里下载的是1.1.11）

下载地址：http://download.qt.io/official_releases/vsaddin/

百度网盘：https://pan.baidu.com/s/1NBnrhm8_SqJ0iPUuyeZ2aQ

三、安装Qt和插件

1、安装Qt和插件很简单，找到对应的.exe文件，运行，然后一直点下一步就行了，默认安装C盘，如果不想安装C盘，修改路径就可以了。

2、安装完之后打开VS2010，会发现多了一个Qt标记。

3、将Qt安装路径下的bin文件添加到系统环境变量。
