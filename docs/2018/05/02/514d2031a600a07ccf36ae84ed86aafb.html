上海证券交易所技术规范白皮书_20170519.docx读后总结

上交所业务包括交易业务、行情业务、通信业务。


一、交易业务

主要是以下5大交易平台：

1. 竞价撮合平台： 为市场提供股票、封闭式基金、开放式基金、交易所交易基金（ETF）、上市开放式基金（LOF）、债券质押式回购等交易业务。

2. 综合业务平台：为市场提供大宗交易、转融通、非公开发行优先股、货币式基金申赎、交易所交易基金（ETF）申赎（含：跨境ETF申赎、货币ETF申赎、黄金ETF现金和实物申赎）、上市开放式基金（LOF）申赎、分级基金分拆和合并、国债预发行、约定购回、报价回购、股票质押式回购、网络投票等交易业务。

3. 期权业务平台：为市场提供股票、指数基金（ETF）等标的物的标准化期权合约交易，目前主要交易产品是50ETF期权。

4. 港股通平台：为市场提供香港联交所规定范围内股票产品的交易业务。

5. 固定收益平台：为市场提供国债、地方债、国开债、企业债、公司债、可转债、可交换债、资产证券化、债券协议回购、私募债等交易业务。

二、行情业务

1. 竞价撮合平台行情： 负责竞价挂牌产品（包括股票、债券、基金）、指数及外部搭载产品的行情发布。目前竞价主要有两类行情向市场提供服务：FAST（高速行情）、Level-2（深度行情）。 FAST行情文件为mktdt00.txt， 老版的上交所行情文件为show2003.dbf 

 （FAST与Level-2有何区别？ FAST是数据传送方式，level-1和level-2现在都用FAST方式传送）

2. 综合业务平台行情：负责综业挂牌产品的行情发布。目前主要包括综业行情文件的转发服务。行情文件为mktdt01.txt

3. 期权业务平台行情： 负责期权业务平台挂牌产品的行情发布，主要包括股票期权合约。目前只有FAST（高速行情）在对市场提供服务。行情文件为mktdt03.txt

4. 港股通平台行情：负责港股通额度和产品状态等行情发布。目前主要包括港股通行情文件的转发服务。行情文件为mktdt04.txt

5. 行情转发：深交所和港交所行情的转发

三、通信业务

技术公司的通信网络接入由地面网（地面广域网、数据中心局域网）、卫星网（宽带双向卫星、宽带广播卫星）接入组成，支持会员及其所属营业部以卫星通信线路和地面通信线路两种不同的通信方式接入，也称“天地备份”接入。

1. 地面广域网：地面交易接入系统具备MSTP、SDH和DDN三种广域线路接入方式，主要用于接入竞价撮合平台、综合业务平台、期权业务平台、港股通平台、固定收益平台等交易业务。

高速地面行情接入系统为用户提供MSTP广域线路接入方式，支持使用行情类程序（UDP）接收FAST（高速行情）。

     MSTP（Multi-Service Transfer Platform）是指基于SDH 平台同时实现TDM、ATM、以太网等业务的接入、处理和传送，提供统一网管的多业务节点。

    SDH（Synchronous Digital Hierarchy，同步数字体制），是不同速度的数位信号的传输提供相应等级的信息结构，可以把SDH简单的理解为集装箱运输，任何厂家的设备只要按这个规格做盒子装货物，那么货物就可以正常运输。

DDN（Digital Data Network，数字数据网）是一种利用光纤、数字微波或卫星等数字传输通道和数字交叉复用设备组成的数字数据传输网。

2. 数据中心局域网

数据中心局域网托管机房上行接入交易系统，采用千兆以太网线路。交易端口为用户提供竞价撮合平台、综合业务平台、期权业务平台、港股通平台、固定收益平台等交易业务接入。

3. 宽带双向卫星

宽带双向卫星系统针对证券通信市场集中交易对大容量卫星通信备份系统需求，为证券公司、基金公司、银行等市场参与者提供证券交易服务的多带宽可选双向卫星通信平台。

4. 宽带广播卫星

宽带广播卫星系统是面向金融市场，为证券公司、基金公司、银行等市场参与者提供证券及期货市场实时行情、盘后数据、公告等数据信息的单向卫星通信系统。

竞价撮合平台FAST行情市场参与者技术实施指引_20141124_SH.doc

   FAST行情系统是基于FAST技术标准，面向市场参与者和广大投资者的新一代实时行情发布系统。该系统在技术上以国际上标准的金融信息交换协议（FIX/FAST）为基础，会话层采用LFIXT协议（轻量级STEP会话层接口）的特点确认数据传输协议，在保持FIX协议灵活性的基础上，将市场数据大幅度压缩，大大提高了处理速度。


    竞价FAST行情数据（mktdt00.txt）将对原show2003.dbf行情中的非交易数据进行剥离，仅保留实时交易相关的数据信息，保证行情数据的精简和传输效率。非交易数据将通过盘后非交易基础信息文件的形式向会员发送。

  FAST行情数据通过EzSR进行接收，EzSR是一款专为基于SR/FAST协议数据接收处理而设计的绿色软件。主要功能包括多播与单点的行情，文件和消息数据处理等。

  EzSR软件的优势包括：直观显示相应业务的数据包接收情况；更加灵活的数据接收业务的参数配置；数据传输协议更加通用，全面支持未来新增数据传输业务等。

  具体EzSR安装配置及功能说明，请参考《FEA_EZSR_MAL_用户手册》。

  EzSR支持将竞价FAST行情落地为两种形式，分别为竞价行情文件形式（mktdt00.txt）和公共数据表形式（marketdata数据库表），会员可根据自身需要选择以任意一种或两种形式同时落地。

  根据实验室测试数据，对于竞价FAST行情3秒一幅增量的发布频率，在2Mbps带宽的线路上接收行情的延时不超过4秒（含3秒一幅发布间隔的时延），在4Mbps带宽的线路上接收行情的延时不超过3.5秒（含3秒一幅发布间隔的时延）。

对于落地为公共数据表形式（marketdata数据库表）的行情，在条件允许的情况下，可以让各系统通过访问数据库表的形式共享数据，这种访问方式行情读取效率较文本复制方式高，如图2.4所示。
