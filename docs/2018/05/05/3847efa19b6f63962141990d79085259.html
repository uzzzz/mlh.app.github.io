2018年全球新兴数字货币交易所骤增，市场迎来“升级换代”潮

  • 柚子社区


2017年数字货币市场的急速升温让交易所们赚得盆满钵满，也迎来了飞一般的发展速度。但由于缺少市场监管和行业约束，全球数字货币交易所纷纷爆出黑客盗窃、监守自盗、巨额上币费、少数庄家联合割韭菜、数据造假等恶意事件。全球数字货币交易所面临着巨大的交易风险、信任风险、投资风险等，部分国家的数字货币交易所甚至面临政策监管风险，如中国已禁止国内数字货币交易所开展业务。

市场发展迫使全球数字货币交易所面临”升级换代”, 技术更加完善、风险防范能力更加强悍、用户体验更好的新兴数字货币交易所纷纷进入市场。据业内人士透露，2018年仅过去的第一季度，全球各国新兴数字货币交易所数量都急速增加，以加拿大为例，2018年初至今已诞生至少5家数字货币交易所，还有3家正在推进，其他国家如新加坡、迪拜、东南亚部分国家、澳大利亚等地区也纷纷开设新兴数字货币交易所。

新兴数字货币交易所在风险防范、交易功能和用户体验方面更加完善。比如，日前宣布在Bibox平台全球首发LinkCoin Token (LKN)的加拿大场外交易所LinkCoin。



LinkCoin是成立于2017年11月、位于加拿大多伦多的C2C模式的场外智能交易平台，由一群国际化专业比特币交易所资深专家、券商金融衍生品设计师团队研发与运营，致力于打造世界级的区块链资产交换平台。

针对目前市面上的场外交易平台无法解决交易对手的信用风险：买方未打款却声称已打款，或卖方已收到付款却不打币等问题，LinkCoin创新设计出了基于以太坊的规避交易对手信用风险的机制，将代售的加密货币加锁，并要求买方提供隐去敏感信息的付款的数字证明，利用智能图像识别技术判断付款数字证明的真伪，在卖方不回复的情况下通过保护基金对买方进行补偿，从而保证了C2C交易流程的无信用风险和更加智能化的交易。LinkCoin还计划未来把用户的交易信息（非敏感性信息）保存于以太网络，而非集中式服务器，保证这些信息不会被平台纂改，为用户提供更加透明化的智能交易平台。


图注  在bibox上购买1000LKN的用户可以随机瓜分百万LKN


除了像LinkCoin这样风险防范功能更加完善的新兴交易所，基于区块链的去中心化交易所也是新兴数字货币交易所中不可忽视的一股力量，包括0x、Kyber、Airswap、stex、Loopring、Etherdelta等。目前除Etherdelta有一定交易规模外，大多数都处于早期开发阶段，而且Etherdelta与中心化交易所的用户体验也相差很远。

面对当前数字货币交易所乱象，全球数字货币市场要想获得进一步发展，作为交易中枢、独立第三方的数字货币交易所，必然面临“升级换代”，比如建立公开透明的市场规则、风险防范以及投资人教育和保护等制度。新兴交易所的快速发展，必将推动全球数字货币交易市场的进一步完善和发展。
