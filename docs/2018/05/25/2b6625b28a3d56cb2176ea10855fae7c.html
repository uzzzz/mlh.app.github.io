为什么会有以太坊智能合约开发实训营？

从2017年开始，在区块链项目如同“星星之火，可以燎原”的势头增长起来，基于智能合约技术建立的区块链项目也日益增长的同时，出现了一个难题：开发者，尤其是智能合约开发者方面的人才，非常稀缺。

很多拥有开发基础的程序员想入门区块链开发都会遇到这样的困惑：

1、从0开始入门，没人教没人带，怎么办？

2、学了不少区块链理论知识，还是不能快速上手？

3、对区块链开发的认知、定位等从业基本功还很不扎实？

4、靠自己摸索能上手，但野路子将来会遇到很大瓶颈，如何打好基础？

针对这一情况，硅谷密探和celer network创始人董沫老师萌生出一个大胆的想法——花3年时间，每年为区块链行业培养300名全球最顶尖的区块链技术开发人才。

这个想法让我们有些激动。有些事，要么不做，要么就干脆做到无以复加。

因此我们在去年的11月特别开办《以太坊智能合约全栈开发入门》实战训练营线上课程。

并且，为了保证每一位学员的学习质量，我们只选择最优秀的、学习动机超强的、无比热爱区块链技术、同时也是我们最有可能提供帮助的开发者和创业者加入智能合约实战营。我们也同样希望，我们提供给每一位加入中美实战特训营学员的价值和帮助是长期的、持续的。

实训营会迎接怎样的挑战呢？

我们在各大网站上所见到的大都是开课后即可学习，学习方式相对自由灵活，所有课程内容和作业、参考文档等一应俱全，但暂时不附带任何教学服务。

而我们精心打造的“实训营”则采用高强度的集中特训制，有特定的学习计划，有老师、助教、班长和班级群全程辅助学习，确保对于每一份作业都进行批改和反馈，完成得不好的会要求重做，甚至长期不完成作业的，可能会被劝退。

这是一个与市面上你可以见到的绝大多数培训和活动不太一样的课程。实战训练营每期为时4周，致力于通过每周两次的线上课程+高强度的课后作业任务，帮助实战营的每一位成员，实现具备在以太坊上开发一个可产品化的智能合约应用dApp的能力。

具体来说，在4周的时间内，我们将会按照每周两个模块的学习进程，通过录播课程视频+课后作业+导师助教答疑的形式，依次带领大家完成以下几个模块的学习——

在48小时内，以小组团队协作的形式，同时开始破解老董部署在以太坊上Ropsten测试网络的一个智能合约，来让学员认识到以及充分实践重要的智能合约安全知识，也是课程的一个结业考试。

整个学习过程强度很大，也会很烧脑。其中，1-7课会通过在线课程+讨论+作业完成，每周预计约需耗时至少5-6小时；实战项目会在课程结业前以小组协作的形式完成。

以太坊智能合约开发实训营适合谁？

我们认为，一个人要想在智能合约开发领域内有能力做出一点有价值的事，你应该需要具备如下条件——

1、掌握Solidity开发环境配置以及Solidity语言相关语法；

2、熟练掌握基于Ethereum的Dapp（去中心化应用）的开发部署；

3、从前端到后端，具备一个对区块链项目技术栈的全局思考能力；

4、其它更高层面的一些东西，如对行业的深度理解和更加深厚的区块链行业资源积累等。

希望上面这四点里，每一个来到智能合约开发实训营的人，我们会尽可能帮你满足前三个条件，打下坚实基础，让你将来可以更好地满足最后一个条件。

因此，如果你想要来参加智能合约开发实训营，首先，你必须要保证在开营的4周内，每周至少可以有6小时以上的时间来完成我们的线上任务，以及课程学习结束后可以参与到最后的项目实战中。

其次，你最好能够满足如下特点——

1、最好对于区块链技术拥有一种近似宗教信仰般的热忱，坚信区块链是一种可以让这个世界变得更加美好的技术；

2、最好是1-5年互联网开发者或者是计算机专业的学生，拥有大二水平的C++/Java/Python/Go 基本知识及使用；

3、必须是系统学习实践过一门面向对象的编程语言，且对Linux基本使用有了解，最好还对基本的Web开发概念性理解以及基本的区块链知识有一定的了解；

4、希望你自信是一个足够有趣、有胆、有能量的人，且拥有至少一项你自信可以胜过大多数人的核心技能。

关于智能合约开发实训营的校友们

迄今为止，我们已经举办了3期实训营，提前半年完成了我们的第一个小目标，累计有300人先后加入且毕业智能合约开发实训营。

而申请者报名表累计每期超过千份，每一个申请者都要经历我们前后至少3-4轮的打分和评审。但，可能也恰恰是这种高标准和高门槛，确保了我们每一次的教学质量都是史上绝无前例的。

如果你填写过我们的报名表却没能通过审核，可能是因为你准备不足，申请资料打动力稍欠；也可能是你的实际编程语言缺乏，可能现阶段更适合加强学习一门面向对象的编程语言；还有可能你已经身居高位，我们认为这门入门级的开发课程的内容已经不足以给予你更多帮助。

无论是以上哪一种可能，我们都希望你能够理解。我们可以确保的是，在对待所有申请者方面，我们都是异常认真的。
