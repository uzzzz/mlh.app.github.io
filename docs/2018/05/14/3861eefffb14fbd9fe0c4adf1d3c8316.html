夸克链(Quark Chain)——区块链3.0新贵

  • 柚子社区

   本文更多的是解读白皮书和其他项目对比,不构成其他投资建议,也不作为其他项目好坏的评定,单纯的从技术方面和创新方面的理解


   前段时间行情有些复苏,大家把关注点重新放回了一级市场,其中一个比较惹人注目的一个项目夸克链受到大家的追捧,既然能这么受观注,想必是有

些出色的地方,抽出一天时间看了下白皮书.


   区块链现在面临的三大问题(安全性/去中心化/高TPS),是各公链面临和解决的问题,其中怎么衡量之间的关系和占比,以及新的解决方式,这个项目可

以说是比较有创新能力的, 有几个关键点比较吸引我.

  

1.多次分片和双层链 (看中)

  对于区块链的两大功能, 确认和记账,夸克采用分治的解决方案,根连用于确认,弹性分片用于记账和处理事务,类似用闪电网络,但不用开关通道和避

免中心化的形成.


2. QuarkChain 基于博弈论 ,用50%的算力解决双花问题和恶意挖矿

3. 多小节点形成集群的超级节点 (看中)

四个节点集群代替四个超全节点


  超级节点的代表不得不提下EOS,虽然不认为EOS是向中心化妥协,单确实有那么点意思, 而夸克链采用的节点集群,不用简历费用超高的超级节点,而且这样的优点在于其中一个节点崩溃,仍然可以验证




4.跨片交易,并且随分片数量线性增加 TPS ( 看中)

交易分为两类, 片内交易和片外交易(跨片), 需要根链确认,进行交易


5.账户管理简单,同一账户的资产在不同分片上用同一个钱包

账户管理采用了主次账户的想法,主账户管理默认分片的地址和余额,次账户管理剩余分片的其他地址

6.采用了EVM,方便以太坊上DAPP的迁移

至于一些应用和技术人员,团队等,是一个考核方面,但我看项目部看名人效应,只要在实实在在为区块链发展和解决问题,即使路走的艰难和漫长都是可以理解的.
