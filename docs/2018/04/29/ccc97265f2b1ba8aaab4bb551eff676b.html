快速使用

现在介绍如何把这个博客变成自己的个人博客
只需两步（假设你现在已经有了github的账号）

第一步 克隆



其中克隆地址为 https://github.com/GitHub-Laziji/GitHub-Laziji.github.io
项目名 {username}.github.io
username 为你的github用户名
点击克隆

第二步 修改配置文件

打开克隆的项目

修改/static/configuration.json 文件的github-username 改为自己的用户名 其他的可以删掉

查看效果

到这里为止 就完成了
浏览器访问 https://{username}.github.io 看看效果吧

获取Token

如果需要发表博客 在 github > settings > Developer settings > Personal access tokens 勾选 gist 和 repo 权限 获取Token
就可以了
