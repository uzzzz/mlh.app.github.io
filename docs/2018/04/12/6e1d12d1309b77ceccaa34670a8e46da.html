PyCharm社区版搭建Django2.0开发环境

工具准备：
（1）Python 3.X 以上版本
https://www.python.org/downloads/windows/
（2）PyCharm
https://www.jetbrains.com/pycharm/

一、创建一个空项目并安装Django2框架

项目创建完毕后，在 Terminal 里输入 pip install Django


看到Successfully installed Django-2.0.4 pytz-2018.4  说明安装完成了！

二、创建网站

输入：
     django-admin startproject myweb



python manage.py startapp myweb

在cmd命令行下输入：python manage.py runserver

此时打开浏览器，输入 http://127.0.0.1:8000 就可以看到了。
