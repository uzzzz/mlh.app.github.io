支付宝支付-当面付DEMO-沙箱测试

前段时间做的网上商城项目由于没有企业账户无法进行微信支付，得知支付宝有沙箱测试环境，特意来运行一下官方的DEMO

去蚂蚁金服注册，认证，下载当面付DEMO这些就不多说了

第一步：配置沙箱应用环境

点击“开放平台-开发者中心-沙箱环境”。进入沙箱环境页面，系统已经自动为你创建一个应用，在基础信息中可以看到应用信息。

在此处下载生成RSA密钥工具

打开工具后能够看见，如下界面

秘钥长度2048的为RSA2秘钥，1024为RSA秘钥，这里我们使用2048，点击生成秘钥

将生成的支付宝公钥复制，填进下面，再复制支付宝公钥

将下载好的DEMO导入IDE 开始填配置文件
open_api_domain = 支付宝网关
mcloud_api_domain = http://mcloudmonitor.com/gateway.do
pid = 商户UID
appid = APPID

private_key =工具生成的私钥
public_key = 工具生成的公钥

alipay_public_key = 刚刚复制的支付宝公钥

第二步：测试

运行项目
看到如下界面

这里我们测试当面付的二维码支付，跳转到订单页面，设置订单信息

生成二维码

使用支付宝沙箱钱包扫码支付

支付成功
