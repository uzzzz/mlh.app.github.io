用cgminer软件使用cpu挖矿，挖Ravencoin（RVN）渡鸦币

  • 柚子社区

我说说我的情况：首先我是一个新手，上学的时候有一些计算机基础，毕业后一直做金融了，今天重操旧业。完全是在挖矿的财富效应上试试身手。只是想锻炼下动手能力，试一下，跑通了再去买专业设备挖矿挣钱。

最近rvn很火，我测试的是如何在CPU上挖RVN。具体RVN情况大家可以看

介绍：http://www.kuangjijia.com/category/885.html

代码：https://github.com/RavenProject


1、我有一台旧的笔记本电脑，是i3的，2G内存，集成显卡（这使得我只能cpu挖矿了）

2、老电脑的xp太旧，我又不想更换操作系统了。

我决心首先装入unbuntu+window xp双系统。

1、第一步，我用diskgenius,把D盘里多于的空间分配了60G装ubuntu。

参考：http://www.diskgenius.cn/help/partresizing.php

2、第二步，做了一个Ubuntu U盘。我先是下载了ubuntu-16.04.3-desktop-i386。我先自己烧制了一个U盘启动的安装程序，把下载的ubuntu烧录进入。这里我参考的的是：

https://jingyan.baidu.com/article/60ccbceb18624464cab197ea.html

3、第三步，安装Ubuntu。

4、第四步，装cgminer。cgminer是一个很好的挖矿软件，其实cpuminer也不错，但我只有ubuntu，而cpuminer只有window版本）。其实cgminer只能用v2.11.4，以后的版本都不支持cpu挖矿了。

以下是精彩部分：

1、下载源码 https://codeload.github.com/ckolivas/cgminer/zip/v2.11.4
2.   解压源码 unzip cgminer-2.11.4.zip //解压zip
3、配置编译环境
apt-get install build-essential libncurses5-dev libcurl4-openssl-dev
4、配置编译

./autogen.sh 

./configure --enable-cpumining --disable-opencl

注意：要用 ./不然找不到文件

5、编译

make
6、运行
./cgminer --cpu-threads 3 -o stratum+tcp://yiimp.eu:3666 -u RQWQaoScemn4uSSiC4TzdeZqjNVP66WGUW -p

--cpu-threads 3 表示使用3个cpu内核。

yiimp.eu是矿池。

网上用的是挖比特币教程，命令不一样。比如他们在 ./cgminer --cpu-threads 3 后面是  -url http://pit.deepbit.net:8332

我们挖的是RVN，要用-o stratum+tcp://矿池名 

-u后面是你的钱包

-p后面是你的密码
