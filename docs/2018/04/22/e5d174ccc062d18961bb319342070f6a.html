【吐血整理】 windows系统下 fbprophet的安装（pyhton语言）

  • 柚子社区

facebook开源了时间序列预测框架prophet，目前支持R语言和python语言。托管在github上：https://github.com/facebookincubator/prophet。本文主要是讲Windows下fbprophet的安装。

系统：windows7 64位  软件：python3.6
第一步：
首先要安装C++编译器Microsoft Visual C++ Build Tools 2015. 下载地址：https://msdn.microsoft.com/en-us/library/ms235639.aspx?__hstc=268264337.be560855f2752462da44d848abeffaee.1524137737397.1524146068915.1524225080060.4&__hssc=268264337.1.1524225080060&__hsfp=3702916517
第二步：
然后安装Anaconda  网址：https://www.anaconda.com/download/#windows
下载及安装说明：https://www.jianshu.com/p/62f155eb6ac5
第三步：
要学会用conda管理环境
创建新环境 在命令行执行 conda create –name predict （我这里创建了一个名为predict的环境，环境的名称可以自己随便起）
切换到环境activate predict
第四步：
安装Gcc编译器
安装说明：https://www.jianshu.com/p/ff24a81f3637
第五步：
命令行执行
activate predict
conda install -c conda-forge fbprophet
第六步：
配置pycharm解释器，先点击file-setting，然后点击右上角的小方块，选择add
 
然后如下图所示，点击现有环境，找解释器的地址。
 
如下图，找到你软件的安装地址，点击ok即可
 
点回了setting界面，一定要记得点击右下角的Apply奥。
