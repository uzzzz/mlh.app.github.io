Docker Compose系列 （一） 安装

本文主要介绍docker-compose的安装，准备环境如下：

  • Ubuntu 16.04.4
  • Docker 1.8

安装

在命令行中运行以下命令

sudo curl -L https://github.com/docker/compose/releases/download/1.21.0/docker-compose-$(uname -s)-$(uname -m) -o /usr/local/bin/docker-compose

其中1.21.0代表版本号，可以依据需要进行修改。uname相关命令是为了获取操作系统名称及电脑类型，便于找到指定的compose

修改运行权限

执行以下命令，将docker-compose设为可执行文件

sudo chmod a+x /usr/local/bin/docker-compose

其中使用a则代表所有用户均可执行此文件

查看版本

docker-compose -v

卸载

卸载docker-compose比较简单，只需要运行以下命令

sudo rm /usr/local/bin/docker-compose

详细安装卸载可以参考官方说明：Docker Compose | Docker Documentation
