（已成功）windows下，VS2012+Qt5.5.1的安装、路径配置、项目配置（其它版本可参考）

  • 柚子社区
一、前期准备
下载安装Qt5.5.1；（不要有中文路径）
下载的版本和vs版本有关系，否则,在最后编辑阶段，不能通过
一定要选择对应vs版本的安装包。如果不开发64bit的程序，选择x86的安装包即可。

下载安装连接Qt和VS的一个插件qt-vs-addin.exe（64位32位通用）。
我用的是VS2012 下载这个版本。
默认已经安装VS2012。

二、安装路径配置

打开VS2012,可以看到菜单栏多出一项QT5。

点击QT5->Qt Options->Add->输入Qt的verson name（Qt5.5.1）及path(包含bin文件夹的目录)->OK

如果出现以下情况
报错：找不到qmake.exe。但是在qt目录下面能找到该文件。

看清楚红色提示，发现目录中有两个\bin目录，只要把链接目录选择到\bin目录的上一级，就可以，问题解决。

三、项目配置

文件 -> 新建一个Qt5 Projects，选择Qt Application，命名test。

右键项目->QT Project Settings->Version->选择Qt5.5.1->OK

编译程序并运行，出现一个名为test的窗口。

恭喜你已经成功啦~(●'◡'●)
