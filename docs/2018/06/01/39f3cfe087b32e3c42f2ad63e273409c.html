免费上路“科学上网”，GitHub学生资源包你值得拥有

说明

  • GitHub资源包里有50$DO账号优惠券，弄好后相当于免费使用一年VPS;
  • 非在校生也可通过文中2.4的邀请链接注册领到10$，相当免费使用2个月；

1、前言

  • GitHub学生资源包（GitHub student pack），是GitHub联合他的一些合作伙伴朋友为我们学生群体提供的一个工具优惠包，旨在给广大学生提供免费使用相关工具动手学习的机会。

2、准备

  • GitHub账号
  • 有效学生证明资料（如学生证）或教育邮箱
  • PayPal账号(下文2.3)
2.1 注册GitHub账号
  1. GitHub官网：传送门；
  2. 官网中填写信息注册，Sige up后会收到账号激活的邮件；
  3. 打开邮件激活账号。
2.2 申请GitHub student pack


  1. GitHub Education官网：传送门；
  2. 点击Get your Pack,输入github账号，再点击Yes,i'm a student.出现下图；
  3. 在框2修改为你的教育邮箱，如没，就上传证明文件的照片（图片清晰没反光，要证明你现在还是在校生）；
  4. 框3填你的 school name（英文），框4选你的毕业时间；
  5. 框5英文填写你打算怎么使用GitHub：范文（建议修改一下）；

There is no doubt that GitHub is a vary useful tool.As a student,i want to improve my programming skills by using open-source.And to complete some projects,i will work with my teammates in GitHub.Of course,if i have a really good project,i will consider contributing back my code to the GitHub community.

  • 点击submit request,邮件通知申请是否通过，一般要1~2天。如果申请没通过，再次确认图片质量及填写信息准确性后，重新申请。

    • 2.3 注册PayPal账号

    资源包申请好后，还要用5$激活服务器提供平台Do（DigitalOcean）的账号
    - PayPal官网：传送门;
    - 注册个购买账户，需要用到身份证等信息，绑定一张能开网银的银行卡；
    - 记住账号密码。

    2.4 注册DigitalOcean账号
    • DigitalOcean官网：传送门；(这是推荐链接，平台会多送10$)；
    • 填写相关信息注册账号，耐心等候网站跳转；
    • 打开邮件激活账号；

    3、使用GitHub student pack中的DO优惠券


    1. GitHub Education官网：传送门；
    2. 再次点击Get your Pack,可能要输入github账号；
    3. 向下拉网页找到Digitalocean项目,复制You code后的优惠码；
    4. 点击上面中的DigitalOcean website，点Login填账号；
    5. Dashboard ——>Go to billing ;（或右上小人头找setting——> billing ）
    6. 在 billing 页面下拉找到Payment methods，付5$来激活账号。
    7. 付款后，应该看到你的 billing history，有15$ (10是赠送的）到账；
    8. 使用优惠包中的50$coupon。通过ticket（客服）人工补给；


    1. support；
    2. Go to tickets
    3. Create new ticket
    4. 参照这样填写一下，记得code替换成你自己的，发个ticket，等待那边的人处理；

    Subject： Can not use my coupon of GitHub’s Student Developer Pack
    Description：Hello,I am a new user here.Just now I got my coupon from GitHub’s Student Pack,but could not activate it.My code is:134679.Please help me activate my code.Thank you!

  • 过1~2天应该会收到这样邮件，你的 billing 会多了50$，这时你应该要去close(关掉)你的ticket，让那边工作人员知道你的问题已解决，那边会很负责跟进你的情况的。

    • 4、 购买、配置VPS

    1. 点击Create,创建一个Droplet(主机)；
    2. 给服务器选择操作系统；(按自己喜好，本人现在在用CentOS)
    3. 选择服务器SIZE，按需选择，一般的选择$5的就够用；（PS:增值不增价，一个月前同价还是512M-1vCPU-20G的）
    4. 选择服务器机房位置，这个直接影响到VPS网速快慢，之前试过Sing1的，装好时很快，一晚后就炸了，目前在用San1还行，可以自己多建几个测网；
    5. 这两个可不选；
    6. 按需改Droptel名称，点击Create;
    7. 完成后，会自动跳转到Dashboard，VPS服务器已经搭建好了。
    8. 打开收到的邮件，这是告诉你服务器登录账号密码，提醒首次登录要修改密码等。重要是IP Address和Password这登录账号密码。

    5 结语

    这是免费的资源，非常适合新手入门“科学上网”，不用承受昂贵的费用。本人衷心希望本文能帮助到有需要的人，如文中有错漏，欢迎留言指出一起讨论，共同学习进步，谢谢。
