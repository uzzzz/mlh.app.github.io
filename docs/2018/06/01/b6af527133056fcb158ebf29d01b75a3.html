在私链上使用Remix和geth控制台部署合约

  • 柚子社区

做一个简单的合约部署

1.打开Remix IDE浏览器： https://remix.ethereum.org，在浏览器创建合约

合约代码如下

pragma solidity ^0.4.0;
contract InfoContract {

   string name;
   uint age;

   function setInfo(string _name, uint _age) public {
       name = _name;
       age = _age;
   }

   function getInfo() public constant returns (string, uint) {
       return (name, age);
   }   

}

2.编译合约

点击右侧栏的Compile模块下的Start to compile按钮进行编译，编译成功后点击Details按钮，出现浮层页面

3.部署

（1）在浮层中找到WEB3DEPLOY , 将WEB3DEPLOY中的内容复制，点击如下图的按钮


在复制好的内容中的gas的后面加上gasPrice:'20000000000'

（2）进入私链geth的控制台，打开cmd命令行，输入以下命令，""中的地址为相应的私链的地址

geth --datadir "D:\Program Files\EthSpace\PrivateChain" --networkid 88 --nodiscover console

这里会遇到一个问题，要在进行部署之前，先将交易的账号进行解锁，不然会出现错误。合约中用到的账号是eth.accounts[0],所以将这个账号解锁

my=eth.accounts[0]

personal.unlockAccount(my)

解锁后，将刚刚复制的内容粘贴到命令行中

开启挖矿

miner.start()

执行成功的会出现以下相似的内容（其中address为合约的地址，transactionHash为hash值）


部署成功后 ，停止挖矿

miner.stop()
