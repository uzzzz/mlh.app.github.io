开发工具atom的使用和web3.js api的使用

上文讲到：

Truffle框架是针对基于以太坊的Solidity语言的一套开发框架。关于Solidity语言后面会继续学习并一同分享。

如果想知道solidity的更多知识可以参考http://solidity.readthedocs.io/en/latest/

或http://www.tryblockchain.org/

今天介绍一下一个开发工具atom的使用和web3.js api的使用，为后面的编程打好铺垫。

一、开发工具atom的使用

1、 这里选择atom，下载地址：https://atom.io/  目前版本是1.27.2

Atom是github维护的，完全免费，并且在github上开放全部的源代码，是基于web技术（Chromium+Node.js）开发的，可以理解为跑在本地的网页。

下载完后点击安装，刚开始的时候比较慢，耐心等待。。。

打开之前生成的truffleProject，具体操作可以参考上一节。

 

注：atom工具类似eclipse，是基于插件的，所以需要安装一些插件可以方便后期的编程操作。

如高亮显示language-ethereum

  代码自动补全：autocomplete-solidity

相同单词高亮显示：highlight-selected

安装插件在File->Settings->Install

安装成功

高亮显示：

 

二、web3.js api

web3.js是以太坊提供的一个javascript库，它封装了以太坊的json rpc api。其中Truffle框架内置了web3.js所以可以直接用。

可以利用web3.js api操作，如

  • web3.currentProvider 获得当前私有链的信息
  • web3.eth.accounts 获得所有的账户信息

 

具体参考http://web3.tryblockchain.org/
