Java搭建区块链开发环境

  • 柚子社区

1.首先我们需要一个虚拟机或者ubuntu 16.04云服务器

    我的环境是ubuntu 16.04

    ok,第一步，跟着我的节奏来

    至于怎么安装ubuntu，大家可自行度娘

    建议搭建选择国内的源（国外的源访问实在实在实在.................太慢了）

    国内的源有阿里，网易，搜狐，清华等等。。。。。。

    在ubuntu里面按Ctrl+Alt+T可以打开终端

    ok ,打开只有输入一下命令

    sudo mkdir -p /usr/local/services/bitcoin（创建一个文件夹）

    sudo apt-get install autoconf pkg-config libtool build-essential libdb++-dev libboost-dev libssl-dev libevent-dev libboost-            system-dev libboost-filesystem-dev libboost-program-options-dev libboost-thread-dev libboost-test-dev libboost-*-dev

    以上命令是我们需要安装的一些 依赖库
 如果你运行这些命令没有没有报错，那么恭喜你，你的依赖库已经安装成功

2.接着我们把环境下载下来

    如果你的ubuntu没有安装过git

    首先输入  sudo apt-get update(更新软件包，如果报错，那你的考虑一下是不是你的源出错了，网上有很多修改源的教程）

    如果软件包更新没有报错，接着输入 sudo apt-get install git 
    安装过程中会问你是否解压，按Y解压就OK。
    安装好了之后输入

    get clone https://github.com/Bitcoin-ABC/bitcoin.git

    下载完毕之后为询问是否解压，y 解压就OK。

    ok.然后进入之前创建的文件夹

    输入cd /usr/local/services/bitcoin 

    进入该文件夹 

    输入 ./autogen.sh

    等待系统运行完毕

    然后输入

    ./configure --prefix=/usr/local/services/bitcoin --with-incompatible-bdb（等待运行完毕）

    继续输入

    make -j 6 &> make.log &(等待运行完毕)

    继续输入

    sudo make install（正式开始安装）

    添加环境变量

    echo 'export PATH=$PATH:/usr/local/services/bitcoin/bin' >> ~/.bashrc 

     OK了，现在已经安装完成.
