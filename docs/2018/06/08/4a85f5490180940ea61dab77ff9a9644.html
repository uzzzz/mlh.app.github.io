在上篇中，熊小猫讲到了专注于做区块链“网关”的Chainlink，本篇将继续分享后两种潜力币，Zebi和Holo.


Zebi


首先来说Zebi.

Zebi有印度以太坊之称，涉足的是“数据主权”领域。简单地说，Zebi试图建立一种区块链网络，让你可以用来储存高价值文件——例如政府文件、健康数据等——并自主决定是否（有偿的）将这些数据给需要它们的相关机构使用，包括政府、医院等。

为什么是实力项目？

Zebi之所以诞生在印度，有其特殊原因。印度人口仅次于中国，得益于互联网普及，社交网络、智能手机的不断增长，以及数字化政策和政府进行的改革，印度的大数据预计将在2020年达到每年3 Zebibytes(Zebibyte = 1024 ^ 7字节)。

与此同时，诸如数据窃取，数据泄露，数据篡改和身份盗窃等数据犯罪正在增加，印度企业和消费者每年要花费超过300亿美元来应对。这无疑是一笔巨巨巨巨款。而区块链的去中心化和不可更改的特点，使其在防数据泄露、防篡改和黑客攻击方面有天然的优势。



在具体实施上，Zebi解决方案由两个相互关联的组件Zebi Chain和Zebi DataGateway组成，可将个人、数据提供商和数据请求者置于一个通用平台上，实现安全，无缝和基于同意的信息交换。

Zebi Chain是 Zebi 区块链解决方案的重要组成部分，并可以安装在大学、教育委员会、企业或政府部门等数据提供者的经营处；这些提供者生成、发布、更新并促进数据请求者所需的数据。Zebi Chain与 Ethereum 区块链结合，为客户们提供私人/公共区块链使用的完全灵活性，并实现 100％的防篡改保护、问责制和透明度。

Zebi Data Gateway 模块是一个由智能合同驱动、不受人为干涉的区块链系统。它连接 Zebi Chain™在数据提供商运营处完成的安装。允许个人和数据请求者通过一个简单的注册流程进行注册，从而将他们带到一个通用平台上，并通过由区块链驱动的 DaaS API 进行安全流畅的数据交换。



Zebi目前已经通过一项专利申请，另外正在申请的两项专利通过了AP和Telangana政府评估，并正在等待财务评估结果，并在印度、美国、新加坡和其他国家向政府和企业提交了提案，技术方面占有一定优势。

印度正处在巨大经济快速增长的风口浪尖上，他们已经拥有快速增长的数字生态系统，而区块链项目将涌向亚非洲发展中国家，了解每个国家和地区的的发展需求，并运用区块链技术的特定解决方案，将会为经济的发展助一臂之力。显而易见，Zebi正是这个为印度政府提供和企业提供特定解决方案的项目。

在之前的一篇文章《祛魅：你对区块链的想象，有可能都是错的》中熊小猫提到，区块链本质上其实就是有特殊规则的数据库，其核心价值在于无需信任、去中心化和不可篡改。除了这些特点之外，它在很多方面是不如传统的中心化数据库的。所以，一个对区块链的真正应用，必然是抓住了这几个痛点的，否则就是伪需求。同时，大数据的泛滥所导致的对数据的不当应用、个人数据主权的丧失、数据彼此之间的割裂是非常大的痛点，从这个角度上来看，Zebi的市场是无比巨大的。

其他因素

  • Zebi团队组成

Zebi是一个年轻创新的组织，由牛津大学、麻省理工学院、斯坦福大学和资深IT人士和思想领袖们共同创立、指导和管理，团队成员有在谷歌、优步、亚马逊、微软和甲骨文等名企有丰富经验。



  • 主要合作伙伴

Zebi的首要客户就是印度政府。目前Zebi已经与印度第 10 大州、人口超过 5000 万的Andhra Pradesh，签署了一份 Zebi Chain多年订购工单，以保护其土地资产。

最近，位于印度安得拉邦的多家酒店开始实施Zebi方案。报道显示甚至是警方为Zebi解决方案开道，当然，警方的出发点主要是关注犯罪问题。这是Zebi非常重要的市场进展，也是区块链技术的重要应用，据了解，截至目前，超过2000家印度酒店已经开始协同Zebi，全面部署其解决方案。

  • 代币——ZCO

Zebi 的代币Zebi Coin（ZCO）作为用于鼓励生态系统内交易结算的效用代币。数据申请者为数据交易支付的 ZCO 将作为交易奖励由数据提供者、个人和Zebi分享。ZCO将激励所有参与者对 Zebi 数据生态系统的使用、贡献和发展。

目前上市时间较短，仅在Qryptos和IDEX交易。



总的来说，Zebi坐拥巨大的市场、精良的团队以及印度政府的背书，在自己的领域里没有什么像样的竞争对手，而且目前还处于一个较低的市值，应该说前途无量。


Holo


第三个也是最后一个要推荐的潜力项目，叫做Holochain。最近引起了许多关注。



为什么是实力项目

关于Holochain首先要强调的是，它不是区块链。

那么它是什么？

简单来说，Holochain是一种开源的、用来构建去中心化应用程序的底层框架。

在区块链中，每个节点都拥有分布式账本相同的最新副本。也因此，区块链存在极高的存储成本和低可扩展性的障碍。Holochain试图打破这些障碍，以便让应用程序开发人员更便宜并且能够快速运行他们的产品。



Holochain与区块链的区别在于，它试图成为一种轻量级架构，即使手机和其他低容量设备也可以运行网络。

要明白为什么Holochain为什么能做到这一点，先一起来看看目前最大的智能合约区块链——以太坊。以太坊的每个智能合约都通过以太坊虚拟机（EVM）运行。不管你的应用程序是一款游戏，还是一个保险应用，都会存储在相同的位置。一切都存储在链上。虽然它们彼此之间可能毫无关联。

Holochain也是去中心化的，因为有分布式节点。在Holochain中，每个节点都可以拥有自己的分类账本。节点A的分类账可能与节点B不同。如果节点A需要将其副本分享给网络，则需要达成共识。Holochain的共识可以看作很像Git（分布式版本控制系统）。在Git中，所有的变化都是通过加密散列链组织起来的（这个听起来像区块链一样）。在这个系统中，如果你信任最新的散列，你可以从任何信息源中找出所有以前的信息，并验证它是否有效。

比特币和以太坊也做同样的事情，那么Holochain的区别在哪里？

Holochain最大的革新之处在于，将全网账本分布式的存储在各个参与的节点上，并通过DHT（分布式）寻址技术保证账本的完整性integrity和可检索性retrievability。

（注：分布式概念简析：类似于电驴迅雷的BT种子，用户在下载的时候进行上传数据，用户越多，新的用户进行下载运行的速度就越快。分布式技术出现的比较早，但由于缺乏有效的激励机制，一直以来没有现象级的爆发。Holochain团队做的事情之一是将分布式技术与区块链奖励机制结合，推动分布式应用的发展。）

基于区块链的网络比如比特币，属于data centric（数据中心），在这种结构中，系统中每个节点都保留和验证同一个账本，根本上是寻求数据上的共识。而Holochain属于agent centric(代理中心），它并不寻求单一的源数据或者某一条区块链上的共识。

通俗一些解释，这意味着什么呢？假设一个代码库由多个开发人员完成，程序员A可以进行更新A，程序员B同时在更新B上工作。如果更新A和B之间没有冲突，则它们可以合并到报告中。或者程序员A可能决定不将更改集成到Git社区中，而是出于自己的原因保留它在本地。

Holochain也是如此。假设Dapp A在整个Holochain网络上运行，但Dapp B决定只想在某个特定的节点集群运行。Dapp A和Dapp B都可以按照自己的意愿来进行。Dapplication A与共享分布式哈希表（DHT，Distributed Hash Table）交互，而Dapp B则不会。

其结果是,系统中的每个节点不需要保存同样的数据。在Dapp A上发生的事情不需要向Dapp B和C等广播。如果John要在社交网络上向Sally发送消息，Jorge不需要知道这一点。

诸如聊天信息，支付渠道和协作文档技术应用程序已具备这些功能。这里唯一的区别是这些程序将去中心化，而不是在Facebook或谷歌上运行。应用程序完全由使用它的人员运行，不需要Web公司的服务器。



Holo官方经常说的一句话是，在holo上每人都是一条链。没错，每人都存储了自己的交易，也就是自己的链，自己的分账本，但自己的分账本也是全网总账的一部分，通过DHT技术和密码学技术让这些分账不可分割不可篡改。每个节点存储的账本都是唯一的，并且是必要的，相对于Blockchain，极大的降低了复本占用的空间和带宽，同时还保留了区块链的优势。

值得注意的是，Holochain只是一个框架，不是一个网络，也就是说，搭建在Holochain上的Dapp可以使用自己的共识机制，并根据自己的安全模式来制定验证规则。Dapp们只需要遵循核心的框架，而不需要维持同样的共识机制。所以无论你想效仿比特币网络还是瑞波，甚至你可以在上面建一个完全和币无关的社交网络。简而言之，只要你将自己的APP连入Holochain，你就可以运营一个Dapp，很简单的成为Holochain中的一个节点.

Holochain试图将用户数据的控制权从大公司手中，交还给用户。试图解决区块链的固有痛点，同时保留去中心化的特性。只有在需要达成共识的时候，才进行数据的交互。通过利用Holochain，开发人员将能够构建新一代去中心化应用程序和服务，这些应用程序和服务不会滥用用户隐私，也不会破坏环境。

代币——HOT

Holochain也有自己的代币，和该公司同名，叫做Holo（简称HOT）.

Holochain支持用户之间的广泛共识或者协议系统。它们的proof of service共识系统，可以实现在一个用户为其他用户提供服务后，获得代币奖励。这尤其适合共享经济。

目前距离发行价已经翻了六倍。市值蹿升速度也比较快。一度升至90名左右。



团队

作为一个开放源代码的项目，Holochain是由许多人以分组的方式合作完成的，在当下的初始阶段，大部分成员来自Holo公司——使用Holochain分布式应用程序网络。

Eric Harris-Braun：耶鲁大学计算机科学学士学位，执行工程师/共同创始人，与Art共同创立MetaCurrency，多个应用程序的顾问和负责人。

在github上找到holochain，也可以看出它们的代码更新相当活跃。令人喜闻乐见。



Holo的整个构建是精细而复杂的，这里只是一个简单的介绍。想要了解更多可以去它们的官网。总的来说，Zebi的理念非常新颖，愿景也很远大。虽然这种新颖的技术确实还存在一些隐忧，但从各个层面看，确实是一匹可以埋伏的黑马。
