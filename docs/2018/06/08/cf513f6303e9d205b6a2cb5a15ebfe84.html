二、 钱包获取

钱包下载地址：https://github.com/XDagger/xdag/releases

 

1. 下载XDag.x64.zip文件，解压，32位系统下载对应的32位应用程序；

2. 在解压文件夹中创建RUNWALLET.bat；

 

3. 编辑bat文件，输出命令：

xdag.exe -d -m 1 xdag.dlpool.cn:6767

Pause

 

4. 双击运行“RUNWALLET.bat”文件，会弹出cmd命令窗口；


 

设置钱包密码，输入密码时候是看不到密码位数的，一定要确保输入正确（需要两次验证密码）；

5. 输出Type random keys:，这个字符串记下来就行；

 

出现上图界面，等待一段时间，矿池系统会分配一个钱包地址。直倒出现


表示钱包地址获取成功。

6. 查看获取的钱包地址：输入“account”，此次获取的钱包地址为：JpsT5ZmQmMU/67hccNzYdPVUpkdF7kcF

 

7. 查看钱包余额：输入“balance”

 

8. 查看其他命令：输入“help”

 

9. 使用xdagwallet.exe查看钱包详情（注：在没有使用cmd命令获取钱包之前是不能使用xdagwallet.exe查看钱包的），pool输入：xdag.dlpool.cn:6767（其他相应的矿池也行），点击“connect”

 

10. 输入之前设置的钱包密码，即能查看到钱包状况，


点击OK之后 


11. 钱包转账：“Transfer”空白栏输入要转账的数量，“To”空白栏输出收款钱包地址，“Send”发送；

 

三、 挖矿软件使用方法

1. CPU挖矿

   直接用钱包挖矿，无需单独下载挖矿工具。

1.1在XDag.x64文件夹建立一个文本文档，命名为“Cpuminer.bat”，编辑内容为：

xdag.exe -d -m 4 xdag.dlpool.cn:6767 

pause

解释：其中“4”，表示使用CPU挖矿的内核数，数字越大算力越高，具体根据自己的CPU核心数设置。

“xdag.dlpool.cn:6767”，表示矿池地址，这个是中国矿池，其他矿池可以相应更改。

 

1.2编辑完bat文件之后，双击运行“Cpuminer.bat”文件，会提示要求输入钱包密码，输入钱包密码，回车，CPU挖矿开始，输入“miners”查看矿工工作情况；也可以到矿池后台查看矿机工作情况，矿池后台：http://xdag.dlpool.cn:777/

 

1.3亦可使用xdagwallet.exe文件挖矿，连接矿池之后，将调节CPU核心数，点击“Apply”即可。


 

2. N卡GPU挖矿

GPU工具下载：https://github.com/jonano614/DaggerGpuMiner/releases

   

2.1下载DaggerGpuMiner.0.2.3.x64.zip文件，解压，在DaggerGpuMiner.0.2.3.x64文件夹新建一个文本文档，命名为“Gpuminer.bat”，编辑文档内容为：


DaggerGpuMiner.exe -G -a c+4GEaIQhROB91MXz5Jp9VYNG82BK1PH -p xdag.dlpool.cn:6767 -t 0 -v 2 -opencl-platform 0 -opencl-devices 0 -nvidia-fix

pause

其中：“c+4GEaIQhROB91MXz5Jp9VYNG82BK1PH”替换为自己的钱包地址，“xdag.dlpool.cn:6767”为矿池，“-nvidia-fix”为因伟达显卡特有命令；

 

2.2编辑完之后，保存，双击“Gpuminer.bat”文件，出现如下界面，表示GPU已经开始挖矿工作，此时可以去矿池后台（http://www.dlpool.cn/）查看矿工工作状态。

 

 

2.3 Nvidia多GPU的例子：

DaggerGpuMiner.exe -G -a c+4GEaIQhROB91MXz5Jp9VYNG82BK1PH -p xdag.dlpool.cn:6767 -opencl-platform 1 -opencl-devices 0 1 2 3

注意：c+4GEaIQhROB91MXz5Jp9VYNG82BK1PH更换成您的钱包。如果您拥有超过4个GPU，则需要强大的CPU以使系统继续为GPU提供新的工作（Intel i5，i7，AMD FX8350，AMD Ryzen 5或更高版本）。否则，系统可能无法充分利用GPU并可能无法响应。

3. A卡GPU挖矿

A卡挖矿设置与N卡基本相似，具体可以参照N的设置方法。

AMD多GPU的例子：

DaggerGpuMiner.exe -G -a WALLET_ADDRESS -p xdag.dlpool.cn:6767 -opencl-platform 0 -opencl-devices 0 1 2 3
