如果技术面告诉你比特币下跌到3000美金，你的基本面信仰还在吗？

  • 柚子社区

来源：币壳讲堂

时隔半年，回忆一下17年末和18年初充斥在币圈各种喧嚣疯狂的声音：

2018是区块链元年，比特币将到20万美金；各种机构正在加速进场，再不上车就晚了；拿住优质币，坐等财务自由；徐小平更是宣称区块链革命已到来，号召传统互联网公司CEO跳入洪流中拥抱革命。

半年后回头看看，现在有多少基本面坚实的优质数字货币能够在这个熊市坚挺？



市值前100的币种中，比特币离高点跌幅已达70%，最坚挺的是USDT，跌幅为16.7%，有32个币种离高点跌幅超过90%，60个币种跌幅超过80%。

我们来看看一些看涨的基本面，区块链技术依然是数字货币背后最强大的基础。

1. 许多主要机构（高盛，纳斯达克和摩根大通）正在安排设立交易组合。

2. 一些最大的风险投资基金和对冲基金正在将其投资组合转变为包含加密数字货币。

3. SEC提出BTC + ETH不是证券。

4. 在Coinbase收购一家金融服务公司成为美国证券交易委员会监管的经纪商的过程中，合格的托管人正在建立稳固的基础设施。顺便说一下，美国证监会已明确指出推出比特币ETF的最大障碍之一是由于该空间缺乏合格的托管人。一旦Coinbase在第三季度/第四季度正式启动为合格保管人，美国证券交易委员会将把比特币ETF放回议程讨论。此外，Coinbase正在收购Paradex，为用户采用全新技术提供服务。此举将显着增加客户通过Coinbase购买的加密货币的数量。

所有这些新闻上的信息都表明为下一波牛市奠定坚实基础。基本面很重要，他们提供了一个长期稳定投资的良好预期。但是，事实是，这半年来，在熊市中，一切都与比特币相关的都流血不止。

那么为什么价格不能反映这种基本面的看涨？

1.加密数字货币极端主义者自二月以来一直高喊“下跌抄底，梭哈抄底，财务自由”，并对历史熊市视而不见。他们乐观地说：“这次不同了，机构资金正在涌入！基本面是如此强大。”是的，巨量的机构资金进来了，但这些机构没有像加密数字货币极端主义者所喊的那样盲目买入，抄底。

这些都是懂得如何赚钱的专业公司。当价格仍然在高位时，他们并不是愚蠢地拿钱进入这个市场。他们拥有强大的现金流动性，并且知道如何现在改变市场，以便以更低的价格进入市场。如果他们不能以更便宜的价格进入交易所，他们会在价格上涨之前摧毁比特币市场，比如通过期货市场做空赚取更多的币，拿到更低价位的筹码。

2.这些在新闻上看到的好的基本面，仅仅是有些人想让大部分人看到的好的基本面，数字货币市场不像股票那样有可以量化的基本面分析计算模型，比如市盈率，资产负债表，股东增持减持情况，财务报表这些基本数据，很多数字货币仅仅只有一个简单的白皮书，一个大概的发展规划大纲。数字货币市场这些不能量化的基本面情况，只能欺骗大部分人的眼睛，让群体陷入非理性疯狂的虚假繁荣幻想中。

3，除了媒体让你们看到好的基本面之外，还有很多你在新闻上看不到到的许多因素可以促成抛售压力：

1. 由于价格下跌，矿工们不得不将挖出来的币卖出投入运营成本，这使得他们的利润率变得越来越低。

2. 当ETH进入疯狂期，ICO公司募集了数十亿美元的资金，因为价格下跌，他们把他们的ETH抛售进市场。

3. 鲸户们很容易地或免费地拿到各种代币，并将它们抛售在市场上。如果通过站台，空投或交易量比赛简单地无成本地拿到这些代币，他们真的认为这些代币有多少价值？

4. 后进场的散户投资者受不了价格的下跌，高买低卖，不可避免地会摆脱市场，造成市场的继续下跌。

现在我们来看看技术面的分析

从周线的各种交易指标上，我们仍然处于技术性深度下跌趋势，均线空头排列，DIF和DEA指标刚刚进入零轴下方，熊途漫漫。到目前为止，几乎所有的k线图都显示了一个具有大量下降三角形的直接熊市通道，熊市会依然持续半年左右，3k到4k这个区间才能止住下跌趋势，目前没有任何交易指标显示健康的向上趋势。不过可能会在6k左右出现强势反弹。我们可能很快就会看到市场做出反应，因为现在市场已经出现了近8周的下跌时间，周线KDJ超跌，日线MACD趋稳，将会有一个日线级别的反弹，反弹高度在7k到8k之间。

忽视技术指标并只看基本面，仅仅因为推动比特币发展的一些良好基本面而继续在每一个反弹点买入进场都只是无稽之谈。是的，我同意，基本面对加密数字货币市场非常有用。但是，除非在交易指标上证明，否则所有东西都是死亡短期反弹，并且继续在市场流血割肉。图表显示下跌的交易量在一个狭窄的范围内，并在支撑位上下探。等待，耐心一点。不要期望在熊市进行交易，否则你会重蹈覆辙。

请记住：基本面在熊市趋势的短期内几乎没有影响。

始终观察k线图上的各种交易指标，而不是以你自己的偏见或感觉来进行交易。如果你没有注意到数字货币的技术分析，并希望根据感觉和新闻上的一些基本面情况来交易获得短期收益，那么我希望你能好运。
