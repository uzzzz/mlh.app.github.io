在第二天的学习中使用Spyder作为开发环境，这里与大家共享：

  • 首先在Windows环境下载安装 Python

  • 选择IDLE

  • 进入Python Shell

  • Python安装成功后，下载安装开发环境Spyder 

源码地址：https://github.com/spyder-ide/spyder/releases
windows请选择.zip
win+R 输入cmd 进入命令行，在spyder根目录下执行setup.py install

  • 到此并没有结束，下篇文章将讲述所依赖的其他模块安装过程！
