（1）Lite版本挖矿教程

程序准备

1.访问Bminer官网，下载Bminer的Windows平台Lite版本。如图所示，数字2版本为Lite版本；

2.找到您下载的Bminer，并解压。

3.打开解压好的文件。

注：Bminer 7.0.0之后支持以太坊（ETH）单挖，Bminer 8.0.0之后支持以太坊（ETH）双挖。

Windows有2种版本，Lite版本通过修改bat文件参数挖矿，Launcher版本通过操作界面修改参数挖矿。

ETH+XVG双挖教程

Bminer 从7.0.0版本开始支持以太坊单挖，从8.0.0支持以太坊和XVG/DCR双挖功能。

Claymore是较为常用的以太坊双挖程序，单挖以太坊费用1%，双挖费用1.5～2%。

Bminer双挖效果比Claymore更好，且单挖以太坊费用仅为0.65%，双挖抽成仅1.3%。

Bminer充分利用以太坊挖矿时闲置资源，在不影响以太坊挖矿速度情况下，达到最佳双挖效率。


本教程以双挖XVG为例，DCR的双挖操作和XVG相同。

步骤1: 右键点击mine_eth_xvg.bat文件后编辑文件；


@echo OFF

REM Change the following address to your ETH addr.

SET ADDRESS= ETH收款钱包地址

REM Change the following address to your XVG addr.

SET ADDRESS_2= XVG收款钱包地址

SET USERNAME=%ADDRESS%.矿工名

SET POOL=eth-us-west1.nanopool.org:9999

SET SCHEME=ethash

SET USERNAME_2=%ADDRESS_2%.矿工名

SET POOL_2=xvg.blake2s.com:8888

SET SCHEME_2=blake2s

START "Bminer: When Crypto-mining Made Fast" bminer.exe -uri %SCHEME%://%USERNAME%@%POOL% -uri2 %SCHEME_2%://%USERNAME_2%@%POOL_2% -api 127.0.0.1:1880


编辑ETH挖矿设置，红色字段：

步骤2: 修改ETH钱包地址(ADRESS字段）为您的收款ETH钱包地址。

步骤3: 修改ETH矿池地址（POOL字段）为您想用的ETH矿池，比如：eth-us-west1.nanopool.org:9999。

步骤4: 修改协议设置（SCHEME字段）为选定ETH矿池相应SCHEME，nanopool的SCHEME字段对应ethash。

注意：不同ETH矿池使用的协议不同，请访问Bminer官网找到常用矿池对应的协议，比如：

F2pool对应的协议字段是ethproxy，nanopool对应的协议字段是ethash。


编辑XVG挖矿设置，蓝字字段：

步骤5: 修改XVG钱包地址(ADRESS_2字段）为您的收款XVG钱包地址。申请XVG钱包地址请参看下小节；

步骤6: 修改XVG矿池地址（POOL_2字段）为您想用的XVG矿池，比如：xvg.blake2s.com:8888

步骤7: 修改协议设置（SCHEME_2字段）为选定XVG矿池相应SCHEMEblake2s。

步骤8: 保存该文件，即可点击运行。

XVG钱包获取步骤：

您可以通过XVG官网下载官方钱包或者在交易所创建钱包生成地址。

下方为币安交易所账号申请步骤：

步骤1.浏览器输入：https://www.binance.com/ 并注册账号；

步骤2.点击Funds菜单下的balance，输入XVG，点击Deposit按钮；

步骤3.下方红框即为XVG的钱包地址；
