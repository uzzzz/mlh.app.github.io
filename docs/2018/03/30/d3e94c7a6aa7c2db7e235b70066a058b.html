技术交流

16年开始关注研究区块链相关知识，主要从事以太坊的技术研究，积累了不少行业经验，后入职火币网进行相关的实践，由于个人精力有限，同时忙很多事情，因此除了博客、微信公众号等公开性发表内容，很少针对每个人进行技术问答，这也是客观条件限制。

但与此同时也为大家提供了一些相应的技术交流和咨询的通道。

QQ技术交流群

群号：659809063，致力于打造国内第一家以太坊技术交流群活跃群，知识分享，结实同行，技术交流，小白进群快速成为专家。广告及炒币勿进，发现必踢。目前群组已超过500人，群组活跃度居高不下。

博客专栏

当然，当你看到此篇文章时，一定可以在csdn博客中找到我的关于区块链的技术专栏：http://blog.csdn.net/column/details/13651.html

达人课《以太坊使用及代币开发实战》

课程地址：http://gitbook.cn/gitchat/column/5ad98d9479e8c577efc7557d

在学习以太坊入门过程中是否遇到过以下问题：

不知如何部署安装 geth 节点？
不知如何进行 geth 节点的命令操作？
不知为何公有链的数据同步、geth 程序问题接连不断？
想调用 JSON-RPC 接口却一头雾水？
想发一套代币却不知如何开发、编写、发布，并在网站上查询到余额和代码？
想用 Java 版本去调用代币执行操作，却不知如何下手？
想开发一套基于以太坊的钱包却不知核心业务如何处理，安全如何保障？
……
如果以上问题遇到一个或多个，那么本系列课程就是为你量身打造的。本课程的目标就是带大家快速上手实战，综合命令操作、代码示例、内容详解、实战经验等多方位信息呈现，以最短的时间达到最高效的学习效果。课程以以太坊为核心，从具体的以太坊节点部署、真实环境同步及问题、以太坊节点 JSON-RPC 接口调用、Token 发行及对应的 Java JSON-RPC 调用和开发数字货币钱包的设计等。

通过本系列文章可以让新手快速入门，进行相应的开发，同时避免一些不必要的资产损失，完成从入门到核心业务开发的蜕变。
