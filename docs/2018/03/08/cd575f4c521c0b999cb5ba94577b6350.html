3.如何把文本文件和图片存储到ipfs

  • 柚子社区

本节讨论如何将文件存储到ipfs上

1. 生成文件

创建一个文件 file1.txt，写入内容为hello ipfs



2. 将file1.txt 增加到ipfs中

通过ipfs add file1.txt，将file1.txt 存放到ipfs


其中 QmcQwhK5AKLToetXRRakKKAF2zWfH7YEDUxZvPXk3qDBgX 是ipfs返回的hash值，我们可以通过该hash值得到文件内容

3.根据hash值查看文件内容

ipfs cat hash值，查看文件内容 

 

4. 通过web查看ipfs的内容

通过http://ipfs.io/ipfs/hash值，查看文件内容

5. 上传图片

上传图片的方式和上传文本是一样的


6. 查看图片

查看图片方式和查看文本也是相同的，也是通过http://ipfs.io/ipfs/hash值



7. 文件的下载

通过ipfs get hash值，完成文件的下载，文本文件和图片都是一样方式
