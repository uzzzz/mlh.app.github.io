05.区块链的那些事儿-Bitcoin源代码的编译与部署

  • 柚子社区

本小节主要讨论基于gitbhub.com进行Bitcoin源代码的编译与安装，从而生成bitcoin

注意以下过程会是个比较漫长的过程，读者要做好心里准备，另外我的操作系统是mac 


1. 安装依赖

首先需要安装相关依赖

brew install automake berkeley-db4 libtool boost --c++11 miniupnpc openssl pkg-config protobuf qt libevent


同时，安装RSVG依赖，在make deploy 生成image时会用到


2. 克隆源代码

把bitcoin的源代码下载下来



3. Config 

配置

./autogen.sh

./configure


4. make

make


5. make deploy

6. 生成Bitcoin Core 客户端

第5步完成后，可以生成Bitcoin Core 客户端


打开Bitcoin Core 会进行欢迎页面，从欢迎页面中可以看到需要203G的空间来下载完整的比特币区块链
