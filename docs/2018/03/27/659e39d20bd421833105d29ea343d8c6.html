安装go

  1. sudo apt install golang（如下图,尝试安装最新的版本apt install golang-1.10）

  2. 环境变量的配置(ubuntu 16.04 桌面版中默认没有安装vim，这里用gedit)


    这里的环境变量参考网上结合自己的理解写了如下实例
    (GOROOT:go的安装根目录,运行环境；GOPATH:你的go工作目录,存放编译后的包和可执行程序,可以不设置)

  3. 运行go version,查看版本

运行go

cd /项目目录,新建一个xx.go文件,内容如下:

package main
import ( "fmt" ) func main(){ fmt.Println("世界，你好！") }

运行(查看go help build看详情)

上述结果一个是重命名，和默认以包名为名生成的可执行程序
