以太坊solidity开发环境部署

  • 柚子社区

基于Ubuntu 16.04

安装Nodejs和npm

sudo apt-get install -y nodejs  
sudo apt-get install npm

Npm更换淘宝源
sudo npm config set registry http://registry.npm.taobao.org

安装truffle

sudo npm install -g truffle

更新nodejs到最新

sudo npm install -g n 

sudo n stable


如果出现 node: No such file or directory
创建软连接 
ln -s /usr/bin/nodejs /usr/bin/node

安装测试环境

（模拟接入以太坊）ganache

命令行版

sudo npm install -g ganache-cli 

Gui版

 truffleframework.com/ganache下载

合约部署与发布

新建一个项目:
mkdir project 
cd project
truffle init  

可以看到部署好了环境

Contracts：放入你的合约文件

Migrations：部署脚本

Test：测试文件夹

Truffle.js/truffle-config.js：truffle配置文件

也可以使用测试用例：

mkdir metacoin

truffle unbox metacoin  

这是一个虚拟货币的例子

配置项目：

1.  打开测试环境（测试时不要关闭）
ganache-cli

2. 将IP和端口配置到truffle.js

 

运行项目

truffle compile     #编译

truffle migrate     #部署到ganache

truffle test     #执行测试脚本
