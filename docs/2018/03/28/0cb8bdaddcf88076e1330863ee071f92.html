Typora及有道云

应工作需要要编写一些简单的甘特图，特将自己所熟悉的编辑器写法记录如下

Typora是一个功能强大的Markdown编辑器,极简而强大。线下写文件推荐使用Typora，如果要线上协同的话可以使用有道云，虽然它分享连接如果短时间多次点击有bug，但是整体还是不错的 。

流程图

原始版

​```flow
st=>start: Start
op=>operation: Your Operation
cond=>condition: Yes or No?
e=>end

st->op->cond
cond(yes)->e
cond(no)->op
​```

typora版

​```mermaid
graph TD
A[Hard edge] -->B(Round edge)
    B --> C{Decision}
    C -->|One| D[Result one]
    C -->|Two| E[Result two]
​```

有道云版

graph TD
A[Hard edge] -->B(Round edge)
    B --> C{Decision}
    C -->|One| D[Result one]
    C -->|Two| E[Result two]

总的来说，有道云应该是将【“`mermaid】写入了正则，省略了该步骤

序列图

基础版

​```sequence
Alice->Bob: Hello Bob, how are you?
Note right of Bob: Bob thinks
Bob-->Alice: I am good thanks!
​```

typora版

​```mermaid
%% Example of sequence diagram
  sequenceDiagram
    Alice->>Bob: Hello Bob, how are you?
    alt is sick
    Bob->>Alice: Not so good :(
    else is well
    Bob->>Alice: Feeling fresh like a daisy
    end
    opt Extra response
    Bob->>Alice: Thanks for asking
    end
​```
