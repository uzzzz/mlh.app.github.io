数字货币交易所BTBTOP简介及活动介绍

  • 柚子社区
今天推荐一家新近上线的平台——BTBTOP，是18年2月底上线的一个交易所，运营的公司是一家新加坡公司，公司专注于数字货币交易所方面的研发，现在已经上线了场外交易的功能，还有比特币和以太坊2种货币，预计在3月底上线USDT，4月份上线币币交易功能。

BTBTOP是一家新的交易所，在产品设计和体验方面还是不错的，有时间大家可以看看。


通过对比可以发现，该平台优先开发的是H5（即手机网站）页面，是通过H5适配做的PC站，所以预估该平台后期会在移动端进行发力。

由于是上线初期，该平台现在的活动力度也是相当大的，主要是以下3个：
1、手续费全免
这个活动的时间从上线开始到3月31号结束，在这个时间内，平台所有的交易都是免手续费的，非常适合一些有卖币需求的用户。

2、实名糖果

这个需要单独的说明下，用户注册后实名即可获得一个价值0.00015比特币的实名糖果，实名后直接进入账户，可出售可提现，看起来不多，可是由于没有像别的平台设置一些前置门槛，可以直接使用，所以是活动力度非常大的，我们可以看到，别的平台的类似活动，要不给的非常少（例如OKEX，给的还不够转账的矿工费），或者就是参与门槛很高，普通用户很难获得（例如火币动辄上百万成交量要求才可动用）。


3、交易红包

这个是实名糖果的补充，引导用户参与平台交易，红包是随机红包，最高可获得0.1BTC（纯运气），实测红包获得最多的是0.0001-0.001之间。也是活动力度非常大的，因为只要充值或者购买，即可激活交易红包，直接获得，无其余门槛，获得后即可提现或者充值。

4、邀请活动

该活动类似其余平台的邀请活动，不过添加了红包功能，首先用户邀请一个好友注册并实名即可获得一个随机红包，红包类型同交易红包，其次如果用户邀请的好友最终交易了，邀请人还可以获得部分好友交易产生手续费，最高可达100%（即全部获得）。

 

由于是新平台上线，活动力度相当给力！
