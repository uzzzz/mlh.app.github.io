ETHOS是一个基于linux系统的挖矿深度定制系统。由于系统相对于windows来说，体积非常的小巧，因此可以直接刻录于U盘，省下了一块硬盘的钱。并且系统占用资源少，对于相同的配置来说，比windows系统更加稳定。ETHOS系统，是外国一个团队制作的。并且U盘售价需要39美元。我们怎么会为开源的linux系统付钱呢。因此我镜像了U盘的内容，并刻录到了自己其他的U盘上做了实验，发现完全可以正常使用，也就意味这不需要再为ETHOS系统付钱了。
功能细节
开机即挖: 自动获取IP地址,无需安装任何驱动, 配置桌面, 编译软件.
最大支持 16 个AMD RX系列GPU: 包括支持 RX Series 功率控制 以及Z170/X/Z270/X/Ryzen 芯片.
最大支持 16 NVIDIA GPUs: 任何 2GB以上 GTX 900 和 GTX 1000 系列.
最大支持 8 AMD R7/R9 系列 GPUs: 任何 2GB以上 HD 7000 系列, 任何 R9 200/300/Fury/Nano.
支持多种币: 包括 Ethereum, Zcash, Monero 以及很多其他可以用显卡挖的币种.
浏览器即可控制: 通过浏览器输入矿机的IP 就可以安装和配置矿机.
支持所有的硬分叉和软分叉:无需额外的区块链存储外设, 矿池和钱包自动处理.
运行稳定: 无故障运行在上千台矿机上
远程配置: 设置矿机远程重启，设置核心频率, 内存频率, 风扇控制, 修改矿池, 以及其他配置.
轻量级配置:稳定工作在最差的5代处理器（只需要2G内存）的机器上
GPU 过热保护: GPUs达到设置的最高温度后会自动关闭
Stratum 支持: 已经自动配置了高效的stratum协议.
自动报告状态: 网页控制面板包含详细的数据统计, 图表, 和事件报告 (example).
便于KVM: 开机自启动控制端，无需鼠标即可控制.
更新便捷: 只需一行命令即可更新到最新版本.
启动快速: 矿工快速启动, 很低的cpu和磁盘占用, 不需要输出空间.
Bios 刷写: 集成了快速刷鞋bios的aitflash工具.

1.1系统更新
2017/11/10 现在 最新版本是1.2.5
如果日后有新版本，可以使用命令：
sudo ethos-update && sleep 5 && r
来更新系统。
然后重启机器。如果重启后发现机器有任何异常，可以使用命令：
sudo ethos-update reupdate && sleep 5 && r
来退回到之前的版本。一般来说，不会有问题的。  （用这个版本稳定就不需要再折腾了，不建议更新了。）
1.2安全设定
系统默认账号是ethos或root，默认密码是live。推荐安装后使用命令：
sudo passwd xxxxx
#xxxxx是希望的密码
来修改密码。
1.3取消远程配置
有些人说我的机器一重启，就换成了别人的钱包，是不是别人做了手脚。其实其ethOS自带的远程配置的原因。虽然远程配置很方便，但是我还是推荐本地配置，毕竟用别人的盗版，还占用别人的服务器资源确实太不好了。可以清空remote.conf文件内的所有字符，或者使用命令：
force-local
来强制使用本地命令
1.4 local.conf配置文件的设定
#使用claymore来挖eth
globalminer claymore
#设置最高温度。有些矿友挖几个小时就停机了，可能就是这个原因。
maxgputemp 85

#钱包 更换为自己的钱包地址

#矿池1 更换为自己的矿池
proxypool1 eth.f2pool.com:8008（鱼池）
#矿池2 更换为自己的矿池

#globalmen 4600
