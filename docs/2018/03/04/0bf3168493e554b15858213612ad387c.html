Sia-UI使用入门及经验分享
作者：家辉，日期：2018-03-04 CSDN博客： http://blog.csdn.net/gobitan

摘要：Sia-UI是Sia的官方提供的文件上传，挖矿及钱包软件。这里分享一下Sia-UI使用的一些经验，供初次使用的朋友参考。

第一部分：环境搭建及简单介绍 
第一步：下载Sia-UI
进入 https://sia.tech/get-started，下载Sia UI的windows版本。文件为Sia-UI-v1.3.1-win32-x64.zip，接近60M。解压到硬盘目录，如G:\blockchain\Sia-UI

第二步：运行Sia-UI
直接双击根目录下的Sia-UI.exe即可运行，界面如下。

界面包括了三个部分：

[1] 最上面的区块同步栏，右侧展示当前区块同步的进度;
[2] 左侧菜单：
  • Files显示上传的文件，也就是你租用的部分;
  • Hosting是你出租的存储空间
  • Wallet是你的Sia钱包
  • Terminal是Sia提供的一个命令行工具
[3] 右侧上面的栏目会根据左侧选择的菜单而不同。

第三步：创建Sia钱包地址
选择左侧的钱包菜单，
注意：每次点击"Recieve Siacoin"都会产生一个新的地址，之前的地址依然可用。查看完整地址列表可到Sia-UI的Terminal中输入"wallet addresses"

第四步：Hosting出租存储空间
在开始出租硬盘之前，钱包里最低至少要有2000SC。目前出租的价格比较低，而且要求7*24小时在线，所以专门搞这个基本没啥赚，甚至会亏。至于对这个有兴趣，而且机器本身就7*24小时运行，且有空余的硬盘资源的可以考虑这样做。如果你经常离线，或者把数据给搞丢了，那么你的抵押的SC币就会被扣掉。

第四步：使用命令行
略，参考第三步中对命令行的使用。

第二部分：经验分享
经验[一]：Sia同步数据库的存放位置的修改
开始使用的时候可能都有疑问，Sia将同步的数据存放在哪儿的。可以通过界面中About页面的"Show Sia Data"中找到。点击它即可定位到数据存放位置。
该目录下有一个叫config.JSON的配置文件，内容如下：
{
    "homePlugin": "Overview",
    "siad": {
        "path": "G:\\blockchain\\Sia-UI\\resources\\app\\Sia\\siad.exe",
        "datadir": "C:\\Users\\dennis\\AppData\\Roaming\\Sia-UI\\sia",
        "rpcaddr": ":9981",
        "hostaddr": ":9982",
        "detached": false,
        "address": "127.0.0.1:9980"
    },
    "closeToTray": true,
    "width": 965,
    "height": 433,
    "x": 131,
    "y": 138,
    "version": "v1.3.1",
    "defaultSiadPath": "G:\\blockchain\\Sia-UI\\resources\\app\\Sia\\siad.exe",
    "siac": {
        "path": "G:\\blockchain\\Sia-UI\\resources\\app\\Sia\\siac.exe"
    },
    "receiveAddresses": [],
    "receiveAddress": "a53343a5460b3bc0f20868a7ec1edc2a9582adc86579a307300ecc2c670daff872f8168ef970",
    "address": null
}
可以修改datadir的值。

经验[二]：Sia区块数据大小
Sia区块同步结束后，总大小大约11G(当前区块高度：144300，2018-03-04)左右。
