游戏上线之初，云鱼官网也会给各位游戏玩家们送福利，只要注册即送云鱼币。

由Accretion Game Studio公司发布的基于区块链技术的云鱼游戏，提出了一个全新的概念，利用区块链技术将游戏资产“鱼”通过智能合约存储在以太坊区块链上。

玩家的游戏资产（即云鱼）保存在区块链上，没有任何人可以篡改、夺取和删除玩家的游戏资产（这根本上区别于中心化的游戏资产）。

毫无疑问，云鱼重新定义了“游戏资产”的归属问题，把游戏资产真正带入“去中心化”的时代。

AGS始终坚信区块链技术代表了一种摆脱游戏物品存储和所有权领域中心化镣铐的方法，云鱼也因此应运而生。

云鱼的目标是推动区块链离开币圈的小圈子，大规模的走向普通人群，让不懂区块链、不是币圈的人也能体验到有别于传统游戏的全新的区块链游戏。

云鱼是一款呆萌的小鱼社交养成游戏，鱼的总量为2100万条上限，这从根本上区别于不限量的资产，稀缺性对于上线后维持高价格和炒作更有保证。

用户可以通过交易市场直接购买获得云鱼，也可以以相应成本透过捕鱼系统捕获云鱼。而钓鱼游戏是一个休闲类游戏和概率游戏结合的游戏，而捕获的方式和概率也是通过智能合约写入以太坊区块链当中的，规则也是公开透明且不可更改的。

可以想象，随着云鱼价格的上涨，参与钓鱼的玩家会增加，钓鱼的难度会提高，钓鱼的成本会上升，进一步刺激云鱼价格的上涨。

有别与CryptoKitties等区块链游戏的是，云鱼不需要靠持续不断的喂养ETH来保持所有权。即使不对云鱼进行任何喂养，它也能在市场化的交易平台中逐步获得升值。

当然用户也可以对云鱼进行喂养，使其获得繁殖权，再通过与平台其他拥有云鱼繁殖权的用户交易，获得繁殖生成的相应代“云鱼”。

而拥有不同基因组的云鱼都会有不同的外形，颜色等等性状，而这些特殊的或是稀缺的基因组决定了每条云鱼在市场上超出想象的价值。

除了云鱼游戏本身，由于云鱼游戏是开放开源基于以太坊区块链的智能合约系统，这意味着不仅官方会继续开发各种游戏和应用，其他基于云鱼智能合约数据的开发者也可以基于本链产生的云鱼的数据特征进行各自的开发，提供各自针对性的游戏或者应用，提供给云鱼的玩家。

比如在游戏上线之初，就有各大游戏厂商入驻我们游戏平台，包括腾讯游戏，网易游戏等。
