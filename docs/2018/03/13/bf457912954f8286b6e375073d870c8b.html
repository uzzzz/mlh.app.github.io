以太坊钱包，手动关联合约地址。

  • 柚子社区

本文 介绍 以太坊 钱包 手动 关联 合约地址。

在 以太坊钱包 的 CONTRACT页面，如图下：



点击 WATCH CONTRACT ，出现 页面 如图下：


分别 输入，合约地址，合约名称，合约 的json 。

合约 的 名称 可以 随便写，但是，合约 地址，和 合约 json ，需要 从 https://etherscan.io/ 这个网站获取。

首先，你需要 知道 你的 合約地址 是 什麽。

创建 代币 的 合约 地址，可以 这样 查看，步骤 如图下：

1、在 钱包 主页 WALLETS 页面，下面有  LATEST TRANSACTIONS ，找到 当时 创建 代币合约 对应 的 Created contract ，点开 后，页面 如下


点击 箭头 指向 的 字，跳转 浏览器，可以 看 到 


红框 中 的 0x 开头 的 地址， 即为 你的 合约 地址，可以 点击查看。




点击 后 查看 


红框中，就是 json 串 ，就是 可以 在 钱包 中 输入 的 json 。
