想要弄懂区块链，2018你不能错过的四个技术进展

  • 柚子社区
2018-02-12  丁丁

如果有关注最近的头条新闻，你也许会得出一个这样的结论—


随着市场的不断起伏，现在说要从货币角度去认真对待加密货币其实还为时过早。


的确，尽管业内那些最知名的开发者们一直在尽最大努力，但世界上最大的加密货币「比特币」从诞生到现在一直处于不稳定状态，而且在使用方面很难(伴随着一定的风险)。



但尽管如此，来自世界各地的开发者与热心人士仍在努力进行改进。


如果顺利的话，他们将解决那些大多数区块链用户已经开始习惯的问题——高昂的手续费和漫长的等待时间。


事实上在未来的一年里，如果你想进一步弄懂区块链，以下四大令人兴奋的技术进展你一定不能错过：


1. 离链信道


有没有存在一种技术能够使得基于区块链技术的交易却完全避免使用区块链？



其实这是长久以来除了区块链以外的有关支付渠道的一个重要想法，最早可以追溯到2015年，但2018年可能终于到了实现的时候。


举例来说，它有点像比特币闪电网络 ( Bitcoin Lightning Network )。



从本质上讲，区块链以外的支付渠道将允许两个人使用任何一种加密货币来回发送小额支付，只有在绝对必要时才在区块链进行结算(高昂的费用和缓慢的交易时间)。


基于这些潜力，这种离链信道的想法正开始变得流行——以太坊开发人员虽然经常与比特币同行意见不一，但他们正在开发相同类型的解决方案。


但这不仅仅是一场竞争，因为还有不少人相信2018年可能会有所不同，因为实际的实时交易可能会发送大量数据。


比特币闪电网络背后的开发人员已经宣布，基于前期成功的测试，这项技术几乎已经准备就绪。


与此同时，以太坊的开发人员也公布了他们的成功测试概念版本Raiden Network，一个更雄心勃勃的版本可能即将问世。


2. 真实赌注


随着加密货币市场的扩张，人们也开始关注到维持加密货币市场所需要的大量电力资源。


虽然相关数据难以具体确定，但作为比特币开采基础的共识协议——工作证明——被定义为能源密集型流程。



因此，人们担心其用电可能对环境产生大规模影响。


这导致了2011年开展了一项新的研究，这个被称为“利益证明”或“通过投票达成共识”的想法虽然确立了，但事实上并没有按照以太坊计划的规模来实现。


因此，期待已久的Casper项目很可能在今年重生，其早期版本也开始显露端倪。在其除夕发布的一个测试版中，Casper的一个支线版本被认为运行良好。



以太坊创始人Vitalik Buterin也表示，他期待这项技术将在将来某个时候与工作证明一起进行测试。


3. 隐私保护


隐私在区块链领域大多数时候都可以说是一个被忽视的承诺，但今年这个问题可能会有所改善。


最值得注意的是零知识证明的进步，Vitalik Buterin称之为：


目前密码学中唯一一个被大肆宣传的东西。



这种证明将变得越来越便宜，部署起来也越来越容易。


这是一种可以隐藏信息而不冒有效性风险的加密技术，其实它已经在很小程度上被应用到以太坊中，这可能会导致一波初创企业以新颖和意想不到的方式来尝试私人智能合同。


此外，在本月早些时候发表的一份白皮书中，出现了一个在不损害信任的情况下实现零知识的系统——这是该技术早期迭代中的一个争论点——这一更新可能会产生令人兴奋的后果。


而随着现有技术的成熟，以隐私为中心的密码货币如门罗币和zcash也将得到改进。


4. 去中心化交易所


随着区块链行业最大的交易都开始面临着难以应对新用户的涌入的困境，越来越多的项目正在开发一种称为去中心化交易所的项目。


去中心化交易所不仅表示一种新的基于浏览器的交易，更多时候其代表着一种软件用户可以在没有中央实体的情况下将一种密码货币与另一种密码货币交换。


到目前为止，硬钱包Ledger已经集成了去中心化交换所Radar Relay，允许用户基于以太坊地交换Token。



虽然功能有限——只有一个钱包支持，而且只能发送基于以太坊的Tokne，但许多业内人士认为，这才是加密货币交易所的未来。
