Mac安装Tesseract-OCR

安装： brew install tesseract

语言库

下载地址：https://github.com/tesseract-ocr/tessdata

简体中文库：chi_sim.traineddata

下载简体中文库chi_sim.traineddata，然后将文件拷贝到到：/usr/local/Cellar/tesseract/3.04.01_2/share/tessdata 目录下即可。
