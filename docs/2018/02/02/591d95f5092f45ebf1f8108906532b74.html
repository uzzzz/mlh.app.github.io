【中国互金协会李礼辉：对区块链总体上持宽容和支持的态度】

 

据中国新闻周刊消息称，中国互联网金融协会区块链研究工作组组长李礼辉表示：当前，监管部门对区块链总体上持宽容和支持的态度，限制了涉嫌违规的炒作行为，在制度建设、数字票据研发方面投入了很大的力量。

 

陀螺评语：区块链项目可以做，不涉及炒币就好。

【“华尔街之狼”Jordan Belfort：比特币在崩溃前或会达到5万美元】

【Coincheck将被律师团起诉，而Coincheck称有能力偿还客户的损失】

 

对于Coincheck黑客事件，5名律师在1月30日组成了受害者律师团，并在2月1日公开了律师团主页。根据主页信息，律师团将由IT企业经营和技术金融证券交易法等等法律专家律师组成，团长为北周士律师。律师团的目标是“为受害者得到应有的救济，并且对虚拟货币实现适当的管理”。第一次诉讼将在2月中旬，参加者必须满足在Coincheck开设过账户，并且在黑客事件发生时在Coincheck存有NEM两个条件。

 

另外，CoinCheck称目前仍有能力偿还客户的损失。

 

陀螺评语：无论是法律还是安全，虚拟货币要解决的问题都不少。

【韩国Bithumb交易所因安全问题遭整改】

 

韩国Bithumb交易所官网显示，由于违反韩国相关法律，交易所收到了韩国通信委员会下发的更正命令。 违反事项包括： 1. 对入侵防护和检测系统的安装和运行的疏忽行为； 2. 没有对存储在计算机上的使用者个人信息进行加密； 3. 没有及时更新相关安全程序； 4. 没有对个人信息进行适当的记录管理。

 

陀螺评语：最近数字货币交易安全问题频现，尤其是日本CoinCheck黑客被盗事件更是给国际敲了次警钟，而对于数字货币交易量全球第一的韩国来说，这更是不能忽视的问题。估计未来韩国在安全性方面的要求将更严格，对交易所的监管力度也会更大。

【韩国副总理金東淵探访小米中关村，强调中韩在区块链技术领域需要合作】

 

今日韩国副总理兼企劃財政部长金東淵（김동연）访问了中国北京高科技开发区小米中关村，介绍了韩国的创新政策并强调了在区块链技术领域的中韩双方合作的必要性。

 

陀螺评语：想到早在2016年12月，韩国宣告首个区块链联盟成立，而这个区块链联盟的成立加上如今的动作，证明了韩国政府发展该国金融科技领域的坚定决心。

【俄罗斯总理：需要对数字货币进行国际化干预】

 

俄罗斯总理表示，我们需要对数字货币进行国际化干预。

 

陀螺评语：现在每个国家对数字货币的相关政策及态度并不一致，要进行国际化干预更多的需要国际化组织的插手。

【印度目前并未禁止比特币使用】

 

之前，媒体引用印度财长阿伦·贾特利（Arun Jaitley）的话，“政府并不认为加密货币是法定货币或硬币，而是会采取一切措施，消除这些加密资产用于非法活动或作为支付系统的一部分。”这被认为是印度将采取禁令禁止比特币使用。

 

事实上，在接受印度国有广播公司Doordarshan 的采访时，Jaitley被问到是否政府正在禁止数字货币，直接翻译如下：我们现在不鼓励人们使用它，目前有一个政府委员会正在研究这个问题，他们将在完成后宣布他们的决定和下一个步骤。对于印度的比特币行业来说，目前一切照旧。此外，印度的税务机关也利用数字货币交易所的帮助追捕逃税者。

 

陀螺评语：利用数字货币交易所的帮助追捕逃税者，实力诠释：对于数字货币的使用“一刀切”并不一定好。
