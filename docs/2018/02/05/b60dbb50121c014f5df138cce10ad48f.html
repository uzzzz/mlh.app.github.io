教你如何搭建一个门罗币（XMR,Monero）矿池

  • 柚子社区

最近区块链这么火，这么多人挖矿，我也就想试试，但是比特币，以太币肯定挖不了了，因为个人电脑算力太低，根本挖不出来，另外比特币的区块链数据现在已经非常大了，同步难度太大。所以就找了一个替代币，门罗币。另外门罗币可以用cpu挖矿，也就是任何电脑都可以挖，不像以太币必须的有大于2g内存的显卡才能挖，挖矿的门槛低很多很多。

挖矿第一步，申请一个钱包地址，有二个方法

  1. 在线申请，简单，快速，无需安装任何软件，无需同步区块，想尝试的最好选择，缺点就是不太安全。提供一个在线申请地址：https://mymonero.com/
  2. 下载官方钱包客户端，生成本地钱包地址。缺点是麻烦，单足够安全。下载地址：https://getmonero.org/downloads/
挖矿第二步，找一个矿池，找矿池的好处是无需同步区块数据，挖矿相对容易，但是但是但是，你得给矿主交税。比如https://supportxmr.com/ 这个你得给矿主缴纳0.6%的税，另外你得挖到超过0.3个门罗币的时候矿池才给你支付一次。也就是说，你的挖超过0.3个门罗币的时候你的钱包里面才会有余额。这样我大概算了一下，以我的渣渣电脑想挖到 0.3 个门罗币得用1000天左右。（附上cpu矿池挖矿客户端软件地址：https://github.com/fireice-uk/xmr-stak-cpu）

如果不想给矿主交税就只能以门罗币官方客户端挖矿了，这样挖矿难度是很大的，忙活半天也不一定能挖到。如果只是尝试这个方案必须的pass

鉴于以上2个方案都不靠谱，我就想为什么没有一个完美的方案？能不能找个不用交税，支付启点又弟的矿池？很遗憾在网上找了半天都没找到。所以没办法只能自己搭建一个了。

首先想一下什么事矿池？
矿池是一个把大家的算力纠结到一起挖矿的软件，然后根据大家提供的算力大小来平均分配挖到的币。
所以矿池挖矿的过程是把我们自己电脑的算力提供给矿主，矿主用我们的算力去挖矿，挖到的矿其实是存在矿主的钱包当中，然后矿主在根据我们提供算力的比例，给我们的钱包支付相应的扣除税率的费用。

所以矿主用什么去挖矿呢？肯定是得用官方客户端了，并且还要同步所有的区块到本地同步完成后才能开始挖矿。
而且这是第一步。

第二步：我们还需要一个矿池软件，矿池软件的作用是集结大家的算力一起挖矿。并且要评价分配挖到的矿。我在github上找了一个开源nodejs版本的矿池软件 ： https://github.com/clintar/node-cryptonote-util  ，但是这里有点bug 直接编译运行有点小问题。所以我fork了另外一个版本  https://github.com/cyyinfo/cryptonote-xmr-pool

编译方法
先安装依赖库

ubuntu 14.04 版本 
sudo apt-get install git redis-server libboost1.55-all-dev nodejs-dev nodejs-legacy npm cmake libssl-dev
ubuntu 16.06 版本 
  
sudo apt-get install git redis-server libboost1.58-all-dev nodejs-dev nodejs-legacy npm cmake libssl-dev

接下来的操作 ubuntu 14.04 和 16.06 版本一样 
git clone https://github.com/clintar/cryptonote-xmr-pool.git pool
cd pool
npm update
接着 配置一下 config.json 

找到其中的  poolAddress 改成自己的钱包地址


在找到  daemon 和 wallet 这2个配置，这2个配置就是官方客户端所在的电脑的ip,和客户单所开放的端口。

接下来就是启动矿池。 

node init.js 

如果启动成功就可以挖矿了。
