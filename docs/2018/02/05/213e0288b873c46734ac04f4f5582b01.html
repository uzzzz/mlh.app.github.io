AMD显卡矿机算力降低原因分析及解决办法

  • 柚子社区

大力矿工

 

自从ETH拔地而起，越来越多的人加入了掘金之路，但运行这么久了，多少有些老矿机会出现掉算力的情况，这就让我们白白损失了大部分算力了，这挖矿这么能开心呢？来，大力矿工来帮你们分析。

原来6卡的AMD显卡矿机，算力160Mhash/s。现在降到120Mhash/s，甚至更低。这类问题，除了矿机老化、电压和网络等外在因素，最主要的一个原因，应该是驱动问题。

原因分析：

  AMD官网，针对挖矿现象，为了提高算力，也是有下功夫的，所以在AMD官网有发布一系列针对对Blockchain优化过的专版驱动，有4系、5系A卡可以显著提升矿机的算力。

虽然我们安装了挖矿专用驱动，但由于A卡DAG每几天就会更新一次，如果驱动没有关闭自动更新功能的话，升级到新版的驱动程序，没有针对挖矿进行优化，矿机算力就自然下降了。

如下有解决方法，希望给矿工带来帮助:

出现算力下降严重的情况，建议大家重装驱动，修复DAG问题。

挖矿优化的驱动程序已经帮大家准备好了：共享在百度网盘：

https://pan.baidu.com/s/1i6fZDS5如下驱动都可以到这下面下载。

一、RX 4xx Polaris核心的显卡推荐安装 AMD 16.12.2版本的驱动

（下载：win10-64bit-radeon-software-crimson-relive- 16.12.2-jan3.exe

  二、非RX 4xx的AMD显卡，推荐安装AMD 15.12版本的驱动（下载 ：amdcleanuputility.exe）；

  三、对于部分 Fiji核心的卡如果驱动不正常，重新安装 16.3.2版本（ 下载详见：radeon-crimson-16.3.2-win10-64bit.exe)

 

注：驱动务必确保一次安装成功，如第不成功，可使用 AMD官方卸载驱动软件amd clean up utility卸载（下载 amdcleanuputility.exe ） ，重新安装。

注意事项：我们安装驱动了，记得关闭驱动程序自动更新。

另外，挖矿请勿用无盘系统，Blockchain专版驱动跟无盘软件有冲突，还是有盘靠谱
