在Eclipse里搭建Go开发的环境

谷歌创造了一个叫做Golang或Go的编程语言。据一些技术分析家说，它将最终完全替代Java。这是一种通用型的语言，可以用来开发任何软件——从普通应用到系统编程。虽然这种语言还不成熟，各种语言特征和规范还在变化，但程序员如今已经用它来从事开发工作了。

下面简单讲述如何 在Eclipse里搭建Go开发的环境。

1、下载Go的安装包

下载网址： https://golang.org/dl/

zip是压缩包，msi是安装包。有32位与64位可供选择。

比如，下载go1.4.2.windows-amd64.msi之后，双击该文件，安装到默认位置C:\Go。

设置环境变量，追加 C:\Go\bin，（该目录下包含三个文件：go.exe、godoc.exe、gofmt.exe）

此时，可以创建一个简单的go程序test.go：

package main

import "fmt"  

func main() {
    fmt. Printf ("Hello, World!\n")
}

fmt里包含格式化输入输出函数。

启动DOS窗口，进入该程序所在目录，利用 go run test.go 就可以编译运行该程序。

当然也可先通过【go build test.go】命令构建可执行文件test.exe，然后再执行。

3、在Eclipse里安装Goclipse插件

单击 Help 菜单下的 Install New Software...


单击【Add...】按钮，在【Add Repository】对话框里输入插件的名称（Name）与位置（Location）

插件网址： http://goclipse.googlecode.com/svn/trunk/goclipse-update-site/


单击【OK】按钮，进入【Install】对话框，给复选框【Goclipse】打上勾。


单击【Next】按钮，按提示进行操作，直到完成插件的安装，然后重启Eclipse。

4、设置Go的环境

单击 Window 菜单下的Preferences，单击左边列表中的Go，设置GOROOT与GOPATH。

单击【New】——【Other...】，弹出【New】对话框：

选择【Go Project】，单击【Next】按钮：

输入项目名，单击【Finish】按钮。
