欢迎阅读IPFS周报!

InterPlanetary File System (IPFS，星际文件系统)是一种新型的，基于内容和身份的，超媒体分发协议。IPFS使得开发完全分布式的应用程序成为可能。它旨在使网络更快、更安全、更开放。因为这是一个相当大范围的任务，在这篇周报中，我们跟踪、汇总了(IPFS)整个生态系统的发展。

最新进展

• 你听说过 IPFS WeeklyCall (IPFS每周电话会议)吗? 每周一下午5点(UTC)你可以与来自世界各地的IPFS贡献者一起面对面参加电话会议;会议中往往会有一名或多名特约演讲者给出短的讲座!下周，也就是12月3日，演讲者是Jacob Heun，他将给我们讲述关于最近一次的JS-Libp2p 发布，以及再下一周 Sander Pick 将与我们分享关于Textile photos的信息。查阅IPFSCommunity Calendar (IPFS社区日历)以获取更多的日期和资讯。

• 如果你是刚进入科技领域的新手，但如果你对去中心化很有激情，那么你应该来申请我们Protocol Labs的 IT Engineerrole (IT 工程师) 一职。 这是一个远程职位，主要职责为 IPFS, Filecoin, libp2p等提供团队合作!

来认识一下David Dahl
　　

MikealRogers, 网络用名 @mikeal， 是NodeConf 和“request”(一个很受欢迎的npm包)的创建者。除了许多其它的成就外，他是Node.js Foundation的创始人之一，同时他现在也是Protocol Labs 社区的领导人。关注Mikeal 的Twitter，了解他对于开源、IPFS以及偶尔关于精美晚餐的独特见解!
