美团打车亮相ChinaJoy，补贴大战即将打响？

文末重磅福利：炒到100万的EOS账号免费抢注

ChinaJoy 2018于8月3日至6日在上海新国际博览中心举办，是国内、乃至全球范围内最大型的游戏、二次元、数字娱乐展会。今年，也有不少区块链游戏的开发商们前往参展。

在ChinaJoy期间，美团打车和京东游戏合作推出游戏主题车，在上海街头亮相可谓赚足了眼球：身着游戏主题服装的司机驾驶主题车接送乘客，让ChinaJoy粉丝们从乘坐开始就拥有了沉浸式的ChinaJoy体验。

 

备受万众期待的美团滴滴补贴大战并没有开始，但这一次的“乘势而为”确实让美团打车火了一把。借由美团打车，我们要聊一聊后现代互联网时代的会员经济，以及区块链时代通证经济的变革。

在我们现代化的网络世界中，对于越来越多的人来说，我们只是想获得一种产品，而无须拥有它。比如我们想要轻松出行，可以选择方便快捷的网约车，而不用购买一辆车承担各种维修、年检、保险等费用。基于此，越来越多的公司正在改变他们的商业模式，从以客户为基础的模式转变为以会员为中心的模式。

 

以会员为导向的商业世界

欢迎来到会员经济的世界！

会员经济的优势在于它允许我们绕过与所有权相关的所有责任和成本，这是一个很大的转变。因为传统经济是围绕所有权原则建立的， 当客户在常规的场景中购买东西，他们即可随心所欲地使用它。

比如你拥有这俩汽车的所有权。你便可以选择用任何颜色来给车身涂漆或更换车身套件。但是，如果汽车发生故障，你必须支付维修费用，还必须缴税、上保险、支付停车位等一系列费用。

但是如果你每周只需要用一、两次车呢？或者甚至一天一两次？短期租赁可能是个好办法。这样，你可以在必要时使用汽车，但不必承担支付所有权成本的责任。

 

商业规则发生了变化

 

幸运的是，互联网使这种访问比以往任何时候都更容易。在过去的几年里，新的网络驱动型公司已经出现，为他们的客户提供了几乎所有领域的便捷替代方案。

例如，马来西亚本土的独角兽Grab，只要我们需要，就可以安排司机接送。我们所需要做的就是预订应用程序，还可以与那些需要乘车的人一起拼车（同滴滴模式）。

 

会员经济如何运作

以会员为导向的业务在公司、网络与用户之间建立了持续的关系。想象一下租车的旧方法。当我们需要参加重大活动，需要聘请司机。在司机提供服务后、完成付款后，此关系即结束。

而会员经济是不同的。当客户选择并信任一家公司时，他们就能从公司与会员网络的持续关系中获得对应的所有会员福利。

以新加坡的Grab公司为例，来研究创建会员经济的5个主要类别:

1. 数字订阅：使用Grab设置帐户无需任何费用。但Netflix等在线企业要求会员支付经常性费用以访问内容、功能或服务。

2. 在线社区：抓住共享目标创建网络- 拼车服务。考虑到这一点，司机和骑手可以相互评价。还有设定了一个会员级别，从普通会员到白银会员，再到黄金会员和白金会员。

3. 忠诚计划：GrabRewards通过为频繁用户创建不同的优惠等级来跟踪和奖励用户忠诚度。

4. 推荐计划：Grab允许你“邀请朋友”并可获得相应的邀请奖励。

5. 补充服务：随着Grab跟踪会员的消费和行为，他们开始提供额外的服务，如GrabDaily，GrabPay等，今后还会有更多。

 

会员经济是过去四分之一世纪建立起的最有价值公司的核心。想想优步，Airbnb，Netflix，LinkedIn，Facebook，亚马逊Prime，Dropbox等等。

然而，随着会员经济继续增长，它逐步向自然垄断的方向发展，使会员网络的所有者有了收取更高费用的定价权——这些所有者明白，即使会员感到不快和沮丧，他们也别无选择。

Grab在东南亚地区收购了优步之后，价格便开始上涨！

所以，可以明确的是，一开始，以会员为导向的业务对客户非常有利。但随着时间的推移，会员享受的福利却并不是那么多了！

到目前为止，大多数会员制企业都是私营或上市公司所有。然而，随着区块链技术的不断成熟，我相信精心设计的分布式标记化会员网络或将改变这场“商业游戏”。

而区块链即将改变围绕它建立的业务的性质。

 

通证经济：区块链的会员经济

 

想象一下，替代了“中心化”公司如Grab，我们创建一个分布式应用程序，名为BlockGrab--由其token持有者共同拥有的合作项目，手中所持有的通证即为“BlockGrab Token”。

每个参与BlockGrab的人，在这个过程中使用和积累Token，实际上都是BlockGrab网络的“所有者”。

当有人想搭车时，BlockGrab软件会扫描区块链的所有列表，过滤符合他要求的车辆。整个流程没有中心化机构的介入。

 

当然了，BlockGrab也可以创建它自己的通证经济：

1. 付款：你可以使用BlockGrab

Token进行付款，而不是使用某种法币。

2. 评论和评分：如果你收到了良好的评价或评分，你将获得更多的BlockGrab Token奖励。

3. 成员资格：当你继续持有更多BlockGrab

Token时，你的会员等级会升级并带来更多好处。

4. 推荐计划：当你继续“邀请朋友”时，整个网络会发展得更大，并为你邀请的每个朋友奖励更多的BlockGrab Token。

5. 补充服务：随着Grab跟踪会员的消费和行为，他们开始提供额外的服务，如GrabDaily，GrabPay等，今后还会有更多。

 

为什么说通证经济是会员经济的下一次革命

 

当会员网络由私人或中心化公司拥有时，价值从网络成员流向持有公司股权的股东手中。

在通证经济中，会员经济建立在区块链之上，客户与公司之间的关系发生了变化。通证持有者将在整个会员网络中占据一席之地，因为所有者和参与者之间没有明确划分，所以在通证经济中更容易调整激励策略。

这是革命性的。通证经济能够提供会员经济的所有好处，而不会随意增加在私人、中心化网络所有权下会员的成本。

更重要的是，这些成本节省可能不仅仅是财务上的，它们还将以保护用户隐私、数据所有权和更有利的网络策略的形式出现。

因此，基于区块链的通证经济模式不会让出租车司机失业，对Uber和Grab等公司是巨大的威胁，新的会员经济让司机直接与客户合作、建立联系。而在这个业务生态的每个人，都将拥有生态的一部分，并共享部分收益。
