188. 买卖股票的最佳时机 IV（JavaScript）

  • 柚子社区

给定一个数组，它的第 i 个元素是一支给定的股票在第 i 天的价格。

设计一个算法来计算你所能获取的最大利润。你最多可以完成 k 笔交易。

注意: 你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

示例 1:

输入: [2,4,1], k = 2
输出: 2
解释: 在第 1 天 (股票价格 = 2) 的时候买入，在第 2 天 (股票价格 = 4) 的时候卖出，这笔交易所能获得利润 = 4-2 = 2 。

示例 2:

输入: [3,2,6,5,0,3], k = 2
输出: 7
解释: 在第 2 天 (股票价格 = 2) 的时候买入，在第 3 天 (股票价格 = 6) 的时候卖出, 这笔交易所能获得利润 = 6-2 = 4 。
     随后，在第 5 天 (股票价格 = 0) 的时候买入，在第 6 天 (股票价格 = 3) 的时候卖出, 这笔交易所能获得利润 = 3-0 = 3 。

思路：

这是这系列的第四题也是最后一题，前三题请看博客：买卖股票的最佳时机 Ⅰ、买卖股票的最佳时机 Ⅱ、买卖股票的最佳时机三。

这道题与第三道题的差别是允许交易的数量为k，是个变量，而第三道题是2。这个依然可以借鉴第三题的解法，设置k个buy和sell，分别代表第k次买卖之后手上的钱。

除了第一次特殊（第一次买时手上的钱为0），剩余的几次可以用循环来处理。

/**
 * @param {number} k
 * @param {number[]} prices
 * @return {number}
 */
var maxProfit = function(k, prices) {
  if (prices.length < 2) return 0;
  if (k === 0) return 0;
  k = Math.min(parseInt(prices.length / 2), k); // k最大只能为股票天数的一半
  let buys = [],
      sells = [];
  for (let i = 0; i < k; ++i) {                 // buy和sell数组初始化
    buys.push(Number.MIN_SAFE_INTEGER);
    sells.push(0);
  }
  for (let price of prices) {
    buys[0] = Math.max(buys[0], - price);
    sells[0] = Math.max(sells[0], buys[0] + price);
    for (let i = 1; i < k; ++i) {
      buys[i] = Math.max(buys[i], sells[i-1] - price);
      sells[i] = Math.max(sells[i], buys[i] + price);
    }
  }
  return sells[sells.length - 1];
};
