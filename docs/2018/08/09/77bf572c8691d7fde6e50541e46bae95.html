Go环境安装

mac下两种安装办法

1, Mac下的brew直接安装Go（推荐）

 brew install go

2, 直接从golang官网下载对应的安装包进行安装。

打开 https://golang.google.cn/dl/ 选择

Apple macOS
macOS 10.8 or later, Intel 64-bit processor
go1.10.3.darwin-amd64.pkg (124MB)

下载安装即可

安装完成后需要进行Path配置

根据个人习惯创建一个包含src、bin、pkg 3个子目录的文件夹。（我这里是$HOME/gospace）

cd ~
sudo vi .bash_profile #如果没有则需要先创建该文件

编辑.bash_profile并保存，编辑内容如下：

#我的GOROOT目录是使用brew安装的目录，如果是使用pkg安装的同学注意调整对应的目录。 
export GOROOT=/usr/local/opt/go/libexec 
export GOPATH=$HOME/gospace
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

linux、windows下安装请下载对应操作系统的安装包，
环境变量的配置基本类时。按照不同操作系统进行配置即可。
