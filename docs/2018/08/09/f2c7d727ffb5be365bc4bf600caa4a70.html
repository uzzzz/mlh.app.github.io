布比 Code Review赏金计划正式启动 让区块链回归代码本身

  • 柚子社区

8月4日，由布比技术社区举办的布比 Code Review赏金计划正式启动，活动针对BUG提出者和优化建议设有相应奖励，奖池高达百万美金。开发者可登录活动页面（codereview.bubi.cn）参赛，复查BUBI Chain X源码，获得奖励，实现自我价值。

 

 

区块链发展至今，其创新的记账方式和去中心化的理念获得了大众的认可，也成为了资本的宠儿，人们越来越多的把目光放在了价值甚至价格部分，忽略了其背后的本源——技术。

 

“我们举办这次活动，主要是为了回归到区块链技术本身，代码复查是开发者在开发过程中非常重要的环节和工具，寻找BUG只是它的作用之一，传播知识，开发者之间的互相交流学习才是它最重要的好处。”布比CTO、也是布比技术社区的发起人王璟表示，“我们希望借由区块链开源的理念，让全球开发者一起对BUBI Chain X源码进行复查，一方面可以优化布比区块链的源码，一方面，我们也希望打造一个纯净的技术交流社区，让国内外的开发者可以基于源码，进行一些技术和学术上的交流和探讨。”

 

技术如果要对人们的生活产生切实的影响，代码只是区块链发展的第一步，场景和应用也至关重要。

“代码复查只是布比技术社区开发者活动第一期，后续我们还有针对高校的毕业设计大赛，针对程序员创业者的孵化大赛，布比在区块链领域深耕多年，在技术、应用、场景包括项目孵化上，都有相应的经验和资源，我们希望通过一系列公开、平等、开放的开发者活动分享交流出去，和全球的各界人士一起，共同研讨区块链应用落地，为区块链行业的发展贡献出自己的一份力量。“王璟表示。

 

据悉，BUBI Chain X是布比技术团队开发的最新版源码，已正式上线并平稳运行2个多月。布比是国内知名的区块链公司，在区块链领域拥有全球第五的专利数量。其自主研发的区块链数字资产网络已经无故障运行2年，有30多个超级节点，服务于1500万使用用户，月交易笔数为300万-800万。其另一产品区块链金融网络，已经无故障运营1年多，目前服务于20多家银行，大型生产企业，月度贷款金额超过10亿人民币。

 

此次Code Review赏金计划是布比技术社区首次举行的开发者活动，指导单位包括信通院云大所（中国信息通信研究院云计算与大数据研究所）、CCF（中国计算机学会）区块链专委会，更有和多所高校以及CSDN、CodingFly等技术社区的合作，是一次聚集政府机构、企业、学术与全球开发者的盛会，希望全球开发者们及各界人士积极参加赏金计划，把握学习与交流的机会，运用先进的技术和理念解决各类区块链应用场景问题，为行业带来更多公平、进步、发展的机会。
