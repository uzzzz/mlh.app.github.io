第二期DBA投票“今日必看”项目分析

  • 柚子社区

一、项目综述

「今日必看」是一个基于区块链的短视频分发平台，以 Token 激励驱动短视频社区生态发展，让优秀内容视频和广告视频在各类用户间直接流转，并以Token实现价值传递。在「今日必看」，用户的付出和贡 献（创造和观看视频内容或广告及其相关交互行为）均可获得相应的回报。

二、项目逻辑

通过把广告主及广 告服务方与广告受众（用户）直接连接起来，去掉广告中间商，进而实现广告商业生态的市场化自由竞争，为帕累托最优提供可能性。通过Token的价值流转和激励，模糊广告、内容、产品、交易等的界限，通过不可篡改的区块链存储技术，实时反映广告或内容的传 达效果及其受欢迎程度，以真实存在的交互（或交易）行为为广告或内容激励提供重要依据，和作为内容产品迭代优化的参考。

三、商业模型

1.广告主

如果需要投放广告，需自备「今日必看」的 Token ：BIK（可通过二级市场购买），广告投放开始及投放结束后的一段时间内需要锁定一定数量的BIK，以作为虚假广告的违约补偿等的备付金；并设定好每用户观看的BIK数量。

2.用户

观看完该广告并完成相关任务后即可获得相应的 BIK，用户拥有的 BIK 可以选择持有以享有今日必看的相关权益，或者购买优质内容享受高品味的生活娱乐，也可以转入交易所卖出以获得其他电子货币或法币。用户也可发布收费或免费的优质内容供其他用户观看获得相应收益和系统激励。

3.系统期激励

用户在使用产品时，产生的有效交互行为（比如观看广告或者内容、创造内容时）均会获得系统激励，可以理解为参与了“挖矿”。

四、一点思考

1.国内ICO的政策风险

BIK为「今日必看」的唯一Token，总量为210亿，其中“21亿用于募资，占总量的 10%；认购比例 1EOS=3200BIK； 1ETH=160000BIK”。白皮书在风险提示栏“项目参与者限制”中说明“因为一些法律问题，本项目不接受任何来自下列区域的居民或公民的参与：中国大陆区域、香港、澳门、美国和其他禁止 ICO 的国家或区域。请禁止ICO的国家的居民或公民务必谨慎对待，必看将不承担任何因此引起的法律责任 ”。白皮书中，没有透漏该项目发起公司注册地，通过“天眼查”和项目创始人，估测可能是“云南睿梵企业管理有限公司”，如果作为一家中国公司，组织类似的私募，会不会有政策风险，值得关注。

2.不愿意“透明”的广告潜规则如何破局

在中国特殊的国情和社情下，很多企业投入巨资进行广告，有些是企业品牌和营销的需求，还有一些是“另有深意”，广告效果已经不是第一优先级，甚至是公司管理层的“集体共识”，主要目的是税务筹划和“各类潜规则费用套取”，这种“郎情妾意”的困局，需要在一个大的“社会环境”中破局。当然，我们相信，潜规则注定会被明规则颠覆，这也是区块链颠覆性价值所在。

结论：

「今日必看」解决了广告行业的痛点，长期看好，如果能避开ICO政策风险，获得基金投资会更有利于项目推进。
