区块链的确是好东西，一般人看不懂也看不透，还笑话玩区块链的人，甚至被说成是传销。都什么眼神啊？多少国家比特币都合法了？不信？爱信不信，反正我是信了。

我为什么要说这些呢，因为昨天发生了一件事。事情的起源还得从我的一篇日记说起。昨天突然一个朋友给我发了一个消息，说区块链都是那些胆大心狠不怕伤人脉的人干的，我他妈哈哈不语，再次验证了我的观点，区块链依旧是风口，只有我们这些圈内人知道。

骏君，我可以爱你更深一些吗？
昨天日记没有赶上写，今天难受了一天，就好像什么东西丢了，再也找不回来了。反思一下，人生也是这样，逝去的一天是再也找不回来的，宇宙中你我太弱小了，不顶用。

其实昨天去见了一位大咖老师，北大清华互联网授课专家骏君，本名孙俊君，资深互联网人，实战派、革新派的网络营销专家，“人格品牌化”理论创立者，《微信营销方法1+2+3》《颠覆与变革：新媒体营销方法》《传统行业的微营销落地执行》等畅销书作者，被搜狐评为“年度最佳网络营销专家”，他有个爱好，就是喜欢粉丝，你对营销感兴趣可以加，微信junjun195758，还可以让你的朋友也都来加，多多益善，这么牛的人，你不加就是你傻了，呵呵，因为我长期无私惯了，我也不想藏着这么好的人脉独享，所以放出来了。

下班后早早出发，北京经常堵车，为了保证准点到达，我就坐了地铁，其实我平时很少出去，因为最近交易所和区块链学院的筹备，出去就是一天，北京太大了，时间耗不起。买一张票进站了，下班高峰期人多啊，一个个全是看手机消遣。反正在我周围没有看到有人利用时间学习，时间就纯浪费掉了，我是觉得把时间都浪费掉了怪可惜没意义。

到了约定地点后，找了一家星巴克，匆匆走去还是打烊了，这不是让我为难吗，整的这么尴尬，还好旁边有家餐厅，我们边吃边说，说什么，互联网啊，区块链啊，骏君老师在互联网领域是很牛的人出了好几本书，我不行，只是对区块链懂得多一些。

前一天就知道骏君老师来北京讲课，又看到了我的日记，就问我知识经济生态如何很好的利用区块链技术，这个对于我简单啊，这几年什么也没做，白皮书倒看了不下400份，我知道一个优质的区块链项目所具备的几个要素，不单单是发个币那么简单。

所以，我针对知识经济生态谈了以下几点：

一、现在是价值互联网时代，通过区块链的加密技术、分布式存储、不可篡改 、公开透明、数据可追溯等让个人知识资产产生可信任的确权、估值、交易。

二、知识生产者身份认证要采取去中心化的身份验证方式，包括信息登记、版权登记、项目规划等。

三、 知识资产上链，拥有者将其知识产品或项目的收益权，在区块链上通过分布式账本，实现知识资产确权和资产收益的公平分配。

四、打造基础的公链配置，这样教育服务机构就可以开发相应的分布式应用Ｄapp。

五、在这个生态圈，以 Token支撑平台激励体系，利用区块链技术，结合知识生产、传播与消费的应用场景，基于智能合约和 token 系统，打造价值互联网赋能知识生产者的基础平台，让知识价值实现资产化，构建知识经济新生态。

总结一下，就是一定要让知识资产上链，这样才是一个牛逼的区块链项目，好像目前还没有人这样做。

而骏君老师愿意帮助我一起推进区块链学院，我们这叫强强联手，其实我不强，但是我得学啊，不能再过老婆孩子热炕头的日子了，人来一趟世界不容易，不折腾着留下点痕迹都对不起自己。

我是君叔，如果你是传统行业老板需要转型区块链，或是你是投资者正在找区块链优质项目，亦或你正在打工但是不甘心现状，都非常欢迎看日记，每天一篇，昨天漏了一篇。
