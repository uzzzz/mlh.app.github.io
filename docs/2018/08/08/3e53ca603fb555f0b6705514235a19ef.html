甲骨文推出Oracle自治交易处理

  • 柚子社区

甲骨文最新发布的“无人驾驶”数据库云服务开创性地应用机器学习和自动化技术，
帮助企业实现空前的成本节约、安全性、可用性和工作效率

北京2018年8月8日电 /美通社/ -- 甲骨文公司董事长兼首席技术官拉里·埃里森 (Larry Ellison) 今日宣布推出最新的 Oracle 自治数据库云服务 -- Oracle 自治交易处理 (Oracle Autonomous Transaction Processing)。借助创新机器学习和自动化技术，Oracle 自治交易处理可实现空前的成本节约、安全性、可用性和工作效率。甲骨文最新的“无人驾驶”数据库云服务专为要求最高的金融、零售、制造及公共部门应用打造，支持高性能交易处理、报告、批处理和分析等复杂的混合负载。Oracle 自治数据库全产品线为企业及组织提供了目前业内最完整、最先进的数据库解决方案。

“Oracle 数据库是迄今为止全球最为卓越的数据库，加入了自治性能后更是如虎添翼，”拉里·埃里森表示，“Oracle自治数据库提供了一个更为可靠和安全的系统，能够抵御数据盗窃，实现高达99.995%的可用性，并且帮助企业机构以及他们的开发者极大程度地提升生产力。”

在过去，创建一个数据库管理系统需要由专家来定制复杂的硬件和软件体系，并对其进行手动维护。Oracle 自治数据库突破性地利用机器学习技术，提供具有自治驾驶、自治安全和自治修复功能，且具备云的规模效益及弹性的数据库服务，为数据管理带来变革。借助该服务，用户可以立即创建新的自治数据库，或轻松升级现有数据库，从而大幅降低成本，并加快上市速度。

作为对现有 Oracle 自治数据仓库服务的补充，Oracle 自治交易处理服务能在单个数据库内支持高性能交易、报告、批处理、物联网和机器学习等复杂的混合负载，大大简化应用的开发和部署，实现实时分析、个性化和基于实时交易数据的欺诈检测。采用该服务的用户将能够：

  • 节约成本：完全自动化状态下的数据库和基础设施运营能为企业节约80%的管理成本，借助高效的数据库自我优化和弹性的按用量购买模式可降低高达90%的运行成本。
     
  • 降低风险：无需停机即可自动应用最新的安全更新，从而消除网络攻击漏洞。可防范系统故障、区域中断和用户错误等所有类型的故障，可用性高达99.995%，相当于每月仅需不到2.5分钟的停机时间（包括维护在内）。此外，Database Vault 可防止管理员偷窥用户数据。
  • 加速创新：数据库管理员可以从数据库维护工作中解脱出来，专注于从数据中挖掘更多价值。开发人员可以即时创建并轻松使用无需手动调优的数据库，节约更多时间。开发人员还可以利用集成的机器学习算法开发实时预测型应用，如个性化购物和欺诈检测。同时，从现有数据库到自治云的升级非常简单，可帮助企业IT轻松、快速地转型至先进、敏捷的云模式。

甲骨文在过去40年一直为全球最复杂、要求最高的企业应用提供支持，Oracle 自治数据库汇集了其中甲骨文积累的丰富经验。它使用与本地部署方案相同的 Oracle 数据库，具有与本地部署数据库相同的功能和接口，因此客户可以轻松采用。自治数据库能够支持任务关键型负载，因为它基于高度成熟的、其他产品无法比拟的 Oracle 数据库技术，如高性能 Exadata 基础设施、支持透明扩展和容错的 Real Application Clusters、Active Data Guard 灾难恢复和 Online Data Evolution。

“DBA 最艰巨的一项任务就是调优和维护任务关键型交易数据库，”IDC 数据管理软件研究副总裁 Carl Olofson 表示，“除了检查统计信息，进行调优和调整，DBA 还需要频繁地安装安全补丁在内的各类补丁，这项工作不仅容易出错，还会中断系统的正常运行。凭借 Oracle 自治交易处理，甲骨文已经将这类风险消除，让 DBA 能够专注于价值更高的任务，提升应用的业务响应能力，同时确保企业数据不受已知漏洞的威胁。”

Oracle 云

Oracle 云是业内最完备、集成的公有云，涵盖 SaaS、PaaS 和 IaaS 的全栈式服务。它支持新的云环境、现有环境和混合环境，适用于所有负载、开发者和数据。Oracle 云为全球195个国家及地区的客户提供将近1000个 SaaS 应用、50个企业级 PaaS 和 IaaS 服务，每天能够处理550亿个交易。
