如何高效学习-阅读笔记

一、增加能量储备

  1. 每周有 3-5 次运动
  2. 每晚有 7-8 小时的睡眠
  3. 吃粗纤维和粗加工的食物，这样的食物可以让你一天的血糖水平保持平稳，避免忽高忽低。
  4. 多喝水
  5. 理想的饮食应该是少吃多餐，一天吃 4-5 餐，每次吃七分饱，这样可以保证你一天内的营养供应持续稳定

二、圆形计划执行

线性的计划是平均安排时间执行工作，而圆形计划安排则先集中小部分时间做大部分工
作，这种计划安排能让你做到有张有弛，而不是死气沉沉，像个机器人。

  1. 一周将7天的活放在六天完成，其中一天什么事都不做。
  2. 每晚有 7-8 小时的睡眠
  3. 晚上不干事，将一天的工作尽量放在早上。
  4. 设定 90 分钟。给定自己 90 分钟，集中精力完成某个学习任务，一旦 90 分钟结束，停 止工作。90 分钟的时间设定能让你学习时注意力更集中。

能量管理推荐书籍：the_power_of_full_engagement 全力以赴-高效能人士的精力管理手册》

三、周/日目标体系

  1. 每周末列一个下周要做的事情清单
  2. 每晚检查周计划，是否完成，并列出下一日的计划

四、养成每日习惯

  1. 设置周期（一个月、一周等）
  2. 一段时间内只坚持这一件事，养成习惯
  3. 达到目标给予自己奖励

如何快速学习（知乎）

（例如，学习新的领域-区块链）

  1. 简单了解，并列出想了解并解决的问题（例如，什么是区块链，采用哪些技术）。
  2. 设立学习目标，有目标的学习才更加快速。
  3. 找到相关学习资料（书籍、博客等）
  4. 根据提出的问题有目的性地阅读资料（不要拿出一本书从头看到尾）
  5. 在阅读资料学习的过程中继续提出自己想了解的问题。
  6. 利用思维导图等方式对自己学到的知识体系进行总结。
  7. 最后，也是最重要的一点。实践！实践！再实践！
