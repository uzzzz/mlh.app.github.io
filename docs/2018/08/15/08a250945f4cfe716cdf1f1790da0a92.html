交易所

1.EOSfinex

EOSfinex是建立在EOSIO解决方案之上的去中心化交易。它目前正在开发，是除了Bitfenix外，世界上领先的加密交易所之一。Bitfenix于2018年2月宣布。根据它们，它将是一个高性能的去中心化交易。它是为了提供一个有利于无信任、透明和快速平台的交易的在线需求。

特点：

  • 高性能与高可信。
  • 基于EOS的区块链解决方案。
  • 承诺处理高容量和快速交易。

状态：开发中。

官网：eosfinex.com/

blog：https://medium.com/bitfinex/announcing-eosfinex-69eea273369f

2.Chaince

Chaince是资产交易平台。目前，它只关注EOS项目。此外，它还称自己是一个优秀的区块链连锁交易平台，作为一种营销手段。Chaince已经启动和运行，可以直接通过他们的网站访问它。它提供了良好的界面和交易能力，而不会失去焦点。它还配备了审计模块，通过已建立的标准审核项目，并确保它符合标准，然后将其列出给交易所。当涉及到费用时，合约也是独一无二的。它不和项目收取任何费用，帮助他们成长。

特点：

  • 去中心化交易平台。
  • 通过不收取任何费用来提高项目成功的机会。
  • 目前只列出了三对交易，BTC/EOS，ETH/EOS和ETH/BTC。

状态：可用的。

官网：chaince.com

blog：https://medium.com/@Chaince

3.OiX

开放式投资交易所（OiX）是为下一代建立的交易平台。他们的目标是构建一个平台，只列出长期持有价值的token。这意味着只有那些有机会服从的token才可以在平台上交易。OiX平台是一个自我调节的token平台。OiX提供的基础设施是安全的，对token交易是理想选择。因此，这意味着他们将尽最大努力遵守ICO，从一开始就遵守标准。一旦完成，项目就成为某种未来的证明。

特点：

  • 一个驱动ICOs更好的交易所，避免从一开始就抱怨。
  • 用户将有机会投资符合特定标准和符合性的ICOs。
  • 良好的资本流动。
  • 提供可见性、可信度和增强的配置文件。

状态：可用。

官网：oix.global

4.Tokena

Tokena是快速、安全和智能的基于EOS的去中心化交易所。它由KEN token提供支持，用于减少在交易所执行的交易佣金。通过使用EOS，它使自己能够超快、安全、去中心化和自动化。它使用智能合约来处理平台上的一切。交易费用也很低，只有0.3%。然而，如果用户使用KEN token，他将不得不支付0.15%的费用。

特点：

  • Tokena基于EOS智能智能合约是安全的。
  • EOS比以太坊更快，因此Tokena也从中受益。
  • 提款和存款是即时的。

状态：开发中。

官网：tokena.co

金融科技

1.Chintai

Chintai是一家基于区块链的金融科技平台。它最好被称为全球去中心化租赁平台。通过这种方式，dApps开发者和社区成员可以通过智能合约来租用区块链带宽。这将有助于通过确保不同用户之间的解决方案的流动来避免破坏行业。dApps也可以租用，业主将在租赁期届满之前得到完全控制。它受益于使dApps持有者通过在EOS区块链链上租赁给其他用户来赚钱。它由Chintai租赁引擎提供动力，采用先进的算法进行正确的交易和执行。

特点：

  • 它提供了巨大的token租赁市场。
  • 用户之间没有交易费用。
  • 高流动性。
  • 安全快速透明。

状态：开发中。

官网：chintai-eos.io

2.HorusPay

HorusPay是另一个与金融科技相关的区块链应用程序。它是一个去中心化的全球支付门户，提供国际供应商和公司之间的安全数据交换。所有的事情都是使用区块链执行的，因此它对受尊重的实体是安全的、可扩展的和透明的。

特点：

  • 公司可以把他们的供应商带到区块链并帮助标准化过程。
  • 完全安全的区块链链共识算法运行它。
  • HorusPay消除了中介，他们可以节省公司收入的40%。

状态：开发中。

官网：horuspay.io

3.Atidium

Atidium是预算和支付管理去中心化的应用程序。应用程序是利用当前的技术，如人工智能和区块链构建的。这对企业和个人都是有益的。通过使用它，你将能够处理你的预算和支付。它也运行它自己的加密ATD。这是一个100%的支付token。这意味着如果你想向别人发送付款，你可以用一个简单的点击。此外，你可以控制一切，包括预算管理。

特点：

  • ATD token带来了必要的货币，使区块链工作。
  • 智能合约可以被编码，以确保自动化的许多任务。
  • 以最好的方式使用ATD token和进行你的预算管理，。

状态：开发中。

官网：atidium.io

游戏

1.Azarus

Azarus是一个透明的平台，让你可以玩和打赌。它是由token支持。视频游戏行业充满了机会，这就是为什么Azarus希望通过区块链技术的帮助使感兴趣的人有能力玩。真正的钱可以被用来玩，一切都由智能合约来控制。它利用透明度和信任，确保每个人都得到平等对待。为了保证Azarus工作，它使用了“rings”的概念。通过创建“rings”（有平台费），任何人都可以建立规则，让玩家按照规则进行游戏。支付是自动发放的。这对于组织者来说非常有用，因为他们可以组织具有完全透明性的事件。组织者还可以以流量付费的方式来发送交易。

特点：

  • 它有桌面和移动应用程序。
  • 组织者可以创建“rings”和他们自己的一套规则。一旦设置，它们就不能被改变。
  • “rings”只能使用支持的游戏创建。
  • 玩家的数据可以使用区块链历史浏览。
  • 用户可以发现挑战，然后参与其中。

状态：可用。

官网：azarus.io

2.Billionaire Token

Billionaire（亿万富翁） Token是一个独特的想法，并采取货币化。现在，你会发现几乎所有的项目都有他们的token，可以通过采矿或股票获得。但是，亿万富翁代币是相反的。主要是围绕抽奖进行。它利用的机制被称为通缩机制。通过使用它，通过去中心化的燃烧器智能合约和去中心化抽奖来烧毁XBL token（与亿万富翁令牌平台相关的token）。这提高了令牌的内在价值，并确保令牌的数量随着时间的推移而减少。

特点：

  • 代币目前在主要交易所上市。
  • 还支持钱包。
  • 你可以设置你的钱包来自动抽奖。
  • 他们目前正处于从以太坊迁移到EOS区块链的过程中。

状态：可用的。

官网：billionairetoken.com

3.DEOS Games

DEOS Games是一个围绕游戏构建的区块链平台。它在其网络上进行区块链博弈，并利用区块链的力量使其更好。目前，预计将有21个比赛类游戏，包括Blackjack（二十一点），Lotto（乐透），Roulette，以及更多。它还允许玩家投票加入下一个应该加入EOS网络的游戏。那么，它到底给这些游戏带来了什么呢？首先，没有与之相关的费用。通常情况下，传统游戏的成本是5%，比特币游戏的成本是1%。它会完全从系统中根除。此外，它提供了最公平的方式玩游戏，有透明度和完整性数据。

特点：

  • 游戏玩家可以即时炫耀他们的胜利。
  • 可以加密你玩的游戏。
  • DEOS Games可提供银行业务。

状态：开发中。

官网：deosgames.com

  • 游戏是独一无二的，你可以玩你自己的方式。你可以专注于经济或战争。由你决定。

状态：开发中。

医疗健康

1.IRYO

IRYO是一个基于医疗保健的区块链应用程序。它正在重新定义全球医疗保健，重点是改善全球卫生数据访问。它在公共的区块链上工作，因此病人的数据可以从任何地方访问。这使得治疗从任何位置都是可行的。这也意味着病人不必随身携带他们的文件。

特点：

  • Iryo Token提供了一个独特的经济模式运行在区块链上。它有助于数据流动和激励医生和病人积极行动。
  • 医疗应用可以建立在IryIO之上，以使现有医疗设施和区块链发挥出最佳效果。
  • 还创建了Iryo诊所，使这个想法向前发展。

状态：开发中。

官网：iryo.io

blog：https://medium.com/iryo-network

身份认定

1.Scatter

Scatter是一个EOS Web扩展，允许你在GO上签名事务。这意味着你不公开你的信息或私钥来签署你的邮件。应用程序充当黑盒。例如，当你必须从购物中购买一个使用密码的电子商品时，你就不必输入任何东西。它将由Scatter完成，直到结束。它也保护你免受诈骗网站试图窃取你的私人密钥。

特点：

  • 所有信息隐藏和安全使用EOS区块链。
  • 机密数据是加密的和安全的。
  • 通过使用去中心化的区块链，用户体验提高到一个新的水平。
  • 一个简单的Web扩展，可以被任何人使用。

状态：可用。

官网：get-scatter.com
