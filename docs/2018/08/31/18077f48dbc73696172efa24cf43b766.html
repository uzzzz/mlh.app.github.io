Atom emmet 插件手动安装

首先我们要获得所需插件的安装包 且把它放入Atom插件包的安装目录下

手动下载 （推荐方式）

https://atom.io/packages 这个是Atom 插件包的地址
进入后搜索要安装的插件包点击版本version 进入后进行下载

解压下载包将整个文件夹放到atom插件包安装目录
C:\Users\Administrator.atom\packages（1、atom默认自动安装到C盘 2这个是我的安装目录）下

有了安装包后开始安装

WIN+R
进入命令行 进入指定安装的插件目录

执行命令

npm install
注：这里需要使用Node 安装所以我们需要安装一个Node.js
https://nodejs.org/en/download/ 下载地址 点击对应OS任意版本进行下载

这里提示报错
2017年2月27日，npm不再支持自签名证书。

因为npm install走的是https协议，需要通过数字证书来保证的

解决方法：
取消ssl验证：npm config set strict-ssl false

如果还没成功，则将npm源更换为国内镜像：
npm config set registry http://registry.cnpmjs.org/

npm config set registry http://registry.npm.taobao.org/

taobaoNPM镜像官网：https://npm.taobao.org/

设置好后，执行安装成功

我们打开ATOM Settings Community Pages
