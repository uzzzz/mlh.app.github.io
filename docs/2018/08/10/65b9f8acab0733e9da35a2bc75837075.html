安装WebStorm和npm转换cnpm

  • 柚子社区

第一步:

按照https://www.cnblogs.com/goldlong/p/8027997.html下载安装nodejs和npm

第二步:

成功之后在解压项目,解压到盘符的根目录下(比如C:  D:  E:)

直接在路径下敲cmd(比如:D:项目名然后回车cmd)

输入命令:

npm config set registry https://registry.npm.taobao.org

npm config list / npm config get registery

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v

第三步:

给项目下的node_modules这个文件夹删除

重新在项目路径下打开cmd命令

输入命令

cnpm install

cnpm run dev
