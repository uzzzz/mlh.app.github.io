亚马逊高管加入Coinbase，动荡的加密行业不断吸纳人才

  • 柚子社区

亚马逊高管加入Coinbase，动荡的加密行业不断吸纳人才，Coinbase迎来一位新的工程副总裁——前亚马逊云高级主管Tim Wagner。

Coinbase雇佣更多高管

Tim Wagner曾是多个亚马逊网络业务单元（Amazon Web Services）的总裁，包括AWS Lambda、亚马逊应用程序接口网关（Amazon API Gateway）和AWS无服务器应用程序库（AWS Serverless App Repository），也在微软担任过几年高管。他于周一入职Coinbase。根据领英个人简介，Tim Wagner将会“帮助创建世界上最开放、安全和可用的金融系统”。他也负责在旧金山、纽约和芝加哥招聘工程人员。

Coinbase首席执行官Brian Armstrong在博客发帖写道：

Tim对于Coinbase的使命充满热情，他与我们的价值观一致，对于领导伟大的技术团队有丰富经验。这些品质与他的工程专业知识，使他成为我们团队的理想补充。

不同行业高管加入Coinbase团队，加强了其作为全球顶尖加密货币交易所的地位。今年早些时候，Coinbase雇佣了前Pershing高管Jeff Horowitz担任首席合规官，前Twitter和Facebook通信总裁Rachael Horwitz担任通信副总裁。

然而，Coinbase不是唯一有招贤纳士新闻的加密货币平台。Winklevoss兄弟的加密货币交易所Gemini最近雇佣了前纽约证券交易所高级主管Robert Cornish担任首席技术官。

加密货币钱包Blockchain雇佣了高盛顶级高管Breanne Madigan担任机构销售和策略主管。Facebook全球支付总监Kahina Van Dyke最近加入Ripple担任新的业务和公司发展副总裁。会计师事务所巨头普华永道（PWC）前总监Ben Ingram离职加入加密货币交易所Bitcoin.com.au 担任首席执行官。

财富500强公司的高管转而进入加密货币行业，更加表明区块链驱动的加密货币在全世界增长的认知度和接受度。汤森路透（Thomson Reuters）调查显示，在汤森路透交易解决方案的400多客户中，20%的金融公司考虑在2018年推出加密货币交易。

高管出走恰逢世界各地多种司法管辖推动建立加密货币和区块链业务的合法框架。2月，瑞士金融市场监管局（FINMA）发布了一组ICO监管指导方针。

泰国和菲律宾最近也宣布了新的代币发行法规。这两个国家的实体寻求进行ICO时，将被要求提交申请文件给各自的监管机构以获得许可。韩国也预计很快推出国内ICO监管框架。
