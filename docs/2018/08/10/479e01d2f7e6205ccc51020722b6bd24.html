分享内容
什么是Curation Markets?

Curation Markets (策展市场) 允许团队协调共同目标（和利益），并从共同创造的价值中获益，Curation Market 采用经济激励的手段，可以为任何主题创造一个众包的“对信息流（或者数据列表）进行筛选、组织、维护、整理和再加工”的市场。

Curation Markets 相当于我们为社区内每一种有价值的信息和内容，创建一个股份制的“公司”，而且这个公司的股份可以被无限分割，股东董事会不受任何地理、政治、肤色、语言、身份的限制，任何成员都可以加入。

最好理解的curation Market的产品就是美国的reddit，今日头条也是 curation 类型的产品。

Curation Market 想要解决什么？

众所周知，我们现在其实是生活在一个信息过载的时代，很多时候信息的接触甚至会比信息本身更加重要，我们对很多的事情的处理，其实往往都会落到对信息的处理上来，比如说我们有的时候想读最好的书，想看最好的电影，想挑选最好吃的餐厅，本质上都是用信息来决策，但很多时候我们信息的决策都非常依赖于社区。

而在信息大爆炸之后，人们迫切地需要一种合适的方式，用来号召社区成员共同进行信息的重新组织和整理。否则，杂乱和重叠的信息充斥太多，很容易会引发一场淹没社区的洪灾。

在这样的背景下，Curation Market想解决的问题，其实是通过创建一系列经过排序的列表，或是一些主题来为人们来提供一个决策依赖的基础：有哪些选项、为选项排序、采取措施。

如果某个社区或社群内有信息的贡献者和管理者，通过他们的努力社群运作得很好并且社群所创造出的信息非常有价值，那么Curation Market就可以为信息的贡献者和管理者获取价值。同时因为整个社区是围绕一些共同目标进行协同的，Curation Market可以将利益相关的人更好地绑定在一起，让利益相关的人对社群进行管理，来决定内容采集者、生产者所应获得的奖励。curation market 可以针对任何的主题，可扩展性非常强。

在信息过载的时代，通过Curation Market对信息进行组织和加工，本身就是一种非常有意义的行为。

它可以解决三个核心问题：

1、让每个成员拥有自己创造的价值：不管是社区内容的创造者还是维护者，都可以从平台中分享集体利益。

2、增加社区的协调性：社区的成员通常都围绕着一个共同目标进行协调合作，但他们并没有一个固定的组织形式，这种松散的团队结构进行协调合作的效率不高，不管是开源项目还是全球变暖的项目。

3、增加共享信息的利益相关性：利用代币来维护社区日益丰富的信息，让社区成员各自的利益更多地与社区内容绑定在一起。

Curation Market 具体是怎么实现的呢？

底层机制：

1、一种可以根据智能合约设定的价格随时（连续）铸造的代币。

2、随着更多的代币开始流通，代币价格会变得越来越贵。

3、购买代币支付的金额（比如eth）集中保存在一个公共钱包里。

4、代币可以在任何时候从有效供应中提现（“烧毁”），随之按比例带走公共钱包里相对应的一部分金钱。

5、代币用于为社区的维护者进行筹款，这些维护者随后按比例对社区的内容和信息进行维护。

核心原理：

拥有 token 的人会尽可能地维护更优质的信息流，以此来吸引更多人的注意力。更多的注意力 ——> token 的价值就会上升。其他人会更希望拥有token，因为他们会想要在一个优质的信息流里展示自己的信息。最终会形成一个正向循环的激励机制。

token机制：

token 持有者需要根据自己持有的 token 数量，按比例对相应数量的内容和信息进行维护。智能合约就相当于 token 的中央银行，通过 eth 买 token ，然后支付的 eth 会被存储在一个金库里。中央银行拥有自己的一套基于算法的货币政策。token 新增的越多，价值就会越高。每个人可以在任何时间，找中央银行把自己的 token 按当前的价格兑换成相应的 eth ，这些 eth 来源于金库。

比如，如果中央银行的货币政策是“每新增1个未结算的token，每个token 价格就上升1个eth ”，那么第1个token可以卖出1个 eth，第10个token 可以卖出 10 个 eth 。如果我是第 5个购买的人，我花了 5 个 eth 购买了1个token，那么，当这个社区第100个人购买 token 的时候，我就可以找中央银行兑 100 个eth 。

更多的机制可以在这套 curation market 的基础上进行叠加，包括投票决定新成员是否能加入社区的机制（也就是一个基于 token 的社区注册机制）、对货币政策采取不同的公式和算法、促进更多人更长久地持有 token 的激励机制等等。

Curation Market的应用：

去中心化的 reddit

信息流相互竞争的社交网络

在去中心化的监管中为区块链协议设定优先级

为用户任意需求制定的个性化清单、可以互相竞争的推荐系统／AI

Curation Market面临的挑战：

1、公地悲剧：由于token的价格是在不断变化的，如何让人们把注意力放在维护信息和内容的工作上，而不是盯着 token 价格的涨跌？可能的答案：用 token 更多地奖励那些第一个或者更早投入到信息内容维护工作上的人。

2、信息流独裁：如果有人购买了大量的 token 在信息流里发布完全不相干的广告内容怎么办？可能的答案：如果他们那么做了，其实就破坏了整个信息流的价值，也就破坏了自己的广告价值。也许可以增加其他的权益上的限制。

Curation Markets带来的启示：

1、Curation Markets 为我们选出最好的算法提供了另一种更直接的方式。因为对信息流的很多处理其实会通过算法来完成，人们又希望手里的 token 会继续增值，因此他们会倾向于把自己的 token 委托给表现最好的那个算法。

2、Curation Markets 也为写算法的人提供了一种更直接的赚钱方式。写一套世界上最好的新闻聚合算法？写一套根据传感器的数据进行预测的算法？如果你真的写出来了，你就可以通过 token 获得巨大的、前所未有的收益。

3、最终的结果可能是：许多小型的基于算法做 Curation 的对冲基金，在公开市场的一次大爆炸。从另一个角度来说——是一种把 AI 直接货币化（monetize）的方式。

简单总结：

Curation Market这种方式可以让我们为每一种有价值的内容、社区或是内容、社区的集合甚至是论坛网站来创建一个“股份制的公司”，社群内的人可以一起来维护它，让它变得有价值，同时每个人也有希望通过这种方式来分配合理的报酬。内容不断被维护得更好，从而吸引更多志同道合的成员加入社区，维护更多更好的内容，如此不断正向循环下去。

讨论分享
Q:Curation Market里面好像并没有什么消耗币的方式， 有的仅仅是大家众筹共建社区， 那么社区内是否需要一些消耗币的玩法（如内容付费等）？

A:curation market 里面消耗币的逻辑是可以大家来定义的，curation market其实是一个非常抽象而简单的模型，完全可以第二次加工。

Q: 新人如何在老的社区里面出头，这种单向的结构，可能会造成新的玩家不愿意进入？

A:新人在老的社区里面可以通过购买代币来获得更多的投票权。

Q:代币的价格不稳定，如何让社区成员比较清晰地认知到币的升值趋向， 交流的都是“正”逻辑，会有其他的逻辑发展方向吗 ？

A:在Curation Market这样的机制下，大家拥有特定的代币来投票给大家想要的主题。用这种想要的主题，他获得的代币越多，在这下面贡献的内容的人也会获得更好的奖励，获得更好的奖励之后又会正向吸引更多的人进入这个社区，从而把代币的价格往上抬。

当然，背后的原因也和Curation Market 具体实现的代币底层设计相关。首先会有一个智能合约，然后在这个智能合约中，代币的价格是会随着智能合约的设定进行变动的定价，代币的价格并不是固定的。而且在 这个智能合约里面流通的代币越多，他的代币价格就会越贵。比如说一个智能合约中规定，当他发行了两个币的时候，第三个币的价格是会上涨的，第四个币的价格他也会比第三个贵，这样的一个价格的设计，使 得币的价格会随着流通量的增加而不断上涨。
