json标注工具与labelme安装

        Mask-RCNN训练的数据集需要用到json文件，所以需要借助标注工具来完成json文件的生成任务。

一、标注工具

1. labelme

       这是python中的库，安装比较方便，且本人认为比较好用。

github代码地址：https://github.com/wkentaro/labelme

本人在win10+python3环境下安装lableme，在cmd中执行：

pip install labelme

安装完成后，直接在cmd中输入labelme，则进入到labelme界面：

2.VIA工具

via标注工具下载地址如下：https://download.csdn.net/download/yql_617540298/10544036

操作方便，可以直接在网页中使用，无须安装，方便生成json文件，可以自行画框进行标注。

下载后，直接点击via.html，在浏览器中直接进行操作，界面如下：

3.其他工具

COCO UI：https://github.com/tylin/coco-ui
