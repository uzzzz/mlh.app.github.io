  • 程序准备

步骤1.访问Bminer官网【https://www.bminer.me/zh/releases/】下载版本。

如图所示，数字1指向版本为Launcher版本，数字2版本为Lite版本。

步骤2.找到您下载的Bminer，并解压。

步骤3.打开解压好的文件。

注：Bminer 9.0.0后支持比原币（Bytom/BTM）挖矿。

Windows有2种版本供下载，Lite版本通过修改bat文件参数挖矿，Launcher版本通过操作界面修改参数挖矿。

  • Lite版本比原币挖矿步骤

步骤1: 打开mine_btm.bat文件；

步骤2: 根据图示修改钱包地址，矿工名。

@echo OFF

REM Change the following address to your BTM addr.

SET ADDRESS=BTM钱包地址（下图显示为1）

SET USERNAME=%ADDRESS%.矿工名（下图显示为2）

SET POOL=btm.f2pool.com:9221

SET SCHEME=tensority

START "Bminer: When Crypto-mining Made Fast" bminer.exe -uri %SCHEME%://%USERNAME%@%POOL% -api 127.0.0.1:1880

 

 

步骤3: 保存后开始运行挖矿，展示如下表示运行正常。

测试机型： P106*1

  • Launcher版本比原币挖矿步骤

步骤1: 打开Launcher文件；

 

步骤2: 按下图顺序，设置算法（2）矿池（3，4）修改钱包地址（5）矿工名（6）。

 

步骤3: 点击开始按钮后，开始挖矿。

测试机型： P106*1
