促进共享经济繁荣的强大利器——IPFS

  • 柚子社区

共享经济正在从一个新鲜事物变成我们生活的一部分，一个新时代被开启了。

说到“共享经济”不得不提的一定是“共享经济”鼻祖罗宾蔡斯女士提出的共享经济的公式：产能过剩＋共享平台＋人人参与。通俗的解释，共享经济就是利用别人暂时不用的、闲置的资源，让每个人参与进来消费，并产生经济效益。

而对于共享经济理念衍生出来的产品和服务，不用我说，大家都很清楚了，包括酒店、汽车、充电宝、雨伞等等。所以今天我们不来谈共享经济具体是啥，如何发展更好？我们剖析一把能让共享经济发展更繁荣的利器——IPFS。

什么是IPFS？

IPFS（The InterPlanetary File System）又称“星际文件系统”，是一种点到点的分布式文件系统，通过底层协议，可以让存储在IPFS系统上的文件。在全世界任何一个地方快速获取，且不受防火墙的影响，可以让我们访问数据的速度更快，更加安全，并且更加开放。

需要注意的是，IPFS即不是项目名称也不是代币名称，而是一种网络传输协议。

IPFS与传统的HTTP协议等中心化存储的方式不同，它实现了真正的分布式存储。还都是基于P2P，形成了点对点传输网络。拥有它，人们可以更容易地连接在一起，共同组成一个全球化的超大网络。

在这张网络中是没有中心节点的，你需要的资源可能在你的邻居那里，也可能在地球另一端。存储在IPFS中的资源，可能分散在世界各地，更可能依据资源的人气，集中在某个热点地区。热点的资源将会更容易获得，访问的速度也会更快。

我们觉得，在不久的将来，IPFS协议极有可能会彻底替代传统的HTTP协议。如果说区块链是对传统互联网技术的一次重塑，那么IPFS则是对传统HTTP传输协议的一次重塑。它也将成为未来所有区块链项目的数据存储基础，为整个区块链产业的发展提供有力支撑。

所以说，IPFS协议的发展空间非常值得期待，很可能逐渐成为未来主流的数据存储方式，构建起一个更快、更安全、更自由的互联网新时代。

IPFS的运行机制

传统的HTTP超文本媒体传输协议，是全世界统一的全局信息格式协议，它的制定为分发和显示信息提供了规范，至今已应用20年。我们今天所访问的几乎所有网站，都是基于HTTP协议的。在HTTP传输协议之下，数据是被集中的储存在服务器上的。

这种简单的中心化存储传输方式，虽然将发布信息的成本降到了最低，但同时也在可分布性和可持久性方面造成了先天的缺失，随着人类数据几何级数的快速增长，中心化的存储方式将很难成为人类知识的永久载体。

中心化的存储也必将出现诸多难以解决的问题，比如存储安全性差，数据有可能被监控、被复制，甚至被篡改；存储成本越变越高，数据越来越难被永久保存；当存储服务器出现突发状况时，数据很可能永久消失，无法被找回；服务器的带宽有限而且昂贵，会造成集中访问时的网络拥堵等。

这些难题，IPFS都能完美地解决。

比如你想在IPFS网络里存储《复仇者联盟3》这部电影。

首先，这段视频数据就会通过特殊的加密算法被分割成了若干小份，然后再分散的存储到世界各地矿工们的存储器里。可能一部分就在你家楼上那哥们的存储器里，还有另一部分可能被存储到了美国的纽约州。

 

其次，IPFS中的容错机制会保证你的数据被复制了足够多数量并存放在不同的地区，即使某一个地区的数据因自然灾害完全被毁，通过其他地区的备份也可以完整恢复你的数据。这就极大的保证了存储在IPFS上的数据安全性，保证你一定能完整看到《复仇者联盟3》。

此外，你也不必担心数据的隐私性，被加密保存的数据小块数据是无法被人查看的，也就是说你的邻居既不清楚他替谁保存了数据，也不清楚保存的数据到底是什么内容，这就比中心化的存储更好的保护隐私。

而且，IPFS在数据传输速度方面也是更有优势的。当你需要读取数据时，所有的存储者会同时为你发送自己保存的那一小块数据，机器接收后自动进行拼接。这样的话，你的下载速度将不再受制于服务器的带宽，而主要取决你的下载带宽，这样自然你的访问速度自然会比中心化的HTTP模式下更快。

IPFS和共享经济

所以说，IPFS是永久的、去中心化保存和共享文件的方法，这是一种内容可寻址、版本化、点对点超媒体的分布式协议。我们可以把它想象成所有文件数据分割多份到众多部用户的电脑里，要的时候可以从各个用户里取回的一个文件共享平台。它集一些成功系统（分布式哈希表、BitTorrent、Git、自认证文件系统）的优势于一身，是一套很厉害的文件存取系统。

 

 

那又为什么说IPFS是一把促进共享经济发展的利器呢？

我们都知道，互联网的初衷就是互通和开放，将所有的资源都有效的整合在一起，共同创造更美好的生活，所以“共享”无疑是互联网中最重要的精神之一。

根据上文所述，IPFS恰恰就是这种精神的强大“卫道者”，因为它所主导的分布式网络系统让我们在不断中心化的网络体系中看到了互联网的初心。

“共享经济”本质上是闲散资源的最大化利用，在“共享”的概念体系中强调的是物品的使用权，弱化了物品的所有权。说白了，这是一种拥有闲置物品的机构或个人，通过将物品的使用权在一定时间内让渡给需求者，以此获得一定报酬的经济模式。

IPFS挖矿网络中的共享也是如此。没有中心化的服务器，都是分散的单个节点，客户存储数据，矿工提供存储空间，客户想要查看数据，矿工搜索并发送相关数据，矿工将自己的存储空间或者检索服务的使用权让渡给客户，客户支付FIL代币。

从这个意义上讲，IPFS挖矿其实就是“共享经济”的延续，是互联网共享精神的体现，顺应了互联网的发展趋势。将来随着它不断进入广泛应用范畴，它肯定能成为促进共享经济发展繁荣的一把强大利器。
