AI、VR、AR、大数据、云计算、区块链、API，这六大科技产物，真的都能被时代接受吗？

  • 柚子社区

 

AI是一个广泛的领域，有些领域已经非常实用，但大多数只是科幻小说的概念。

VR技术与IT产业没有多大关系，它依赖于脑科学的发展。现有的“VR”是伪VR，技术路线是错误的，没有游戏。

 

AR的大部分需求可以通过IT行业的技术来解决。它比VR更可靠。你可以参考智能手机技术的发展。对于那些更科幻的人，你可以参考VR，所以没有游戏。

 

大数据的社会价值不容小觑，这有点像天网的意思，不仅如此，大数据技术还应用于许多地方（如国家治理），并产生了真正的社会和经济价值。

 

云计算是一个古老的概念。这是互联网时代大型机概念的新发展。虽然历史正在向上发展，但云计算具有大型机的各种优势，大型机的各种缺点并没有下降，因此历史的螺旋式注定要将其关闭 - 云计算的速度无法跟上智能手机的速度。怎么可以看出大型机无法跟上PC速度的历史发展？云计算是一个更集中的大型机，智能手机更是个人电脑。

 

区块链没戏，当比特币被炒作时它也是火，但火并不意味着它是有用的。有段时间郁金香也非常火，但谁能说郁金香有很好的前景呢？让我们来谈谈它，区块链技术本身就是一种非常低级的基础设施技术，它通常用于解决大规模分布式网络节点之间的信任问题，但问题在于会遇到这种情况，很少有单位需要解决这个问题，大多数有这种需求的单位倾向于选择有利于改善其控制的技术。（举一个粗略的例子，腾讯更喜欢在自己的控制下使用它的Q币，而不喜欢使用不受控制的比特币），所以人类对区块链技术的需求只存在于一定的最小范围内，需求量小，就意味着价值低，所以没戏。

 

API应用程序接口是一组定义、程序及协议的集合，通过 API 接口实现计算机软件之间的相互通信。API 的一个主要功能是提供通用功能集。程序员通过使用 API 函数开发应用程序，从而可以避免编写无用程序，以减轻编程任务。现在的社会上，API已经应用在各行各业上面，比如说生活缴费行业就有很多的API接口，“赛合一数据”的话费、流量、加油卡水电煤API接口等，对很多的公司提供了便利，这样的作用在这个互联网时代，是不可估量的。

 

AI/VR/AR／大数据/API，这五种技术如果发展得成熟了，它们就是能够改变人类社会发展历史进程甚至改变人类文明形式的史诗技术，妥妥的潜力股。相比之下，云计算和块链只是昙花一现，它们可以在不影响全局的情况下添加到蛋糕中，仅仅如此而已。
