eos最新版的本地环境搭建（v1.0）

  • 柚子社区

啊...eos更新换代是真快，转眼间当年的教程就不能用里，今天闲着没事，做个eos最新版的本地环境搭建的教程，简单轻快版的

首先呢，是下载源码

git clone https://github.com/EOSIO/eos —recursive

然后就会在主目录下生成eos文件夹

cd eos ，进入eos文件夹

输入ls 我们来看看里面都有什么

 

理论上呢，里面没有build文件夹的，现在这个是我搞好了的，好了进入eos文件夹

./eosio_build.sh

自动构建源码

之后就会出现build文件夹了

进入build  输入sudo make install

然后cd返回主文件夹

cd .local/share/eosio/nodeos/config

vim config.ini

开始修改config.ini文件

enable-stale-production=false,这里要把false改成true，也就是enable-stale-production=true.

producer-name =   eosio,

在produce-name后面加上eosio就可以

注意producer-name是被注释掉的，要把#去掉

最后在后面加这几行

 

  • plugin = eosio::producer_plugin
  • plugin = eosio::wallet_api_plugin
  • plugin = eosio::chain_api_plugin
  • plugin = eosio::http_plugin

然后去eos/build/programs/nodoes里./nodeos

就开始生产区块了，没有报错，这就是eosio的最新版本地配置方法
