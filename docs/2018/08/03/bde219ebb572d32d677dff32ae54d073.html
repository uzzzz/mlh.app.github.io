来自Mirantis和富士通的超级计算机和OpenStack云系统

  • 柚子社区


8月2日，Mirantis和富士通宣布，两家公司合作创建了用于北海道大学Information Initiative Center的一个跨学科大规模计算系统，其理论峰值性能为4.0 petaflops，包括超级计算机系统和云系统。该系统预计将于2018年12月投入使用。

北海道大学正在扩大教育和研究的范围。北海道大学Information Initiative Center提供了一个学术信息平台，作为全日本大学教师、研究生和研究人员进行学术研究的多用途设施。

“OpenStack由于其灵活的软件定义基础设施和强大的API，非常适合高性能计算（HPC）。”Mirantis的首席营销官和联合创始人Boris Renski说，“Mirantis Cloud Platform（MCP）专门针对HPC用例进行了调优，支持数据科学、高级模拟、人工智能和机器学习、物联网、区块链等，成为北海道大学的选择。”

北海道大学Information Initiative Center目前运营着北海道Acedemic Cloud——该云由跨学科的大型计算系统组成，包括超级计算机系统、云系统、PB级数据科学统一云存储。为了将其跨学科的大规模计算系统升级到北海道大学High-Performance Inter-Cloud，它正在构建一个超级计算机系统，以将现有系统的处理性能提高2000％，并建立全国范围的广域分布式云系统（也可以用于云和网络的研究）。它将进一步推进科学和技术模拟、人工智能、大数据和数据科学等现有领域的使用和研究，并将提供一个能够满足更广泛需求的环境，因为人们接受了培训来使用大规模的计算系统进行开创性的研究。

“北海道大学High-Performance Inter-Cloud是一个先进研究的转型研究推广平台，有机地整合了全国范围内部署的超级计算机系统和云间系统，并加速了各种类型的研究项目，如人工智能、大数据和物联网，以及来自日本各大学、研究机构和私营部门的大规模计算科学模拟。”北海道大学信息促进中心主任Masaharu Munetomo博士说，“MCP提供基于OpenStack和Kubernetes的灵活、可调的平台，持续提供基础设施即代码，以便持续、快速、可靠地获取Mirantis和富士通的新创新。”

“Mirantis Cloud Platform带有基础设施即代码，可实现云基础设施的安全、轻松的生命周期管理。”富士通平台软件事业部任务关键系统支持部门负责人Masahide Abe表示，“我相信完整的开源云基础设施软件是那些期望从系统基础设施获得敏捷性和可靠性的用户的最佳选择。富士通和Mirantis共同提供不断发展的云基础设施软件——该软件结合了先进的开源技术，并结合了高质量的服务和支持。”

新系统将包括：


超级计算机系统：

——新的超级计算机系统是一个大型并行计算机系统，包括子系统A（由1004个配备英特尔至强可扩展系列处理器的PRIMERGY CX2550 M4 x86服务器组成）、子系统B（由288个配备英特尔XeonPhi处理器的富士通服务器PRIMERGY CX1640 M1 x86服务器组成）。每个计算节点都用最新的高速互连英特尔Omni-Path Architecture连接，可实现高度并行的计算性能。

云系统：

——新的云系统拥有64个x86服务器，包括富士通服务器PRIMERGY RX2540 M4和NVIDIA Tesla V100（最新的GPU计算卡），部署在北海道大学，还有7个远程站点位于日本关东、关西和九州地区。将这些服务器连接到SINET5学术主干网，使其成为从北海道到九州的全国性广域分布式云系统。


——云平台使用MCP，其核心是OpenStack Pike版本，提供最新的研究环境。


——旨在灵活满足各种需求。


——富士通Software Technical Computing Suite是富士通用于大规模系统的HPC中间件。该系统可为作业提供最佳控制和节点占用，从而提高各种预期操作的便利性。它还提供灵活的电力控制，通过限制每个用户消耗的最大电力并打开或关闭节点的电力来降低耗电量。
