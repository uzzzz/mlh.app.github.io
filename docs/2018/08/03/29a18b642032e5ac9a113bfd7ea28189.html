Skrumble Network推出区块链测试版dApp应用，重新定义端对端通讯

  • 柚子社区

Ally是一个基于区块链的去中心化应用(dApp)，用于端对端通讯，旨在让用户的隐私和数据所有权得到真正的重视
 
多伦多--(美国商业资讯)--专注于通讯业务、致力于简化人们联系和签约方式的公共区块链Skrumble Network (SKM)今日宣布，公司在全球发布了其首款面向iOS和安卓系统的试用版去中心化应用Ally。Ally目前处于中心化测试阶段，在不久后也就是2018年9月将实现去中心化。Ally是业界首款多用途移动媒体和通讯应用，能够让全球用户自由、匿名、安全地进行通讯。与Facebook、Telegram和微信不同的是，新的去中心化应用基于Skrumble Network公共区块链，因此使用的是社区自有存储，且不设有中央服务器。此举确保了任何外部机构都无权审查、拦截或操控任何数据。
 

Skrumble Network联席创始人Eric Lifson表示：“我们十分高兴能够见证我们的项目所取得的里程碑，并借此实现了连接全球社区的共同愿景。目前，人们迫切需要一个能够将其隐私、数据安全以及自由放在首位的通讯平台，我们通过打造Ally来解决这一问题，从而创造一个能够让所有人都感到安全的环境。”
 
Ally（试用版）的一些主要特色包括：
 

  • 信息定时删除
  • 私人、公共和群发功能
  • 最多支持50人同时视频
  • 流媒体直播
  • 应用内置钱包
  • 聊天内置加密货币转移功能

 
7万多名社区成员已经预注册参与Ally的测试版。Ally的去中心化测试版有望于2018年9月在测试网(TestNet)发布，预计将于2019年初在主网(MainNet)发布。
 
Skrumble Network由Skrumble Technologies Inc提供支持，已经吸引了超过15家世界级风投合作伙伴，以及多名知名行业顶级顾问，包括Anthony Di Iorio（以太坊 、Jaxx & Decentral联合创始人）、Jeff Pulver（VOIP先驱，Vonage联合创始人）、 Jinius Tu（Aion首席技术官）、Redouane Elkamhi博士（多伦多大学罗特曼管理学院金融学副教授）和Kevin Hsu（Block VC创始人兼执行合伙人）。
 
SKM代币目前可在Gate.io、LBank、Hotbit、CoinBene和Fatbtc进行交易。
 
关于Skrumble Network
 
Skrumble Network (SKM)是一家新成立的创新公共区块链，并针对安全的以通讯为中心的连接和交易以及去中心化应用进行了优化，为开发人员打造各类应用提供通讯层。由于没有中间机构或中央服务器主机来审查、拦截或操控任何数据，Skrumble Network的区块链和去中心化应用旨在提升数据隐私的保护力度，并帮助真正实现全球范围内的通讯民主化。
