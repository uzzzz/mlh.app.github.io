一、产品分析

产品概述：

Ultrain团队于2017年在杭州成立，致力于通过其可编程技术基础设施构建高性能分布式区块链平台，加速推动构建一个高性能、高服务质量、低成本的支持大规模应用的商业生态系统。

行业痛点：

Ultrain认为现有的区块链公链都无法支持大规模的商业应用，主要是因为：

1、系统交易性能地下，btc的tps是7笔/s，eth的tps是20笔/s，而支付宝的峰值是26.5W笔/s，两者相差近四万倍。
2、智能合约性能低下，代码行数限制、执行时间限制、执行顺序限制，使得很多试图币改的行业都不能切入。

技术创新：

1、共识创新：

超脑链的共识框架是 R-POS，也就是随机可信证明机制，整体采用了主侧链的多架构体系，主链采用随机可信拜占庭共识，理论 TPS能达到2万，确认时间为15秒；侧链使用随机可信权益证明，理论 TPS 为3000，确认时间为2秒。

民主投票制，社群群众+人工智能+机器学习，每一次对不同治理方案的选择，都采用民主投票的方式确定方案，每人拥有的票数，按其锁仓的Ugas的数量进行分配，并由Ultrain的治理AI协调投票结果并最终自动执行。

2、密码学创新：

每轮形成共识的时候，只需要使用整个网络中1%的算力（CPU、GPU、内存、硬盘、带宽等）就可以实现铸币，而剩下的99%算力都可以以信任计算的形式提供出来，通过Ultrain的管理和调度，对外服务。而不再像btc、eth两大公链那样铸币，消耗大量的社会资源。

3、智能合约创新：

每一个基于Ultrain都可以发行自己的token，设定自己token的货币体系，选择不同的共识机制，专属矿机。Ugas是用于购买运行智能合约所必须的算力token，类似eth的gas，是Ultrain的计费和结算工具。

Ultrain在JavaScript语言的基础上，设计了智能合约的编程语言，包括提供标准代码库支持；通过模板技术提供一系列标准化的智能合约模块；以及通过机器学习的实现智能合约形式化及口语化编程实践，降低普通人的学习曲线。任何人简单培训后，可以通畅的阅读、编写、审核智能合约。

产品进度：

2017年10月到2018年6月为内部网络运行，对外服务 API 构建，发布智能合约开发框架 Robin，根据黑马网报道，该项目在2018年7月11日推出两个产品更新：

1）基于R-PoS共识机制的概念验证网络；
2）开发者友好的DApp集成开发框架Robin。

产品上线落后于进度一个多月，但是在官网公布的 GitHub 代码中并未搜索到 Robin 相关内容。Ultrain相比于其他公链项目，最核心的区别是创新的共识机制，解决性能问题和挖矿的能源消耗。

路线图可行性：

超脑链的计划路线为通过搭建公链来做应用生态，同时结合人工智能和机器学习，以及物联网来为 Dapps 服务。从目前产品开发情况来看，整体难度较大人工智能和物联网并不成熟。路线图中大部分功能的视线需要结合大环境的发展，包括人工智能技术和物联网技术的成熟才能实现。

三、代币分析

代币发售：

总量：10亿
本轮出售比例：私募轮1亿枚（10%）

代币单价：0.2USD

本轮募资：2000万美元

锁仓规则：投资人在token登陆交易所时release 25%，随后每6个月释放25%，锁仓期合计一年半。

使用用途：

UGas 是超脑链的经济体系的基础，其主要有如下用途：

1、参与铸币与算力的节点，都必须抵押 UGas；在选择铸币节点和算力节点时，持有UGas 数量越多，选中几率越大，但呈边际效益递减特性；
2、使用超脑链算力时，需要支付 UGas 做为使用费；
3、使用超脑链上的第三方服务组件时，需要支付 UGas 做为使用费。

点评： 根据官方白皮书披露，项目Token的用途是算力的计费和结算。代币的分配情况，根据路线图，到2019年4月官方售卖矿机，而私募占比10%，锁仓近一年半。整体上项目前期代币筹码比较集中。在整个超脑经济生态中，参与方包括矿工，超脑链开发运营团队，DApp 应用开发者，DApp 使用者，交易所投资者这几种角色。

那么项目代币的增值就来自于其DApp开发和使用者的增量，只有项目上的应用增量时，矿工的算力提供才能产生更多的价值，而DApp的使用者增量的因素就是DApp的价值体现了。本质上也就是公链的性质，就是看项目的底层技术、生态情况、吞吐量、安全性、可拓展性及商业化的规模。

六、青水评级总结：

1、梦想有点大，Ultrain准备在公链、智能合约、物联网、人工智能等全领域展开，步子太大，目前每个领域都有一定的技术难度，想取得突破，怕现在这么点人不够看。

2、想法很好，但是TPS只有每秒2W，就显得不够吸引人。毕竟现在很多公链比如夸克都百万TPS。

3、Ultrain的团队大部分都来自阿里巴巴，部分成员有关于区块链或互联网的专利，技术背景很强大，同时其创始团队有着亮眼的政治背景。团队没有创业成功的经验，不过如果或多或少能蹭到阿里巴巴资源的话，毕竟算是嫡系，潜力就很大。

4、融资能力强大，种子轮融资400万人民币，天使轮融资几千万人民币，最近一波2000万美元融资，融资是一波比一波亮眼，实际估值都快2亿美元，顶级VC排排坐，团队肯定不差钱，也不差资源。

4、路线图很一般，虽然如期完成第一阶段的开发平台的发布，酷炫的视频中展示了智能合约的可操作性，团队的执行力还是不错的。

5、10%私募，15%分配给团队，50%用来挖矿，没有预挖。而2019年4月才推出矿机，这样看来代币分配就不是很合理，如果在矿机之前，代币进入市场，可被操作性就很大。
