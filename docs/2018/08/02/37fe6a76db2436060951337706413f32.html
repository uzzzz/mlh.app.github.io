在过去的几年中，互联网审查长期存在。在2011年的阿拉伯之春期间，互联网审查宣布“它将使公民抗议的浪潮彻底终结，使公民保持沉默”。这一举动使得Facebook和Twitter这两个大众用来协作彼此的主要社交网络完全失效。这清楚地表明了我们的互联网有多么的缓慢、脆弱以及健忘。

如何通过去中心化的互联网实现网络自由（西班牙语）

 

另一方面，我们有IPFS。

IPFS是一种新的点对点超媒体协议，旨在补充甚至取代现在掌控网络的HTTP协议。HTTP存在以下问题：今天，当您访问网站时，您的浏览器必须直接连接到为该网站提供服务的计算机，即使这些服务器距离遥远，传输过程中需要占用大量带宽。因为每个网络都有一个对等协议，而每个数据提供商都要收取费用，因此每个网络跳（ network hop）都花钱给数据提供商并浪费带宽，。更糟糕的是，HTTP一次只能从一台计算机下载文件，而不是同时从多台计算机中获取文件。-  TechCrunch

 

总结一下，通过权力下放和IPFS， 我们可以避免审查、扣押、访问控制，让事情变得更容易，更快速地适应社会需求 ; 它也带给人们更多的自由，从而增强自我。

 

事实上，虽然今天的每个人都在一边讨论着权力下放，一边围绕权力来建立公司。正如已经证明的那样，没有很多真正的去中心化成功案例。
