3.安装VS编译器过程中遇到问题是：

大神推荐的安装包GRMSDK_EN_DVD.iso，安装不成功。

报错：Setup could not find the file WinSDK_amd64\WinSDK_amd64.msi

查找攻略发现下面这篇文章可以借鉴

http://blog.sina.com.cn/s/blog_b7770ea80101ge5j.html

我的解决方案包括两点：
（1）.vc++的开发运行环境，凡是大于V10.0的版本，都需要卸载，这样才能安装MS SDK。

控制面板下程序卸载中，找到这些东西，全部卸载。另外卸载有可能会影响其他程序的功能，可以在安装完成qt vs编译环境后，再把vc++ redistributable安装回来。

（2）大神推荐的安装包GRMSDK_EN_DVD.iso，不一定适合你的电脑环境。

这里我安装了为微软提供的web安装版，能够自动匹配下载。

链接：
https://download.microsoft.com/download/A/6/A/A6AC035D-DA3F-4F0C-ADA4-37C8E5D34E3D/winsdk_web.exe

4.以上的winsdk安装完成后，就可以安装qt库了，具体请参见大神攻略。

5.qt库安装完成后，安装qt creator开发环境。

6.新建一个qt控制台程序，试试hello world吧。

祝各位好运！
