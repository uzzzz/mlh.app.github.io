区块链是什么

区块链是分布式数据存储、点对点传输、共识机制、加密算法等计算机技术的新型应用模式。所谓共识机制是区块链系统中实现不同节点之间建立信任、获取权益的数学算法。
区块链（Blockchain）是比特币的一个重要概念，它本质上是一个去中介化的数据库，同时作为比特币的底层技术。区块链是一串使用密码学方法相关联产生的数据块，每一个数据块中包含了一次比特币网络交易的信息，用于验证其信息的有效性（防伪）和生成下一个区块。—— [ 百度百科 ]

这是比较官方的解释，简单点理解区块链就是个计算机技术的新应用模式，就是个新应用模式或者说新的技术玩法。

为什么说区块链一定会火

刚也说了这是个技术新玩法，那么我们来拆解下它这里几个核心价值点：

价值项 技术点
去中心化 点对点网络
智能合约 加密算法，共识机制
分布式账单 公有链，私有链，联盟链
数字货币 BTC,ETH等

这里针对以上名词不做赘述，大家可以针对性查阅资料了解，也正是基于去中心化和智能合约两大核心价值，才造就了如今层出不穷的数字货币，当然，现在的币圈龙狗同行，鸡凤齐飞，是骡子是马，还需要静待洗牌完毕才能揭晓。但单单一个数字货币这一个区块链的直接衍生品就已经搞的币圈一天，人间十年了，可想而知，当两年后区块链应用纷纷落地之时，那必将是互联网的时代变革。

初链-TrueChain

背景介绍　不废话，直接点：
初链希望在保持去中心化本质的同时，尽可能提升效率。其核心设想在于PoW与 PBFT 相结合的混合共识机制。下面说说它的几大核心价值点：

支持无限节点进入
上面也说了初链的优势在于尽可能提升效率。对于互联网电商、即时通讯软件、双边交易平台等大规模商业应用，支持大规模和不断增加的用户数目是公链承载它们的必备条件。PBFT 的通信复杂度决定了参与决策的节点只能保持在极有限的范围内。而利用 PoW 可以接纳无限节点的特性与之结合可以弥补这一弱势。
安全性
PBFT 由于无法保证全体节点参与决策，所以存在比较强的道德风险及安全隐患。少量节点的不作为或宕机可能导致数据被篡改或全链瘫痪的隐患。以 PoW为基础，选举产生 PBFT 节点的混合共识机制设计可以保证 PBFT 节点出现题时及时进行重新选举，并对 PBFT 节点进行实时监督。
高性能
用户交易被及时记录是公链可用性及安全性的保障。PBFT
接点的通讯效率足以支持 10,000-100,000 TPS（每秒交易处理
量）。可以保证多个智能合约或商业应用同时处理交易时全链通
讯不受到阻塞，账本按时间戳先后顺序准确记录交易。

还有很关键的一点，在初链的白皮书上有明确说明：

无论是现在的测试网还是未来上线的主网，初链将一直保持
向所有用户免费开放的准则。初链始终认为，公链是面向所有用
户的基础设施，而非牟利工具。为每位用户提供更贴合需求、使
用便捷、低成本的公链开发工具不只能为未来初链寻找合适盈利
来源开拓空间，更有助于整个公链开发乃至区块链行业的长足发
展。

技术架构

初链的技术架构自下而上共分为四层：一是底层服务，包含
P2P 网络、LevelDB 数据库、密码学算法、分片优化；二是核心
层，包含区块链模块、TRUE 模块、共识算法模块；三是 API 接
口层，包括 TVM 虚拟机、gRPC 通讯协议；四是智能合约层。
具体设计请参阅初链技术黄皮书或 github 开源代码
https://github.com/truechain。
