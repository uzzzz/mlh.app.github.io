nodeJs——用express命令创建nodeJs新项目

  • 柚子社区

1、安装node.js

      来自菜鸟教程(很详细)     教程地址：http://www.runoob.com/nodejs/nodejs-install-setup.html

2、npm安装

      node安装时有自带的npm已经安装 但 不是最新版本的npm

      查看node、npm版本号：

              win+R快捷键打开，输入命令cmd，确定

       

      输入 命令node -v   npm -v 查看相对应版本号

      

      用命令将npm更新到最新版本（如果需要可以将npm更新最新版本，不需要可以不更新）

            npm install -g npm

            更新易报错，可根据报错提示信息相应去解决

            （博主当时运行项目报错，解决命令：npm rebuild node-sass --force）

3、下载最新版本express

      运行以下命令：

      npm install -g express  

      npm install -g express-generator

      

4、创建简洁而灵活的Web应用框架--Express

      cd 项目目录

      express 项目名称

      

      添加成功后，所在文件夹

      

      根据提示执行命令

            cd 项目名称

            npm install

      执行命令后  项目文件新增文件夹node_modules和文件package-lock.json

      执行npm start运行项目

      

浏览器访问--http://127.0.0.1:3000/

出现以下画面即成功
