Brave 狮子王浏览器，撕裂式创新？

  • 柚子社区

在许多的区块链峰会上，都有VC给大家建议如何去评估一个项目，譬如说：落地没，创始团队，顾问团队，项目是否解决痛点，Token结构等等的评估，那今天带给大家一个带有明星光环的项目「BAT」，目前已经有超过200万的月活用户，犹如一颗正在升起的美丽恒星！

Mozilla 前CEO，也就是火狐浏览器的母公司，Javascript 发起人，Brendan Eich。光光一位创办人，「BAT」就是所谓的明星阵容了，BAT 不是我们国内说的那三大巨头，而是 Basic Attention Token，基本注意力币。

而 BAT 的主力产品当然就是这个 Brave browser 狮子王浏览器，虽然团队声明，BAT 未来也可能在其他浏览器或是应用中可以使用，但是我们今天就以狮子王浏览器为主来介绍，浏览器 下载连结 。

首先我们用从用户的角度来看起：

  • 运行速度极快，最高可达到Chrome and Safari的八倍，并且完全屏蔽广告
  • 防止网页追踪，提供深度安全保护，不支持userscript，用户无法自定义界面
  • 使用机器学习来对用户进行注意力测量，未来用户可以透过看广告来收获 Basic attention token，当然也可以完全不看广告，内容作者/内容商同时间可以获得 BAT 作为奖励
  • 与 DuckDuckGo 合作，你搜寻的行为就是你搜寻的，不再是google的行为资料库
  • 安装 MetaMask 之后，可以进行相关的支付或是智能合约的运作
  • 简单来说，我从 Chrome 换成 Brave，从免费切换到赚币

再来从商业的维度来看起：

  • 商业结构：用户，内容商/作者，广告商，用BAT代币串连
  • 广告必须购买Token来头放广告， Token使用就会被销毁掉，持有BAT的用户可以享受到升值，或是透过使用Brave浏览器获得BAT
  • 不需聘用任何会计与出纳，可以没有财务报表，在公开帐本里，任何人都可以查到相关代币流向
  • 用户可以选用 BAT 捐赠月费方案，所有的代币将会投放给首批 BAT 认证的内容作者
  • 有可能是少数在2018年就可以通过代币来创造生态系的项目

狮子王面临的问题：

  • 哪来的内容生产者？目前与Youtube上的内容发布者合作进行分红，但是Youtube未来可能会有相对应的措施，其他的内容生产平台未来也有可能集体抵制。
  • 商业模式的撕裂？互联网现有的模式，搜集使用者习惯，透过用户的使用状况提供精准广告的这个前提，被撕裂了，换句话说，使用Brave浏览器的人越多，将会影响越多广告商，以流量为主要运营模式的网站，都将在狮子王浏览器里被彻底的破坏。
  • 许多网站的功能没有针对Brave做出最佳化，导致体验出问题。
  • 如果要使用 Metamask 网页钱包，还不确定 Brave 浏览器的安全性能到什么层级。
  • Brave 的插件开发，社区运营都是极大的挑战。同时间要面对 Google 这道墙，将看广告的选择权还给用户，这肯定不是一条正常人会选择的路子！

而随著我们对于 Brave 团队的长期观察，虽然项目进度一直在往前推进，但是现实的残酷也告诉了我们，Google的版图，影响力之大，一般人很难撼动。目前 BAT 合作的对象多是 Youtuber，甚至还有说唱天王五角的加盟，但是 Youtube 的态度是什么，目前还不明朗。而目前大部分狮子王浏览器的使用者集中在北美与欧洲，算是集结了一群比较中高端的用户，而这个阻挡所有广告，追踪器的浏览器，在互联网圈子里面能够拿到多少盟友，在区块链圈子又能抢到多少盟友呢，请跟著链榜一起追踪 BAT 最新动态吧。
