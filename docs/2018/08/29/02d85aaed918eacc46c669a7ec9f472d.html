一、NodeJS安装

使用nvm管理Nodejs。
nvm 是Nodejs 版本管理器，常用命令如下：

nvm install [Nodejs 版本] 安装某版本的Nodejs
nvm uninstall [Nodejs 版本] 安装某版本的Nodejs
nvm use [Nodejs 版本] 使用某版本的Nodejs。
nvm ls 查看本机安装的所有的 Nodejs, 并高亮当前使用的版本
