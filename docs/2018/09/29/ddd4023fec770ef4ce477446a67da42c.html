思想之路

在上一个活动中，我学习了很多。区块链的技术发展之快，不在我的预料之中。

小伙伴们似乎有点着急“落地问题”，这个问题我看来，可以说有值得投入的地方。但是我回去之后看了一些文档，关心得比较多的却是另外一个问题。

在活动上我们已经使用了“物理世界”“数字世界”“通证”之类的词。回来之后，我的兴趣又掉回去我原来担心的一些事情上……一系列复杂的脑力运动之后，我总结出了这篇文章。希望开启一些新的领域。

我们这里最终是要建立概念定义并且说问题。因为问题就是我的意思。(我的工作是把这些整理成形式)。注意：我并不声明这些东西是我原创的。相关的资料参考，我也略去了。

关系之路

数字世界和物质世界同一性

使用Linux的同学应该对“环境”并不陌生。在民生领域，这个词也是有用的。比如说“大环境”，“商业环境”“历史社会环境”，如此等等。

经过我的研究，这个词就在世的意义来说，差不多是“一切存在着的对象、中介的总和”。更低一点，比如又可以理解为“周围的地方和事物”，“所处的情况和条件”，如此等等。

这是现实世界的环境，跟区块链、通证、计算机电脑之类的有什么关系呢？有关系。因为，计算机中的环境，通常又被理解为“各种变量的总和”。而各种变量，就是可以总为物理世界各种对象因素的中介或代表。

举个例子，支付宝余额在阿里巴巴集团服务器中的那个数字，代表你的支付宝余额。你花钱，减了那个数字；你存钱，那个数字增了；那个数字为0，你支付宝没钱了。

微信发出去的数字包上的数字，可能是代表着一些话、一些声音，你说了什么话，微信发出去什么数字；你收到了什么数字，经过了微信的解码、处理，变为你以为的收到的“信息”“声音”；这两个数字没事，说话的差不多等于听到的，这两个数字哪搞错了，听不到、说不出、或者什么地方不对了。

又比如，在百度上浏览什么，各种各种的数据信息的展现，又比如你填入的动作，你的动作转换为具有效力的意思表示，或者什么东西。最后在计算机里面的，终究是一堆数字、变量过他们的改变。

总之，变量的修改跟某种现实的效力，是有着统一性的。现实世界的环境，和数字世界的环境，具有统一性或者说同一性。(而且夸大来说，现实世界的差异性、矛盾性同样在数字世界以什么方式存在着)。

区块链世界必然要联系现实物理世界

马上我们来看，就根本的信息结构的意义来说，区块链是一堆数字，而互联网也是一堆数字，所不同的是，互联网世界的数字由于人的干涉、成熟，已经有了成熟的发展；也就是说，成了精了。

于是我们就发现现在伦理方面的一个问题，某种程度来说，区块链的问题，人工智能的问题，说到底就是人类怕这些东西“又成了精”了。这个问题在政治方面的麻烦，今天似乎还是没有解决的。

但是抱着发展技术为人民服务来说，我们看看大伙现在的脑子里是什么。

第一个问题是我们希望区块链能够用在为人民做好事造福人民为人民服务产生价值创造劳动岗位促进人类命运共同体的发展还是什么科学民主文明进步之类的地方。(问题1)

第二个问题是这第一个问题怎么能做到。(问题2)

注意这两个问题都很困难。第一个问题不妨说为区块链的用途，第二问题不妨说为区块链的用途如何必然可能。

我们的猜想：必然要在联系现实世界中可能。

问题之路

建立严格的区块链理论体系：社会、物理和数字的联动

就一个最低的意义来说，社会是我们最广泛的关心对象。就一个低的意义来说，物理，或者说自然界，又是我们能力的某种极端。

直观就可以发现，上面我们说的问题1和问题2必须要被合并为“数字世界的问题”，消解它自己在社会、自然之中。

也就是说，需要社会、物理和数字的联动。

发展有能量的区块链社会实践：三阶环境论

我们回头看看现实。

首先，整个物理现实世界，本身就是一个环境，我们不妨就叫做“一阶环境”〔First Order Environment〕。

然后比特币的区块链上，不妨看为是一个“二阶环境”〔Secondary Order Environment〕。

然后把用户，或者叫人心，看做又一个环境，不妨叫做“三阶环境”〔Third Order Environment〕。

那么，区块链的作用过程，不妨就看做是一阶环境、二阶环境、三阶环境的相互作用过程。

三阶环境不通过二阶环境直接作用一阶环境：比如直接去买足球票。

三阶环境通过二阶环境作用一阶环境：比如买区块链上的足球票。

一阶环境决定二阶环境和三阶环境：已有现实条件使得人们用不用怎么用或者怎么看区块链。

二阶环境和三阶环境反作用于一阶环境：比如区块链技术对现实世界的能动改造。

区块链较能公式：若一个实践不通过二阶环境的效能是a，通过二阶环境的效能是b，那么区块链较能量 c=b-a 。(部分a,b可能可以期望为无穷)

而区块链的效力问题，某种程度上就是区块链实践的问题：三阶环境的操作怎么能够通过二阶环境的实践，在一阶环境得到效果。
