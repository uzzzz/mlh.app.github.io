Metamask安装

Metamask是以太坊的一个插件形式的客户端，可以通过chrome的网上应用商店进行安装，chromey应用商店网址如下：

https://chrome.google.com/webstore/category/extensions?hl=zh-CN

 

在应用商店界面左上角，输入metamask，单击搜索按钮，在出现的页面MetaMask项里，出来扩展程序，点击添加到chrome按钮，安装完成，如下页面所示：

在chrome的工具条里会出现一个小狐狸的图标，MetaMask客户端已经成功安装。

 

MetaMask的使用

网络选择

点击小狐狸小图标，点击出来的界面右上角的下拉箭头,选择连接的网络，出现如下图所示：

 

出来的界面作简要的说明：

Main Ethereum Network：是以太坊的主网，选择是客户端默认的选择的。

Ropsten Test Network ,Kovan Test Network,Rinkeby Test Network：这是公网上的以太坊的网络。

Custom RPC：我们可以自定义连接本地节点.

创建账号

输入助记词，输入密码，密码最少是8位字符，即可创建账号。助记词务必记住和保密，遗忘密码时，可以通过助记词找回。

创建完账号，输入登录密码，进入如下界面：

上面界面左边箭头所指，即为刚创建的以太坊账号，也即是公钥。

 

存入

有了以太坊的账号，那么我们就可以从别的账号上，通过转账的方式往这个账户存入以太币。

转出

把账户里的比特币转出，可以点击上面界面上的send按钮，出现如下界面：

 

From： 默认当前账号。(不用填写)

To:     填入对方账户。

Amount：填入比特币的数量。

Gas fee: 转账费用，自动生成，费用越高，转账就越快，因为矿工会优先处理费用高的交易，可以点击进行调整。

填写完成，点击next按钮进行下一步操作即可。
