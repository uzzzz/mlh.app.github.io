冠希老师看了会流泪，分布式存储爱我你怕了吗？

如今全球数据储存量呈现爆炸式增长，企业及互联网数据以每年50%的速率在增长，据预测，到2020年，全球数据量将达到35ZB，等于80亿块4TB硬盘。数据结构变化给储存系统带来新的挑战。非结构化数据在储存系统中所占据比例已接近80%。

除了企业对信息的储存拥有比较大的需求，个人的云储存需求也不容小觑。而传统的储存已经不能满足日渐庞大的信息储存需求。

 

传统信息储存将我们至关重要的个人信息放在巨头的手中，黑客得到我们的个人信息简直如探囊取物，不要说将电脑交给修理店的冠希老师，即使是苹果这样的巨头公司，云储存iCloud也爆发过几次信任危机。

作为企业与信息储存相关的，安全当然是第一位的。不过因为庞大的数据储存，不得不还要加入考虑的就是费用问题。

 

好在，web3.0时代的到来带来了与区块链天然契合的分布式储存——直击储存量，安全性，费用庞大的三大问题。

 

web3.0分布式储存——扩展储存量    

传统分布式存储是一个将硬件或软件组件分布在不同的网络计算机，彼此之间仅仅通过消息传递进行通信和协调的系统。

以前就只能把文件储存在一个单一的集中服务器中，有了区块链技术就能把文件储存在全球范围内上千种不同的设备中。

 

它的工作原理如下：你上传的每一份文件都会被分开储存进若干小内存块中，每一块小内存块又会被储存在网络中若干不同的设备中。哪一块内存块储存在哪一个设备中都被记录在区域链中。如果你想取回文件，系统会根据指令将其重组。

 

web3.0分布式储存——提高安全性   

研究结果显示，2016年，在1540万美国消费者中被偷走的美元共计160亿，而去年同期，1310万美国消费者中被偷走的美元共计为153亿。在过去的六年里，身份盗窃者已经偷走的财产共计1070亿美元。黑客们皆钟爱于利用在网上相对容易地获取大量数据。

 

这是由于数据一直被储存在集中的数据库中,所以,任何黑客的攻击将导致大量的用户数据暴露。

尽管区块链承诺提供更好的安全性，但考虑区块链实现的一些风险仍然很重要。与人们的看法相反，区块链确实是可以被破解的。Sybil攻击、路由攻击和DDoS攻击都为正在出现的区块链项目带来了重大的安全挑战。

 

对于达成一致意见的算法项目，在区块链网络上由矿工执行的51%的攻击已被证明对许多项目造成重大的安全问题。比如，边缘地区，以及其他项目，都面临着常规的51%的攻击。

 

即使有这些问题，区块链仍然比传统的数据存储安全更有优势。正如上面所提到的，传统的数据存储解决方案严重依赖于集中的数据库来维护安全性。对于黑客来说，攻击目标更加明确。一旦黑客成功地在一个集中的数据库上执行，黑客就可以访问大量的数据。然而，使用区块链和分布式台账技术，破解要难得多。例如，在Storj中，由于数据被加密并分布在大量的数据库中，只有少量的数据可以被侵入。

 

web3.0分布式储存——降低成本

为了比较形象的说明收费，我们直接对比了传统储存大佬Dropbox和web3.0储存新秀Storj的收费标准。

DropBox的使用费，对个人有2GB免费储存额度，超过这个限度不管多少每月会收取8.87美元。

企业服务有2TB的内存，每位用户每月收取11.09美元。

Storj的数据存储费是每1GB/月0.015美元，使用1 TB时，每月会花费15美元。所以不超过2GB的情况下是Dropbox更为划算（有点打脸）。

面向公司的情况，对比没有人员限制的Storj，Dropbox每用户每月11.09美元的2TB简直是在剥削。

数据下载费——0.05美元/每1 GB下载，好在Storj的云存储平台在注册1年后，可免费进行数据存储和数据下载，高达25 GB。

代币分配：吸引矿工

要说起web3.0与2.0的储存最大的不同，还是在于分布式储存是点对点的参与，让更多闲置的储存资源得以利用，其中最重要的一点就是奖励机制。

接下来说的是IPFS，星际文件系统(InterPlanetary File System)是一种内容可寻址、版本化、点对点超媒体的分布式存储、传输协议系统。其代币名字是 FIL，总共有20亿枚。分配方案为：70%作为挖矿的回报，按照6年分发一半的计算方式，前6年可以挖的FIL数量为7亿，平均每天32万个FIL。加上线性递减的方式，挖矿初期每天的区块奖励保守估计会在35万个FIL。15%作为研发费用, 6年逐步解禁；10%分配给投资者；5%预留给基金会。

FileCoin 的矿工分两种 ：1、储存矿工（存储文件），2、检索矿工（寻找文件） 。根据FileCoin 白皮书描述存储矿工提供存储空间来挖矿，可以挖到区块奖励，存储空间越大，奖励越高。而检索矿工只能接受用户的检索（寻找）文件的订单获得订单奖励，并不能获得区块奖励。

 

共识证明：构建生态

Filecoin主要的价值是借着发行代币，让矿工有钱可赚来参与整个储存的生态系建设。而在生态良好的运营中，更重要的还有共识机制这一重要的环节。

 

前面说过，FileCoin的系统中有储存矿工和检索矿工这两种。为了确保他们的工作证明，以及确定他们不是恶意破坏的攻击者，网络采用了时空证明(Proof-of-Spacetime) 和复制证明(Proof-of-Replication)。

时空证明作为存储矿工需要提交抵押品，而时空证明可以证实存储矿工是否有安份守己地存储好数据，若存储矿工做出对系统有害的事情，抵押品将会被全部或一部分被没收。复制证明则是改良了的PoS(Proof-of-Storage)。PoRep可以证明数据是被矿工存储，而且可以保证每份数据的存储都是独立的，因此可以防止女巫攻击，外源攻击和生成攻击。

 

凭借着安全性能高、成本低廉以及良好的生态，分布式储存势必会在不远的未来替代web2.0存储。这场与区块链技术结合的“完美的联姻”，一定会给我们的生活带来更多的便利，期待躺在家里通过访问网站就能看超清大片的那一天。
