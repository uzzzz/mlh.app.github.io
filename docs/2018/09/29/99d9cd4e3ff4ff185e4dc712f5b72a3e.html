fabric-ca-server的两种启动方式

  • 柚子社区

一、本地启动
 

fabric-ca-server start -b admin:adminpw



-b选项为管理员提供注册ID和密码;
如果未使用“ldap.enabled”设置启用LDAP，则必须执行此操作。
在本地目录中会创建名为fabric-ca-server-config.yaml的默认配置文件，该文件可以自定义。

二、通过docker启动：

  • 进入下面网址找到对应的fabric-ca版本  https://hub.docker.com/r/hyperledger/fabric-ca/tags/
  • 然后pull下来。
  • 新建ca的yaml文件(在GOPATH/src/github.com/hyperledger/caDemo下)

 

fabric-ca-server:
  image: hyperledger/fabric-ca:latest
  container_name: fabric-ca-server
  ports:
    - "7054:7054"
  environment:
    - FABRIC_CA_HOME=/etc/hyperledger/fabric-ca-server
  volumes:
    - "填写你的执行目录:/etc/hyperledger/fabric-ca-server"
  command: sh -c 'fabric-ca-server start -b admin:adminpw'

 

  •  执行以下命令：
docker-compose -f docker-compose-ca.yaml up
