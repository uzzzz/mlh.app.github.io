下载安装golang

乌班图的话下载安装golang的过程比较容易。使用以下命令：

sudo apt-get install golang

运行以上命令需要输入用户密码。然后等着安装完就可以了。

可以使用以下命令来确定安装成功。

go version

配置环境变量

然后我们需要建立一个go语言的工作区。并把它放到GOPATH里面。GOBIN是运行go install命令时告诉计算机应该把生成的可运行文件放到哪里。运行以下命令：

mkdir ~/workspace
mkdir ~/workspace/bin
mkdir ~/workspace/pkg
mkdir ~/workspace/src
echo 'export GOPATH="$HOME/workspace"' >> ~/.bashrc
echo 'export GOBIN="$HOME/workspace/bin"' >> ~/.bashrc
source ~/.bashrc

workspace是工作区的名字，你完全可以使用自己喜欢的名字。src用来保存源码。pkg用来保存各种包。bin用来保存可运行文件。然后运行

go env

上图我们看到GOPATH和GOBIN那里已经被修改为我们的创建的workspace文件夹以及bin文件夹。
