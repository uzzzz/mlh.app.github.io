最初开始了解区块链是《区块链：定义未来金融与紧急新格局》，在概念和前景中对区块链有了许多印象，不过那时也就是了解了解。最近想学习一下Dapp的开发，然后就开始了对去区块链的一系列学习。上过CSDN上的一节免费区块链课程，当时有点跟不上，感觉老师的逻辑也不是很清楚。所以后来我就在网上找了许多资料开始一步步的学习。

这次主要是学习了陈琳前辈的关于区块链的系列课程。通过这次的学习，我完成了对构建本地私链，创建账户，挖矿，转账，并在上面进行合约的编写和发布的学习。

在学习过程中，遇到了一些问题：

  1. miner.start() 返回null，但是挖矿任然成功。
  2. 挖矿不成功，首先需要两个交易账户，其次在新版本中需要在启动私有连的时候增加–dev.peroid -1,此时返回的任然是null，但是可以通过日志看见挖矿仍在进行中。
  3. 课程中提到的安装本地remix-ide，通过github上的项目下载npm install或则直接npm install remix-idex -g 都会报错。我找了网上的多种解决方法，升级node和npm版本等都没有解决，最后为了能在本地也能编写智能合约，下载了mist和Ethereum Wallet。目前没有发现这两个的区别，他们都能编写只能合约并发布，不过在mist中会推荐使用Ethereum Wallet
  4. 还有一个比较困扰的问题，网上的很多只能合约直接copy然后去编译总会报错，各种错误，最容易遇到的是（如下），这种的解决方法就是在函数名后面加上一个external可以避免，想要更深的了解得学习一下solidity语法。
