一、牛X到爆的游戏机制

游戏是这么玩的： 在游戏网站上，只要花费0.0047ETH（目前的价格），就可以买到一把钥匙（key），买多少把key随意。

1、一局游戏开始,系统自动按 24 小时开始倒计时。 2、玩家可以花eth 买入一个或者多个 key，key 就是分红权,key 越多，得到的分红越多。每次买Key，还有机会抽取类似刮刮乐的小奖。 3、只要有人成功购入 key 时，游戏倒计时便会被重置，即重新从 24小时开始倒计时。 4、玩家购买key 所花费的 eth 会充入总奖金池（包含刮刮乐，最终大奖等等），之前买入过 key 的玩家会从总奖金池获得分红。 5、随着游戏的进行，key 的价格会越来越高，购买一个 key 所需的eth会变多。 6、没人购买key 游戏倒计时便不会被重置，24 小时倒数完毕，游戏结束。 7、游戏结束后，假设总共募集到10万ETH，那么其中25%左右的ETH（2.5万ETH）作为大奖金池； 8、最后一个购买 key 的玩家为最终获胜者，其赢得大奖金池里 48%的 ETH（2.5万*48%），2%分配给社区基金会； 9、 大奖金池剩余的 50%所有玩家按比例分配。

还有些其他的细节，比如购买时可以选择队伍（蛇，鲸，牛，熊），获得不同的能力，有分红比例高的，有抽奖比例高的等等，就不赘述了，因为这并不重要。

五. 小白如何参与到 Fomo 3D 游戏中？

1.注意！注意！注意！先要注册Metamask钱包。

参与游戏前，必须要准备好 Metamask 钱包（https://metamask.io/）。

Chrome 、Opera浏览器都有插件可以使用。不过因为众所周知的原因，安装过程需要科学上网。

安装钱包完成后，请务必从交易所或已有 ETH 钱包中，转账一笔希望花费的ETH费用到 Metamask 钱包中来参与游戏。

2. 准备好 Metamask 后，复制链接 http://exitscam.me/project 并在浏览器中打开， 进入游戏页面。

3.选择队伍，这个队伍代表你 key 金额的分配方案，为了最大化收益请选蛇队，可以帮你获得更多的利润返还。

 

4.输入要买的Key数量，输入后可以看到右侧自动计算出，总计需花费多少数量的 ETH。

 

5.选择 Send ETH并在Metamask钱包中确认。

 

6.购买key成功后，即可享受躺赚。在右侧版面可以看到账户中持有的key数量，和当前收到的收益。

 

如果希望直接使用账户中得到的金额，再购买key，则可以点击左侧面板的「Use Vault」按钮购买。

 

 

如果希望将收益提出，则只需要在左侧面板点击「Vault」，执行提币操作。提币收到的ETH不会减少key数量，因此提币后依然根据 key 持续收到奖励。
