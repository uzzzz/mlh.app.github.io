在投资市场中，如果有一个投资者跑赢了市场，那么必定会有人亏钱，这是个很简单的道理。


那么，在一个公平且规范的投资环境中，每个投资者都可以平等的获取投资信息，谁能根据这些信息作出更好的预测，谁就是这场博弈的最终赢家。


然而，加密货币市场就好像是一个从未被开发的西部大荒野，在这个市场中，并不存在什么公平竞争——一部分人的获利必定是以牺牲另一部分人的利益为代价的。


今天，我们就来看看那些跟你在同一个赛场竞技、将你变成韭菜，并且你永远都不可能战胜的三种加密货币的投资者吧！



类型一：内幕型交易者


美国证券交易法第10b5-1条例将内幕交易定义为：掌握了非公开信息的任何一种幕后证券交易。在传统的交易市场中，内幕交易是非法的。2010年的一篇研究论文表示，内幕交易者在股市中的确能够持续获利。


然而，就在过去一年的加密货币市场中，我们就已经发现了很多引人注目的内幕交易案例，比如以下几个。


Coinbase：比特币现金（BCH）事件


2017年12月19日，比特币公司Coinbase在Twitter上宣布，将比特币现金（BCH）纳入交易所的交易范畴。但是在公布这一消息之前，比特币现金的交易量和价格就开始飙升。难怪连圈内大佬都发了这条语带讽刺的微博：


Jacob Mooallem：在比特币公司正式宣布这一消息之前，比特币现金的交易量出现飙升，巧的很啊！

3月1日，比特币公司Coinbase因涉嫌内幕交易，遭到了集体诉讼。


韩国金融监督服务（FSS）事件


没错，即便是监管机构，也会因涉嫌内幕交易被调查。韩国FSS官员事先就已经知道即将实施新的加密货币交易限制这一政策，但是，在声明正式公布之前，他们依然进行疯狂的交易。


1月18日，韩国FSS的负责人崔炯锡（Choi Hyung-sik）被证实参与了违规交易。尽管被抓了个现行，但另一位FSS官员回应说，“从技术上来说，现在还没有任何针对虚拟货币交易的道德或行为法律法规，因此，很难对崔炯锡进行相应的处罚”。


上述两个例子只是大家所熟知的案例。实际上，加密货币市场的内幕交易十分猖獗。大多数情况下，交易所宣布纳入一种新的加密货币之前，该加密货币的价格和交易量就会飙升。

类型二：巨鲸型玩家


﻿﻿

不是指《白鲸》里的那种鲸


巨鲸只是一种比喻手法，比喻那些能够调用大量资本操纵市场的投资者，也可以称作庄家。


大多数加密投资者将这些庄家看做是恶魔，他们也从来没见过真正的庄家大佬，但是他们却都说是拜这些庄家所赐，市场才会有这么大幅度的波动。


在某些情况下，这些投资者的看法并没错。最近，某个学术研究表明，加密货币交易市场中确实存在大规模的价格操纵，导致市场大幅度的波动。例如，在2013年，某一单个投资者在两个月内就将比特币的价格从150美元拉升至1000美元。就在几周前，也有一篇论文显示，有庄家在利用大量的泰达币（USDT）来操纵比特币的价格。


这些庄家大佬凭什么能够操纵市场？靠的就是以下四种方法：止损策略、杠杆玩法、拉高出货、洗牌交易。


止损策略


在加密货币交易的过程中，庄家故意压低价格，以此来触发投资者赎回止损。然后，庄家再反过来以低价买入，静待市场回暖后卖出，大赚一笔。


这种策略比较适用于交易量低、成交量小的加密货币。只要庄家手中拥有足够多的加密货币，他们就可以诱发一系列的卖出订单，来压低价格。


为了便于理解，我们来想象下面这一场景：


某加密货币当前价格为150美元，有10个比特币（BTC）的交易价格介于110美元到150美元之间；还有10个BTC订单价格介于90美元到110美元之间。


现在，庄家想将比特币的价格压到100美元以下。但是对于一些加密货币投资者来说，100美元可能就是他们的止损点。


庄家们为了压价，进行了以下操作：


1. 提交10个BTC的卖出订单，将当前价格从150美元压到110美元；

2. 保持住抛售的压力，其它普通投资者看到这种情况，就会自然地抛售持有份额；

3. 在不了解内幕的情况下，普通投资者的止损价格很快就低于100美元，很快地，会造成价格的进一步下跌；

4. 这时候，庄家再以90美元甚至更低的价格买入所有的止损订单，坐等市场回暖。

5. 市场回暖后，庄家果断卖出，大赚一笔！


杠杆玩法


这种操纵手法一般只有交易所能玩转。我们来看看Bitmex交易所是如何操纵比特币的价格的：


1. 一名投资者投入100美元，其杠杆是100倍，那么现在市场价值为1万美元，Bitmex交易所将清算价格定为9,900美元（即市场价格减去保证金）；

2. 如果价值跌至9,950美元，Bitmex交易所就会强制清算，而其市场价格仅比最初入市价格低50美元（即5%）；

3. 当市场价格达到清算价格时，Bitmex交易所就会以9,900美元的价格强制卖出；

4. 强制清算后，投资者不仅失去了全部的保证金，还要支付100倍杠杆率所涉及的巨额费用。

由于交易所很清楚触发清算的价格，他们利用机器人来控制市场价格的变动。

十分可疑的是，低成交量和横盘整理之后，往往会出现成交量和价格的大幅度上涨，而这些波动通常与Bitmex交易所的清算相吻合。


然而，需要说明的是，没有任何证据表明Bitmex交易所参与了价格操纵。但可疑的是，交易量较低的交易时段之后，交易量就会激增。当这种情况发生时，清算除了让投资者背负一大笔交易费用之外，再没有别的益处了。

谎骗交易——拉高出货


谎骗交易，即在订单成交之前故意取消当前的出价，以此向投资者发出虚假信号。让我来看看他们的获利手段：


1. 在成交量较小的时候，庄家发出一个大单买入信号，意在向市场发出看涨信号；

2. 当投资者完成了几笔交易后，庄家取消了整个买入单；

3. 价格上涨后，庄家卖出加密货币，以此获利。


当然，反过来也可以。通过大量的下卖单，庄家发出看跌的信号，诱导投资者以更低的价格卖出加密货币。


﻿﻿


洗牌交易


最后一个是洗牌交易，即庄家同时买入和卖出，这样一来，庄家就可以：


1. 人为地拉升交易量，发出看涨信号；

2. 用小额订单操来纵市场价格。


洗牌交易往往很难被人发现，因为这看起来和真实交易十分相似。


然而，在7月27日，Bitfinex交易所在比特币现金分叉期间，又神不知鬼不觉的进行了一场洗牌交易。在分叉时，所有比特币持有人都将获得同等数量的比特币现金。


为了守住比特币的保证金，Bitfinex交易所不得不对这些数字进行细分，公告如下：


当在世界统一时间戳（UTC）发生第一个分叉区块时，比特币现金将被分配到比特币钱包余额中（预计将在2017年8月1日发布）。


分配方法如下：


  • 所有比特币钱包余额都将接收到比特币现金；

  • BTC / USD的保证金多头和XXX / BTC的保证金空头将不会收到比特币现金；

  • BTC / USD的保证金空头和XXX / BTC的保证金多头将不会支付比特币现金；

  • 比特币放贷方将收到比特币现金。


由于在分叉期间要保证保证金头寸中比特币现金的净额，上述方法可能导致Bitfinex交易所在比特币现金中出现盈余或亏损。 因此，我们将以社会化分配系数的形式解决这种差异。 例如，目前，平台上的多头多于空头，导致分配系数为1.091（这意味着对于每个符合条件的比特币用户，将将获得1.091 个比特币现金）。实际系数将在分配时重新计算。


对于机构来说，这些规则也是可以玩转的。由于Bitfinex交易所不能够通过空头头寸获得分叉之后的比特币现金，因此投资者可以简单的买入10个比特币和10个空头比特币。这样一来，投资者就可以在没有任何比特币价格波动风险的情况下，得到免费的比特币现金。如果比特币价格下跌，这些空头将抵消损失；如果比特币价格飙升，利润将抵消空头头寸。


在7月27日，交易所的多头多于空头，分配系数为1.091。但是，到了8月1日的时候，分配系数降低到0.7757，如下图所示：


﻿﻿


在分叉之前，交易所创造了大量的空头头寸。 当空头大幅度增加时，价格并没有像往常一样下跌，反而上涨了。


更为可疑的是，在分叉的最后一个时间戳，空头头寸减少了24,000单。

这里的价格操纵如此明显，Bitfinex交易所不得不承认，并发布了一份洗牌交易的声明。



类型三：抬价型玩家


在介绍完内幕交易和庄家后，我们来介绍最后一种类型的投资者——抬价型玩家，即购买便宜的加密货币，人为的哄抬价格，然后以较高的价格卖出，以此获利。


﻿﻿


加密货币市场中有很多这样的玩家，这里列举其中的几个：


  • Big Pump Signal（82,184个成员）

  • VIP Signal Strategy （24,138名会员）

  • PumpKing社区（11,124名成员）

  • Crypto4Pumps（13,954名会员）

  • AltTheWay（8,350名会员）


我们来看看拉高出货的策略：


1. 当这些玩家发现了一款易于操作和销售的加密货币时，也就是说这类加密货币有强大的社区、广告潜力、小额订单和较低的交易量；

2. 这时，他们就开始秘密积攒这种加密货币，与此同时，不会影响到它的市场价格；

3. 这些玩家将价格拉升信号传递给他们的会员，这些会员则需要支付300美元以上的入会费，来获取这些信号；

4. 然后，会员将这一信号告诉那些容易上当受骗的投资者，开启第一波拉升。他们告诉这些投资者，由于“新的网站更新”或“新的合作伙伴关系公告”，加密货币的价格将会上涨；

5. 随着价格的上涨，玩家们就开始抛售手中持有的加密货币。

6.一旦加密币被卖出，他们就会立刻将信号传递给付费会员，开始出售加密货币；

7. 于是，价格开始下跌……这就像是一个曲面游戏，反应最慢的投资者就彻底输了。

“内部群”抬价对话


我之所以要介绍这类靠拉高出货赚钱的玩家，是因为他们是唯一能够持续盈利的团体，他们拥有最大的控制权和最高的影响力。因此，会员们会愿意支付300美元的入会费。而其它买入者的情况就更加糟糕，他们被错误的诱导，买入一个看似有前途、价值被低估的加密货币。这些人并不了解这背后的阴谋，因此很快就会变成韭菜。



不想当韭菜，我该怎么办？


最简单的方法就是：停止交易。你越想投机，你就越容易被利用。


投机交易是一场零和游戏，有人赢就有人输。在这个不公平的市场中，上面说的这些大玩家拥有着常人所没有的优势，普通投资者也只能是输家。


这就是为什么我认为普通投资者应该只投资指数。如果你看好加密货币市场的技术和前景，为什么还要去投机呢？ 不妨只持有整个加密货币市场的一小块份额。过去15年的股票市场已经证明了，指数可以成功的击败95％的专业投资者。


投资市场的指数最好的一点就是，它能够保证投资者在这个波动较大的市场中获利，而频繁的交易做不到这一点。


最后，我想告诉你，在这个加密货币市场中的确存在一些投资者对这个无监管的市场进行人为操纵，并不是想制造恐慌。我只是希望投资者能够了解这个自由和毫无管制的市场究竟意味着什么。在这个市场中，我们并没有受到美国证券交易委员会或任何其他制裁机构的保护。尽管这个市场中有着无限的自由，并且给你了一个快速创新的生存空间，但是，这同时也意味着所有的犯规同样也是一种公平竞争！


加密货币市场是一个充满各种辉煌和险恶的新世界！


如果你想抓住一夜暴富的机会，那么你一定要先做好准备，迎接这些不按套路出牌的不正当玩法！
