苹果开发者账号续费提示“Unable to verify mobile phone number”

  • 柚子社区

      最近发现公司的APP在App Store搜索不到了，看了下原来是开发者账号过期了，于是开始续费操作（顺便说一句，开发者账号过期之后app会被暂时下架，重新续费后就又可以搜索到了）。

    1、登录网址：https://appstoreconnect.apple.com  然后点击登录

    2、正常情况下点击Renew Membership就可以顺利进入付费界面，结果查询发现4月之后苹果账号需要添加Phone Number


    3、弹出Your Apple ID needs to be updated。其实就是添加一个电话号码，查询网上好几个解决办法，有忘记密保重新更换appid的：https://www.jianshu.com/p/8d25d194af61   最简单的方式就是添加电话号码。


 4、打开电脑中的系统偏好设置，选择icloud，进去之后选择退出原来账号，登录开发者账号appid，点击账户详细信息。


   5、点击联系信息，选则➕号。


  6、选择要添加联系信息的类型。选择电话号码，点下一步输入手机收到的验证码就可以了。然后重新登录https://appstoreconnect.apple.com，就不会再出现之前那个界面了。
