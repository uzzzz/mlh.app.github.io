安装Go语言，测试时候的报错

  • 柚子社区

声明，linux环境是centos7.4，需要安装的是1.10.3版本的go

1：需要先安装1.4版本的go语言

git clone https://github.com/golang/go.git

cd go

git branch

git checkout realease-branch.go1.4

git branch

cd src/

./all.bash

 

Build complete; skipping tests. To force tests, set GO14TESTS=1 and re-run, but expect some failures.

 

解决办法：export GO14TESTS=1

 

./all.bash

 

2：接下来安装1.10.3版本的go

wget https://studygolang.com/dl/golang/go1.10.3.linux-amd64.tar.gz

tar -xvf go1.10.3.linux-amd64.tar.gz

设置环境变量

$vim /etc/profile

添加

export GOROOT=$HOME/go

export GOPATH=$HOME/gopath

export PATH=$PATH:$GOROOT/bin

保存

esc

:wq

 

cd go/src

./all.bash
