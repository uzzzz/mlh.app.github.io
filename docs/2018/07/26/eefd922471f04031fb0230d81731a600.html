一、Node

下载地址：
NodeJS下载地址：https://nodejs.org/en/download/

NodeJs先有来个版本。分别是LTS和Current，LTS版本是平滑过度的版本（更适合大众群体），Current版本是属于全新版本。如果你的NodeJS应用使用的比较老的SDK开发，LTS版本能让你平滑升级到新版本，如果你的NdoeJS应用使用的是全新的SDK开发选择Current版本即可

下载页面可选择LTS和Current版本，然后下载对应平台的安装包即可。 

  

安装

Windows

Windows下安装NodeJS，只需要运行node-vX.X.X-x64.msi安装文件，根据安装向导，一路Next完成安装即可。

安装完成后，使用Windows键 + R，然后在运行中输入cmd，开启命令行后，在命令行输入node -v。

如果成功打印版本NodeJS安装成功，如果输入命令无法找到的提示，请将NodeJS安装目录的bin目录配置到环境变量的path中。

NPM

npm是随NodeJS一并安装的包管理工具。通过该工具对NodeJS进行模块包的安装卸载升级。

npm 常用命令行：

npm -v 查看版本号
npm config help 查看设置配置
npm install <Module Name>   安装模块 比如 npm install gulp （安装gulp模块）
npm install <Module Name> -g 全局安装模块
npm update <Module Name>  更新模块
npm search <Module Name> 搜索模块
npm uninstall <Module Name> 卸载模块
npm install npm -g 升级npm
npm list 或 npm ll 或 npm la 或 npm ls  查看所有已经安装的模块
