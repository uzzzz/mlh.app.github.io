web3j通过命令行工具新建钱包、更新钱包密码及转账

  • 柚子社区

生成一个新的以太坊钱包：

$ web3j wallet create

更新现有钱包的密码：

$ web3j wallet update <walletfile>

将以太币发送到另一个地址：

$ web3j wallet send <walletfile> 0x<address>|<ensName>
