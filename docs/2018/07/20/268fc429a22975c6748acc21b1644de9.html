Ubuntu18.04 Shadowsocks代理，谷歌相关配置

  • 柚子社区

ubuntu不像windows系统中，有shadowsocks的客户端，所以相对比较麻烦。不过可以借助shawsocks-qt5来变成图像界面。

第一步：下载shadowsocks-Qt5包进行运行。不要用网上的这三个命令

sudo add-apt-repository ppa:hzwhuang/ss-qt5

sudo apt-get update

sudo apt-get install shadowsocks-qt5

会报一些错误，而且很难解决。 所以建议去下载Github上的Shadowsocks-Qt5-3.0.1-x86_64.AppImage。网址为：https://github.com/shadowsocks/shadowsocks-qt5/releases

第二步：打开终端，运行命令

chmod a+x Shadowsocks-Qt5-x86_64.AppImage
./Shadowsocks-Qt5-x86_64.AppImage

进行配置Shadowsocks-qt5,可以点击文件选项，选择从gui-config.json导入连接进行连接，也可以选择连接选项手动输入账户密码，加密方式连接。

至此，shadowsocks方全部配置好了，注意，此时的shadowsocks在运行是因为命令在运行，一旦关闭了终端，shadowsocks就不起作用了。所以我们可以借助screen命令让它命令在后台运行，关闭了终端后shadowsocks也可以运行。

第三步：进行电脑配置，打开右上角的设置，如下图，点击网络，选择右方的网络代理，把禁用改为手动，将HTTP代理处右边的数字改为自己服务器的代理端口号。下面的FTP代理和Socks主机和我的相同即可。到此就完全结束，打开谷歌，就可以上网了耶哈哈哈哈
