使用A卡挖矿刷BIOS教程

  • 柚子社区

　　首先说明，使用A卡挖矿的话，ETH是最佳的选择，性价比最高，如果你只想挖ETH的话，建议购买A卡，如果你想挖更多币种，建议你购买N卡。使用A卡挖矿的话，想要提升A卡挖矿的算力，就必须要刷BIOS，首先要准备的工具：ATIWinFlash, Radeon BIOS Editor , WinMD5。(获取工具可以添加比特威风官方QQ群：730963933获取)。

　　首先，打开ATIWinFlash，并保存原始的BIOS，这里尽量要多保存几个。

       

　 第二，下载用作种子的BIOS 文件，可以在QQ群730963933中获取，也可以在下面的网站中下载。http://techpowerup.com/vgabios/index.php?page=1&architecture=ATI&manufacturer=Powercolor&model=HD+4870&interface=PCI-E&memSize=512。选择匹配的种子文件下载。

　　第三，打开Radeon BIOS Editor ，在Additional features界面载入下载好的种子文件。

　　第四，保存RBE文件

　　第五，退出Radeon BIOS Editor ，重新进入，在Additional features界面载入第一步保存的原始BIOS

　　第六，载入第四步保存的RBE文件，破解超频上限.注意下面几幅图的区别.

　　第七，保存更改后的BIOS

　　第八，重复第一步到第七步，制作保存多个BIOS。

　　第九，打开WinMD5,按提示将制作的几个BIOS拖进去校验.用一样的原始BIOS和一样的种子文件制作出来的几个BIOS,校验码应该是一样的。

　　第十，关闭所有应用程序和杀毒软件,打开ATIWinFlash,载入修改后并经过校验无误的BIOS。

　　第十一，点击"程序",等待几秒钟后,提示"VBIOS编程成功"。

　　第十二，按提示关机重启,卸载原来的显卡驱动,再重装。装完后,重启进CCC控制中心检查是否成功破解超频上限,然后可以进行各方面的测试。

　　如果在载入原始BIOS后,直接修改核心和显存的默认频率和超频上限的话,有时会发生装驱动时无法识别显卡的情况,所以需要使用别的显卡的RBE文件来修改核心和显存的默认频率和超频上限。刷过BIOS的显卡,在高频下未必稳定,可以多做几种BIOS,测试一下哪一种是最稳定的。另外，刷BIOS一定要注意不能刷的频率过高，有可能会将显卡刷坏，一旦在运行过程中出现花屏的线想，那很有可能是显卡刷出问题了。这一点不如N卡方便耐用。
