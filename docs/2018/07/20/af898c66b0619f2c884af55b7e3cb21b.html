fedora20 下安装gotty的步骤

  • 柚子社区

本文记录了fedora20下安装gotty的过程。

根据网上各种关于gotty的安装说明，如文：https://linux.cn/article-8445-1.html，基本上gotty的安装可以分为三部分：

(1)安装Go语言环境

(2)配置环境变量

(3)安装gotty

下面记录一下自己在安装配置过程中遇到的问题和解决方法。

由于直接从Go官网上下载相关安装包网速很慢，所以第一次选择了直接yum安装golang(在fedora20下不建议这么做，因为这样安装的golang版本太低，在后面安装gotty时会遇到问题，报找不到context)。后面直接下载了go的tar包，解压到/usr/local下。同时配置环境变量，在~/.bashrc中添加：

export GOPATH=/Home/xxx/workspace/go
export PATH=$PATH:/usr/local/go/bin

其中，GOPATH用于指定自己的go工作目录，不要和go的安装目录相同即可。

可以通过go version来查看go是否安装好。

go version

 

接下去是安装gotty。有go get直接安装和go install安装两种方式，都能成功(前提是go版本不要太低)。

先说go get。

go get -v github.com/yudai/gotty

这种方式下直接装成功可用。此时会在之前配置的工作目录的bin下有gotty。进行简单测试：

下面说第二种方法，go install安装。方法为：将github下载的东西放到之前配置的GOPATH的src路径下，进行go install即可。若安装过程中出现找不到package的时候，将对应的package移动到其要求的路径下即可，package都在src中。
