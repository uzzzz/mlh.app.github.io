Github 学生包包含很多开发者工具，比较吸引的是 DigitalOcean 的 50美元 代金券和 NameCheap 的免费 .me 域名和 Comodo 的 SSL 证书。

 

申请条件

年龄大于 13 岁并正在进行可能被授予学位或学历的课程，意味着你从初中开始就可以申请学生包了。你只需要一个学校提供的电子邮箱地址，或有效的学生证件或是任何官方入学证明。

 

GitHub学生包申请教程

一、进入申请地址：https://education.github.com/pack/ ，点击 Get your pack 

二、然后提示你要登陆注册

如果没有github账户，需要注册一下。

三、再回到要申请的地方，点蓝色框

四、填写学校的edu邮箱，如果没有的话就需要上传学生证

下面是一个申请理由的模板：

I am a junior student from China, and I used Github to share my code with my classmates and friends since I entered university.

五、等待一到两个工作日（我是秒通过）

看看你的邮件，一封名为 Powerup get! Welcome to the Student Developer Pack. 的邮件，证明你成功了！

六，打开 https://education.github.com/pack/offers ，你已经拿到优惠了！
