上次大概讲了一下代币的概念和它的价值，这次来给大家介绍一下以太坊IDE－remix，我们后续的代币就是基于remix进行开发

一. remix是什么？

简单来讲就是以太坊的开发编译环境，基于网页开发，免下载免安装，即开即用，大家先看看它长得啥样，有个大概印象。附链接: http://remix.ethereum.org

二. remix界面使用介绍

初看起来remix好像还蛮复杂，怎么这么多乱七八糟的，其实大家用到的只有三个界面，我们姑且称它们为界面ABC三兄弟，超级简单有没有

界面兄弟Ａ　－　代码界面

里面放的就是以太坊智能合约代码，用solidity语言写的，什么？不懂智能合约？什么？不懂solidity？没关系，这个系列的主题是教您发代币，拷贝复制您总会把，我们把代码写好，您到时直接拷贝不就行了。什么？您想自个写智能合约？咳咳。。。后面我们柏拉图团队会专门推出手把手教你写智能合约系列，这次咱们先发币、发币。

界面兄弟Ｂ　－　编译界面 compile

点击编译界面中的start to compile按钮，Ａ中的solidity代码就会编译成以太坊虚拟机可执行的二进制代码，源码有错误的话，下面也会将错误显示出来。ＯＫ，总结起来就两个字，《点它》。

界面兄弟Ｃ　－　运行调试界面 Run

别看上面这么多界面，又是Settings、又是Analysis什么的，其实大家用到的除了compile就是Run了，至于具体的用法，我们后面会用一个简单的智能合约教大家如何在本地部署，好了，今天就到这里，下次见;
