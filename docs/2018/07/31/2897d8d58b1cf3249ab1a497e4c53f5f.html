“链圈”是其实就是数字货币的技术极客的统称。 相对币圈各种大佬。各种富翁来言就微不足道的多了。今天就看一个区块链技术老司机如何看待币圈生存法则。

目录大纲：

  1. 币圈利润最高的三块业务
  2. 区块链存在的核心价值-互联网信任危机
  3. 熵增、熵减、技术见真知
  4. 区块链学习网站

首先来看下币圈利润最高的有三块业务：

1.数字货币交易所

代表： 火币、OK、币安。后起之秀 Fcoin。

正所谓币圈一天人间一年，短短一年过去了，数字货币交易所从未停止洗牌。

按照非小号官网2018年7月27当日统计，仅币安网的24h成交量达到了人民币972614万。注意这里的单位。按照交易所1%的手续费，日收入正常在1亿元人民币左右。而这个交易所仅仅是2017年第三季度成立的而已。

所谓币圈的传说实际上只是一种比较容易理解的说辞。换种科学的行业解释是这样的，因为行业的利润率远远高于他的门槛，也就是熵增太快。造成行业的失控的因素增加。越是前期靠的就是胆量，魄力，很多人甚至在不知不觉中已经成为了币圈千万富翁，可以行业却越发不稳定。如何降低熵值 ，华为老总任正非给出了非常好的解释：“当企业利润过高的时候，我们就需要降低企业的利润，这实际是在保护我们自己，别的公司感觉没有利润可图的时候，也就不会跟你抢占市场了，避免了竞争”。所以华为企业市场竞争的核心都是在降低生产成本，同时提供更好的服务，也因此30年来一直增长。而入今日大家看到 拼多多上市首日就超过300亿市值，快赶超京东、淘宝，再对比30年来风雨无阻的华为，大家大概就能猜到背后的逻辑了。

2.数字货币矿机/矿池 

代表：比特大陆

目前最大的矿机生产也是矿池所在地-比特大陆，早已经占据了比特币45%左右的算力，比特大陆比特币最大的生产厂商，交易所也是其下游的服务提供商。可想而之里面的利润所在。周围很多土豪老板朋友不懂投资数字货币，直接投资矿机，也都在去年行情好的时候一个个赚的盆满钵满。

3.比特币早期投资者

代表：李老师   、凌晨三点半（听说入群手续费就是一年10个比特币）

典型的网红经济，一人影响一个行业。 不得不说互联网时代带给一个人价值魅力的极限体现。笔者正是因为李老师的谈话录音，而被他独到的见解所折服，而非他拥有多少个币，入股了多少个交易所。

信息大爆炸

      

在互联网发展的第二个阶段，当网络应用设快速迭代更新新走上正轨的时候。网络层开始出现了信息大爆炸，信息不对称等现象。人人都是自媒体，也很多人赚着信息不对称的钱。这个年代的网民早已有了选择恐惧症。所有在消费过程中微商才出现，而后是拼多多，信任成本高昂，我们只好选择被动被砍，反正都是买。这背后正是互联网深深的信任危机。

互联网信任危机

人们大都无法接受主动吃苦，但是又不得不被动受罪。多么令人无奈的人生独白。

当撒谎，行骗的成本低到如此地步。加之和平年代大家由盲目拜金，导致了大部分人为了赚钱不停的再秀着自己的道德底线。武汉疫苗的事件更是突破了我们之前的所有认知，其性质完全不亚于南京大屠杀。即便如此，互联网网红大佬除了刘哥再未有其他英雄豪杰发言。大家都享受在这个人造的谎言天堂之中，被动享受这些本来不属于我们的苦难。民众的拜金更是推波助澜。 最后无非是温水煮青蛙， 而大家还都欢快的在里面享受、蹦跶，当苦难来临时便早已无力反抗 ，早已不知根本原因。

弱肉强食无可厚非，但起码我们有选择是否相信别人的权利。当我们很信任你，却被你坑的时候，那真的不能说你能力有多强，而是说明你脸皮有多厚。所以我们看待一段财富的时候，也希望大家能够思考下他背后的逻辑。这里并不期望每个人都有超强的独立思考能力，但最起码对金钱要由自己的底线和认知，要知道对方的钱来路是否正，这一点，听闻日本社会是非常好的一个代表。

三维世界中 ，能量守恒定律仍然是成立的。盲目的拜金就是纵容这个社会的失衡，最终温水煮青蛙，你就是温水里蹦跶的青蛙。

熵增、熵减、技术见真知

当所有币圈大佬抢破头皮争抢行业，使其不断熵增的时候。那些技术大大们都在为这个行业持续做出熵减的贡献，新技术，新模式共享，开放的探讨将真正引领这个行业。Fcoin的出现更是验证这一逻辑所代表的事物客观规律。 开源和技术社区永远都是这个时代我们的最好归宿。加入一个高质量的技术社群，用技术产生影响力改变时代，将是我们研发人员的未来和宿命。
