Dapp开发环境搭建(windows)

  • 柚子社区

                     

  • 安装Dapp开发环境
  1. 安装node.js

安装完毕，打开控制台窗口，检查是否成功

Node -v

  1. 安装节点仿真器

>npm install -g ganache-cli

验证安装成功

>ganache-cli

3. 安装solidity编译器

  > npm install -g solc

  >solcjs –version

   

 

  1. 安装web3

>npm install -g web3@0.20.2

.>node -p ‘require(“web3”)

  1. 安装truffle

>npm install -g truffle

>truffle.emd version

  1. 安装webpack

>npm install -g webpack@3.11.0

>webpack -v

  • 构建示例项目
  1. 新建DApp项目

mkdir demo

cd demo

  1. 用webpack模版初始化项目骨架结构

>truffle.cmd unbox webpack

  1. 安装项目依赖的NPM包

  1. 修改truffle配置

如果使用图形版的ganache，不需要修改truffle.js配置文件，否则，需要在truffle.js中，修改port为8545，因为ganache-cli在8545端口监听

  1. 启动节点

启动节点仿真器，以便部署合约并执行交易

>ganache-cli

     6. 编译合约

       > truffle.cmd compile

       

  1. 部署合约

>truffle.cmd migrate

 

  1. 启动DApp

>npm run dev

  1. 运行成功

  

  1.  如果要让别的机器访问Dapp应用

在package.json修改

  1.  保存后，重新启动Dapp 把防火墙关闭

在另一台机器的浏览器打开

访问成功
