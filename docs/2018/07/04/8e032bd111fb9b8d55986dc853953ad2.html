身份对人的重要性

目前面临的问题

      在现实世界中，我们用像驾驶执照这样的文件来证明我们的身份。这些文件证明了我们的事实，比如我们的名字，年龄，或者眼睛的颜色。但在互联网上，驾照并不存在。相反，为了证明我们的身份，我们不得不使用用户名密码系统的拼凑。
      因为我们处理的是如此多的孤立系统，通常是在日常的基础上，我们的个人信息最终会被复制并在互联网上销售，损害我们的隐私，破坏我们的安全。


Sovrin解决方案

      Sovrin是一种分布式的、全球性的自我主权认同的公共设施。“自我主权”指的是任何个人、组织或事物的终生便携式的身份。每个人都可以很好的使用这个智能身份。拥有一个自我主权的身份允许持有者以一种隐私安全的方式提供可验证的凭证。这些凭证可以代表各种各样的东西，比如机票或驾照。Sovrin身份将改变目前破碎的在线身份识别系统。Sovrin身份将降低交易成本，保护人们的个人信息，限制网络犯罪的机会，并简化从医疗保健到银行到物联网再到选民欺诈等领域的身份挑战。

 Indy Node简介

      这个代码库实现了运行节点（验证器和/或观察者）的所有功能，这些节点在分布式账本上提供了一个自我主权的身份生态系统。这是Indy的核心项目;随着时间的推移，除了indy-sdk，所有其他的indy项目可能会逐渐弱化。

Indy拥有自己的基于RBFT的分布式账本。

Indy与Sovrin的关系


     Indy代码库是独立于Sovrin，但通常与之相关。Sovrin是一个用于身份的公共工具，它是建立在这个代码库之上的。安装Sovrin包（例如，sudo apt install Sovrin）的人可以使用Sovrin的治理和信任框架与Indy验证器池进行集成。但是，可以使用一个不同的网络来使用Indy节点，使用社区所选择的任何约定。
