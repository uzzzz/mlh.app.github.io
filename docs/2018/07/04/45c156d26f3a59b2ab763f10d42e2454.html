富强资本创始人 Jack:优胜劣汰更有利于行业的发展

  • 柚子社区

2018年6月24日，是个周末，投资了MT、PNT、ONE、DAGX、BHT等知名区块链项目的Jack，被链视界临时调整访谈时间到下午后，却仍然如约而至，先给这种奋斗在最前线的区块链投资者点赞。


链视界：您的过往经历是怎样的？


Jack : 我是富强资本的创始人，富强资本成立于2007年，至今已经11年了，在过去11年里我们投资了很多个传统项目，比如创业黑马、易津资本、湘村黑猪、点我吧、九曳物流，做的很不好的是小鸣单车，一共50多个项目。


链视界：您是如何了解并进入区块链领域？


Jack : 早在上海第一台BTCPOS机在IC咖啡馆发布展示会上时，我就接触了第一个应用比特币，但那个时候我们都认为比特币是炒作和交易行为，并没有和区块链的发展联系在一起。这几年区块链的发展让我们看的越来越清晰，2017年是价值互联网的第一年，未来将有95%的资产会上链，公司行为灭亡。9.4事件之后比特币的价格降了一半，对我们来说是一个很好的机会，就正式All in区块链领域。


链视界：进入区块链领域之后您都做了哪些研究和项目？


Jack : 进来了之后我们投资了MT、PNT、FSN、RSK、DFN、ONE、GOX、LT、DAGX、BHT、FTI、FPAY等区块链项目，基本都是一级市场进入，EOS在二级市场也参与了一些。


链视界：在您投资的这些项目里，方便分享一下回报吗？


Jack : 已经退出的在5倍以上，未退出的预计综合收益5倍。今年也在寻找百倍项目，投100个或者找100个团队来投。


链视界：基本都是当前市场上非常优质的项目，您是如何发现这些项目 ？


Jack : 区块链讲究的是去中心化，分布式，我们的项目源也都来自不同的社区参与者，他们分布在全球，都是区块链里的精英，来自项目、交易所、媒体、投资机构等。我们的目标是发展108人，就是投资108将。


链视界：您选择项目的标准是什么？


Jack : 我们把项目进行分类，如存储、公链等是基础层，支付、钱包、交易所等是中间层，银行、保险、社交、商城等是应用层。我们希望在每一个层面上都有所行为，但在选择的时候会看他们有没有用户基础和区块链技术结合，并且是长期可使用的平台。


链视界：您如何看待去中心、分布式技术在当前的应用？


Jack : 我认为未来一定是去中心化的时代，尽管当前还早，但也已经有很多应用诞生了，所以我们一直都在看分布式的项目，比如去中心的新闻、视频、存储、商城等。在过去大半年里，我们在投资上也尝试了去中心投资模式，我们希望把它产品化，这也是我们为什么投资EATT的原因，因为它就是EATT的原型。


链视界：您刚才有提到EATT是否方便讲一下这个项目？


Jack : EATT.IO 是基于以太坊协议的区块链价值发掘引擎，致力于利用区块链理念建立去中心化的投资和支付平台。最终目标是建立一家全球共享社区自治的去中心化商城，目前商城开发正在持续进行中。


链视界：作为去中心化的投资平台，它的业务流程是怎样的？


Jack : 分布式投资团队或者个人提交项目，通过9个节点投票规则，通过则自动到账，通不过则失败。


链视界：既然是投资团队参与，是否涉及到配资，以及绩效考核？


Jack : 不涉及到配资，个人也可以提交，但有一定的门槛比如100W的EATT，具有多年的投资经验，或者技术方向的人才。没有严格的绩效考核，但是平台有竞赛系统，节点和后备节点之间会有替换。


链视界：这样的项目团队是怎样的？有什么样的规划？


Jack : 目前10几个人发起，以后会完全去中心。除了对项目资金支持外，我们有论坛、内部交流、联盟都可以给项目提供支持。拥有EATT的投资人意味着拥有EATT所有投资项目的权益。


链视界：您认为它可以做成的原因是什么？


Jack : 时机、布局、行业的熟悉程度、机制等。


链视界：接下来半年，您认为什么类型的项目会比较热门？


Jack : EOS、支付的一键发币、聊天社交工具、IPFS这样的储存等。


链视界：在您的业务之外，可以和我们的读者分享点什么吗？


Jack : 近期最热的是交易所大战，有人说是千所大战，大战并不可怕，这是在促进优胜劣汰，包括钱包、媒体也都将面临竞争。在优胜劣汰之后必将团队重组，这只会更有利于行业的发展。
