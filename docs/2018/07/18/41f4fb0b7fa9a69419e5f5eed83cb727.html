支付宝，微信的提现即时到账。【转账给支付宝用户服务】【企业付款】

1.应用场景

当存在用户发起提现无需审核时 ，当用户作为服务提供者向企业提供服务时，等等场景。
企业需要向用户付款，缺又不想走繁琐的审核程序，通过第三方支付平台转账接口向用户直接转账，即时到账。

2.微信【企业付款】- 付款到零钱

1.开通条件
用户商户号需要满足已经入驻90天以上并且最近30天没有不正常交易。
2.付款资金
默认使用商户号的账户余额进行支付，如果已经开通运营号，则使用运营号的余额进行支付。
3.相关说明
  1. 使用openid + appid付款
  2. 不支持付款给未实名认证的用户
  3. 同一个用户每日付款限额2万
  4. 同一个商户号每天最多出账100万，会根据风控策略调整
  5. 默认每天最多向用户支付10次，但是用户可以自己调整

tips：我有遇到过单日出账30万-50万就出现问题的，主要原因是支付频次太高，本案例仅供参考。

3.微信【企业付款】- 付款到银行卡

1.开通条件
用户商户号需要满足已经入驻90天以上并且最近30天没有不正常交易。
2.相关说明
  1. 支持17家银行
  2. T+1日到账，不能实时，一般1-3天
  3. 需要手续费，总金额的0.1%，最低1元，最高25元
  4. 每个商户号每天最多出款100万，给单个用户最多出款5万
