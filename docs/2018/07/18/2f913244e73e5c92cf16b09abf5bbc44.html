数字货币交易所 | 冰与火的六月攻坚（小结篇）

  • 柚子社区

六月的夏天来得又早又匆忙，朋友圈盛夏摄影展看了一天又一天，瓦蓝瓦蓝的天空没有一丝云彩，炽热的太阳烧烤着大地，河里的水烫手，地里的土冒烟，就连蜻蜓都只敢贴着树荫飞翔，仿佛害怕阳光晒伤了自己的翅膀。

无独有偶，数字货币交易市场也与这天气相得益彰，只不过火热中蕴含着丝丝冰凉。

6月2日，火币区块链研究院发布了《全球区块链产业全景与趋势报告（2018上半年）》，将区块链的发展划分为了三个层次和阶段：第一是层分布式通讯、传输网络；第二层是分布式账本；第三层是分布式、开放经济生态。目前，区块链发展正处于第三阶段的技术攻坚阶段。

是大Boss来了吗？跟着柒姐，一起走进区块链生态的上游——数字货币交易所，盘点和思考六月天。

 

曾经有个分布式架构师说：数字货币交易所是比特币发展到出现非定向交易后的必然产物，是超出中本聪的意料之外的。比特币的设计初期，有些类似朋友圈内使用的东西，就是点对点的，我发给你、你发给我一个token的一种方式。所以功能非常单薄和不可靠，即便是增加了POW的那种共识方式也是一种弥补而已。这种定向发送token的方式，更适用于转帐和支付。

事情的发展往往都是超出预想的、复杂的，以及人性的多面性和商业多样性，造成了目前各种coin也出现同样的问题。当比特币的认知度越来越高的时候，token之间的交换并不仅仅满足于熟人或者预先知道地址的双方转帐和支付，非定向交易的现象开始出现，恰好这时，数字货币交易所应运而生了，和传统交易市场一样，同样需要具有挂牌和撮合机制。或许可以说，数字货币交易所的诞生也是对比特币这些的coin做的另外一个“补丁”。

存在即为合理，数字货币交易所的起步虽然不早，但是发展却非常得迅猛。甚至在我们未曾熟悉的时候，这样的新闻跳入眼帘，“400亿美元一天，秒杀沪深200家上市公司总和；仅用了6个月，币安是谁？”全球排名前Top5的币安、OKEX、火币等一众领先者，以天为单位刷新着我们的认知。

# 交易所黑马 FCoin 横空出世 #

交易挖矿、持币分红、高额邀请返利、变相 ICO......日交易量超过 240 亿人民币的FCoin创造了一个奇迹，虽然不是第一个交易即挖矿，却是将 Token 网络经济玩得最溜的。FCoin身体力行得证明了：“交易深度是交易所的核心，绝对的交易深度意味着绝对的价值。”

 

传统互联网世界里，无论是电商、还是共享经济、社交媒体，竞争就是不断融资、现金补贴用户，烧成市场第一。FCoin真是将移动互联网玩烂了的“烧钱补贴抢流量”的玩法变为了“发放平台币”，越早期参与者收益越大、补贴越多，还把币安、火币等“利润回购平台币”模式直接提升为“收入分成”，而且保底 80% 的收入会分配给平台币持币者，给了早期用户极大的激励。

对于任何一个具有网络效应的商业模式，早期用户都是最有价值的，他们是内测用户，是市场传播者，是早期失败风险的承担者之一，但他们在经济收益上和后期用户几乎并无差别。通证经济的最大价值就是激励早期网络效应，Token 给了我们另一种思路，既然互联网都喜欢说羊毛出在猪身上，那为什么羊毛不能就出在羊身上？

# 黑客袭击 #

2017年韩国虚拟货币交易所首次受到黑客攻击的是Youbit，当时损失的金额为1,000亿韩元；韩国交易所第二次受到黑客攻击的是Bithumb，其当时黑客盗取了31,506名用户信息与4,981个账户信息；第三次受到黑客攻击的是前不久的Coinrail，其损失金额为400亿韩元；最近一次是6月20日，受到黑客攻击的交易所是Birhum，其已经第二次被攻击，损失金额为350亿韩元。2017年至今韩国虚拟货币交易所损失金额为1,750亿韩元，约合1.58亿美元，丢失用户、账户信息总量为36,487条。

 

 

据中央日报报道，韩国科学技术信息通信部和韩国互联网振兴院正在对Coinrail、Bithumb交易所被盗事件进行调查，黑客利用的手段初步定为邮件恶性代码。科学技术信息通信部已确认Coinrail、Bithumb交易所被盗事件发生前全体员工收到大量邮件，通过恶性代码攻击了交易所的热钱包。交易所的部分加密货币（10～20%）保存在热钱包，而大部分加密货币（80～90%）保存在断开互联网的冷钱包。密钥和钱包的安全是区块链安全1.0的中心，智能合约是区块链安全2.0的中心。目前大多数黑客时间还是使用传统攻击手段。

康奈尔大学区块链研究员Emin Gün Sirer教授发推声称，“明年将会有一场大规模利用EOS漏洞的黑客攻击，如果EOS使用其仲裁人来逆转黑客攻击，其危机将蔓延至下游。将会有涉及开发人员和区块生产商的诉讼威胁。”Emin Gün Sirer还补充到，“考虑到开发人员处理关键安全问题的方式，出现这一攻击将很可能是不可避免的。”（CCN，6月19日）

相对于传统交易所，数字货币交易所需要具备的要素更多，比如技术资源、运营管理、金融机制、口碑品牌以及社群等。如何活下来的关键是建立共识机制，不断地教育用户，不管是对投资者交易的保护，还是做好项目方的评审。交易所的根本是用户，有什么样的用户就要有相应的商业模式，每个交易所都有不同的定位，目前有规模的数字货币交易所不超过一千家。交易所的关闭和新生将会发生频繁的更替。

•  爱尔兰商业银行开始制裁该国加密货币交易所

据insidebitcoins消息，Amarach Research和Red Flag最近进行的一项研究显示，爱尔兰目前持有加密货币的人数约为12万人，约占总人口的2.5%。这一数字在四年的时间里翻了三番。然而，爱尔兰加密货币的增长势头可能面临阻碍。据称，爱尔兰各商业银行已经开始制裁该国的加密货币交易所。爱尔兰加密货币交易所们要么不得不停止业务，要么不得不从国外寻找银行合作伙伴。

•  柬埔寨BTK交易所新规：项目未获合法执照 不予上币

柬埔寨数字资产交易所(BTK比特王)6月23日在其官网发表声明称：为落实柬埔寨国家银行、证券交易委员会、国家警察总署关于肃清ICO乱象的联合申明，BTK决定对目前已在柬埔寨当地注册，但尚未取得相关主管部门颁发执照的项目，将拒绝接受上币申请，以此配合柬埔寨国家政府监管,履行各项义务。

•  日本加密货币监管机构勒令FSHO交易所关闭运营

日本加密货币监管机构（FSA）在其历史上首次拒绝了该国加密货币交易所的重新运营申请。今年3月，日本FSA对FSHO和Bitstation加密货币交易所实施了一个月的暂停令，称其操作存在违规行为。此前FSHO还曾两次被监管机构勒令整改。但根据一份报告，FSA最终还是得出结论，他们认为FSHO交易所不能满足新的监管准则，因为其系统安全远低于标准。因此勒令FSHO交易所关闭运营。

So，数字货币交易所向监管、合规进一步迈进。

•  日本加密货币交易所协会的自律方案已出炉。该机构致力于保障用户的资产和使交易价格更加透明。在该方案中，规定禁止交易新的匿名货币。像门罗币、DASH、Zcash等加密货币很有可能会退出历史舞台。自律规则方案共有100页。该方案还要求设置负责分类管理的部门和防止洗钱对策，因此大大增加了交易所的负担。部分交易所也表示不满:“这种体制已像金融商品交易法一样严格了”。日本加密货币交易所协会不断加强金融厅的业务改善和停止命令，以加强人们对市场的信任度。但是，如何真正有效进行合适的交易价格和分类管理等问题还有待解决，对加密货币用户的保护只是第一步。

•  由于韩国政府加强了反洗钱(AML)监管，韩国主要数字货币交易所都表示自愿遵守。其最大的数字货币交易所正在自愿升级反洗钱和内部控制系统。Bithumb 阻止了 11个国家使用其平台，并减少了不使用实名制的账户的取款限额。Kakao支持的Upbit 采用了汤森路透的系统来达到同样的目的。23家加密交易所也同意遵守韩国区块链行业协会制定的自律标准。据报道，银行未能达到合规指导原则。

 

目前，已有至少40余个国家发表了对ICO和加密货币交易所的观点，日本和美国等国向合规化交易所迈进，特别是美国和加拿大正在试图将ICO纳入证券法律管辖内。

 

数字货币交易所作为新生事物，尚未定型，中心化、去中心化、多中心化的各种尝试和构想正在实施中。但至少有一点可以肯定，数字货币交易所的使命之一就是让项目方认清现实，交易所设立标准，督促项目更好发展，行业还有很多机会，传统行业的人进场是件很容易的事情。Come on，欢迎来到互联网的下半场！
