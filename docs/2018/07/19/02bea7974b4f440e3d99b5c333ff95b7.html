cesium学习：cesiumjs开源三维GIS下载及离线部署

  • 柚子社区

可在线学习cesium示例和api说明，但三维gis打开网页慢，等待时间长，可以对示例和文档进行离线安装，本地查看示例

在线学习网站：https://cesiumjs.org/。

离线部署过程：1.在github上下载最新的cesium项目压缩包文件

                          1.1 在https://github.com/ 上搜索cesium

                      1.2 选择JavaScript语言的cesium包

             1.3 下载压缩包

2.利用cmd进行环境配置安装

  2.1需要下载node,一个JavaScript engine ，下载地址：https://nodejs.org/en/，默认方式安装

  2.2打开cmd，我的文件放到了e盘，命令：E:\>cd cesiummaster

  2.3 命令：E:\cesiummaster>npm install，需要等待安装2分钟。

  2.4 命令：E:\cesiummaster>node server.js ，根据提示对没有安装成功的模块重新安装，直到提示：

Cesium development server running locally.  Connect to http://localhost:8080/，说明安装成功。

3.打开离线文件,如果想修改8080端口，则打开server.js进行修改，node server.js重新启动服务。
