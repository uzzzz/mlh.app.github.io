CLEARink 荣获SID 2019 显示周“公众之选”大奖

  • 柚子社区

其展示的电子纸2.0获得大获好评

加利佛尼亚圣何塞--(美国商业资讯)--CLEARink Displays 荣幸地宣布，在刚刚过去的国际信息显示学会（Society for Information Display, SID）2019 显示周展会上荣获“公众之选”（People’s Choice Award）大奖。

SID 在其于加州圣何塞 McEnery举办的第56届国际座谈会、讲座及展会即2019显示周展会上，宣布了首届“公众之选”的获奖者，该奖项由数千与会人员票选而出，对象是展会上在未来几年内将会在美国和全球上架的新兴产品。

“我谨代表 SID 祝贺 CLEARink Displays 在显示周国际座谈会及展会上荣获首届‘公众之选’大奖，”SID 主席 Helge Seetzen 博士表示，“这是业内最高荣誉之一，获奖者是由来自全球的与会者从超过200家展商中选出的。”

CLEARink Displays 受专利保护的电泳全内反射（eTIR） 技术，是革命性的新型显示技术，具有低能耗、日光下可阅读、彩色及播放视频的特点。CLEARink 在显示周上展示的9.7寸平板显示屏以教育和其他应用的平板及PC为目标市场。该平板的高对比度和4096色恰如其分地模拟了教科书的体验。能播放达30 fps 的视频，不仅能支持教育类视频的播放，也能支持无延迟快速翻页、网页浏览、快速笔记，并在平板上模拟白板的体验。其驱动电压低于5V，能耗极低。同时227dpi的精度，可支持任何语言的细线条书写绘画，并能展示地图和图像的精致细节。

“能够获得显示周首届‘公众之选’大奖是我们的荣幸，”CLEARink Displays 的CEO，Frank Christiaens 表示，“我们非常感谢 SID 和业内与会者的投票，在过去的一年中，我们的团队取得了很大的进展，我们有最佳的反射式显示、阵容强大的投资人团队和超过三亿美金的业务预订，我们的愿景是建立最有价值的显示技术公司，解决市场的痛点，为行业和社会带来深远的影响。”

关于2019显示周

第五十六届SID国际座谈会暨展会，及2019显示周于5月12-17日在加州圣何塞会展中心举行。显示周是全球领先的行业活动，重点关注新兴的电子显示和视觉信息技术的发展、与产品的集成以及商业化。显示器周吸引了来自整个研发、工程、设计、制造、供应链、营销、销售和金融生态系统以及商业和消费者终端用户市场的与会者，它提供了无与伦比的学习机会、市场动向、采购、市场路线图，以及建立了职业和业务增长的联系。

关于SID

国际信息显示学会（Society for Information Display, SID）是显示器行业唯一的专业组织。由于专注电子显示技术和视觉信息技术的进步， SID为行业合作、交流、培训提供了独一无二的平台，同时展示了业内最新的技术和产品。SID的会员都是显示器行业技术和业务的专家，涵盖了开发、设计、制造、 应用及市场营销等方面。为了推广行业及学术方面的技术进展，同时让消费者了解显示技术的重要性，SID每年举办包括显示周在内的超过十次大型展会，让行业和学界济济一堂展示和塑造这个行业的未来。SID全球总部位于1475 S. Bascom Ave., Ste. 114, Campbell, CA95008。详情请见www.sid.org。

关于 CLEARink Displays

CLEARink 是反射式显示技术的领导者，开发了广泛应用于平板、手提电脑、可穿戴设备、物联网显示及电子货架标签（ESL）、户外标识及机动车的反射显示技术。CLEARink 的专利保护 TIR 技术是现有反射显示屏的一个绝佳替代品，它能够提供无与伦比的视频和彩色效果，而能耗仅是 LCD 显示技术的一小部分，并拥有极具竞争力的成本结构。公司成立于2012年，总部位于美国加利福尼亚州 Fremont，在深圳、温哥华和 Eindhoven 设有办公室，已从显示器行业的战略投资方处筹集到超过2500万美元的融资，其中包括德国默克公司 （道琼斯：MRK.DE）、3M（纽交所：MMM）和联想创投（0992.HK）。
