自考总结-2019年5月

  • 柚子社区

本次自考在5月17号落下了帷幕.
总结一下这次的自考吧!

操作系统

第二次复习操作系统,大到每一章都做笔记,写错题,总结知识,小到每一句话都注意,毕竟第二次考试嘛!应该更加的仔细.其中也总结了一些学习上的经验.
@ 层层递进,逐步学习.总分总的结构也适合.
这次学习的时候,先每一章进行讨论和复习,然后汇总复习,最后再进行每一章每一节每一句话的复习和把握.这样的复习方式,如果认真进行的话,真的非常的不错喔!
@ 如果想要获得知识的话,需要仔细的理解.
有的时候,遇到的问题,不懂,就会让别人给自己讲题,自己仔细的在做一遍相应类型的题,确保自己会做了.
@ 严格要求自己
有的时候,因为项目的事情会不想去讨论,但是想到别人都去,自己也就暂时放下了项目,去认真对待自考了.

数据库

这一次没有跟着大部队学习,因为上一次没有过嘛,所以有一个小的团队,四个人一起学习.
但是后来由于项目的事情,经常缺席,最后的一周有一些紧张了,所以每天自己花时间去复习的.好像这是第一次自己花时间去学习,很爽其实,因为在项目组实践了很多关于sql,数据库的东西,就觉得数据库没有这么难啦!

@ 不懂的知识,反复的去复习,加深印象,产生共鸣.
@ 自己知道自己的不足,不能盲目的去学习!
@ 多多实践比什么都重要.这次的数据库能够过,我觉得很大一部分是实践的功劳,真正的懂得了原理,就不怕了,.不然不懂得原理就会很怕!
