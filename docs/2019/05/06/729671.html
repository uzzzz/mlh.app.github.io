华为将在剑桥ARM总部附近建芯片厂，2021年投入运营

据英国媒体报道，华为计划在剑桥城外建设一家可容纳400名员工的芯片研发工厂，主要业务是开发用于宽带网络的芯片，预计将在2021年投入运营。

工厂将位于英国芯片业的中心地带，距离ARM控股公司总部仅15分钟车程。华为在英国雇佣了数千名员工，其中约120人在剑桥工作。

2018年2月该公司曾宣布，将在未来5年内，向英国投资30亿英镑。（新浪科技）

查网龄送流量限制携号转网？中国移动否认

4月30日起，中国移动悄然推出了一项查网龄送流量的活动：发送短信“2019”到10086，中国移动会赠送三个月流量，根据网龄长短、使用品牌情况不同，用户可获得1-10GB不等的免费流量。

根据实测，次日也确实收到了移动发来的“优惠到账提醒”，提示参加营销活动获赠的国内上网流量分2个月到账，第1个月返还赠送的XMB已经到账，有效期到2019年5月31日。

不过，网友们纷纷晒出薅羊毛成功的同时，也有人质疑，中国移动会通过这一优惠活动限制携号转网，甚至有人晒出了疑似携号转网受限的短信。

对此，中国移动10086客服热线回复称，此次活动只是单纯的赠送流量，并没有任何额外条件，不会限制用户携号转网。

小米公布一季度手机出货量反驳调研机构

5月2日晚间，小米集团发布公告称，2019年第一季度，公司智能手机出货量超过2750万台。

小米在公告中称，部分调研机构关于我们2019年第一季度智能手机出货量的预测数据有误、显失公允，与真实情况相去甚远。

未免投资者遭受误导，公司特此公告，根据我们的管理账目，我们于2019年第一季度智能手机出货量超过2750万台。


周鸿祎卸任北京奇安信科技有限公司董事职位

5月4日早间消息，天眼查数据显示，4月30日，360公司董事长兼CEO周鸿祎周鸿祎退出北京奇安信科技有限公司董事职位，并新增刘红锦为董事。

360公司于4月13日披露公告称，将对外转让其所持北京奇安信科技有限公司的全部股权。360公司此前是奇安信第二大股东，所持奇安信股权占其总股权的22.59%。（新浪科技）

谷歌高层对性骚扰举报者进行报复？大批员工罢工静坐以示抗议

据《每日邮报》5月2日报道，数百名谷歌员工在美国和海外的公司办公室举行静坐，抗议公司管理层对举报性骚扰的人进行报复。

这次静坐抗议是在美国东部时间上午11点开始在谷歌公司进行的。谷歌否认其公司政策是对投诉员工进行报复。

该活动是去年11月谷歌全球员工大规模罢工的策划者组织的。这场发生在11月1日的罢工是为了抗议谷歌向该公司一名被控性行为不端的高级开发人员支付的巨额款项。

iPhone在中国出货量跌30%

日前，苹果发布了2019年第二财季财报。财报显示，苹果第二财季营收为580亿美元，同比下滑5%；净利润为115.61亿美元，同比下滑16%。

中国区销售额同比下降21%，而据市场调研机构Canalys发布的关于中国智能手机市场的最新出货量统计报告显示，2019年第一季度，苹果iPhone在中国出货量同比下降了30%。

尽管围绕苹果的iPhone硬件销售的数据都是下跌，但苹果服务业数据依然在增长，从去年99亿美元上升至115亿。

根据财报数据，iPhone贡献的收入当前只占53.5%了，这个数字在去年同期涨到了61.4%，上一季度又涨到了61.7%。

而服务业创新高，收入占比从上季度的13 % 大涨至20%，成为苹果营收的第二大支柱。（钛媒体）

特斯拉向员工发布邮件，警告他们停止泄露公司信息

5月4日讯，特斯拉的安全团队向员工们发送了一封邮件，警告他们停止泄露公司信息。

CNBC获悉的这封邮件已经得到了几名特斯拉员工的证实。邮件警告称，那些“为了看到我们失败而愿意做任何事情的人”正在将特斯拉员工作为“目标”，以通过社交网络和其他方式来获取信息。

这封邮件提醒特斯拉的员工们，他们已经签署了保密协议，并发出了警告，称特斯拉将对那些不当泄露专有商业信息或违反我们一致同意的保密义务的人采取行动。这将包括通知雇佣、就损失提起索赔，甚至是刑事控告。

这封邮件的“剑锋”某种程度上指向了员工向媒体泄露信息，称1月份，一名特斯拉的员工被确认在推特上向记者们分享了机密商业信息，其中包括产品号码。

邮件还表示，最近，一个特斯拉的员工因为在社交媒体上发布了一个内部会议的电话号码而被开除了。（腾讯证券）

微软引入了远程开发扩展，使VS Code上的远程开发更容易

5月2日，微软宣布推出VS Code远程开发扩展包，使开发人员能够使用容器，远程计算机或Windows子系统Linux（WSL）作为功能齐全的开发环境。

目前，开发人员需要使用内部构建（Insiders build）进行远程开发，直到稳定版本可用为止。 内部构建是每天发布的带有最新特性和Bug修复的版本。（Linux公社）
