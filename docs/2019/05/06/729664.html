44 岁的微软下一步是什么？

这个 5 月，技术圈的盛事尤其之多，最受关注的莫过于 Google I/O 和微软 Build。一年一度的 Microsoft Build 大会将在今年 5 月 6 号到 5 月 8 号举行，行业巨头的一举一动，将影响着整个行业的发展。在大会前夕，微软官方发布公告，公布了 Azure 的更新，介绍了对 AI、混合现实、IoT、区块链等领域的技术支持。


AI


在去年的 Microsoft Build 大会上，微软表示“云为先，AI 为先”，并发布了 Project Brainwave 预览版本，并将其整合到 Microsoft Azure 机器学习服务中。Project Brainwave 部署在 FPGA 的 Intel 芯片上，使得 Azure 成为了最高效的 AI 云计算平台。在今年的 Build 大会的公告中，微软针对 Nvidia TensorRT 和 Intel nGraph 作出优化，增加 ONNX 运行时支持，更加高效地进行运算。

不仅如此，为了更好地为开发者赋能。微软在机器学习认知服务中，添加了一个叫作 Decision 的服务。这个服务包括之前发布的异常检测（Anomaly Detector API），现在又添加了个性化（Personlizer）和内容整合（Content Moderator），能够让认知服务专注于做出更好的决策。基于认知服务，微软也在开发认知搜索功能，可以让开发者无论是面对结构化数据，还是非结构化数据，无论是在使用交互式数据可视化 BI 工具，还是处理机器学习模型，都表现得更加"博学"。微软也将提供此服务的预览版本。

今年的大会中，将会包含手写体识别、表单识别及自动输入、语音服务等相关内容，敬请关注。


混合现实


前两年， AR/VR 在公众关注中，迎来了高速的初期发展，现在依然有不少消息，但大多数已经变得沉默，鲜有消息。由微软主导的 MR 情况也与之类似。 HoloLens 刚发布的时候，惊艳四方，但最后也逐渐离开了公众的视线。现在，微软将以 3500 经美元的价格 或者 99 美元/月的价格，为开发者提供 HoloLens 2 的开发版本，虽然微软仍未公布何时能够提供 HoloLens 2 的开发版本。如果你购买了开发者服务，微软将向你提供一台 HoloLens 2 的开发设备和能够在 Azure 混合现实服务中使用的 500 美金代金券。

值得关注的是，Epic Games 的首席 CEO 表示，他们将支持 HeloLens 2，虚幻引擎 4 将在 5 月底上线支持 HeloLens 2 的版本。


IoT


AIoT 也是现如今一个火热的领域，在这片领地上，怎么能少得了微软的身影。针对 IoT ，微软将推出一种新的数据库模型，叫作 Azure SQL Database Edge，它可以在运算力受限的设备上运行。让人惊讶的是，它使用了和 SQL Server 相同的核心编程环境，支持 ARM 或者 X86 的设备，并且同时支持在线和离线的设备。这让 IoT 的开发更加简单。如果你以为这样就结束了，那你将错得很严重。不仅如此，微软还推出了一种新的开方式建模语言，让 IoT 设备可以即插即用，让 IoT 设备无缝的连接在云端。

在微软的技术服务的支持下，开发者可以更加专注自己的垂直领域，做出更多低成本、理解用户的产品。相信在不久的将来，IoT 硬件会更快的发展。


区块链


早在 2015 年的时候，微软就投入了巨大的人力和物力在区块链上。去年，微软发布了 Azure 区块链平台，开发者可以在 Azure 区块链网络中建立自己的链。经过一年的发展，Azure 区块链能够给开发者提供更好的服务。

为了更好的为开发者服务，微软与摩根大通（J.P.Morgan），打造了企业级分布式账本和智能合约平台 Quorum，基于以太坊分布式账本协议开发而成，为金融服务行业提供以太坊许可链方案。不仅如此，微软还为自家的开发工具 Visual Studio Code 提供了区块链插件，更加方便创建和使用 Quorum 方案。
