蚂蚁金服总裁助理毛军华去世，终年41岁

5月1日消息，蚂蚁金服方面向腾讯《一线》确认，蚂蚁金服总裁助理毛军华，于5月1日晨间在香港逝世，终年41岁。

公开资料显示，毛军华2002年毕业于清华大学，获得经济学硕士学位，并曾获得巴黎工程师学院工商管理硕士学位。

2003年加入中金公司，主要负责对亚太区金融行业的研究。加入蚂蚁金服前，任中国国际金融有限公司研究部董事总经理。（腾讯一线）

网速1.5Gbps，华为首款5G手机海外上市：售价6600元

5月1日，华为携手运营商Sunrise正式在瑞士推出了Mate 20 X 5G版，这也是华为首款上市的5G手机，其售价是997法郎，约合人民币6600元。

对于5G手机来说，基带至关重要，而随着Mate 20 X 5G版的上市，也意味着华为已经完成了巴龙5000基带的所有测试，并且已经达到了可以商用的水平。

该款手机预计6月在国内上市，且国内售价比国外便宜。

卢浮宫将推达芬奇500周年纪念展，HTC成官方合作伙伴

近日，HTC官方宣布将以虚拟现实体验合作伙伴的身份，支持法国卢浮宫于2019年10月24日至2020年2月24日举办的达芬奇逝世500周年纪念特展。

据了解，2019年作为李奥纳多·达芬奇逝世于法国的500周年，对卢浮宫意义重大，毕竟馆内典藏了全世界数量最多的达芬奇画作。

因此，此次纪念特展从10年前便已开始筹备，其中便包括采用新的科学方法来检视卢浮宫馆藏画作。

传华为计划发布5G电视机：最快在今年

据报道，华为计划最早今年发布5G电视机。不过目前，华为没有予以置评。

根据早前消息，华为有可能会在今年4月份发布旗下首款智能电视（智能大屏）产品，其中，55英寸的屏幕供应商为京东方，65英寸的屏幕供应商则是华星光电。代工厂则是京东方收购的苏州高创。

不过现在有消息人士透露，华为电视要延迟发布了，主要原因是“软件还没有Ready”。同时该人士还表示“华为会有两条腿走路，一方面是做自有品牌，另一方面是在其他电视厂商中植入系统进行运营”。

英国法庭判处阿桑奇50周监禁 称其“违反保释规定”

当地时间4月1日上午，英国伦敦南华克皇家法院判处维基解密创始人阿桑奇50周监禁，称他因寻求和获得厄瓜多尔的政治庇护而“违反了保释规定”。

据报道，当地时间2日上午，伦敦威斯敏斯特法院还将就美国引渡阿桑奇的请求举行听证会。（海外网）

库克首谈5G iPhone：暂不考虑，需仔细评估

5月1日消息，苹果公布了新财报。财报显示，苹果公司第二财季大中华区营收为102.18亿美元，低于去年同期的130.24亿美元。

在随后的电话会议当中，库克被问及“苹果的5G策略是什么”、“什么时候会推出5G产品”等问题时，苹果CEO库克首次回应了相关问题。

他表示，5G目前还不是苹果考虑的问题。其称苹果公司将仔细评估各类新技术，然后在合适的时间和条件下尽快推出产品。

库克还补充了一下，其表示新技术的转化和应用会对成本造成一定压力，并且技术存在着不同的方向，比如DRAM和NAND，他们目前无法预测下一个新技术的方向究竟是什么，还要看多因素的发展情况而定。（快科技）

传闻下代iPhone将标配快速充电器

4月30日，据日本博客Mac Otakara报道，苹果公司将在今年晚些时候推出三款新iPhone手机，每款手机可能都会标配18W USB-C电源适配器，以及一条USB-C转Lightning快充线。

这传闻实际去年iPhone XS系列发布前就有了，甚至连18W充电头的图片都曝光出来，但结果让人大失所望：XS系列仍是5W慢充，那个18W充电头确实存在，但它是新iPad Pro的标配。

“祖传5W充电器”已经成了iPhone最大的短板之一，尽管iPhone 8开始，苹果就给自己的手机加上了快速充电支持，可在在30分钟内给iPhone充电50%。

但标配仍是5W慢充，人们不得不再花一笔钱单独购买充电头和线。这是Mac Otakara几个月来第二次分享同样的传闻，希望这次是真的。

Facebook上线"暗恋"功能

近日，Facebook在F8大会上宣布推出一项名为“暗恋（Secret Crush）”的新功能，将允许Facebook Dating用户选择多达9个他们想要表达兴趣的朋友。

该功能将只在Facebook已经推出其现有约会服务的国家提供，包括哥伦比亚、加拿大、泰国、阿根廷和墨西哥，以及今天在亚洲和南美推出的14个新国家，这其中并没有美国。（新浪财经）

Google Chrome正式版74.0.3729.131 发布

5月1日，Google发布了新版Chrome浏览器的74正式版升级，版本号提升到74.0.3729.131，为所有支持的桌面平台带来了2个重要的安全修复。

Google Chrome 74于上周正式推出，因此这一版本是主版本的第一次更新。Google发布了有关此版本中解决的安全漏洞的一些详细信息，并解释将在稍后发布更多详细信息，直到“大多数用户更新了修复程序”。（cnBeta.COM）

Fedora 30正式发布，GNOME 3.32作为默认桌面环境

Fedora 30已按照其最初的发布时间表按时发布，并且正在打包大量新功能。Fedora Workstation 30，Fedora Server 30现在正在发布。

它展示了最好的桌面环境GNOME。Linux之父、备受争议的Linus Torvalds都选择Fedora作为他的日常操作系统，这应该不足为奇。（Linux公社 ）
