实验要求：

  1. 完成Virtual box平台安装，会应用相关操作；
  2. 在virtual box虚拟平台上部署Fuel Master节点；
  3. 在virtual box虚拟平台上部署计算节点Computer；
  4. 在virtual box虚拟平台上部署控制节点Controller；
  5. 在web控制台中测试并配置各节点；

实验环境：

  1. 满足实验要求的PC端；

    |>四核|>8G|>200G|

  2. Virtual box虚拟平台；

    |IP地址| 10.20.0.1/24 | 172.16.0.1/24 |192.168.100.1/24

  3. 正常的Fuel master节点；

  4. 正常的computer和Controller节点；

  5. 使用火狐或者Chrome浏览器；

任务二：Fuel Master的安装及配置
2.1、安装环境检查
2.1.1在Virtual box中创建新的虚拟机作为Fuel节点的底层环境

2.1.2硬件配置要求
内存：4G
CPU：2核2线程
硬盘：80G
网络：
网卡 网卡1 网卡2 网卡3
IP地址 10.20.0.1/24 172.16.0.1/24 192.168.100.1/24

2.1.3、创建虚拟机

2.1.4、选择内存大小为4096MB，点击“下一步”继续安装。

2.1.5、选择“现在创建虚拟硬盘”，点击“下一步”继续安装。

2.1.6、选择要创建的硬盘的格式，点击“下一步”继续安装。


2.1.7、设置硬盘大小为80.00GB

2.1.8、勾选网络，则可以进行网络安装。

2.1.9、配置Fuel Master节点的网络环境。



2.1.10、选择镜像文件；

2.2.3、进入Virtual Box主界面，点击开启Fuel Master，

2.2.6、安装成功界面
