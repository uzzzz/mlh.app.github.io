追与求（一）知分寸，守进退

  • 柚子社区

           最近两星期都在忙着搞大事情，结果文笔也拉下来，正好五一把拉下的文笔给补上，结果写了很多都感觉不行。真是不进则退啊😂。想着上次，程序员的聊天养成手册（2）关系模型章节写得还不是很详细，就继续在写点啥，这章节的主题是【知分寸，守进退】。

           就像程序员一样，在长沙，3-4k是新手水平，代表你能完成基本的功能点了，算是程序员入门了。5-7k算是中级水平，代表着你能独立完成子功能模块，但是没法自己架构整个软件系统。8K以上代表你踏入高级的水平了，已经能够独挡一面，能独立架构整个软件的架构，能够规划拆分子模块并指导初中级程序员完成工作。每一个水平阶段所需要的技术、业务的水平自己应该去了解，为自己定制一份合理的职业规划。不然一个新手面试开8K工资，公司HR只会觉得是异想天开。感情也是如此，从不认识到最终的情侣关系，彼此会走过那些阶段，每个阶段所需的要素有那些，都应该是我们值得思考的地方。不然就会想好人卡一样，你很好只是我们不合适。这里的问题就很明显，明明没有突破交往阶段的门槛，就表白了，做出了超过目前阶段关系的举动，这是不合适的。

            在 程序员的聊天养成手册（2）关系模型这篇中讲到四阶段以及突破几段关系的三个门槛。

            四阶段：无兴趣阶段、自在相处阶段、交往阶段、感情稳定阶段。

            三门槛：注意力门槛、交往门槛、伴侣门槛。

     ​    ​   如图所示，可以分为从   陌生人  =>   朋友   =>   交往   => 伴侣，只有知道每个阶段意味着什么，我们才能做出合理的决策与规划，不至于做出不合适的举动让对方感到不适。对于这几个阶段，这里就不详细展开了，有兴趣可以看这篇文章 程序员的聊天养成手册（2）关系模型。这一篇写得时候匆匆忙忙的，所以写得不是很详细，但是大致讲述了几个阶段的关系的含义。不过关于突破阶段关系的几个门槛讲的不是很详细，不过门槛问题有点长，在后续的章节已函数公式的形式说说突破门槛的要素，今天就写个前言。

            这里首先就要说说关于女性感情的三个阶段。

            分别是：感性时期  =>  实质时期  =>  母亲时期

            感性时期：女性在二十二三岁以下时，初识人间，对人世间的一切都带着好奇，所以这个阶段的女生更在意异性的有趣度。你如果能和对方多谈谈旅行游记啊、工作经历啊、人生哲理啊、高大帅气啊  ，女生都会对你着迷，因为此时还尚年轻，不用考虑太多的未来呀、工作呀、结婚呀实质性问题。更多的是对未来的好奇以及世间的有趣。感觉、有趣、刺激、高大帅气等，在次阶段的女性眼中比重会比较大。

            实质时期：当女性在二十四五以上时，也慢慢的踏入了一个得考虑婚嫁的年级，此时会更看中能否给自己一个稳定的家，一个良好的生活质量。所以男方的经济能力、工作地位、是否有房等等实质性的能够给予一定生活稳定感的条件，在此时的女性眼中比重会比较大。

            母亲时期：这个不用多说，当女性升级为母亲的时候，注意力会从伴侣转移到自己的子嗣上。这个和主题不太相关，就不多说。

           既然知道了女性在不同时期的需求，那就应该有针对性的采取不同的策略。最要不得的就是，一些男生明明自己可能无趣、木讷、平凡甚至要求多，确在感情失败后，责备女生不懂欣赏【内在】、【老实人】甚至是责备女生【拜金】，一些女生也有同样的问题，明明自己并不突出，甚至毛病还一大堆，确在感情失败后，责备男生【肤浅】、【不成熟】等等毛病。

           虽然这无分对错，只是彼此看待问题的角度不同，从而导致矛盾的产生。但是你不能说他们的立场就一定是错的，不过违背道德和法律就另说了。但是我始终觉得，人呀，应该常常去自我反省，去找出失败的根源问题所在，不然类似的问题就回想跑马灯一样，一次又一次的重复。但同时这也是件复杂困难的事情，毕竟每个人的原生家庭、成长环境、过往的经历等等都不同，这也导致了每个人是独立的个体，你很难用一个标准去衡量所有的人。但这也是一个值得我们深思的问题。

           当然，另一个思考方向是，就想之前章节说的【提到苹果，你会想到软硬整合、高工艺，提到华为，你会想到自主研发，提到小米，你会想到性价比。所以如果中意精制，你可以入手苹果，如果你是商务、国企政要你可以选择华为，你是硬件的发烧友，你可以选择小米，每个企业都有他的品牌价值，让人在选择的时候，很明确的自己要知道选择某种品牌，让人一目了然。】每个人的品牌价值都不同的，若把自己放置在需求最强的市场，那也是一种提升价值加权认定的方法。不过这没啥好说的，就继续说说今天的问题。

           回到主题，那这和【知分寸，守进退】有什么关系呢？那就是我一直觉得无论工作上还是在感情上，自己在什么阶段，就拿多少的工资，低了对不起自己，高了，超过自身能力，自己也有非常大的鸭梨。感情上也是，如果还在无兴趣阶段，就不要盲目的请人家吃饭看电影，此时连朋友都不是，做出了超出关系的言行与举动，会让人感到不适，从而拒绝你。没到突破交往阶段的时候，就不要盲目表白，不然好人卡就在来的路上了。可是如果不做点什么，那岂不是一辈子都没有交集了？没错，确实应该做点什么，让女生注意到你，比起盲目的请客吃饭与表白，我更觉得应该花点时间去思考，对方是如何看待自己？

           所以关于相处的分寸感，就变得更为重要。只有知道自己处于哪个阶段，才能用正确的起手式开始一段良好的关系。

           耽搁了两个多星期，结果肚子里的墨水完全提不上来，今天就写点鸡汤吧😂。
