新增功能


AWS CloudFormation

在由光环新网运营的 AWS 中国（北京）区域和

由西云数据运营的 AWS 中国（宁夏）区域使用

AWS CloudFormation 在 Amazon API Gateway 中

自动创建 WebSocket API


现在，您可以使用 AWS CloudFormation 模板在 Amazon API Gateway 中构建 WebSocket API。AWS CloudFormation 提供了一种通用语言，用于在云环境中跨所有区域和账户描述和预置所有基础设施资源，从而简化在云中构建应用程序的方式。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/automate-websocket-api-creation-in-amazon-api-gateway-using-aws-/

AWS Config 

AWS Config 为由光环新网运营的 AWS 中国（北京）区域的 AWS Config 规则增加了默认限制


AWS Config 很高兴地宣布，为由光环新网运营的 AWS 中国（北京）区域的客户 AWS Config 规则增加了默认限制。现在，默认情况下，客户可以为每个地区的每个帐户创建 150 个 AWS Config 规则，在这之前，只能为每个地区的每个帐户创建 50 个 AWS Config 规则。可以在 AWS Service Limits 页面上找到 AWS 服务限制的完整列表。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/aws-config-increases-default-limit-for-aws-config-rules-in-aws-beijing/

Amazon IAM

IAM 角色现已在由光环新网运营的

AWS 中国（北京）区域面向 Amazon GameLift 提供


自今日起，借助 Amazon GameLift（一种为基于会话的多人游戏部署、操作和扩展专用游戏服务器的托管服务），您可以利用自己拥有的 IAM 角色来允许 Amazon GameLift 访问在由光环新网运营的 AWS 中国（北京）区域提供的其他服务，例如 Amazon CloudWatch 或 Amazon DynamoDB。现在，您可以更灵活地控制如何将游戏服务器进程连接到在由光环新网运营的 AWS 中国（北京）区域提供的其他 AWS 服务。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/iam-roles-now-available-for-amazon-gamelift-in-aws-china--beijin/

Amazon RDS 

Amazon RDS for Oracle 现在支持

通过 Amazon S3 集成提供数据导入和导出功能


自今日起，Amazon Relational Database (RDS) for Oracle 支持与 Amazon Simple Storage Service (S3) 集成来提供数据导入和导出功能。借助此功能，RDS Oracle 客户可以在其 RDS Oracle 数据库实例和 Amazon S3 之间轻松、高效且安全地传输数据。此功能现已在由光环新网运营的 AWS 中国（北京）区域和由西云数据运营的 AWS 中国（宁夏）区域推出。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/Amazon-RDS-for-Oracle-Now-Supports-Amazon-S3-Integration-for-Data-Ingress-and-Egress-Capabilities/

AWS CloudFormation

AWS CloudFormation 现已在由光环新网运营的

AWS 中国（北京）区域及由西云数据运营的

AWS 中国（宁夏）区域支持启动模板


现在，您可以在 AWS CloudFormation 模板中将启动模板与 Amazon EC2 Auto Scaling 组和 Amazon EC2 Spot 队列结合使用。启动模板是一项用于对您的 EC2 实例启动请求进行模板化的新功能。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/aws-cloudformation-now-supports-launch-templates/

AWS Config

AWS Config 规则现已在

由西云数据运营的 AWS 中国（宁夏）区域推出


AWS Config 现已支持在由西云数据运营的 AWS 中国（宁夏）区域使用 AWS Config 规则。AWS Config 提供预构建的 AWS Config 规则，用于评估 AWS 资源的配置。AWS Config 规则还会评估托管实例（包括 Amazon EC2 实例和在本地运行的服务器）中的软件。您可以从在启动时提供的 53 个托管 AWS Config 规则中选择，以评估 AWS 资源配置中的更改。您还可以创建自定义规则来定义资源配置的内部最佳实践和指南。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/AWS-config-rules-now-available-in-AWS-China-Ningxia-Region-Operated-by-NWCD/

AD Connector 

AD Connector 现已在

由光环新网运营的 AWS 中国（北京）区域和

由西云数据运营的 AWS 中国（宁夏）区域推出


借助 AD Connector，您可以将现有的自管式 Microsoft Active Directory (AD) 提供的身份用于兼容的 AWS 应用程序。AD Connector 是一个代理，可将 AD 请求从 AWS 应用程序定向到您的 Microsoft AD，而无需在 AWS 云服务中缓存信息。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/ad-connector-is-now-available-in-aws-china-beijing-region-operated-by-sinnet-and-aws-china-ningxia-region-operated-by-nwcd/

Amazon SQS

Amazon SQS FIFO 队列现已在

由西云数据运营的 AWS 中国（宁夏）区域推出


Amazon SQS 是一种完全托管的消息队列服务，让您可以轻松分离和扩展微服务、分布式系统和无服务器应用程序。Amazon SQS FIFO 队列可以补充 Amazon SQS 标准队列，因为它能够提供最佳的排序和“最少一次”交付功能。FIFO 队列具有与标准队列相同的功能，但提供了支持排序和恰好处理一次的额外优势。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/amazon-sqs-fifo-queues-now-available-in-eu-stockholm-region-and-china-ningxia-region-operated-by-nwcd/

 AWS IoT Device Management

使用在由光环新网运营的 AWS 中国（北京）区域可用的 AWS IoT Device Management 的新功能生成队列指标


AWS IoT Device Management 是一项服务，您可以借助该服务大规模载入、组织、监控和远程管理互联设备。在 AWS IoT Device Management 中，队列索引功能使您可以为队列中的每个设备的注册表、设备影子和连接状态编制索引，并根据这些属性的任意组合搜索设备。您可以使用队列索引功能查询哪些设备正在运行特定版本的固件，或查询哪些设备主动连接到 AWS IoT。 



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/generate-fleet-metrics-with-new-capabilities-of-aws-iot-device-management/

Amazon S3 Glacier

由光环新网运营的 AWS 中国（北京）区域和

由西云数据运营的 AWS 中国（宁夏）区域全面推出 Amazon S3 Glacier Deep Archive 存储


Amazon Web Services 全面推出 Amazon S3 Glacier Deep Archive，这是一个新的 Amazon S3 存储类，可针对一年内很少访问的长期保留数据进行安全持久的对象存储。S3 Glacier Deep Archive 的存储价格为 ￥ 0.012 /GB/月起，可实现云中最低成本存储，价格比本地磁带库或异地存档中的数据存储与维护成本还低。要开始使用 S3 Glacier Deep Archive，请访问：

https://www.amazonaws.cn/en/s3/



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/s3-glacier-deep-archive/

Amazon EC2

宣布支持挑选 Amazon EC2 计划事件的时间


今天我们宣布支持您挑选账户中 EC2 计划事件的实施时间，让您更加灵活地管理 EC2 实例。



查看更多信息请访问：

https://www.amazonaws.cn/en/new/2019/announcin-the-ability-to-pick-the-time-for-amazon-ec2-scheduled-events/
