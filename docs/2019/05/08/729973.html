小米回应，有品特殊激励需放弃年终奖：这是新激励方式试点，仅限核心主管

AWS设立中国香港数据中心


最近AWS宣布在中国开通三个由西云数据运营的Amazon CloudFront站点，这三个新站点分别位于北京、上海和宁夏中卫，将为中国客户提供更好的体验，包括更快的内容分发、更高的安全性。


其中三个站点通过专用的私有网络相互连接，并连接到光环新网运营的AWS中国（北京）区域和西云数据运营的AWS中国（宁夏）区域。Amazon CloudFront是一种快速的CDN服务，截至目前AWS在全球已有169个Amazon CloudFront服务节点。


图片来源网络


与此同时，AWS宣布AWS亚太（香港）区域正式开放。AWS亚太（香港）区域是AWS在亚太区及中国的第八个区域，其它现有区域包括北京（由光环新网运营）、孟买、宁夏（由西云数据运营）、首尔、新加坡、悉尼以及东京。


荷兰运营商KPN允许华为有限参与5G网络建设


日前KPN宣布，其核心5G设备将使用一家西方供应商，但与华为签署了一份初步协议将对其无线电和天线网络进行现代化改造。


声明中，这家荷兰运营商表示在“审查了其固定和移动网络的所有部分，并评估了哪些供应商最能实现其网络雄心”之后，在5G的推出中采用了多供应商政策。


图片来源网络


该公司与华为签约升级其无线网络的决定，可能会随着我国政府对供应商做出的任何决定而改变或取消；目前KPN没有透露哪家公司有可能赢得其核心网络合同，但指出该公司最近宣布正在与诺基亚合作开发固定业务。


该运营商表示，其供应商政策将使其能够“在考虑到对重要基础设施保护的评估不断演变的情况下，满足快速实现5G移动网络现代化的需要”。与此同时，华为的政策与英国当局本周在一份泄露的文件中概述的政策类似。在这份文件中，华为获准从事5G的一些工作，但与核心网络的敏感部分无关。


巨人网络任命聂志明为CTO 成立人工智能实验室


巨人网络日前确认，聂志明已经加入巨人网络并出任首席技术官（CTO），分管技术中台、前沿技术、AI实验室等部门，全面负责公司技术工作。同时巨人网络透露，已在内部人工智能（AI）实验室、前沿技术部等部门，这些部门都由聂志明分管。


图片来源网络


在巨人网络最新发布的财报中称，未来将对互联网领域的前沿性技术展开积极探索和布局，包括人工智能、大数据、区块链、云端游戏产品研发运营等领域。


资料显示，在加入巨人网络之前，聂志明曾任职腾讯互动娱乐事业群研发部总经理，负责公共技术平台建设、游戏产品安全保护及品质管理、事业群内部研发侧效率工具建设和知识管理等业务。加入腾讯前，聂志明先后在北京金洪恩、北京同方北美从事相关工作，在互联网技术研发与管理、游戏产品品质管理、前沿技术探索等领域拥有超过18年经验。


Windows 10五月更新最少需要32GB空间


微软Windows 10五月更新即将的到来，目前在发布预览通道已经有用户收到RTM版更新，不过临近发布，微软也偷偷修改了Windows 10五月更新的最低存储空间要求，从原先最低16GB猛涨到32GB。


图片来源网络


相比于此前的Windows 10版本，新版的Windows 10系统对存储空间要求越来越大，不管是32位还是64位版本，五月更新最低要求一律都是32GB，这可能与此前微软推出的Windows沙箱功能有很大关系。


虽然目前很多电脑和笔记本电脑预配的SSD已经普遍在128GB至512GB之间，但是对于一些老款的笔记本电脑来讲，Windows 10五月更新安装起来还是非常吃力的，就算是128GB的SSD，现在来看也很吃力。


谷歌在 Chrome 74 中试验内置模块 KV Storage


谷歌最近宣布，计划在Chrome的未来版本中交付两个新的WICG提案。其中KV Storage（键-值存储）试图提供localStorage的便利，但性能更好；其目的是利用导入映射提案将其作为内置模块的第一个示例交付，JavaScript生态系统在本地存储和缓存数据方面还存在一些缺陷。


图片来源网络


据了解，Web Incubator CG （WICG）最近将KV Storage提升为W3C IndexedDB规范的一部分，KV Storage类似于localStorage，但更现代、并在IndexedDB之上。该规范要求浏览器厂商将其作为内置模块实现，并使用IndexedDB作为其后台存储。


传苹果挖角英特尔5G工程师：负责芯片架构


一份刚被公开的电子邮件中称，今年 2 月份苹果“挖走”了英特尔负责开发调制解调器芯片的工程师乌玛山卡·斯亚咖依（Umashankar Thyagarajan）。通过查证，斯亚咖依的 LinkedIn 的确显示他目前在苹果负责芯片架构。


在这封电子邮件中，英特尔高管梅萨斯·阿梅加（Messay Amerga）和阿贝·乔什（Abhay Joshi）称，斯亚咖依在开发英特尔调制解调器芯片（用于 2018 款 iPhone）过程中扮演了关键角色。


诺基亚贝尔与中国一汽签署5G合作


日前，诺基亚贝尔宣布与中国一汽签署战略合作备忘录，联合开展智慧网联车辆、工业物联网(IIoT)、智能制造及数字化转型等方面的研发。


根据协议，诺基亚贝尔将为一汽长春生产基地部署5G试验网，用以支持自动驾驶及网联车辆应用研发。此外还将与一汽合作开发数据中心、云、大数据及5G网络切片等重要解决方案，助力一汽加快数字化转型。


有资料显示，诺基亚贝尔还将提供端到端系统集成，利用自动AGV（自动导引车辆）、AR/VR机器视觉及AI质量检测等自动化解决方案来帮助一汽推进智能制造。


W3C 发布 WebXR 规范草案，用于开发沉浸式 Web 应用程序


万维网联盟（W3C）最近发布了WebXR规范草案。WebXR Device API旨在为开发者提供用于开发沉浸式应用程序的接口，让他们通过这些接口开发出基于Web的、舒适的、有吸引力的沉浸式应用程序。


开发者可以通过WebXR API来创建XR体验，其中XR包括了增强现实（AR）、虚拟现实（VR）和最近出现的沉浸式技术。


目前很多 VR 开发人员使用 Unreal 和 Unity 来开发交互式的虚拟世界，而程序员们通常使用 C# 之类的语言来开发这些虚拟世界的底层逻辑，WebXR 构建在 WebVR 之上，并想要取代这些。


它的目标是帮助开发者使用 JavaScript 来开发 VR、AR 和其他沉浸式应用程序；这样一来，开发人员就可以开发出可以利用 Android ARCore 和 iOS ARKit 的 Web 应用程序。
