AWS 亚太（香港）区域落地后，亚马逊 CTO 撰文“揭秘”……

  • 柚子社区

4 月 25 日，Amazon Web Services, Inc. (AWS) 宣布 AWS 亚太（香港）区域正式开放。至此，AWS 在全球已有 21 个区域，提供 64 个可用区。同日，亚马逊首席技术官兼副总裁沃纳·威格尔博士在自己的个人博客上，撰文讲述了关于 AWS 亚太（香港）区域的那些事儿 ——

沃纳·威格尔博士

亚马逊首席技术官兼副总裁

沃纳·威格尔博士（Dr. Werner Vogels）现任亚马逊公司首席技术官兼副总裁，负责推动公司以客户为中心的技术愿景。作为亚马逊云计算背后的力量之一，他热衷于帮助年轻企业达到全球规模并快速发展转化为数字化的企业。

今天，我很高兴地向各位宣布 AWS 亚太（香港）区域正式开放。即日起，AWS 客户可利用全新的 AWS 亚太（香港）区域，为香港的终端用户提供低延迟的服务，并满足任何数据本地化需求。

AWS 亚太（香港）区域是 AWS 在亚太地区及中国大陆的第八个活跃的 AWS 区域，其它现有区域包括北京、孟买、宁夏、首尔、新加坡、悉尼和东京。随着 AWS 亚太（香港）区域正式投入运营，AWS 现已在全球 21 个地理区域内运营着 64 个可用区，并已宣布计划在巴林、开普敦、雅加达和米兰增加 4 个 AWS 区域和 12 个可用区 。


这一新区域的成立为我们在香港特别行政区的持续投入锦上添花。AWS 于 2008 年在香港推出首个 Amazon CloudFront 站点，使客户能够以低延迟向终端用户提供内容。此后，AWS 在香港又增加了两个站点，并开设了首个 AWS Direct Connect 接入点。


2013 年，AWS 在香港设立办公室。如今，该办公室设立了广泛的职能部门，包括业务拓展、解决方案架构、专业服务、技术支持、技术布道、基础设施工程和业务运营。为了助力香港下一代企业茁壮成长，AWS 还为香港的初创公司提供支持。


2013 年，AWS 推出了全球性的 AWS Activate 计划，为香港的初创公司提供专业指导，与 AWS 专家进行一对一的讨论，以及在线培训、自主进度动手实验室、AWS Support、精选第三方公司提供的特色促销产品服务以及高达 10 万美元的 AWS 促销积分等无偿服务。


香港的企业正将其关键任务工作负载迁移到 AWS，以推动成本节约，加快创新和上市步伐。这些企业客户包括：嘉里物流、SC Digital Solutions、Kopi Incorporation、偌亚奥国际集团、香港医思医疗集团、Wiz Connected 等。


嘉里物流

自 2015 年以来，嘉里物流一直在 AWS 上运行。作为服务于国际品牌的全球主要物流服务提供商，该公司使用 AWS 全球网络与 53 个国家和地区进行连接。目前，嘉里物流在全球范围内管理着 6,000 万平方英尺的土地及物流设施。


嘉里物流集团信息技术总监 Wilson Lee 表示：

AWS 为我们提供可靠、灵活的服务，帮助我们通过技术创新真正实现业务目标。这使我们能够以最好的状态为客户提供服务，反过来又促进了集团业务的扩张和长期增长。通过使用 AWS，我们成功将集中式的组织架构转变为多区域、多环境架构，为客户创造了更大价值。我们很高兴能够使用 AWS 亚太（香港）区域服务，我们期待使用 AWS 产品进行更多尝试，以便为我们的终端用户提供更多服务。


SC Digital Solutions

SC Digital Solutions 是由渣打银行（Standard Chartered Bank）、香港通讯科技公司电讯盈科（PCCW）、香港电讯（Hong Kong Telecommunications）和中国旅游服务提供商携程旗下金融科技子公司携程金融（Ctrip Finance）联合成立的一家合资企业。SC Digital Solutions 获得了香港授予的首批三张虚拟银行牌照之一，并且正在利用 AWS 服务，为客户提供一个全面的云原生银行业务平台。


渣打银行虚拟银行首席信息官 Andrew Farmer 表示：

AWS 是一家致力于为客户提供安全、可靠和灵活的服务的知名公司，提供全面的安全性和合规性工具，进一步帮助客户降低风险。在内部，AWS 自助服务工具帮助我们的开发运营（DevOps）和工程团队增强快速创新的能力。这反过来又帮助我们在银行内建立起良好的创业文化。AWS 提供了一个云原生生态系统，通过开放的 API 与同类最佳的第三方产品相集成。AWS 拥有前沿创新的传统。


Kopi Incorporation

Kopi Incorporation 成立于 2009 年，总部设在香港，该公司使零售、安保和物业管理公司等客户能够对其场所内的流动情况进行监控，使客户能够进行分析并获取洞察，从而为制定决策（如零售货架摆放位置）提供信息，最大限度地提高销量和收益。该公司使用 Amazon EC2 运行 Redis 数据存储及其管理和基本分析应用程序。


Kopi Incorporation 总监 Samuel Au 表示：

AWS 的可扩展性使我们能够获得诸如体育场馆、其它音乐场馆和博物馆等领域的潜在客户。我们可以为音乐会等活动期间的需求高峰提供支持，并随着需求的减少而自动缩减。


偌亚奥国际集团

（Royale International Group）

自 1980 年以来，总部位于香港的偌亚奥国际集团（Royale International Group）一直提供全球快递服务。该公司将全球网络与本地业务相结合，为客户提供安全的端到端交付解决方案。通过 AWS 的可扩展性和简化管理，该集团在香港 7-11 便利店的新快递服务实现大幅增长，作为在线购物的收取点，这些便利店越来越受到欢迎。该公司正在利用 Amazon API Gateway 来确保 7-11 便利店的经理、在线商家和快递员掌握有关其货件的最新信息。


偌亚奥国际集团项目经理 Steven Kwok 表示：

在 AWS 上运行平台的前六个月，我们实现了 300% 的业务增长。我们现在可以将消费者从各个商家购买的商品运送到香港的 370 家 7-11 便利店。


香港医思医疗集团（UMH）

香港医思医疗集团（UMH）提供涉及医学美容、健康检查服务、牙科、足科、整形外科和中医药等领域的一站式解决方案。通过在 AWS 上运行的 SAP S/4HANA Finance，UMH 可以在以往一半的时间内生成关键财务报告，同时为高管提供公司业务绩效的实时洞察。


香港医思医疗集团副首席信息官 Michael Ngai 表示：

在 AWS 上部署 SAP S/4HANA Finance 之后，我们生成第一季度报告的速度比旧财务报告系统（主要是手动生成）提高了 50%。


WiZ Connected

WiZ Connected 由在香港的法国企业家创立，采用自有品牌销售基于无线网络（Wi-Fi）的产品，同时将其物联网平台授权给其它电气和照明供应商。WiZ Connected 联合标签品牌集成和部署了全球超过 25 个国家的、数百种室内和室外照明设备及电气设备。


WiZ Connected 的云端运营经理 Gerard Leung 表示：

自 2016 年我们的物联网平台成立以来，我们一直将 AWS 技术作为坚实的云支柱。我们为数千名日常用户和数十万台互联设备提供服务，我们的平台经证明是稳健、可靠和反应迅速的。同时，AWS 服务的灵活性帮助我们将运营成本降至最低，使我们以前所未有的速度，轻松完成基础设施的横向和纵向拓展。随着越来越多服务的面世，特别是即将推出的 AWS 亚太（香港）区域，我们对 AWS 所带来的机遇以及为我们在亚洲快速发展提供的支持感到无比振奋。”


香港大学专业进修学院

香港大学于 1957 年 5 月成立校外课程部（Extra-Mural Studies Department）作为其教学部门的延伸，该部门后发展为香港大学专业进修学院（HKU SPACE）。自创校以来，学院所提供的专业及持续教育课程在香港及中国内地的报读人数已超过 270 万。该学院表示，在 2018 年已有 125 名学生报读云计算架构基础证书课程——香港首个为应对日益增长的云计算专业人员需求而开设的云计算课程。


香港大学专业进修学院院长李经文教授表示：

我们的目标是帮助学生在当今瞬息万变的世界做好职业定位，并为未来发展作好准备。我们预计，学生人数将在未来十二个月翻一番，届时将达到 250 人。同时，专业进修学院也正致力于在今年成为 AWS APN 培训合作伙伴，将学生带到真实的商业社区中去，并将课程内容扩展至人工智能（AI）、机器学习（ML）和大数据等领域。我们希望与 AWS 共同推动香港人才利用互联网经济带来的新机遇。


数码港

数码港是一个云集超过 1,300 家科技公司的创新数码社群，由香港政府全资拥有的香港数码港管理有限公司管理。数码港的愿景是推动数码科技发展，为香港缔造崭新的经济动力。数码港一直致力于帮助青年、初创公司和企业家与策略伙伴建立联系，以推进其业务的发展。


数码港主席林家礼博士表示：

香港一直是亚太区域内创新和科技中心的领头羊。数码港致力于培育初创公司，并协助中小企业和大型企业走在创新和科技的最前沿。AWS 亚太（香港）区域的开放将使不同规模和种类的企业都能利用 AWS 的一流数据基础设施来推动其数字化转型之旅。数码港很高兴能够继续与 AWS 密切合作，透过 AWS Educate 为香港培育更多云计算和数据分析的专业人才。全新的 AWS 亚太（香港）区域将加速香港成为亚洲领先的智慧城市的步伐，助力企业抓住新经济带来的无数机遇。

AWS 亚太（香港）区域现已正式开放。我们很高兴能够提供从基础技术（如计算、存储和网络）到更高级的服务（如容器和无服务器计算）等的全面的服务组合。我们期待未来继续扩大这一业务组合，为客户提供更多服务。
