跨境说加入 APN 培训合作伙伴计划，今日开启首期课程！

  • 柚子社区


4 月 26 日上午，琴澳金融科技大讲堂暨第一期金融科技主题青年新锐训练营顺利开展，活动由横琴新区管理委员会金融服务局主办，珠海横琴跨境说网络科技有限公司承办。


本期“跨境说金融科技主题青年新锐训练营”旨在面向珠澳金融、科技新锐，为学员提供顶级的学习资源和深度知识，让学员有机会了解最佳实践，帮助学员提升在行业应用中的理论和实践水平，同时推进产学研深度结合。学员可在训练营的帮助下准备 AWS 认证考试，通过业界认可的证书来验证技术水平。

开幕式上，横琴新区管理委员会金融服务局新兴金融发展部部长张梦如女士简要介绍了横琴金融的发展情况和人才政策，希望借由此次训练营，吸引很多港澳青年及粤港澳大湾区学员前来参与，促进横琴新区在金融方面的高效发展。同时，横琴新区也大力支持 AWS 携手跨境说利用横琴的优质资源提供更专业、更全面、更与时俱进的人才培训服务活动，推动金融、科技、产业深度融合。


珠海横琴跨境说网络科技有限公司（以下简称“跨境说”）已加入 AWS 合作伙伴网络（以下简称“APN”）培训合作伙伴计划，该计划使 AWS 授权讲师能够在授权范围内提供 AWS 开发的培训。同时，跨境说打造的“第一期金融科技主题青年新锐训练营”圆满结束，珠海市横琴新区管理委员会金融服务局局长池腾辉、横琴创新金融研修院执行院长唐士奇、AWS 大中华区培训与认证总监张涛和跨境说 CEO 周运贤出席仪式现场，并亲临体验第一期金融科技主题青年新锐训练营。

揭牌仪式于 26 日上午 9 时 30 分在横琴·澳门青年创业谷举行，仪式旨在宣布跨境说正式加入成为 APN 培训合作伙伴。同时，跨境说作为 AWS 大中华区人才培训的 APN 培训合作伙伴，为 AWS 的培训与认证业务提供支持。

AWS 合作伙伴网络计划是 AWS 推出的一项全球性合作伙伴计划，通过提供业务、技术、营销和上市支持，致力于帮助 APN 合作伙伴构建基于 AWS 的成功业务或解决方案。为保证 APN 合作伙伴整体水平， AWS 建立了一套严格的筛选条件体系和考察流程，获得成员资格必须满足相应的技术标准和准入门槛，并具备卓越的核心团队实力和良好的 IT 建设基础。


跨境说经过历时半年的严格考察审核，最终凭借天然的业务契合度、先进的技术水平，扎实的团队基础、丰富的技术成果以及与时俱进的管理理念，在激烈的竞争中脱颖而出，成为位于粤港澳大湾区的 APN 培训合作伙伴。此次揭牌仪式，既是对跨境说整体水平的肯定，也是跨境说迈入新征程的重要起点，其重要性不言而喻。


训练营特邀来自 AWS 大中华区培训与认证部的 AWS 资深技术讲师——赵恩子为总顾问兼学术导师，讲师赵恩子获得了多项 AWS 专业认证，拥有十余年的 IT 行业从业经验，包括多年的 IT 大型系统架构师及顾问经验，曾为诸多世界五百强客户设计大型网络及 IT 基础架构，参与过众多不同行业的客户标准化定制，对企业网络及数据中心的架构规范、设计、建设、运维及管理有深刻的理解与认知。


另有珠海市横琴新区管理委员会金融服务局官方成员，跨境说 CEO 周运贤领衔的跨境说内部技术团队和外部高校导师团队参与，跨境说内部技术和讲师团队，作为 AWS 云计算技术讲师，共同为全世界提供云服务，而高校导师团队成员则均来自澳门科技大学、中山大学等国内知名高校。

本期训练营的课程体系由 AWS 和跨境说 AWS 主题专家联合设计，课程内容将围绕新一轮科技革命与数字经济、金融科技、AWS Business Essentials 和 AWS Technical Essentials 四个模块展开，分为理论知识、案例剖析、实战操作三级。理论知识主打基础理论知识梳理，案例剖析则帮助参加培训的人员深层剖析行业应用知识，而实践课程由训练营基于真实世界场景精心设计，力争贴近产业一线需求，提高参加培训人才的实践能力。


在授课模式上，训练营采取线上+线下、横琴和澳门两地授课的模式，初期将在横琴·澳门青年创意谷 17 栋运营，于本年末扩展至横琴·澳门未来中心运营。横琴·澳门未来中心将不仅为横琴新区，也为澳门地区、珠海地区、合作区域提供高素质人才，承载着促进区域发展和新兴产业发展人才培育储备的重任。
