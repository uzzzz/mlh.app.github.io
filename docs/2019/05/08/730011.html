AWS 在线研讨会

AWS 在线研讨会是常年举办的系列线上技术课程，话题涵盖：快速上手实操、前沿技术话题、行业技术应用、核心服务详解、培训与认证等。

下周将与大家分享的是

Just Do IoT ——

AWS IoT 的服务生态和实践架构分享

今天，物联网的发展很快，体量迅速增长，各种产业都在拥抱物联网，推动物联网的应用和落地，然而整个物联网产业链也足够碎片化、复杂化。物联网的发展不只是依赖一个消息代理的服务终端，要想在物联网时代浪潮中存活，需要与新一代信息技术的融合和突破，其中，物联网、人工智能、大数据、云计算、边缘计算、区块链等技术成为其中主要的通用、赋能技术。


我们将在本次介绍 AWS IoT 如何给物联网产业赋能，了解 AWS IoT 的服务生态，并且通过 Demo 演示和典型场景的架构分析，了解 AWS IoT 平台的服务与集成，为您的物联网应用部署和创新“加速度”。

学习要点

1. AWS IoT 服务架构综述 —— 如何进行设备接入、消息处理和与消息推送/数据库/存储服务的集成；


2. Demo 演示 —— 利用上述 AWS IoT 服务搭建用于温度控制的物联网系统架构；


3. 经典 IoT 场景架构分析。
