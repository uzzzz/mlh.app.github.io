创响全球峰会

我们将邀请 AWS 全球投资机构业务拓展总监，海内外头部创业公司 CXO 以及知名跨国公司代表在“全球竞争环境下的中国创业生态”、“中国创业公司出海”以及“教育科技创新”等热点话题上展开深入探讨。同时，AWS 出海加速营项目将在大会上闪亮揭幕，期待您的参与！


腾云出海分会场

这些年来，AWS 帮助了许多优秀的初创企业（例如 Apus、猎豹移动、YeahMobi 等等）成功拓展海外业务。此次会议 AWS 邀请了在广告、电商、社交、游戏等热门出海行业的优秀创业公司客户代表现身说法，分享他们在这些行业的出海经验。行业落地干货满满，精彩内容不容错过！


科技赋能教育创新分会场

我们特别邀请来自 AWS 教育行业的优秀客户代表，分享基于 AWS 的人工智能、大数据等新技术在教育领域的创新应用。来自海外的 EdVentures 创始人、教育界资深投资人也将莅临现场，分享全球教育初创企业的成功经验和行业投资趋势。


作为 AWS EdStart 教育初创加速计划全球路演系列之北京站，本活动还将邀请 6 家本土优秀教育科技初创公司分享其创新产品理念和成长历程。

AWS EdStart 教育初创计划是一个全球化的社区，致力于聚集众多志同道合的伙伴，探讨解决教育行业的复杂问题。这将是您与教育科技行业初创者及投资者进行面对面交流沟通的绝佳平台，我们诚邀您一起探索教育科技初创公司成长之路，解析行业投资热点，分享创新教育产品背后的故事。
