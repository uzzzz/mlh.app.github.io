AWS 到底有什么“魔力”，引无数英雄竞折腰？

AWS 不仅能够帮助游戏开发商在开发阶段提升创新力，还能够提供大型多用户游戏基础架构的管理能力、可扩展能力以及实时的在线服务运营能力；当然，还同时集成了语音和大规模机器学习能力。

重磅嘉宾

孙国良

2013 年加入网易游戏，曾负责《天谕》、《猎魂觉醒》、《楚留香》等游戏项目运维工作，之后负责游戏出海对接公有云的运维架构以及混合云管理平台设计，专注于混合云上业务解决方案和最佳实践的沉淀积累。

★ 精彩抢先看：《网易游戏海外 AWS 实践分享》

AWS 作为网易游戏海外上云的方案之一，我们在《阴阳师》、《荒野行动》、《第五人格》、《明日之后》等多款游戏出海运营过程中积累了实践经验，希望与大家一起探讨如何利用 AWS 全球基础设施构建全球通服游戏架构，以及在这过程中遇到的技术问题和做出的优化。同时，也为大家介绍准备引入的预研新技术对游戏业务的进一步优化。


王磊

珠海沙盒网络技术合伙人及技术总监。负责珠海沙盒网络研发团队的管理、公司游戏和应用的架构、研发及运维等等。沙盒网络成立于 2015 年，公司致力于打造全球范围内的在线游戏创作平台，旨在帮助普通玩家将创意转化为游戏。此前，沙盒网络以 Blockman 系列为代表的产品在全球范围内取得了玩家的广泛青睐，目前拥有近 5,000 万注册用户。

★ 精彩抢先看：《一个小游戏平台架构的演化》

在这个演讲中，将为各位介绍沙盒网络的小游戏平台架构的演化过程以及背后的原因和思考，并与大家一起讨论小游戏平台是如何充分利用 AWS 的云服务获得快速发展的。最后，我们将展望下一步如何通过利用 AWS 的云服务获得进一步发展。



张愚生

现任金科系统架构师，负责游戏中台系统、广告系统架构设计，多年的服务端研发经验。目前为金科游戏平台技术负责人。

★ 精彩抢先看：《AWS 的云原生游戏中台系统架构》

近年来，“云原生”的概念和相关技术越来越多地受到业界的关注，但又应该如何落地到实际的游戏架构中去呢？我们将与大家分享，金科文化如何在 AWS 的平台上为《会说话的汤姆猫》游戏产品家族打造一个云原生的游戏中台系统架构，内容涵盖前端接入、数据传输、数据处理和分析、国内外数据同步等层面的设计指标与架构思路，当然，还有在这个过程中我们所“趟过的坑”，以及对未来架构的展望，都将一一分享探讨。


姚光华

复旦大学硕士，现任声网 Agora 游戏行业负责人，负责游戏行业相关解决方案的产品架构师工作，专注于声网 Agora 在泛娱乐领域的前沿应用与方案设计，在游戏直播、云游戏领域有丰富产品经验。

★ 精彩抢先看：《下一代游戏沟通工具》

实时音视频功能几乎已经成为了在线多人游戏的标配，从强竞技类 MOBA 到开放世界 FPS，玩家已经不能接受低效的 IM 沟通。海内外各大游戏社区的成功也直接印证了游戏玩家的社交需求非常旺盛。根据经验和数据，游戏内的社交功能体验的好坏可以直接影响玩家的留存和粘性，如何做好游戏沟通工具，如何跳出现有沟通方式直接拥抱下一代平台，是一个值得思考的课题。
