All-In！华夏航空全面采用 AWS 构建云基础架构

  • 柚子社区


2019 年 4 月 10 日，亚马逊通技术服务（北京）有限公司（以下简称 “AWS” ）宣布华夏航空股份有限公司（以下简称“华夏航空”）选择由西云数据运营的 AWS 中国（宁夏）区域构建其云基础架构。

华夏航空于 2006 年正式成立，在贵阳、重庆、大连、呼和浩特、西安建立了五个运营基地。公司一直致力于编织中国立体交通体系中最薄弱的部分 - 支线航空网络。目前，公司机队规模达 45 架，飞往全国 100 多个航点，与国内主流干线航空公司、政府、机场通力合作，为中国新兴城市居民带来更经济便捷的交通选择，也为区域经济带来无限生机。 


作为中国第一家专门从事支线航空客货运输的航空公司，华夏航空在 2018 年将中转 2.0 系统、营销、内部管理、人力资源系统和一些业务系统搬到云上，实现了系统的高可用、高可靠以及成本的显著降低。有鉴于此，华夏航空决定到 2020 年将业务系统全面迁移上云，涵盖核心业务运营与保障、飞行管理、安保管理以及周边的业务管理系统等，成为国内率先采用全云策略的航空公司之一。

中转 2.0 为华夏航空独有的机票创新销售方式，以灵活整合多航段行程，通过地服提供便捷中转服务，提升用户体验，并通过 OTA（在线旅行社）给用户接近直飞的便捷购买方式和票价优惠。


随着中转集票业务增长，之前的虚拟化环境部署因硬件设备老旧以及网络出口带宽不足，严重影响 OTA 的业务对接成功率和客户购票体验。采用 AWS 云后，OTA 的日均访问量提升了 43.8%，日均流量增长了 22.3%，云服务稳定性得到了质的提升，有效支撑了中转业务在春运期间的销售增量，实现了系统零故障。


跟以往 IDC 租用方式相比，采用 AWS 云服务租用的方式，基本上每一个业务上云的成本都是降低的，有的可以节约大概 50% 以上，大幅节约了项目建设与支撑成本。


华夏航空首席信息官蒋涵如表示：

“AWS 的技术领先性、高可靠性、创新的思维，以及客户至尚的原则，让我们进一步确信了将所有业务迁移上云的可行性，也确定了用云平台代替我们的基础设施建设，用云技术来作为数字化转型的工具。2018 年进行管理平台迁移的经验，让我们有信心在 2019 年将公司最核心的运行和保障业务，也就是我们的业务引擎迁上云。”

AWS 全球副总裁、大中华区执行董事容永康补充道：

“云计算为航空公司提供了高扩展的基础架构，提供了高可用性和业务创新机会。AWS 非常高兴帮助华夏航空充分发挥了云计算的势能，用云技术取代传统 IT，改造和优化整个系统，拓展业务成长机会。我们期待着与拥有创新基因的华夏航空一起，运用全云策略，打造航空业数字转型的经典案例。”
