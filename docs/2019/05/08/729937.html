倒计时 4 天！年度开发者盛会 Unite Shanghai 2019 全日程揭晓（附表）

  • 柚子社区


技术沉淀，华丽蜕变。跨越十四个春秋，Unity 在不断迭代的过程中已成为全球知名的游戏开发引擎，于此，其不仅可以为游戏开发者、发行商们带来底层工具的支持，更能够帮助各类的创作者、艺术家们实现各行各业高效、低成本的技术解决方案。然而技术苍穹之下，意味着学无止境。对于 Unity 开发者而言，想要快速了解 Unity 的最新技术知识与使用技巧、获取各种爆款游戏与应用研发经验与教训最为便捷的渠道之一无疑是每年最具规模且备受期待的 Unite 大会。作为已有 12 年历史的 Unite 大会，现已成为游戏、VR/AR、MR 等行业中最具权威性和影响力的活动。

如今，Unite Shanghai 2019 即将于2019年5月10日-12日强势来袭，本次大会依然将会秉持「开发大众化、解决疑难问题、助力开发者获得成功」三大宗旨，邀请来自业界顶尖的开发者、技术专家、艺术家、美术师和技术引领者，在为期 3 天的日程中为全球的开发者带来超过 50 小时、100+ 场精彩的技术和行业前瞻性演讲，并在展现最新的 Unity 成就的同时，还将共同探讨 Made With Unity 的高品质、跨平台的交互式内容以及探索基于 Unity 打造更多的可能性。

日前，大会主办方放出了大会全日程，接下来，将为你揭晓本届 Unite 的最强阵容与最精彩的内容呈现。


从重磅嘉宾齐聚的 Keynote 一窥 Unity 开发者生态系统全景


功能强大的游戏引擎 Unity，虽然得以让开发者们从 Android、iOS 等平台的原生开发中解脱出来，但是在制作 RPG、STG、FPS、SLG 等类型的游戏过程中，开发者还将面临高品质内容的打造、性能优化、常用工具上手等多重挑战。为此，在大会开幕的第一天，Unity 将率领其官方的王牌讲师团队针对具备初级 Unity 和 C# 开发经验的开发者们，带来性能优化、最新模块化设计、面向数据编程等多维度的实战教学。

而在为期近一天的 Training Day 结束之后，最值得关注的 Keynote 将会隆重开启。现场，Unity CEO John Riccitiello、Unity CMO Clive Downie、Unity大中华区总经理兼全球副总裁张俊波、Unity韩国总经理 Suzie Kim、腾讯天美J3工作室技术副总监郭智、腾讯云战略合作部总监梁璐、腾讯云战略合作部总经理庆雪辉、Samsung高级工程师 Joonyong Park、OtherSide Entertainment图形工程师 Elizabeth LeGros、Unity北美区技术总监 John Elliot等重磅级嘉宾将带来 Unity 最新的技术与工具，解读在游戏、汽车制造、影视动画和建筑不同行业中 Unity 的未来发展趋势。


探索爆款游戏的诞生，领略 Unity 在多重行业的技术魅力


紧接着，在第二天的 Unite Shanghai 2019 大会上，Unity 将为我们呈现四大干货满满的技术专场，从 Unity 高清渲染管线、Scriptable Render Pipeline（SRP）、Unity Ting Mode 、Timeline、次时代地形工具、面向数据的技术堆栈（DOTS）等必备技能出发，为参会者全面解读 Unity 最新技术的应用难点，并详解打造高品质、次时代场景的游戏和影视作品的过程，以助力开发者可以快速有效地解决新工具上手问题。与此同时，来自业界的一线技术专家还将带来最为热门的《胡闹厨房2》、《Crow: The Legend》、《异教徒》、《FGO》、《头号玩家》、《不可思议之梦蝶》、《球球大作战》等案例制作经验分享。

1、技术专场1——Unity 制霸影视动画、汽车制造、游戏多行业

2、技术专场 2—— 快速掌握 Unity 技术革新，为用户体验升级

3、技术专场3——Unity 与 VR/AR 结合的最佳案例

4、技术专场4——爆款游戏的优化升级

5、教育峰会——新技术驱动下的教育实践与人才储备

除此之外，Unity 还将于 5 月 11 日举行为期半天的教育峰会。现场，Unity 全球高级教育经理Ellen Flaherty、Unity 大中华区教育总经理蔡美玲，将带来焕然一新的 Unity 教育新品， 并分享 Unity 为人才、教育等方面支持的最新进展。同时，各大高校的技术专家们还将深入探讨在新技术驱动下的教育实践。


人工智能、VR/AR、云计算等创新技术与 Unity 的融合与碰撞


在人工智能、大数据、云计算、VR/AR 等技术深度融合之际，各大科技巨头纷纷开始加速转型，而在此浪潮之下的 Unity 开发者们，又该如何顺流而上？在 Unite Shanghai 2019 第三天的技术专场中，你将从中探寻到最适合自己的答案。

第三天技术专场的议题将主要围绕 Unity C# Job System、轻量级渲染管线（LWRP）、引擎内部的序列化机制、实时光线追踪技术、增量式垃圾回收、UIWidgets、DOTS 等技术热点展开，并通过《使命召唤手游》、《闪耀暖暖》、《CF穿越火线》的技术案例剖析标杆 3A 级手游引擎实践之路。同时，开发者也将见证在工业革命 4.0 时代，Unity 是如何在与 VR/AR/MR、AI、5G以及云计算技术融合的过程中为行业横扫硬件性能、低生产力、高成本等传统技术障碍的。

1、技术专场1——Unity 技术引擎的升级与演化

2、技术专场 2——迎战云计算、AI，Unity 华丽地蜕变

3、技术专场 3——详解 Unity 跨平台的最佳实践

4、技术专场 4——基于 Unity 性能优化的绝佳技巧


百闻不如一见，百见不如一干


当然今年的 Unite Shanghai 2019 精彩远远不止如此。在现场，除了可以聆听到精彩纷呈的游戏与跨行业解决方案的主题演讲，开发者们还可以充分体验 Unity 的精品游戏与应用。在 Made With Unity 展区，来自全国顶尖的优秀作品将横跨移动、PC、VR/AR 等多个平台，囊括动作、解谜、仙侠、角色扮演、射击、休闲和虚拟仿真等类型，让大家深刻体验高品质创作所散发的技术魅力。而今年不仅是游戏开发者的盛宴，更是创作者、艺术家们展现梦想的舞台。

技术长路，道法万千，我们已为你备好技术的饕餮盛宴，静待君来。更多 Unity 精彩内容与你相约 Unite Shanghai 2019：https://unite2019.csdn.net/
