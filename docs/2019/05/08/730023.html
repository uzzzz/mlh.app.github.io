未来的信息技术业务：2019 年德勤首席信息官晚宴

  • 柚子社区


2019 年 4 月 11 日，德勤携手 AWS 连续第三年主办了首席信息官晚宴。作为“德勤首席信息官计划”中的旗舰活动，该晚宴旨在帮助资深信息技术领导人对业务、职能及其管理职务的挑战和变化做到“知于前，行于前”。


当天，约有 30 位分别来自国际国内著名公司的首席信息官齐聚上海柏悦酒店，同德勤全球和德勤中国的信息技术和数字化领导者，还有来自 AWS 大中华区的高管，共同探讨了未来信息技术业务的发展所带来的趋势和机遇。

德勤管理咨询的客户行业及市场领导人黄伟强先生，通过介绍信息技术的未来业务开启了晚宴。AWS 中国合作伙伴生态系统总经理汪涌先生致欢迎词，并在演讲中展望了云技术对企业产生的颠覆性影响。

AWS 首席云计算企业顾问张侠博士也从企业信息技术的未来这个角度分享了自己的看法。他指出：

随着数字化转型进程的加快，几乎每个企业都不可避免地需要面临改变，从陈旧、僵化、缓慢的旧信息技术，变为创新、敏捷、高效的新信息技术，这一过程蕴含着巨大的机遇。无论是加快业务转型和创新的速度，或是以更敏捷的方法快速拥抱层出不穷的新技术，甚至借助容器、无服务器以及机器学习等技术实现应用程序和信息技术基础架构的革新，这一切都少不了云平台的推动。而 AWS 始终致力于借助遍布全球的基础架构，为不同需求的企业提供支持。从基础架构到核心服务、从应用到管理、从物联网到人工智能……AWS 丰富的服务和广泛的全球覆盖为企业转型提供了鼎力支持。

在随后的晚宴中，德勤 EMEA 技术研究和观察领导人 Hans van Grieken 先生，带领宾客们了解了“德勤 2019 科技趋势”中的重要发现。在场各位都热情地参与互动，并发表了诸多优秀观点和有趣评论。 

近年来，云技术平台在企业数字化转型中发挥着越来越重要的作用。 作为 AWS 合作伙伴网络（APN）全球主要合作伙伴之一，德勤管理咨询致力于帮助企业在整体业务创新和转型中使用诸如云计算和大数据等数字化工具，并一直积极为客户探索和提供创新解决方案。

德勤和 AWS 将继续加强在大数据、分析、云迁移和网络安全领域的合作。双方皆希望不仅能为中国公司，也能为全球公司解决数字化转型需求。在为企业整合数字化核心时，我们能通过合作，同时帮助企业进行大规模的快速创新，开发新的商业模式，拥抱新的数字化未来。
