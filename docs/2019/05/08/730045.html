当我们向往一场说走就走的旅行，随手定下心仪的酒店然后享受旅行所带来的乐趣时，你有没有想过，是什么成就了这种便利？而这种便利的背后，又有着怎样的故事？


在这里，也许你会找到答案。


本期大咖专访的嘉宾是德比软件（上海）有限公司（“德比软件”）的联合创始人兼 CTO 藕锋。作为一家拥有全球 18 万家酒店数据的公司，德比软件的成功背后有着什么秘密武器？纵横旅游技术领域 20 多年，藕锋究竟是怎样的一个人，又是如何与另一位创始人张焕杰一同一步步谱写了德比软件在行业内的传奇？

有一种合作，叫彼此成就

德比软件的故事始于硅谷。随着互联网的不断发展，藕锋与张焕杰在硅谷相识，随后一拍即合共同创立了德比软件。他们深刻认识到，只要是与旅游相关的企业都会有自己的系统，这些系统会通过互联网彼此对接，而德比软件要做的事情便是，构建一个巨大的网络，为全球酒店集团及其分销商提供高质量的数据对接服务。


如果说有些企业与 AWS 的合作是一场美丽的邂逅，那么德比软件与 AWS 更像是一对志同道合、共同进步的老友。


当传统的全球分销系统遭遇挑战，德比软件选择了利用 AWS 建立全球分布式数据交换体系，以更低的成本和高度的稳定性实现全球酒店数据的实时、智能化同步，为不同地区的分销渠道和酒店提供便利快捷的服务。之所以选择 AWS，藕锋表示，一是 AWS 全球分布式架构以及服务全球的理念和模式与德比软件不谋而合，二是 AWS 云服务在全球颇受认可的实力与地位。


“德比软件与 AWS 不仅是一种客户关系，更是一种合作伙伴的关系，具有极其重要的战略意义。” 藕锋表示。由于双方合作较早，见证了彼此的成长，默契程度也是与日俱增。特殊的业务模式使双方在构建数据平台方面相互促进，是一个彼此成就，成为对方最佳实践的过程。“未来，我们将采用 AWS 更多的产品与服务，同时在数据分析方面深度合作，对业务作出预测，为决策提供支撑。”


作为一名技术大拿，藕锋对企业上云也有着深刻的见解。藕锋认为，虽然现在很多企业的上云规划被云计算的可靠性、安全性、成熟度，以及上云成本和传统系统的迁移难度等等原因所掣肘，但云计算的高效率和资源共享所带来的收益是显而易见的，因此“企业迟早都会上云”。


有一种选择，叫另辟蹊径

特立独行从来都不是一件容易的事，尤其是当我们习惯了墨守成规，或者出于失败所支配的恐惧。然而，引路靠贵人，走路靠个人。路，终究是要靠自己选择并走出来的。


德比软件的坚持在于不分国籍的因才适用、始终面向全球市场的战略和绝不抄袭他人的商业模式。在越来越多人热衷于复制他人成功模式的市场环境中，这种坚持显得愈发难能可贵。这是德比软件能够在激烈的市场竞争中脱颖而出的原动力，而其背后，实际是一种敢于不走寻常路的勇气和另辟蹊径的睿智，同时也是一种创新。


创新，也正是 AWS 根植于血液并且一直在践行的事情。不论是 AWS 自身的成长发展史，还是与各行各业客户的合作，AWS 的创新脚步从未停歇。就像 AWS 与德比软件，正是因为敢于另辟蹊径，摒弃了为网络或运行平台投资硬件的传统思维，创新性地选择了利用 AWS 广泛的全球云服务资源来建立全球分布式数据交换体系，才能铸就德比软件在全球市场开拓中前沿的优势。


作为公司创始人，“不走前人路”不仅是藕锋在工作中的坚持，也早已融入了他自己的日常生活中。在我们很多人还在烦恼于工作与生活两者之间只能“厚此薄彼”不可兼得的时候，特立独行的藕锋却把工作和生活过得相映成趣。

据藕锋介绍，出于工作原因，自己经常往返于中美两国并辗转于世界各地，久而久之，造就了他入乡随俗、随遇而安的生活态度。生活中藕锋也爱好旅游，每当去到一个地方，便会切换当地人角色，让自己融入当地的人文和生活。与常人不同的是，对于藕锋而言，旅游并不仅仅是享乐。“我在哪里？我从哪里来？将到哪里去？” 带着这样的哲学思维，他会去深入思考当地特殊人文背后的成因，既能体味不一样的生活乐趣，又可以为进一步改善工作、实现服务本土化的业务目标提供助益。一如 AWS 所提倡的工作与生活和谐互融的状态，彼此之间相映生辉，岂不乐哉？


有一种信念，叫持之以恒

谈起对生活的感悟，藕锋表示：“越来越发现，任何事情想要做好，都需要经得起时间和耐性的考验。”


的确如此。所谓“行百里者半九十”，很多人都在黎明前的黑暗中选择放弃，殊不知光明就在眼前。努力与耕耘自然重要，坚持不懈、持之以恒却也是不可或缺的。不管做任何事情，越接近成功的时候越是困难，同样也越需要坚持。


藕锋是一个从不轻言放弃的人。除了读书学习，藕锋还有一个爱好：徒步。作为一项极具挑战性的运动，徒步不仅仅是体力的考验，更是一场毅力跟耐力的较量。凡事欲速则不达，唯有坚持方得始终。一年两次的徒步旅行，带给藕锋的不只是体质的增强，还有坚韧的意志和深入骨髓的持之以恒精神。而只有当坚持成为一种习惯，沿途的风景才会显得格外美丽。

藕锋于 2018 年 9 月在加拿大班夫国家公园徒步

工作中亦如是。无论是面对 “9·11 事件”美国旅游业的萧条给公司带来的冲击，还是公司拓展过程中全球分销网络扩展所面临的挑战，藕锋从未想过放弃，也正是这份坚持，铸就了德比软件如今的成功。


当被问及对于未来的想法时，藕锋笑言：“未来自己希望能够更具耐心，也希望这份耐心能够带来好的结果。”


谁无暴风劲雨时，守得云开见月明。只要坚守“创新”和“坚忍”两颗初心，我们有理由相信结果将不负所期。而在未来双方的进一步合作中，AWS 也将赋予德比软件在开发新技术与产品方面更高的创新性和灵活性，一同凌云而上，收获更大的成功。
