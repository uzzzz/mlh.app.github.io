2019 年 5 月 6 日，一年一度的微软 Build 开发者大会正式到来。


微软 CEO Satya Nadella介绍了微软的全新平台架构：以 Azure 为基础，支撑起 Microsoft 365、Microsoft Dynamics 365 & Power Platform 和 Microsoft Gaming 三大平台。显然，Azure 已经成为微软所有业务的根基和支撑性业务，它的重要性再次得到了无与伦比的强调。



在本次大会上，微软宣布了一项全新的服务——Microsoft Graph Data Connect（中文名为 Microsoft Graph 数据连接服务），它可以允许用一种新的方式来构建 Microsoft Graph。


除此，为了帮助团队更好地协同创造，微软在本次大会上推出了Fluid Framework（Fluid 框架）。这是一个基于网页的新型平台，提供了组件化的文档模型以共享的互动体验。


对于微软来说，Build 大会逐渐面向开发者；而普通消费者用户更为关心的 Windows 10 不仅不再出现在 Keynote 中，甚至也只是成为 Microsoft 365 的一个组成部分；而在具体功能的更新上，属于 Windows 10 自身的功能越来越少，它更像是一个躯壳，或者是一个容器，承载着微软在 AI 和云计算方面的用户端口。


华为拟在英剑桥新建半导体研发基地 与ARM做邻居


据国外媒体报道，华为计划在英国剑桥建造一家半导体研发工厂，距ARM控股(Arm Holdings)总部仅15分钟车程。


图片来源网络


据悉，华为研发基地预计到2021年投入运行，最多雇用400人，投资额等详细信息没有透露。外媒分析称对于正在讨论采用华为通信设备利弊的英国，华为可能打算强调经济贡献，毕竟在2018年2月曾宣布，到截至2023年的5年里，华为已经在英国投资30亿英镑。华为在半导体设计方面获得ARM控股的协助，有观点认为今后双方将进一步加深合作。



国内工业SaaS最大笔：黑湖智造完成B轮1.5亿元融资


日前，工业流程协作和数据分析软件公司「黑湖智造」完成1.5亿元人民币的B轮融资，这是目前为止国内工业SaaS领域的最大笔融资。


本轮融资获得了多家基金和企业的投资意向，金沙江创投、BAI（贝塔斯曼亚洲投资基金）行使“优先投资权”领投，GGV纪源资本、真格基金等多家机构跟投，募集资金将用于人才引进、产品创新、业务拓展等方面。


图片来源网络


据了解，黑湖智造是一家致力于通过云计算、物联网等前沿技术帮助中国工厂高效率、低成本地推进数字化转型的科技公司，在2016年成立于上海，其核心产品「黑湖智造协同系统」和「黑湖智造MI」是工业领域流程协作和数据分析工具的显著创新。


NGINX Plus 18 版本发布，支持动态证书加载


近日，NGINX已经发布了 NGINX Plus 18 版本（R18），这是它们一体化的负载平衡器、内容缓存和web服务器。该版本的特性包括支持动态证书加载、对OpenID Connect实现的增强以及为虚拟服务器指定端口范围的能力。


据悉，通过支持动态证书加载，SSL/TLS证书现在可以按需加载，不需要在配置文件中明确地列出。根据TLS握手期间由服务器名称指示（Server Name Indication，简称SNI）提供的主机名，NGINX Plus现在可以动态地加载正确的证书，允许在单个服务配置下托管多个安全网站。


在NGINX Plus之前的版本中，每个安全主机名都需要在配置文件中有自己的server代码块，静态地把证书和私钥指定为磁盘上的文件。这也意味着，当添加新主机名时，都必须重新加载配置。

类脑芯片所需能量是最先进计算系统的1/10？真的假的？


据新研究，一种类似电池的设备可以作为计算系统中的人工突触，用来模仿大脑的效率和学习能力。大脑只需要很少的能量就能学习和记忆大量信息的这种能力，激发了类脑或神经形态计算机的研究。


目前一个研究团队，之前开发了这种计算机的一部分：一种充当人工突触的设备，模仿神经元在大脑中的通信方式。在《科学》杂志上，该团队报告称这些设备的9个原型阵列在处理速度、能效、重复性和耐用性方面表现优于预期。


图片来源网络


未来，团队成员希望将他们的人工突触与传统电子设备相结合，希望可以成为推动小型设备人工智能学习向前发展的一步。


欧洲议会选举前Facebook设立作战室 打击虚假信息


据外媒报道，随着欧洲议会选举活动临近，社交网络巨头Facebook设立了一个作战室，以监控是否有虚假信息、虚假账户和违反该社交网络规则的行为来干预选举，这项努力是为了防止可能影响选举的大规模虚假宣传活动。


作战室与Facebook在2018年10月美国2018年中期选举之前和巴西选举之前设立的作战室相似，当时那个作战室已于去年11月底关闭。在今年印度大选之前，Facebook还在德里设立了一个类似的作战室。


图片来源网络


报道称这间位于Facebook爱尔兰欧洲总部的新作战室将会一直运作下去，直到将于5月23日至26日举行的欧洲议会选举活动结束。


英国将Apple Pay和Google Pay支付接入政府服务


英国政府近日宣布，将在gov.uk服务中接纳Google Pay和苹果支付的支付方式；此前在英国，Apple Pay和Google Pay的移动支付服务都只能用于商业交易。


开通这两种支付方式的政务服务包括全球通关服务（Global Entry Service）、基本在线公开和禁止服务检查、注册旅行者服务（Registered Traveller Service）和电子免签服务（Electronic Visa弃权）。在初步试验后，英国政府将进一步在政务服务中推广移动支付，包括地方政府、警察和NHS组织。


首例！诊断肾脏疾病，AI技术平台获FDA突破性设备认定


一直以来，Renalytix AI是一家肾脏疾病AI诊断公司，旨在开发基于人工智能的肾脏疾病临床诊断技术。该公司近日宣布，其主要诊断产品KidneyIntelX已获美国FDA突破性设备认定（Breakthrough Device designation），这也是第一家在AI肾脏疾病诊断领域获得FDA突破性设备认定的公司。


有报道称，KidneyIntelX主要用来对2型糖尿病性肾病患者进行诊断，以控制进展迅速的病情并改善临床管理。最终目的是减少美国医疗系统每年用于慢性和终末期肾脏疾病的巨额费用，而这一费用最高可达约1140亿美元，十分惊人！
