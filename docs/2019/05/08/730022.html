推荐人：费良宏

AWS 首席云计算技术顾问

《Pro Python 3》


学习 Python 3 下如何优化编程技巧和实用方法，以成为更高效和创造性的 Python 程序员。程序员 Python 3 进阶必读。

《Python Deep Learning》


快速了解深度学习以及热门的GAN 、VAE以及强化学习。



《The Rust Programming Language》


这是由 Rust 核心团队的两名成员撰写的 Rust 无可争议的首选指南。程序员学习Rust 必备！

《Microservices in Action》


从架构师而不是程序员的视角了解构建和部署基于微服务的应用程序的实用书籍。是架构师微服务实践理论最好的指南。



《Technology Strategy Patterns》


了解应用结构化思维将业务环境与 IT 架构相关联。这是一本不多见的关于“如何成为 CTO”的最佳读物。


推荐人：张侠博士

AWS 首席云计算企业战略顾问

《Blockchain Revolution》


一本全面、有代表性地介绍了区块链革命的书，对于区块链支持、比特币、加密货币部分都有很好的解说。




《Introduction to Artificial Intelligence》


人工智能概论，全面地介绍了计算机模仿人类思考的过程，概述了过去二十年人工智能领域的主要进展、取得的成绩和方法，是一本综合介绍人工智能并值得参考的书。

《Ahead in the Cloud》


来自 AWS 企业战略团队的作品，讲述企业领导应该如何从技术、人才、资源、文化等各个方面利用云计算帮助企业转型，为企业创新做出贡献。



《 How to create a mind》


纽约时报的畅销书，介绍了人类整个思考过程的秘密，讲述了人类认知过程中对下一代人工智能可能造成的影响，是一本有影响力的未来学家写的关于人工智能的书。

《Rise of the Robots》


纽约时报的另一本畅销书 – 机器人的崛起。这本书中讲了机器人在下一个时代会怎样影响我们的社会生活，比如可能对人的工作产生什么影响，被评为“2015 商业技术类最有趣的一本书”。
