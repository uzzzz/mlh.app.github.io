数据备份及大数据分析应用方案研讨会


随着由西云数据运营的 AWS 中国（宁夏）区域的落地以及第三个可用区的建成，客户在中国已经可以使用基于三个可用区的 AWS 服务，如 Amazon Aurora 等。


同时，在中国通过 AWS 云服务进行多区域部署也成为可能，您可以充分利用多区域部署带来的优势，实现跨区域级别的容灾、终端用户就近接入以及获得更多成本优势！


我们将通过本次研讨会向您介绍：

  • AWS 中国（宁夏）区域

  • AWS 数据库服务

  • AWS 两个区域间的数据迁移和数据同步技术

  • 多区域部署的应用场景及案例分享


当然，我们也安排了专业的 AWS 解决方案架构师坐镇现场，在 Q&A 环节一次性为您解答所有疑惑，让您的上云决定再无顾虑，搓手期待！
