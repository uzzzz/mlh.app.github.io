“富人靠科技，穷人靠变异”，基因背后到底有什么秘密？

  • 柚子社区


“富人靠科技，穷人靠变异”

在超级英雄电影中，既有钢铁侠这样的“氪金大佬”

也有美队这样靠超级士兵血清改造基因的“逆天产物”

虽然电影终究只是一场“魔幻现实”，但在这个时代，科技创造出的奇迹已经数不胜数，一手造就蜘蛛侠、美国队长的“上帝密码”——基因也早已被人们所熟悉。大概近几年最著名的基因案例，便是好莱坞影星安吉丽娜·朱莉通过基因测序，获悉了自己罹患乳腺癌的几率非常高，于是主动选择了预防性的双侧乳腺切除手术，从而为自己“一绝后患”。


如果基因的便利只为名人富人而展开，那绝对有失公允；而想让基因工程惠及到世界上的大多数人，又绝非易事，成都生命基线科技有限公司 COO 凌之浩对此就深有体会。对以基因为业务的企业来说，如果还使用传统的模式进行运作，面临的挑战将会十分艰巨。 






成都生命基线科技 COO

凌之浩


毕业于德国马克思普朗克生态研究所，师从美国科学院院士 Ian.T.Baldwin，被国内基因科技领域的迅速发展吸引而回国。他所在的成都生命基线科技有限公司成立于 2015 年，致力于用互联网技术加速基因科技在科学研究和大众健康领域的应用。


生命基线成立于 2015 年，致力于用互联网技术加速基因科技在科学研究和大众健康领域的应用，目前主要有两大业务：


• “简基”

以基因检测和精准体检为前端，提供健康风控服务，主要针对企业员工；


• “基因帮”

提供试剂耗材和 NGS 服务的一站式线上平台，主要针对全国范围的高校、科研院所科研人员。


从科研领域到企业运作，凌志浩和生命基线需要一个足够给力的“伙伴” ——

灵活搭配-适配企业特定环境

在公司运作的过程中，数据不断累积。企业既需要解决存储问题，也要让数据可以轻松调取便于分析，但是如何在保障用户高敏感基因数据安全的前提下，同时使企业获得高速处理基因数据的能力？ 

成本节约-支持企业业务需求

按照目前的测序技术，要相对完整的覆盖一个人的全基因组序列，大约需要产出约 90G 的原始数据。在传统模式下，面对日益增长的样本量分析需求，企业只能选择同时开启几百甚至上千台服务器，这样的成本支出显然不是好事。而且，大数据分析场景中的高通量数据传输以及如何将结果以更加高效的方式呈现给终端用户，也是生命基线所必须面临的问题。在这种情况下，生命基线开始寻求通过云来继续提供优质服务。


在试用过一些超算和云计算平台后，凌之浩最终决定，对于需要强大计算能力的算法模型研发，使用 Amazon Web Services（AWS）服务。

生命基线可根据业务需求自动进行弹性扩展所需要的实例，通过 AWS 命令行工具和丰富的 API 种类，可以用一条简单的命令，实现对上百台计算实例的管理。最后，再利用 Amazon S3 存储桶来储存样本数据和最终结果，通过生命周期管理，将不常用的数据再归档到更为便宜的 Amazon Glacier 中去。AWS 这种技术不仅缩短了计算的时间，而且还极大地节省了经济成本和管理成本。


这一模式非常适合测试和开发环境，特别是对于像成都生命基线这样具有大数据分析的场景，使用 Spot 实例的价格比普通包年或者按需实例的价格还要便宜。


现在，只要花上几分钟时间，凌之浩的团队便可以部署上百台服务器实现计算分析任务，并在计算后自动关闭，这种可根据业务需求自动进行的弹性扩展，极大降低了运维压力。

使用 AWS 让我们相比于之前节省了大量研发成本，另外 AWS 提供的多种 API 接口很方便开发人员去接入。当遇到什么问题时，AWS 这边也会很快响应，进行技术支持。

作为一家创业公司的 COO，这是凌之浩格外看重的一点。



目前，基因市场仍处于“宝宝期”。云计算等先进技术固然降低了其门槛，可人们的主动健康意识依然普遍淡薄。譬如很多人知道烟酒对健康有害，却一点也不在意，往往是有了切身之痛或者在身边人的经历触及到自己后，才会被动重视疾病预防。



在云计算的助力之下，以疾病预防为目的的基因检测这项技术如今已经能够“飞入寻常百姓家”，为我们每一个人揭示曾经遥不可及的“生命密码”。如果可以用科技改变生活，甚至改变命运，何乐而不为呢？
