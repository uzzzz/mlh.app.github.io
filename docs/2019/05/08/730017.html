AWS 宣布开通由西云数据运营的三个 Amazon CloudFront 站点！

  • 柚子社区

4 月 23 日，亚马逊旗下公司 Amazon Web Services, Inc. （AWS） 宣布，通过与宁夏西云数据科技有限公司（简称“西云数据”）协作，在中国开通三个由西云数据运营的 Amazon CloudFront 站点，分别位于北京、上海和宁夏中卫。新站点将为中国客户提供更好的体验，包括更快的内容分发和更高的安全性。至此，全球已有 169 个 Amazon CloudFront 服务节点。

Amazon CloudFront 是一种快速的 CDN（内容分发网络）服务，可以安全地分发数据、视频和应用程序，还可以通过 API（应用编程接口），在保证低延迟、高速度传输的同时实现高级的功能，所有一切都在开发者友好的环境中实现。

三个 Amazon CloudFront 站点通过专用的私有网络相互连接，并连接到由光环新网运营的 AWS 中国（北京）区域和由西云数据运营的 AWS 中国（宁夏）区域。


Amazon CloudFront 支持的使用场景包括视频分发、电子商务、手机银行、软件下载、音乐下载、流畅地交付频繁访问的图像和对象文件等。

 

客户可以在其中国区域的控制台中创建 Amazon CloudFront China 分发版，向最终用户分发内容。通过本地边缘站点的使用，客户位于中国境内的源服务可以降低 31% 的延迟。同时，根据计划，明年将在中国增加更多 Amazon CloudFront 站点。
