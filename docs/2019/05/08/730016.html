关于 Amazon CloudFront 的所有问题，这个专题研讨会给你答案

  • 柚子社区

Amazon CloudFront 宣布在中国推出 CloudFront，并在由宁夏西云数据科技有限公司（简称“西云数据”）运营的北京、上海和中卫区域增设三个新的边缘站点（POP）。客户现在能够以更低的延迟及更高的可用性和安全性，向中国大陆的最终观看者提供内容。在中国使用 Amazon CloudFront 服务时，许多用户可以看到延迟已经减少了 31%。

在本次研讨会上，Amazon CloudFront 产品管理、业务开发和解决方案架构师团队，将为客户介绍：

• 如何使用 CDN 进行架构设计

• Amazon CloudFront 在中国的独特功能

• 如何开始使用该服务并优化其应用和内容交付
