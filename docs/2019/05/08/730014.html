主题演讲

当天，我们邀请了 AWS 首席云计算技术顾问费良宏，为大家带来《AWS 云助您创建新 IT 架构和应用》的主题演讲，现场聆听大咖演讲的宝贵机会，不容错过！

费良宏

AWS 首席云计算技术顾问

费良宏主要专注于云计算以及互联网等技术领域，致力于帮助中国的开发者构建基于云计算的新一代的互联网应用。其拥有超过 20 年在 IT 行业以及软件开发领域的工作经验；曾经任职于 Microsoft、Apple 等知名企业，任职架构师、技术顾问等职务，参与过多个大型软件项目的设计、开发与项目管理，并于 2009 年开始，从事与云计算领域相关的技术工作；研究方向主要是软件架构的演进，尤其是云计算方面的架构实践，并擅长 Web 应用、移动应用以及机器学习等的开发。

客户案例分享

本次活动，我们还特别邀请到了烟台欣和企业食品有限公司夏军东先生为大家带来实战案例分享，详细为您介绍 AWS 是如何助力欣和企业数字化转型的。

夏军东 

烟台欣和企业平台管理高级经理


2006 年加入欣和，主要负责集团基础架构设计与规划，曾参与过集团 SAP 实施、集团网络规划、虚拟化、云迁移、DEVOPS 等项目，对于云架构规划迁移、云原生应用、云成本控制等有丰富经验。


6

金牌讲师

AWSome Day 2019 众星云集，特别邀请 AWS 资深技术讲师黄涛，循序渐进地为大家讲解 AWS 上的架构设计、软件开发及系统管理和大数据运用等。黄老师擅长通过提纲挈领、知识面切割和分块深入的方法，配合案例解释、经验分享、热烈讨论和动手实作来帮助客户快速、高效地掌握 AWS 云服务技术。

黄涛

AWS 资深技术讲师

专门负责面向广大 AWS 客户和开发者提供专业化的培训服务，拥有多年云平台开发、应用架构和部署的实践经验。在软件开发领域积累了丰富的架构、编码、软件工程和管理心得。多年高性能运算和云计算的实战经历为他带来对高性能运算、云计算和应用架设的深刻理解；参与初创的历程也让他对小规模事业拥有更真实的体会。


相信通过本次课程，您将具备基本的云技术能力，并且能够掌握在 AWS 平台上构建安全且高扩展性的服务平台，从而为你的公司创造更多利润与佳绩！
