大众汽车集团总部位于沃尔夫斯堡，是全球领先的汽车制造商之一，同时也是欧洲最大的汽车制造商。集团目前拥有来自 7 个欧洲国家的 12 大汽车品牌。此外，大众汽车还提供广泛的金融服务，包括经销商和客户融资、租赁、银行和保险活动以及车队管理。集团在 20 个欧洲国家以及美洲、亚洲和非洲的 11 个国家设有 122 家生产工厂，在全球拥有 664,496 名员工从事汽车生产、汽车相关服务或其他业务领域工作，汽车销往 153 个国家和地区。


这样一位汽车工业领域的巨头的名字，最近正和 AWS 一起在全球媒体的科技版面刷屏！“频上热搜”的背后，究竟是什么“大事”正在发生？

近日，Amazon Web Services（AWS）和大众汽车集团宣布，双方已经签署一项为期数年的全球协议，根据协议内容，双方将共同打造“大众汽车工业云”——一个基于云的工业数字化生产平台，从而转变大众汽车的制造和物流流程。

大众汽车将利用 AWS 服务组合所带来的广度和深度，包括物联网、机器学习、分析和计算服务等，来提高工厂效率及运行时间，增强生产灵活性并改善车辆质量。

大众汽车工业云将汇集大众汽车 122 家制造厂的实时数据，管理装配设备的整体效率，并跟踪零部件和车辆。大众汽车将使用 AWS 物联网服务套件，包括 AWS IoT Greengrass、AWS IoT Core、AWS IoT Analytics 和 AWS IoT SiteWise 等，来查找、收集、组织数据，并对来自工厂车间的数据进行复杂分析。AWS 物联网服务将为各工厂的生产运营提供全新见解，从而优化生产并提高流程效率。

大众汽车还将基于 Amazon S3，构建一个全公司范围的数据湖来分析数据，通过识别生产和浪费方面的差距，来找出运营趋势、改进预测并简化运营流程。


大众汽车还将使用 Amazon SageMaker 全面托管服务，为开发人员和数据科学家提供快速构建、培训和部署机器学习模型的能力，以优化其所有工厂中的机器和设备的运行流程。


此外，大众汽车还计划使用 AWS Outposts 在企业内部部署本地 AWS 服务、基础设施和运营模式等，针对延迟敏感的应用程序，提供工厂和云之间的无缝切换功能，带来一致的混合体验。


保时捷执行董事会主席，大众汽车管理委员会成员

Oliver Blume

我们将继续把加强生产作为大众汽车的一个关键竞争因素，与 AWS 的战略合作将为此奠定良好基础。大众集团拥有全球汽车生产制造方面的专业知识，AWS 拥有云技术专业知识，双方的合作相得益彰。借助全球工业平台，我们希望创建一个不断增长的、具有透明度和效率的工业生态系统，为所有相关各方创造利益。
