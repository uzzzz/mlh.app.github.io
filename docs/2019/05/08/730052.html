由西云数据运营的 AWS 中国（宁夏）区域现已提供 AWS Elemental MediaConvert。AWS Elemental MediaConvert 是一款具有广播级功能的基于文件的视频转码服务。借助该服务，您能够轻松创建视频点播 （VOD） 内容，实现大规模的广播和多屏幕传输。该服务将高级视频和音频功能与简单的 Web 服务界面相结合，并采用按需付费的定价模式。借助 AWS Elemental MediaConvert，您可以专注于提供引人注目的媒体体验，而无需执行复杂的视频处理基础设施构建和运营操作。



AWS Elemental MediaConvert

使用优势

广播级功能

借助 AWS Elemental MediaConvert，您可以使用各种 Internet 和专业媒体格式来制作在任何设备上看起来都非常棒的高质量视频输出。AWS Elemental MediaConvert 支持超高清分辨率、高动态范围视频、图形叠加、高级音频功能、内容保护和隐藏字幕，可提供全套工具来提供高品质的观看体验。

可靠且易于管理

AWS Elemental MediaConvert 不需要对底层基础设施进行任何设置、管理或维护。您只需提交您想要进行的视频处理设置任务即可开始使用，无需花费时间或资源来管理转码基础设施。AWS Elemental MediaConvert 不仅可以预置处理任务所需的资源，而且还可以自动监控它们，因此您不必担心可靠性问题。

简单且可预测的定价

AWS Elemental MediaConvert 让客户可以在云中创建高质量的端到端视频处理工作流，而无需为视频处理基础设施进行前期投资或资本支出。您只需根据处理视频所持续的时间和您使用的功能付费即可。



AWS Elemental MediaConvert

工作原理

将您需要转码及合成的来源文件上传到 Amazon S3 中，包括视频文件、音频文件、字幕文件、叠加图像以及可能的不完整的 IMP 补充文件等。启动转码任务，在任务定义中声明您的输入和输出详细选项，MediaConvert 将输出到 Amazon S3，您可以利用 Amazon S3 托管静态网站进行播放，或者进一步分发到 Amazon CloudFront 等 CDN 服务以加速播放。


AWS Elemental MediaConvert 可输出为 HLS、MPEG-DASH、Microsoft 平滑流式、CMAF 等在线播放文件堆栈，以及 MPEG-4 或者 QuickTime 文件。输出码率可动态调整。利用时间码可以同步音频与字幕，或剪辑视频。在输出流中可以加入 SCTE-35 标记以插播广告等。每一个转码任务可以包含多组输入及输出内容。您可以使用 AWS 内置或自定义的任务模版以及输出预设来简化任务创建。

AWS Elemental MediaConvert

快速上手

使用 AWS Elemental MediaConvert 非常简单，您可以访问控制台管理您的转码任务，也可以使用 AWS CLI 通过命令行或使用 SDK 开发您自己的应用进行管理。


管理控制台入口



轻松创建转码任务

点击“入门”或者在任务菜单选择“创建任务”，即可看到以下界面。填写 Amazon S3 中的输入文件地址；如果必要的话，可以添加多个输入源。


点击输出组右侧的添加按钮，选择输出格式：


在目标字段中指定转码结果存放到 Amazon S3 中的位置。


然后选中“Output1”，在输出设置的“Preset”字段里，选择输出预设，对于流媒体输出，在段修饰符里填入“$dt$”为每个分片文件名加上时间作为后缀。对于多个输出，填写不同的名称修饰符以区分文件名。


在作业设置里，选择预先创建好的 IAM 角色，该角色需要有对输入和输出的 Amazon S3 存储桶有读写权限。然后点击创建按钮，即提交了本次转码作业。


您可以在任务列表界面观察任务状态，待任务完成，即可查看 Amazon S3 中的输出文件。


您可以在浏览器中通过输出文件的 Amazon S3 地址来访问转码后的视频，需开通输出文件的 Amazon S3 存储桶的托管静态网站设置，并取消存储桶上的公有 ACL 访问限制。


至此，我们就完成了一次完整的视频转码操作，并基于 Amazon S3 提供了一个简单的 VOD 视频点播服务。我们可以使用 AWS Lambda 函数来调用 MediaConvert 服务，并在源文件上传到 Amazon S3 时自动触发，从而以无服务器方式完成视频转码，为 VOD 视频点播业务提供高品质的内容。


创建输出预设

在输出预设界面，点击“创建预设”按钮。


填写预设名称，选择容器。


填写更多视频输出的详细设置。必要的话，加入各种预处理器，比如叠加图像，时间码等。


填写音频输出的详细设置。最后点击“创建”即可。


创建任务模版

在任务模版界面，点击“创建模版”。


填入模版名称，添加输入和输出组，填写作业设置。这些操作都与创建一次转码任务相同，这里就不赘述了。
