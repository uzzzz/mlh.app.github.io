其它笔记小技巧：

  1. 做好笔记的前提：

    1. 要做好假设，把笔记写成复习时一看就能清楚明白地内容。
    2. 考虑复习时会遇到地状况：如果记录得太详细会增加复习上得负担，那么就要减少相应得笔记，只记录重要的内容。

  2. 做笔记的正确做法：

    1. 字写大、行距一行便修改；
    2. 只记在右页，留出左页；
    3. 若资料/讲义不会多次翻阅，则笔记直接写在笔记本上；
    4. 用手写方式做笔记。因为是大脑思路整理的过程，更符合实际需求；

  3. 同时使用两本笔记本：

    1. A4活页本：
      1. 每页分三部分。在右页上左右距离3.5厘米处画竖线：左侧记录分类、中间记录普通笔记、右侧记录行动方案（如要买的某本老师提到的重要书籍）；
    2. A5薄型笔记本(50页左右)：
      1. 随身携带、把A5上重要资料剪贴到A4上；
      2. 整本分三部分。
        1. 前10页：粘贴旧随身笔记本上的重要笔记 及 与这个月相关且重要的内容；
        2. 中间30页：正常使用的笔记内容；
        3. 后10页：随手记录的空间，如待办事项、临时准备；

  4. 课堂上的笔记技巧：

    1. 目的：考前再翻一次，就有把握拿到分数。
    2. 避免：
      1. 使用录音笔；
      2. 一字不漏的记录板书；
      3. 不重复作业；（即不先写一次潦草笔记，再写一次漂亮笔记）
    3. 不明白地方：用问好和三个阶段进行划分；（下课查一查？、问同学？？、问老师？？？）

  5. 用做笔记来提升记忆力：

    1. 回顾笔记，看自己在什么地方会犯错：
      1. 写错答案的地方：上面画“X”，旁边写上正确答案；
      2. 烂熟于心、不需要再复习部分：直接打上一个大大的“X”，提醒自己不重复复习；
    2. 笔盒：
      1. 蓝色可擦笔：重要事情；
      2. 黑色可擦笔：不重要事情；
    3. 四色荧光笔：
      1. 红色：一看就懂；
      2. 绿色：看完要略加思考；
      3. 黄色：以前看过，但不是特别了解；
      4. 蓝色：完全陌生，完全不了解；
    4. 四重圆圈法（应用于错题集或试卷上）：
      1. 红色：第一次复习没办法回答；
      2. 绿色：第二次复习仍然无法回答；
      3. 黄色：第三次复习无法回答；
      4. 蓝色：第四次复习无法回答；
      5. “X”：无需再复习；

  6. 灰姑娘之心日记（待办事项清单）：

    1. 步骤：每天开始工作前，把待办事项列出来。若有追加，就写在清单后面；
    2. 特点：每天只指定两个目标；
    3. 分类：
      1. 红色：工作；
      2. 绿色：学习或阅读；
      3. 黄色：有趣的事情；
      4. 蓝色：杂事；
    4. 备注：当天未完成待办事项，写在红色便利贴上，贴在明天的灰姑娘之心日记上；
    5. 便利贴的颜色分类：
      1. 红色：马上要做，且很重要；
      2. 绿色：肯定要做，但不需要马上做；
      3. 黄色：半年到一年内做完；
      4. 蓝色：想试试看，但不做也无所谓；
    6. 便利贴的大小：
      1. 小型：一些关键词；（如书名、简单的行动）
      2. 中型：动词+名词；（如做一个excel表格）
      3. 大型：写需要短文来表达的事情；（如在某日期给甲方提供某资料）

  7. 其它笔记小技巧：

    1. 每一句话长度：不超过一行，如果太长就控制在两行以内；
    2. 记录非A即B类型内容：先列举出结论，再补齐需要注意的重点——B（非A）；
    3. 以问答形式记忆信息：每一页最多放四道问答题；
    4. 在笔记上贴上喜欢的贴纸；
