今天初学了python的微信自动回复机器人课程。
首先是利用导入python包wxpy。
使用cmd

pip install wxpy

输入代码pip install wxpy即可完成自动下载
然后新建python项目
一.首先编写一个微信的扫码登录以及获取用户所有好友的测试代码
1.导入wxpy包

from wxpy import*

2.扫码登录

#扫码登录
bot=Bot()

3获取登录用户的所有好友

#获取好友
my_friends=bot.friends()
print(my_friends)

4将获取到的好友列表输入到一个frends.txt中

for friend in my_friends:
    with open("frends.txt","a",encoding="utf_8")as f:
        f.write(str(friend)+"\n")

二.编写微信自动回复机器人
1.首先在图灵网站注册机器人http://www.tuling123.com/member/robot/index.jhtml
2.同上述步骤导包和登录

from wxpy import*
#扫码登录
bot=Bot()

3.可以在这里规定自动回复谁的信息，把好友的备注写入my_friend

my_friend = ensure_one(bot.search('明信片夫妇'))

4.初始化机器人，将自己创建的图灵机器人key输入

#初始化机器人（api_key：图灵）
tuling=Tuling(api_key="f39bd8846fb94436ab9f4d024671e20a")
#自动回复

5.设置自动回复，并且打印收到的信息，同时用户可以使用控制台输入回复的内容同时可以回复

@bot.register(my_friend)
def aoto_reply_all(msg):
    tuling.do_reply(msg)
    print("接受: "+str(msg))
    s=input("输入回复内容:")
    msg.reply(s)

6.机器人开始运行

#开始运行机器人
bot.join()
