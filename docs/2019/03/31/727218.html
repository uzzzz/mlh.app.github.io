①get请求

import requests         ##导入requests 库
res = requests.get('https://www.baidu.com')    ##get('解析网址')
print(res.text)          ##打印内容（HTML）
print(res.status_code)  ##打印 状态码
print(type(res.text))   ##打印 type
print(res.cookies)      ##打印网页cookies

②抓取二进制数据

图片、音频、视频等文件实际上都是由二进制码组成的，由于特定的解析方式，我们才看到我们看到的图片视频等，因此同样的我们只要获取他们的二进制码就同样可以获取相应的图片视频等。这里介绍一下最基本的获取然后保存。

import requests      ##导入requests 库
res = requests.get('https://timgsa.baidu.com/timg?image&quality=80&size=b9999_10000&sec=1553972483306&di=187905c240257964db04ed37f81e22cb&imgtype=0&src=http%3A%2F%2Fwww.shangdu.com%2Fuimg%2F5csxamSv5HcAGX1u9EzzMYV9LogXlpAevZWCsVNFOtcFqilPK7.jpg')    ##从网上随便找了一张图片的地址 
print(res.content)
with open('风景.jpg','wb') as f:   #风景.jpg ：为保存在文件的相对地址并且命名为风景.jpg ，wb 为write byte 写入二进制
    f.write(res.content)
