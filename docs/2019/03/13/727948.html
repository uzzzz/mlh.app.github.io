itchat_op_pc

介绍

使用itchat微信库，利用文件助手发送命令操作电脑，包括调用视频查看使用者，截屏，发送文件等。

软件架构

  • Python 3.6.1
  • itchat
  • cv2
  • PIL

安装教程

下载后修改config.ini

[email]
sender= 发件人邮箱账号
pass = 发件人邮箱密码
addressee = 收件人邮箱账号，可以发送给自己

使用说明

  1. py -3 wechat.py

  2. 扫码登录
