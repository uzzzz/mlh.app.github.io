golang的下载与安装使用---图文教程

  • 柚子社区

1.首先去官网下载：

可以去Go官网中国 https://studygolang.com/dl

 

选择自己需要的版本

我下载的是go1.12.1.windows-amd64.msi

2.下载完成之后进行安装

选择安装位置，我选择在

D:\Go

，其他就是一直点下一步就可以了。

安装之后看一下目录结构


安装好了之后会直接将D:\Go\bin添加到系统环境变量Path中，

自己检查一下 若没有则自己手动添加。

3.同时会新建用户环境变量GOPATH

值为%USERPROFILE%\go，我的GOPATH的值即为C:\Users\zifeiy\go

 

4.检查一下

打开命令行：输入go version查看go版本

 

5.使用  第一个go程序

在桌面新建一个hello.go文件  加入以下内容

package main

import "fmt"

func main() {
    fmt.Printf("hello,world!\n")
}

打开命令行 进入文件所在目录 输入 go run hello.go 编译并且运行

成功打印hello world

至此：已经下载和配置好了go 并且成功运行第一个go程序。
