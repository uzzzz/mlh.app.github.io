300块买来的道理之2019蓝桥杯反思

  • 柚子社区

本次蓝桥杯只拿二等奖近期的原因在于我的应试能力很差而又不注重练习而导致的。
上面的话总结的很正确，刚和班里得了一等奖的同学交流了一下，人家迷宫没做出来，但是倒数第二题和第三题做出来了，他们在洛谷以及别的网站上刷了一个月的题。虽然只做了二三十道，但里面还有一些提高组这题，这就是为什么人家碰到难题有信心去挑战未知，有这个能力让自己能够去对付自己没有碰到过的敌人，并从中找出一定规律使得他们能够对一个问题拥有更多的完成度。而我，四个月没刷题，比赛前几周别人问我题，我写一道dfs写了半个小时，足以证明刷题的重要性。
最核心的问题在于，是第三题那一类题，是我应该得不到分的题，但，机会还是出现了，并且让细心的人抓住了它（这既是一种幸运，也是一种不幸）。第三题考察的范围是一些小型的数论知识，比如模，整除，最大公约数，最小公倍数之类的知识，由于我只准备了三天左右，因此我既没有选择去系统的学习它们，也没有选者去累积它们，而是直接放弃了他们，因此，当我一看到什么广义斐波拉契数列的前20多万项的和时,我就直接放弃了它，又在检查题的时候连看它一眼的意识都没有，这奖丢的活该，送分题送到面前了，却在去理解题意之前就被吓跑了，真是可笑。
原来倒数第二题也是这样的原因，我一看到后缀表达式这几个字就直接不看题目了…。
中期的原因则是因为准备的不够充分，给女朋友做毕业设计耗费了大量时间。
长期的原因则是因为我报了一个月后压根就没准备去了，临时抱佛脚是为了学分去的。
以上长，中，短三个原因任意一个不发生，都不会错失奖项，但偏偏事实就是这么残酷。
我学习到了什么：

  1. 但凡是决定去做一件事情，在决定之前就要想好，有没有可能有另外一件更重要的事情突然插进来，要提前想好会不会有更重要的事情打断它的进程，或者做这个事情还不如去做另一件事情。要么就别做，要决定去做就要为它分配时间，定计划，然后至死不渝的去完成，不要再给自己找借口。
  2. 做题，尤其是那种并非是i+1（i是匹配于自己能力的难度）难度的题，而是i或i以下，尤其是i以下类型的题，要想清楚，比如本次比赛第一题，找最值即可，我去写暴力，还tom（去掉o）花式的写了两种写法来保证这个五分题的正确性，我有这个功夫不如去看下第三题
  3. 算法比赛的检查有一定必要性，但我过渡检查了，我最后四十分钟应该拿去挑战更多的题目的，因为我前面的题做的时候就以及在反复检查了，可谓是不该检查的地方过渡检查，该检查的地方不去检查
  4. 2第二题花了太多时间，这种题应该手算，我万万没想到我在编写一个通用的解题程序过程中遇到了自己没想到的问题，结果白白浪费了太多时间。
  5. 但凡涉及大量时间和金钱的活动一定慎重再慎重，要把所花的时间估计出来。比如我完全可以告诉我女朋友，我帮你写一个星期的代码，把最核心的东西写出来，其余你自己做，这样就不至于后面花了四个星期去做一些枯燥无聊的工作。
