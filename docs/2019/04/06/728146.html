搬砖套利是指各个区块链资产交易平台存在差价时，通过在低价平台买入，在高价平台卖出获取利差的行为，MGC Token 采用ATS高频量化交易。

1、单线套利

A交易所ETH价格是$300，B交易所ETH价格是$350，如何进行套利？

赚取差价：

A交易所充值，买入ETH，使用$300，然后提取ETH充值到B交易所，卖出ETH，得到$350，获利$50

代币数量增加

B交易所卖出ETH，得到的350充值到A交易所，能在A交易所买1.166个以太坊

2、对冲套利（双向套利）

在各个交易所储备多种主流数字资产进行对冲套利

举例：

A交易所ETH价格是$300，B交易所ETH价格是$350

A交易所买入一个ETH使用$300，在B交易所卖出一个ETH获得$350

所持代币数量不变，获利$50

那么在传统人工单线操作会有以下几个问题：

  1. 法币充值、提现问题
  2. “时差”问题
  3. 交易手续费和矿工费问题

这样看来利润几乎不大，并且存在风险。

Automated trade system智能系统如何搬砖

ATS可以实时追踪查询各大交易所数据，并且同时进行高效精确分析对比，捕捉交易平台差价以及交易量，系统自动识别发出指令决定是否执行套利任务，并且在第一时间自行启动套利交易。

  • 币币交易双线对冲套利

用「币币交易对」进行交易，实现搬砖套利，比如用BTC/ETH交易对。比如Bithumb交易所和BitMEX交易所，都有BTC/ETH交易对，如果两个平台存在差价，那么ATS就会进行搬砖套利。

币币交易的好处是交易所充值和提币都不需要手续费，如果提现法币的话，存在手续费，大概为0.5%左右。

这样法币充值、提现的问题就顺利避开了。

  • “时差”问题

举例：

发现A交易所和B交易所存在BTC/ETH交易对存在利差：

A交易所1BTC = 15ETH

B交易所1BTC = 10ETH

想得到更多想获得更多的BTC，按照下面的流程：

往A交易所充值1BTC，换15ETH；

15ETH提现到钱包（交易所之间不能直接转币所以要先提现到钱包）

将钱包中的15ETH转账到B交易所，换1.5BTC；

如果价差一直存在，可以继续进行套利，直至没有价差

为了便于计算，上面的BTC/ETH举例，目前实际情况是1BTC≈20ETH；

 

传统人工搬砖弊端

充值、提现和交易所有过程做了零手续费的举例。在发现价差后，传统搬砖需要做充值-交易-提现-充值-交易5个步骤才能完成一次搬砖。这个过程需要大约半小时甚至更长的时间才能完成，90%的时间都花在充值和提现上，有非常大的可能还没完成一个搬砖过程，两边的价差就消失了，甚至可能出现逆差。还有搬砖过程中产生矿工费、交易费

MGC Token如何操作

同时往多个交易所充值BTC、ETH、USDT等

在两个平台都有BTC和ETH，这时候发现如果发现两个交易所的存在价差：

假设刚开始你在A和B交易所都充值1BTC和15ETH。

A交易所1BTC = 15ETH

B交易所1BTC = 10ETH

那么ATS快速的将A交易所的1BTC换成15ETH，B交易所的15ETH换成1.5BTC，那么在这个操作之后将收益0.5BTC；

盘点资产：

A交易所资产：30ETH；

B交易所资产：2.5BTC ；

如果在BTC数量不变的情况，让ETH增加，A交易所还是操作不变，在B交易所用10ETH换1BTC;

盘点资产：

A交易所资产：30ETH；

B交易所资产：2BTC ，5ETH。

我们叫这种搬砖叫对冲套利，前面说那种搬砖叫单线套利，进行对比就看出对冲套利的优势：

“无时差”

对冲套利节省了代币的提现、充值打包区块的时间，交易所上的交易只是一种记账，只要捕捉数据成功，ATS系统几乎可以在瞬间完成；

在不减少原有数字代币数量情况下，让交易对中一个或多个代币的数量增加。

“低币耗”

双线搬砖在平台只需要做好数字资产储备，ATS每一次搬砖后，等价格反转或进行该币种套利。避免反复充值、提现对代币的消耗。真正做到不受币种价格涨跌，实现高频量化交易进行套利

“双线套利”

对冲搬砖操作完成之后，如果两边平台利差一直是同一个方向的，比如实施双线搬砖时：

A交易所1BTC = 15ETH；

B交易所1BTC = 10ETH；

通过搬砖，资产分布变成：

A交易所资产：30ETH；

B交易所资产：2.5BTC；

这样做的风险前面讲过了：没有抓住利差时间完成搬砖，很有可能利差就消失了，所以人工操作很难把控时间。

ATS采用高频量化双线对冲套利节省充值、提币的矿工打包时间和打包费用，做到“无时差”搬砖，ATS对冲操作，瞬间完成交易。

搬砖费用和套利空间

上面为了计算方便，将手续费忽略了，交易手续费和矿工费是必须考虑的事情。

要实现套利必须满足，套利价差大于双边的交易手续费。

比如：

A交易所ETH/BTC为：0.05

B交易所ETH/BTC为：0.048

两边的价差为4.16%,大于两边交易所的手续费0.4%，约有3.6%的套利空间，单次交易就可以获得如此多利润！ATS智能理财系统是24小时运转，实时进行捕捉交易，只要大于0.4%的价差，都可以进行发出指令执行套利任务！

MGC Token选择交易所标准

1安全性

2市场深度

3交易对

4交易总量

安全性：是指交易所是否安全可靠，所具备的国际公誉性，交易所是否有过安全或信用污点受众比较广的交易所相对更安全。

 

市场深度：是由在同一价位上所有未成交的买单或者卖单构成的。市场是深度越深，相对价格波动越小，在“配置”交易所时，如果安全性有保障，建议配置一家市场深度浅的交易所，价格波动会更大，当出现明显差价是，现在深度深度浅的交易所，迅速操作，然后马上在市场深度深交易所做对冲操作，那边价格还没来的波动，套利已经完成。

交易对：是指用一种资产去定价另一种资产，比如用BTC去定价ETH，就形成了一个BTC与ETH的交易对。两个交易所都这样的交易对，那么就能做双线交易。

MGC Token持有的资产放在多个交易所，这样会有更多的搬砖机会。

交易总量：各个交易所的交易频率和交易总量决定着这个交易所产生利差的概率和次数，选择相对应交易总量较高的交易所进行ATS系统操作

选择交易对

选择交易对，主要考虑选的币种是否值得长线有，说人话就是，就是这个币种是否会被越来越多的人使用、持有。

另外一个指标是，是否短期波动相对较大，波动越大套利空间越大。

那么ATS智能理财系统根据上面的原则对配置多个交易对，获得更多的搬砖套利的机会。
