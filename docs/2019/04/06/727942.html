part1: 区块链入门

新人须知

1.什么是区块链？
区块链是点对点网路，互联网技术革命科技，采用区块数据安全可信网络架构，区块链的特征有去中心化、不可篡改、共识机制、分布式数据存储、点对点网络。
2 什么是比特币？
比特币（BitCoin）的概念最初由中本聪提出，是第一个全球去中心化加密数字货币。
3 什么是去中心化？
是没有任何机构操控，依靠一套去中心化的发起机制，逐步将数据写入这种机制中，所有人共同维护一个数据库，系统自动设定奖励机制和挖矿机制。
4 什么是去中心化钱包？
钱包是存储、支付的工具，做到离线存储，所有交易都是在区块链上进行，用户的资产都在链上，不会记录用户的资产信息和私钥，私钥一旦丢失平台将无法协助找回，需务必要保管好资产的私钥和助记词。
5 什么是助记词？
助记词是用来做什么的？助记词是私钥的另一种表现形式，可以帮助用户记忆复杂的私钥，其生成的过程是通过算法而来，而用户需要备份保管好自己的助记词。
6 什么是私钥？私钥有什么用？
私钥为一种加密算法，使用私钥来加密或者是解密数据，同时私钥也是持有数字资产的唯一证明，所以用户需保管好自己的私钥，一旦丢失MGC Token将无法提供找回。
7 为什么要预留一部分矿工费？
用户的转账都是由旷工来执行的，所以在转账的时候需要确保有足够的矿工费。

Part5 MGC ATS 智能搬砖体系

23 MGC Token ATS 是什么？
ATS 即为计算机程序软件名称，全称为：Automated trade system，通过ATS技术解决人工交易时差问题，ATS可以准确深度分析市场行情，在产生超过0.4%差价之上都可已进行操作，并且不受币市价格涨跌影响获得盈利的目的。
24 ATS 高频量化交易是什么？
是一种量化程序，采用的是全球最先进的数学模型代替传统人工进行识别操作交易，采用的方式为7 X 24小时不间断进行套利，交易方式有双向对冲、币币交易双线交易，跨平台搬砖、趋势交易等。
25 ATS 智能理财系统的利润是多少？
月收益9%-25%，平均值在10%-15%左右。
26 ATS 最低门槛和封顶限制分别是多少？
ATS最低100美金等值代币即可开启，ATS一共分为三个级别，单个账户最高可开启10000美金等值代币获得ATS收益。


27 MGC Token 搬砖套利是真实的吗？
MGC Token 采用全球顶尖计算机技术来进行高频量化交易，ATS智能可以实时捕捉各个交易所产生的利差，并在第一时间发出指令完成交易操作，ATS智能系统24小时无休息运转，凡是开启ATS智能搬砖后，套利过程公开透明，用户可以实时观看后台定时更新的交易数据，搬砖套利记录真实存在。
28 MGC Token 智能搬砖是如何操作的？
简单来说是通过各个交易所低买高卖进行套利，ATS采用高频量化技术，可以理解为计算机程序软件，在不受币市价格涨跌，实现超高稳定收益，将收益的一部分给到用户。
29 没有主流数字货币，怎么开启ATS？
用户通过法币交易在各大交易所购买主流货币转入，再通过提币至 MGC Token 钱包，进行操作开启ATS搬砖套利即可获取相应收益。
30 如何将主流货币转入MGC Token?
用户需要将自己在交易所或其他钱包的主流货币进行提币操作，转入MGC Token 智能钱包。完成操作后等待区块处理即可，到账之后用户可选择存入ATS智能理财系统，还可以将 MGC Token 支持的8种主流钱包导入 MGC Token 钱包，MGC Token 将为你提供稳定增值服务，开启获利，复利倍增。
