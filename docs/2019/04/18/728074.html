二、市场分析

1. 市场背景

以上数据来自艾瑞咨询

随着国内经济的高速发展，人均收入水平也跟着逐渐涨高。

根据中国统计局发布数据显示：2017年上半年全国居民家庭月平均收入水平为6250元，其中90后与00后家庭收入高于全国居民家庭的70.8%。

如上左图显示：90后及00后群体家庭月收入在5000元以上的占比高达75%。

经济环境的变化也使得用户的消费习惯发生改变，据艾瑞数据显示：规模高达2.8亿的90后及00后的消费群体中，文娱消费占比占总支出的28.9%。其中24岁以下的用户中，视频服务和游戏服务等在线娱乐产品占用这群用户的娱乐消费时间的比例分别达到27.1%和9.2%

以上数据来自艾瑞咨询

90后及00后，这类群体所追求的娱乐也与过去截然不同，而青年亚文化也随之而诞生。

这种文化是反对传统的一元性，强调自我和多元观点表达，追求的是新奇，轻松和简单。

而动漫文化作为一种典型的青年亚文化，随着青少年群体的消费能力以及传播能力的加强，促使动漫相关产业在文化产业中的占比稳步上扬，动漫产业的产值在持续快速增长。

以2018年为例，国内的动漫产值已达1747亿元。在线动漫市场也有逐步上涨趋势，至2018年为止，在线动漫内容市场规模已达141.6亿元。

二次元产业主要分三个部分：

  1. 上游主要为内容设计/制作方面，包括盈利性的PGC公司，如：动画公司、漫画公司、游戏公司等，同时也包括非营利性的UGC部分，如原创、同人二次创作等。

  2. 中游包括两部分：一是具备发行交易功能的版权代理业务；二是具备渠道功能的内容/社交平台，如动漫电视台、杂志、出版社等传统渠道，如视频网站动漫频道、动漫视频/阅读网站、APP等新兴渠道等。

  3. 下游主要为商品授权及衍生品业务，包括：IP在不同市场渗透的商品授权业务，以及众多形式的漫展、同人展、舞台剧、主题乐园、演唱会、周边制作及贩售等衍生品业务。


从盈利模式来看，上游微利，中游多元，下游利高是二次元产业的主要特征。

而动漫产业属于二次元行业中重要的一环，其盈利模式也与之相似。

2. BILIBILI在二次元市场的位置

以上内容来自艾瑞咨询

在众多偏向动漫市场的网站与软件之中，BILIBILI最初以其出色的社区运营，丰富的二次元内容，以及无广告即可观看大多数的新番动漫等多多优点，吸引了大量二次元爱好者进驻BILIBILI。

BILIBILI最初是由ACG（即Animation、Comic、Game、动漫、漫画、御宅向游戏）而启发的一个内容社区，到如今已经发展成为具有多个模块的综合性娱乐平台，也是目前国内最为活跃的二次元社区之一。其囊括了二次元市场的中下游，其线上通过创作激励，吸引更多UP主不断更新优质内容，同时通过弹幕社交服务，保持平台的用户活跃度。

线下通过各个漫展的宣传，以及自主打造的BML(Bilibili Macro Link)和BW（Bilibili World）等动漫线下活动，将BILIBILI这个自主品牌打响。

2. 用户画像

2.1 目标用户

  • 二次元爱好者/御宅族：希望能够有个平台，能够观看新番动漫，了解二次元资讯。

  • UGC创作者：希望能够有平台能够让自己分享内容，吸引粉丝，从而获得认同感和成就感。

  • 网络视频爱好者：希望有个内容比较丰富的平台，能够满足自己日常消磨时间的需求。

《用户体验要素》（Jesse james Garret）这本书内对一个产品的用户体验要素可以从抽象到具体，从5五个层面进行分析（如上图）。

而互联网产品作为需要严密贴合用户需求的产品，这样的分析逻辑也非常适合对互联网产品进行分析，所以以下，将从战略层、范围层、结构层、框架层，以及表现层对BILIBILI app进行产品分析。

1. 战略层分析——用户需求和产品目标

1.1 用户需求

基于以上用户分析我们可以得知，BILIBILI的用户需求主要为四个方面：

  • 从内容上，BILIBILI需要满足用户对以二次元为主题内容观看以及其他综合性内容的补充，B站的主要用户群体是二次元爱好者，B站的内容也应该以二次元为主题——也即是新番，动漫相关原创等内容，同时还需其他综合性的内容补充吸引非二次元爱好者的用户。

  • 从体验上，BILIBILI以其弹幕文化而出名，而且视频画质高，观看视频无广告等皆是BILIBILI自身拥有的优势所在。

  • 从社交上，通过频道对内容主题的划分，使得用户更加轻松找到自己所需的内容和群体，便于认识更多的同好之人。

  • 从创作上，B站自身拥有过亿的用户群体，巨大的流量吸引着众多的内容创作工作者进驻B站，而且还通过创作激励政策，鼓励更多UP主更新更多有趣的内容。

1.2 产品目标

基于以上的用户需求分析，BILIBILI的产品目标应为：以二次元文化为主题，打造一款适合年轻人的潮流文化娱乐社区。

1.2.1 基于战略层的个人建议

相对于其他的视频网站，B站的核心竞争力主要呈现在以上几点。

BILIBILI以其独特的文化吸引着众多用户，这些B站特有的文化也是增加产品的用户粘性所在。

然而，当前的B站，随着大量新用户的入驻以及产品内容板块的扩充，B站的二次元文化主题被严重的稀释了。B站为了吸引更多的用户，添加了很多三次元内容的板块，而这些三次元内容的增加，导致站内二次元相关内容的比例缩减。

以首页推荐界面为例：目前首页推荐视频内容大部分为三次元相关视频，之前B站独特的鬼畜，MMD等相关内容基本很少出现，而且内容多以搬运内容为主，内容呈同质化现象等，这些内容都比较大的影响B站核心用户（即二次元爱好者和原创UP主）的的使用体验，减弱核心用户的黏性。

基于此，本人基于战略层的建议是：及时调整二次元于三次元内容的比例，加强原创内容的推送，鼓励用户创作内容，限制其他内容的搬运。

2. 范围层分析——内容需求和功能需求

2.1 内容需求

结合以上用户分析以及战略层的产品分析，BILIBILI的内容需求可分为：用户的内容需求，以及产品的内容需求。

其中用户内容需求可大致分三个部分：

  • 二次元内容：主要是以ACGN(即动漫，漫画，游戏，轻小说)为主，内容来源主要为来自日本以及国创的原创内容以及根据这部分内容的再创作内容。这些内容是二次元文化的核心，也是BILIBILI的主题相关内容。

  • 5次元内容：主要为二次元内容的衍生内容，比如手办，偶像等基于二次元的文化核心从此在三次元，也就是现实世界中所创作的实际存在产品。

  • 三次元内容：基于现实世界的内容创作，也就是我们所熟知的主流文化相关内容。

而从产品运营层面，BILIBILI内所需包含的产品内容需求则大致为以下方面：

  • 用户原创内容：通过原创或二次创作，增加站内视频内容的丰富性，吸引更多新用户的同时，加强产品的用户黏性。

  • 用户社交：通过内容来组织用户的社交活动，促进用户之间的互动交流。

  • 流量变现：把积累的流量变现，增加产品的营收。

  • 信息收集：搜集用户的个人信息，浏览痕迹，把用户进行标签化，方便运营和推送相关内容，提供用户的产品体验感。

2.3 基于范围层的个人建议

正如上述战略层分析的建议所描述，B站目前核心的二次元文化主题被稀释，内容呈同质化现象，个人认为其主要原因是二次元相关UGC内容占比减少，内容推送机制不善。

具体分析为以下几点：

创作机制政策的不完善：

根据B站的创作机制,，UP主的收益主要根据上传视频的数据，比如说：播放数作资格判断，再根据内容流行度，用户喜欢度等进行综合判断，计算UP主的创作收益。

但是，二次元的这种亚文化并非属于大众流行文化，其受众数量也逊色于主流文化内容。况且，仅凭播放数、点赞数等相关数据并不能判断视频的质量度。更何况，搬运视频，微创作视频相对于站内原创视频由于有原视频的流行加持，更容易获得创作激励。

内容审查制度不完善：

站内对于搬运视频监管力度不足，导致标题党，营销号盛行，站内内容质量下降。

内容推送机制不完善：

站内的内容推荐，主要根据用户最近浏览记录，以及热门度进行视频推送。

以首页推荐界面的内容推送为例：每次刷新会更新十个视频，其比例大约是“4个最近浏览相关视频+4个站内热门视频+2个文章/直播/信息流广告”，基于上述两点原因，导致推送内容质量度低，UP主的原创内容没有展现，内容同质化严重。

基于以上原因，本人提出建议如下：

  1. 加强内容审查，在视频结束后提供用户评分功能（参照NICONICO的番剧评分），对低分视频进行及时审查，删除。

  2. 增加上传视频的时长限制，减少超短时长的视频数量。

  3. 优化视频的数据限制，比如用户观看多少分钟后才计算播放数以及允许用户点赞，转发，收藏等相关数据。

  4. 创作激励政策倾斜，B站是以二次元文化为核心的视频社交网站，针对二次元相关的原创视频进行政策倾斜，加大激励力度。

  5. 增加用户标签功能，在用户注册成功后，提供标签供用户选择感兴趣的内容标签，根据用户标签推送相应内容（参照twitter,facebook）

  6. 完善推送机制，建议推送机制是3个关注UP主相关内容（推送用户未观看内容，最近更新优先展示）+4个用户标签相关内容（推送高评分内容）+2个二次元相关内容+1个直播/文字/信息流广告（参照youtube），这样做鼓励用户和UP主之间建立更紧密的社交关系，使得原创内容得以展示，保证UP主的利益。

  7. 添加拉黑功能，用户拉黑UP主后将不再展示该UP主的相关内容。

3. 结构层分析——信息结构和交互设计

3.1 信息架构图

从图中可知：BILIBILI的信息结构主要分成5大模块，分别是：首页、频道、动态、会员购、我的。

  • 会员购模块：主要是BILIBILI的商城功能，以其自身庞大的平台流量为基础，通过提供二次元相关的衍生产品的购买服务，提供B站营收。这个模块主要用来打通二次元的下游市场。

  • 我的模块：则是用户中心功能，方便用户管理个人信息。

3.3 交互设计分析

页面布局：

B站的页面布局采用的是分割线设计，把不同的内容分割展示，这种布局模式能够方便用户了解界面，赋予页面内容组织性。

手势操作：

BILIBILI的手势操作比较简单，一般来说，只需要一根大拇指即可完成对整个产品的大部分操作，方便用户比较容易上手使用产品。

页面跳转：

页面跳转应符合用户的视觉顺序，一般人的视觉顺序都是“左→右”，“上→下”。

而Bilibili的页面跳转主要有四种方式：

  • 直接跳转：这种跳转方式使用得比较少，仅在选择底部TAB导航标签时出现。

  • 左右跳转：这种跳转方式是BILIBILI产品内使用最多的跳转方式，其使用的地方一个在于通过左右滑动实现分段式的二级导航页面的切换，另一个则是站内大部分的单击选择所发生的页面切换都是左右跳转，点击确定是从右往左跳转，返回则是从左往右跳转。

  • 翻转：视频打开全屏，页面呈翻转跳转样式。

  • 搜索框入口跳转：点击就给与获取光标的感知，所已做好做简单的联动过渡效果。

  • 添加会员等级权限，在用户中心增加等级权限说明。

  • 增加用户奖励政策，鼓励用户进行评分，点赞等操作，将其以每日任务形式推出，在个人中心增加每日任务并说明奖励政策。(比如：加快升级，限量提供付费番剧播放次数等)

  • 在个人中心添加个人标签选项，方便用户添加或更换标签，添加“不感兴趣“和”最近想看“的两个标签库，方便定位用户兴趣，方便推送用户想看的内容。

基于交互设计方面，本人还想提出如下建议：

  • 增加视频小窗播放功能，方便用户进行观看和选择的双向操作。（目前只有直播有这个功能）

4. 框架层分析——界面设计，导航设计&信息设计

4.1 界面设计

登陆界面：

从上图可见：B站的登陆界面非常简洁，仅有注册、登陆、忘记密码三个按钮和两个输入文本框，其中登陆和注册按钮颜色不同，且登陆按钮置于右侧，这方便用户快速找到登陆按钮的同时，方便了多数用户的右手使用习惯，仅用右手就能完成登陆操作。

值得一提的是：输入框上面的吉祥物22娘和33娘，在输入账户时，是睁着眼睛；而输入密码是则会自动捂住眼睛，这样的细节设计是能够让用户会心一笑的交互体验。

内容展示界面：

内容展示界面主要采用网格式和垂直列表式分布，这样做是因为B站是视频社交产品，其图片和视频的展示比较多，这种采用图文结合的展示方法有利于充分利用整个空间，而且每次展示内容个数符合交互设计的“7±2”原则，内容排版从上至下，从左至右，符合用户的浏览习惯 。用户能够准确找到自己喜欢的内容的同时，也不至于在庞大的视频推荐中选择困难。

播放界面：

非全屏播放时界面分成三个部分：视频播放内容、视频相关信息以及相关视频推荐，整体分别从上往下排列。

这样的界面设计的好处有以下几点：

  • 视频播放主要以观看为主要，置于最上方方便吸引用户的注意力，同时还能防止因为用户用手操作手机而影响用视频观看。

  • 视频相关信息主要以文字为主，置于中间位置方便用户一眼便能了解整个视频的大致信息，同时这里的手势操作主要以单击为主，用户操作次数较少，并不会手势操作不方便的影响。

  • 相关视频推荐置于最下方，这个位置拇指最容易进行操作，这类则考虑到用户在观看视频的同时，还会进行其他内容查找的动作。这类的手势操作有滑动和点击，主要以滑动为主，频率一般为一次以上，考虑到用户操作方便，把相关视频推荐放在底部最为合适。

全屏播放时，界面的功能按钮分布如下：

  • 用户在视频观看中经常使用的功能按钮置于最下方（比如暂停，弹幕，画质，进度条），方便用户对视频进行功能上的操作。

  • 界面右侧是截图，GIF制作，以及锁屏等功能键。

  • 界面右上方为点赞，踩，投币，转发等社交相关功能键。

  • 界面右上方则是退出键以及视频标题。

4.2 导航设计

B站的导航设计模式主要有两种：“tab标签式的一级导航+分段式”和“宫格式的二级导航”

  • 全局导航：底部的5个TAB，囊括了B站产品的五大信息模块，是模块入口扁平画，方便客户使用产品，全局导航按钮置于最下方，方便用户的手势操作。

  • 局部导航：首页上方有4个分段式导航控件，其占据空间不大，方面界面展示更多内容，这类的界面切换主要通过左右滑动的手势进行，置于上方不会影响用户的手势操作。

  • 辅助导航：频道模块则采用宫格式导航，把全站内容划分成32个模块，能够给客户展示大量的内容，这样的设计能够同时显示大量的功能，使得功能扁平化，方便在有限的空间内最大程度的展现。

4.3 信息设计

内容信息：

视频内容信息主要通过卡片的样式来展示，卡片包含的信息有图片、标题、标签、up主、时长、播放数等相关信息。卡片式设计能够以较少的空间展示大量的内容，同时整体视觉整洁有序，方便用户选择。图片和标题是主要内容，占据卡片的大部分空间，这样的设计方便用户在第一眼就能大致判断视频内容信息，减少客户的选择和考虑时间。

用户个人信息：

用户个人中心主要分成四个板块：用户资料、用户创作、用户的产品使用和用户服务。

这四个板块基本记录了用户使用产品的整个过程，包括了：用户的社交信息、产出信息和输入信息。

个人主页则主要反馈了用户的产品使用信息，能把用户感兴趣的内容收集到一个页面，方便用户关注和寻找。

错误提示信息：

以上三个状态分别是“未登陆”、“页面加载失败”、“搜索内容失败”的三个提醒信息界面。

  • 未登陆时，会在界面内容的最上方有一个专门的卡片提醒用户登陆。

  • 加载失败时，会有提醒界面告诉用户可能时网络问题等原因，建议再次刷新。

  • 搜索内容失败时，会告知用户目前站内没有相关内容。

这几个界面都通过图文结合的结合的形式把提醒信息传达给用户，利用吉祥物22娘和33娘的可爱卡通形象，在一定程度上可以缓解用户在操作失败后内心的失落感。

同时，用特定的姿势配合文字信息能够更好地传递给客户，比如：未登陆提醒卡片内有22娘和33娘的招手姿态，配合文字告知用户登陆后更多精彩。再或者如搜索内容失败，以两个吉祥物失败倒地的姿态配以文字以及颜文字“T T”传递给客户她们两个一样感同身受的样子，缓解了用户的失落感。

5. 表现层分析

范围层主要表现产品的视觉设计效果，以下主要分析打开页面和推荐首页的视觉效果设计：

1）打开B站，首先看到的时左上图的非常吸引用户眼球的卡通形象，这里面包含的元素有22娘，33娘以及啤酒，B站通过这个图片来直接向客户传递“哔哩哔哩 (゜-゜)つロ 干杯~”的产品SLAGON信息，强调了B站的二次元文化的主题。

2）用户的目光驻点停留再手机的中心位置，打开软件后，产品跳转的第一个界面是首页的推荐界面，推荐界面的中心部分是视频推荐。这样可以直接把用户的主要转移到内容之上，这个界面有如下特点：

  • 内容展示呈网格式集中分布，满足交互设计中的“接近原则”，促使用户集中注意力。

  • 每个内容以卡片式展示，图文结合，小范围空间展示大量内容，方便客户认知。

  • 内容之间通过留白的方式进行区分，这是最近交互设计的趋势，能够让这个界面内容展示显得自然而不突兀。

  • 一次展示6个内容内容展示，满足交互设计中的“7±2法则“和“希克定律”，给用户适当量的选择，减轻用户的决策成本。

  • 对于未登陆用户，内容置顶层有提醒登陆卡片，方便用户直接跳转登陆页面。

3）当用户目光焦点移动时，根据“左→右，上→下”的用户习惯，软件的上下方分别是导航栏。

最上方主要是搜索框，这里采用了粉红色的色调，提醒用户这里的功能很重要；最下方是软件的一级导航栏，其常用性从左到右依次减少。底色是白色，图标和文字采用了灰色的色调，相对于黑色而言，灰色会显得端正而不会过于严肃，同时图标设计非常可爱，这样不仅能反映出这部分按钮的重要性，也能让用户感到亲近，突出导航入口扁平化的效果。

4）总的来说，整个页面采用了粉红色和白色的色调，给与用户一种小清新的感觉，比较符合御宅族及二次元文化爱好者的视觉需求。整个界面的内容分布页能够符合交互设计的需求，不会让用户感觉视觉疲劳和选择困难。
