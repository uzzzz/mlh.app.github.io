正文

个人感觉，对我而言一边实习一边准备秋招比较合适，就投递了几个实习生岗位，很巧的是前后两次面试的公司都是美团，且都为酒旅事业部，两次面试都拿到了实习 offer，正式岗面完还在等消息中…

一、周边游频道实习生

一共两面，共计 90min，感觉过程不是很流畅，面试官很疲惫，我也是。

问题梳理：

  1. 介绍之前两段实习经历，详述产品形态，对自己所负责的部分进行追问。

  2. 说一说自己的优点。

  3. 有没有问题问我。

  4. 为什么想做产品。

复盘总结：

对自己之前的工作要比较熟悉，面试之前自己重新回想准备清楚，因为你不知道面试官会问到这么深入，当时面试官直接下载了我提到的 APP，让我指明哪部分是自己负责。每个点都会详细的问，为什么这么做，有什么根据，做了之后有什么改变，这样的方案是最优的吗，会不会有这样那样的负面影响，当时有没有考虑到，如何评价这个产品等等…

整体状态一定要放松，在这场面试中，我越来越紧张，越来越疲惫，我感觉这种情绪也会影响到面试官，她如果得不到自己想要的答案，会持续的追问，每次追问都是面试者的机会。之后的面试抱着一种佛系的心态，表现就比这次好一些。

二、境外度假实习生

只有一面，30min 左右，基本上就是聊了之前的实习经历。

问题梳理：

  1. 第一段实习经历，产品介绍，自己做的比较主要的工作有哪些？如何构思设计？

  2. 第二段实习经历，平时工作流程是怎样的？当时写的需求文档包括哪几部分？

  3. 为什么做产品？会不会 SQL？

复盘总结：

  1. 放松，放松，对我这种一紧张就结巴的人来说，这一点是最重要的，这次面试之前抱着不过就拉到的想法，反而感觉相对放松些，能表达出更多的内容。

  2. 美团这边的话，产品会 SQL 好像比较有优势，两次都问到这个问题，可惜我不会。

三、校招-助理产品经理

校招找了亲友内推，投递时选择了酒旅事业部，去年也投了内推，结果没通过，今年还比较幸运，直接可以面试，不用经历简历筛选、笔试。

问题梳理：

  • 一面比较顺畅，主要是针对个人经历进行提问，常规实习内容提问之外，你认为最有成就感的事是什么，有没有你觉得比较好的需求，被老大否定？你知道为什么否定吗？

  • 二面难一些，面试官在听我描述某个需求时，从中找更加详细的点进行提问，与一面时自己讲别人听的形式有所区别，会从产品设计的角度提问做需求的原因，持续追问，很明显段位更高，与一面考察内容不同。二面，工作场景问题，如果开发执意不做某个需求，该作何处理？产品设计问题，以提高点击量为目的，你认为电影搜索结果中，不同信息展现的优先级是怎样的？

  • 三面，总监面，主要是他在说，我在听 233…问了你觉得产品工作中比较重要的事是什么？因为个人对酒旅产品使用不多，问了为什么外卖中满减、折扣不能同时使用？好像没有别的了。

  • 四面，HR 面，主要问了以后的职业规划，家庭情况，offer 情况，期望薪资等。

复盘总结：

  • 之前工作的总结也需要以结构化的方式进行表述，这样听者会比较清晰的get到你的点，这个需要提前做好功课。

  • 回答问题前，自己先思考一下面试官问问题的意图，然后快速梳理回答的要点，产品设计类问题，要从需求出发，再考虑产品设计，不能本末倒置，做任何事先从目的出发。

  • 在回答问题时，可以多结合自己过往经历，自己的优点进行说明，谨记 star 法则，不要仅仅是回答面试官的问题，面试官有时候想问题也挺费劲，还不如自己多说点东西，说不定某个点就能触动他们。

  •  一定要准备好问面试官的问题，每一面他们都会问，你还有什么问题吗，最后总监面和HR面，问题没准备好，导致问的问题有点尬，个人觉得有点减分。
