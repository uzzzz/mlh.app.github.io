2018年，一名16岁美国少女在与朋友们玩闹时，被自己的闺蜜从18米高的桥梁上推落，导致6根肋骨骨折、两肺部刺伤、喉部出现血肿、肩部旋转肌腱撕裂、全身淤青。

推这个动作本身不会造成太大伤害，但配合周边的环境，结果就不一定了。

最近，美国法院宣布对推人女孩的判决：入狱2天、罚款300美元、社区劳动38天，且两年内不许与受害女孩联系。不少网友认为——罚轻了。

其实，我们的人生里多多少少、或轻或重的都推过别人，没推过人的人生就不是完整的人生，为什么我们总想推别人一把呢？

推人的乐趣

作为地球上唯一双手解放，能独立行走的动物，我们人类是从骨子里钟爱推人。由于人类行走模式的不同，人类摔倒的姿势比其它动物都更夸张，至少幅度更大。

所以在人类社会里，推就成了常见的肢体接触......

表达愤怒推一推：

故意伤害推一推：

论地铁防护门的重要性

测试平衡推一推：

你们让以后的人工智能怎么想？

嬉戏玩闹推一推：

乘虚而推

因为人类太热衷于推人，现在连喵星、汪星人都学会并熟练掌握这个技能了。

四两推千斤


推其不备

推这个动作如此普遍，甚至传染给其它动物，导致人类热爱推人的根源是什么呢？ 

推人的心理

直立行走，解放双手，给了我们推人的机会，但心里没有推人的念头，也不会动手推别人。除了那些恶意致人死伤的神经病，生活中还有一些小罪恶会令人将自己罪恶的小爪子伸向别人......

找乐子的本性

人类是一种特别不耐寂寞的动物，最怕无聊，所以我们会想尽办法找些乐子来满足自己。那么乐子从何处来？

乐子是一种精神食粮，可以通过创造或吸纳来获得：吸纳是通过聚会、喝酒、闲聊、看电视等获得短暂的愉悦；创造是通过自己的双手来塑造某种东西或完善某种技能，像绘画、写作、雕刻、演奏以及各类体育项目都可以令人愉悦，且时效更长。

啥？上面两个你都不感兴趣？那捉弄别人的第三种找乐子方式可能满足你。除了上面两种途径，释放人性中的“小恶”就成了人类找乐子的第三种途径，几乎所有的人，都曾做过“小恶”，比如下楼时推一把前面的同学，伸脚绊一下路过的朋友，戴个面具吓吓毫无防备的室友......

我不是没溜儿的人，我没溜儿起来不是人

我们骨子里都是“没有乐子会死星人”。而且没溜儿这事儿属于传染病，在人群中扩散的很快。

习得性没溜儿

常言道：近朱者赤近墨者黑。人类生来是善于学习的动物，不止身边人的举动，就连电视上、电影里的“壮举”都是人类乐于模仿的内容。

在荧屏上充斥着各种主人公走投无路，跳水求生的桥段时，广大少年儿童以及没常识的成年人就会天真的认为这些事不会造成伤害。

你以为人跳水时的样子：

危险动作，请勿模仿及鼓动别人模仿！

其实人跳水的样子：

头撞到跳台或岩石，哪个比较疼？

还可能是这样子：

有些国家的运动员都不敢保证自己的入水姿势正确

好吧！这张是合成的假图，但道理是真的

水虽然是液体，却依然可以让人受伤，因为野外水域的地形复杂，有可能撞上水底的石头（及其他人的头），游泳馆（深度够）的水池虽然安全，入水姿势不对（横拍），依然会受伤，就像曾经在跳水节目里发生事故的韩庚，即使运动神经出众，也会在“拍”水后当场昏厥。

同样的高度，即使你横拍下去毫发无伤，也不要推别人下水。因为人与人的体格强壮度不一样，即使你自己跳水后感觉OK，别人也可能轻则骨折、重则致命。当然，更缺德的是那些专门拿别人做实验的人。

缺德的好奇心

有些人虽然不是科学家，但天生有一颗随时随地做实验的心，好奇是人类的天性，据说婴儿总喜欢把手里的东西扔到地上，心理学家认为这是婴儿在感受自由落地的运动。

何止人类想做这种低成本的实验？

随着年纪的增长，我们的实验团队、材料、创意也日新月异，心理成熟度完全跟不上坏主意的发展速度。

慢慢地，我们期望看到更大质量物体的自由落地运动，比如你的好友或室友，所以总想在他们下楼时推他们一把。

不得不承认，人类与楼梯产生的摩擦力反应，很有趣，具有多样性......

侧摔（找只喵星人客串一下）


滚摔


遇到这样的，不如好奇一下一会自己怎么死吧

比起上述出于好奇心产生的推人冲动，有一种推人心理更要不得！

无脑服从性

有些网友为这个推人闺蜜叫屈，表示闺蜜是在“女孩的要求”下推她下去的。

确实，很多人类都有服从心里，容易轻信他人的话，他们或许是年龄小、缺乏分析问题的能力、没有独立思考能力，总之很容易受到他人的煽动，即使被推女孩没有要求闺蜜推自己一把，闺蜜也可能在众人的要求下把她推下去，还有些人甚至可能在众人的要求下自己跳下去......

为什么每个人出生时都有自己的脑子？就是为了让每个人都有独立思考的能力。为什么有些人就是不能独立思考？

推人的后果

生活中，朋友之间除了物理上的推，还可以有精神上的推。

之前奚梦瑶逼王嘉尔吃辣椒，在王嘉尔明确表示自己不能吃之后，奚梦瑶还是“推”了他一把。

汗如潮水

被逼吃辣椒的王嘉尔反应可以说是“立等可见”，还有一种立竿见影的反应叫做过敏，酒精过敏的林志玲，曾经就被那英逼酒，虽然林志玲表示自己喝酒会过敏，但那英不信，觉得她装，于是林志玲被那英“推”着干了一杯酒，现场演示了一把教科书级别的酒精过敏。

脸上、前胸红疹+红肿

那英可能不知道，人虽然是杂食动物，体质却各不相同，会对各种常见食物过敏：海鲜、鸡蛋、花生、荞麦......都是能导致部分人轻则（短暂）毁容，重则害命的“健康美食”。

很多人觉得自己能承受的，就是别人能承受的，这种用自己的标准衡量所有人的思维是人类的思维bug，很多人由于阅历浅不信别人会过敏，或好奇别人过敏什么样子，总喜欢强人所难的“推”别人一把。

这种擅长“推人”的人，也许本性不恶，却真是大写加粗的愚蠢。

当你的“推”要了朋友的命时，这还算恶作剧吗？

小编认为蠢人比狼人更可怕，因为他们能一脸无辜的毁掉一个人的生活，且防不胜防，所以愚蠢也是一种反社会的行为。
