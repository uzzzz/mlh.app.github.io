最近，研究人员正在努力证明，我们的胃、肠道和身体其它部位发生的事情对我们的生活质量有着巨大的影响，而且这种影响往往是无形的。


我们消化道和皮肤上的细菌、真菌、病毒和原生动物——也就是我们所说的微生物群落，或微生物组——可能在决定我们的健康、幸福感、情绪、寿命甚至体重方面发挥着巨大的作用。微生物组研究人员认为，在决定个人命运方面，微生物组可能发挥着与DNA同样重要的作用。


和DNA一样，微生物组也非常复杂。它由超过100万亿生物组成，包含的遗传物质是人类基因组的200倍。仅仅分析一个人的肠道菌群就可能需要一生的时间。



而在人工智能（AI）和最先进的计算硬件的帮助下，这项任务已经缩短到几分钟，只需扫视一眼，就能识别人体生物群系中的每一种微生物，检测人眼看不到的模式和变化，并分析它们之间的相互作用。


“美国国立卫生研究院人类微生物组计划”这样的微生物组研究的目标就是证明对人类微生物组进行表征的可行性，以期更好地诊断和治疗疾病。

说到技术和人类健康，遗传学近年来最受瞩目。


人类基因组计划的DNA测序项目于2003年完成，绘制了由30亿个化学碱基对组成的20500个人类基因的图谱。现在，科学家们可以了解哪些基因位于DNA链中的哪个位置，以及它们的功能可能是什么，从而为每个患者量身定制医学治疗方案。


近年来，为了了解人类健康及改善健康的途径，微生物组受到了越来越多的关注。尽管微生物组早已为人所知，但其复杂的构成限制了研究人员理解它的能力。直到现在也是如此。


我们的身体至少包括和人类一样多的微生物细胞——这些细胞在组成和结构上与我们身体的细胞有所不同。事实上，微生物组本身就是一个有生命的有机体，重约5磅。这些细胞充斥着我们的消化道；内部器官；以及我们的嘴，鼻子，眼睛和皮肤。微生物群落的组成因人而异，它由数百万个基因组成，人们普遍认为，它们与人体基因组以及免疫系统相互作用，从而产生健康或生病的条件。



基因组学和生物组学就像两个复杂的谜题一样，可能会揭示出许多造就人类的秘密，正如美光执行副总裁兼首席商务官Sumit Sadana在最近于旧金山举行的 Micron Insight’18大会上所述，“这些就是研究人员试图破解的密码。”



微生物组如何定义我们？



基因组学和生物组学研究最令人兴奋和困惑的方面之一就是它们可能提供通向未来的窗口。我们的DNA和微生物组如何定义现在和未来的我们？它们为我们的健康做出了哪些预测?我们容易患哪些疾病？


在遗传学中，科学家寻找一个基因或一组基因的缺陷，以确定一个人患某些疾病（如阿尔茨海默氏症或糖尿病）的可能性。未来，治疗方式可能包括替换或修复这些突变基因（也被称为基因疗法）。不过，就目前而言，我们只能依靠基因；我们能做的最好的事情就是选择能提高我们健康几率的生活方式。



然而，一个人的微生物组可能会通过饮食变化、某些药物甚至锻炼而迅速改变。因此，能够通过检查肠道来发现正在发展或即将发生的疾病的迹象——这对于改善健康，甚至延长寿命，都是非常宝贵的。


利用人工智能和机器学习，科学研究旨在随着时间的推移捕捉尽可能多的人的微生物学特征，以了解哪些变化预示着潜伏的疾病。研究人员也可能通过操纵生物群系而找到预防甚至治愈疾病的方法。


这个理念就是：既然每个微生物组各不相同，那么每种治疗方式都可以根据个体基因突变和生物构成而进行调整。因此，研究人员正努力从简单的诊断转向预防和个性化医疗。


预测性：人们认为，在生病前的几个月甚至几年，人体微生物组就会发生变化。但是饮食、压力、锻炼和其它因素也会改变微生物组。医学科学家正在努力了解哪些变化是由于生活方式的差异而造成的，哪些预示着新生疾病。检查健康人的微生物组可以提供一个基线，以便分析任何随时间而发生的变化。


预防性：研究人员希望利用充斥人体系统的微生物种类及其活动，帮助提出有可能预防糖尿病、癌症和痴呆症等疾病的建议。研究人员希望把基因数据与微生物组数据相结合，作为预防保健的两连击。


个性化：到目前为止，“个性化”或“精准”医疗主要依赖于家族史和DNA分析。在“个性化”或“精准”医疗中，治疗方法因个人独特的生物学特性而异。研究人员正在通过来自微生物组的数据，寻求更精确的治疗方案。


由于每个人的生物群系中都有数以万亿计的微生物需要检测，因此分析速度一直低于预期。随着先进计算机系统以及美光内存驱动的人工智能的进步，研究人员有望在几分钟内完成人类需要多年才能完成的事情。


让我们来看看技术是如何加快基因组测序并降低成本的吧！根据美国国家人类基因组研究所的数据：

  • 第一次人类基因组测序开始于1990年，耗时13年完成，耗资10亿美元。

  • 10年后，DNA测序耗时两三天，花费5000美元。

  • 如今，对一个人的完整基因组进行测序只需要20分钟，花费仅600美元。


“随着内存技术的进步，完成这项对人类健康至关重要的分析的时间和成本只会越来越少。微生物组分析也是如此，通过更快、容量更大和更复杂的人工智能技术，分析微生物组成为可能。”

美光业务拓展经理EricCaward

微生物组到底能帮人类解决什么健康问题？



研究人员正在研究微生物样本并针对健康诊断和护理进行分析。不久之后，护理人员除了采集我们的血液和尿液，可能会采集我们的微生物组样本，作为常规健康检查的一部分。



然而，要想以有意义的方式分析这些样本，科学家们需要一个数据库。美国国立卫生研究院的“我们所有人”计划旨在采集100多万人的数据，包括微生物组样本数据，以造福精准医疗。美国国立卫生研究院现已中断的人体微生物组项目进行了多项研究，探索肠道菌群在人类健康的各个方面所发挥的作用。其它许多项目正在进行中，以下领域有望得到改进：


改善免疫系统。微生物作为免疫系统调节器的作用已经被广泛接受。根据科学网站ScienceNordic上的一篇文章，从我们的婴儿时代开始，肠道中的细菌就开始教我们的免疫系统“如何才能表现好”。这些微生物也会影响我们消化道和其它器官的炎症程度。


预防霍乱。来自杜克大学、麻省总医院和孟加拉国达卡国际腹泻病研究中心的研究人员正在使用机器学习算法来检测生活在消化系统中的细菌群落的模式——人类用肉眼肯定看不到这些模式。由于并非所有接触过霍乱的人都会感染这种导致腹泻的疾病，人工智能可以帮助研究人员了解哪些人有患病风险，以及患病原因。它还可以帮助开发疫苗。


药物的相互作用。根据美国国家医药图书馆的摘要，我们假设微生物组的构成会影响我们对药物治疗的反应以及我们对某些药物的代谢程度。


老化。微生物组与健康衰老之间的关系是加州大学圣地亚哥分校和IBM研究院健康生活人工智能中心关注的焦点之一。该中心研究基因、环境因素、日常习惯和人体微生物组对老年人认知能力的影响。该研究已经证实，帕金森症及其它与衰老有关的疾病在微生物组中有一定的基础。该中心使用人工智能技术对研究和抽样提供的大量数据进行筛选和分析。



道理我都懂，但这能对个人产生什么影响？



人工智能算法可以帮助医学领域更好地理解微生物组的复杂组成和相互作用，并找到人类研究人员无法找到的关联。最终可能在治疗包括糖尿病和癌症在内的一系列疾病方面取得突破。然而，目前的分析仅限于把一个人的微生物组与数据库中的其它微生物组进行比较。


微生物组研究可帮助预防糖尿病


美光公司员工PatriciaReiter委托提供这项服务的一家公司对她的微生物组进行分析。参与该项目时，客户首先要在邮件中寄去少量粪便物质，然后就会收到书面分析报告。Reiter之所以选择这项服务，是因为她有消化问题。这份报告让她大吃一惊：尽管她是一名鱼素者（不吃肉但是可以吃鱼类和海产），但该公司认为她没有摄入足够多样化的食物。


Reiter 说：“他们排除了很多可能性，包括乳糜泻和克罗恩病。我一度对这个监测过程持怀疑态度，我想一定是某种疾病或什么东西引起了我的消化问题。让我安心的是，检测结果让我知道我的身体并没有患这些病的倾向，但愿增加饮食多样性能让患病风险会更低。”


特别值得一提的是，专家建议她增加“益生元”（芦笋，香蕉，燕麦和苹果）和“益生菌”（酸奶，康普茶，泡菜和其它发酵食品和饮料）食品的摄入量。一些推荐的食物——如菊芋和牛蒡根——在她看来很奇怪，但她希望膳食的改变能改善她的整体健康状况。



美光： 人工智能支撑医疗技术变革



要分析这些复杂微生物的组成、结构和相互作用，人类需要机器的帮助。幸运的是，带宽、内存和速度方面的进步正在为人工智能技术铺平了道路——其中美光处于领先地位。



为了满足这些新功能，美光开发了三种重要的内存产品：低功耗、高密度DRAM；快速NAND闪存；以及3DXPoint™技术。



Sadana说：“美光的产品组合是云数据中心人工智能最全面的产品组合，它涵盖了方方面面。”


研究人员希望告别“一刀切”方案、告别猜测。基因组学、生物组学、活检、虚拟和预测诊断以及其它人工智能辅助领域致力于实现比以往更具预防性、预测性和个性化的精准医疗，从而让人们过上更长久、更美好、更幸福的生活。


Sadana 说：“人工智能将改变医学。”


而这些变革，现在正在由内而外地发生着。
