Mathpix将截图转换成LaTeX可编辑文本

  • 柚子社区

在 Linux、macOS、Windows 系统下使用 Mathpix 可以截取到复杂数学方程式的截图，并可以转换成 LaTeX 可编辑文本，在选写学术及科学文献时经常会用到。编写 LaTeX 文档最麻烦的部分是那些复杂的方程式，使用 Mathpix 就可以搞行了，同时使用 Mathpix 可以截取数学方程式的截图，它会立即为你提供 LaTeX 代码。然后，你可以在你最喜欢的 LaTeX 编辑器中使用此代码。Mathpix 是一个免费使用的工具，但它不是开源软件。macOS 和 Windows 版本直接下载安装包进行安装，在 Linux 系统中可以使用 Snap 安装方式安装，即不用去下载 Mathpix 软件包也能在 Linux 下安装。

Mathpix下载链接

https://mathpix.com/

在 Windows 及 macOS 下安装 Mathpix

直接到 Mathpix 网站下载 exe 文件或者 dmg 文件就可以安装了。

 

在 Linux 下安装 Mathpix 的方法

Mathpix 适用于 Linux、macOS、Windows 和 iOS。暂时还没有 Android 应用。在 Linux 下 Mathpix 有一个 Snap 包。这意味着如果你在 Linux 发行版上启用了 Snap 支持（至少有4100款Ubuntu Linux软件都支持使用snap安装），你可以用这个简单命令安装 Mathpix：

sudo snap install mathpix-snipping-tool


使用 Mathpix 的方法

使用 Mathpix 非常简单。安装 Mathpix 后，打开该工具。你会在顶部面板中找到它。你可以使用键盘快捷键 Ctrl+Alt+M （也可以自定义设置）开始使用 Mathpix 截图。它会立即将方程图片转换为 LaTeX 代码。代码将被复制到剪贴板中，然后你可以将其粘贴到 LaTeX 编辑器中。

结论

Mathpix的光学字符识别技术正在被像 WolframAlpha、微软、谷歌等许多公司用于在处理数学符号时提升工具的图像识别能力。总而言之，Mathpix对学生和学者来说是一个很好的工具。它是免费使用的，如果你在找在LaTeX中处理数学符号的工具，那就选择Mathpix 吧，因为容易安装及使用。
