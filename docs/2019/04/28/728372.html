ElegantBook 书籍模板更新（v3.07）

  • 柚子社区


ElegantBook 模板基于 LaTeX 的 book 类，专为 LaTeX 书籍而设计。作为 ElegantLaTeX 最为经典的模板，其定理样式、章节样式和封面被人熟知。


如果你有其他问题、建议或者报告 bug，可以给我们留言。如果你想了解更多由 ElegantLaTeX 项目组设计的模板，请访问


https://github.com/ElegantLaTeX/。


本次更新为 ElegantBook 3.07 版本，此次更新主要涉及到颜色主题等内容。原先颜色主题仅保留了默认的绿色主题，其他颜色主题均做了更新，并新增了灰色与黑色颜色主题，方便与简化的定理模式结合。并且将章节和定理的颜色分开，从原先的 3 颜色控制变成 4 颜色控制。


并且这次更新引入 simple 和 fancy 两种定理模式，方便对于较为密集定理类环境的书籍排版。

简化定理模式说明：

在以往，如果一本书中连续出现过多定理，定义或者其他定理类环境，整个书籍会显得非常多区块，这也是最早 ElegantBook 作为 ElegantNote 改良版出现的原因。


但是，即便在 3.x 版本，这个问题其实也存在，不过因为 3.x 版本采用了 tcolorbox 之后，有很大改善。在这个版本，我们将引入 simple 和 fancy 两种定理模式，其中，fancy 模式为之前的定理模式，也就是 tcolorbox 所定义的定理类环境。而 simple 模式由 amsthm 宏包经过双重定义之后得到的简化定理样式。

更新内容


此次更新内容：

  • 删除自定义字体（系统依赖），未来将不会再考虑。

  • 增加灰色和黑色颜色主题。

  • 更新原有的颜色主题，但保留原默认主题。

  • 可以隐藏章标题后的 base 图案。

  • 加入 simple 和 fancy 定理模式。

  • 提供选项支持隐藏 solution 和 proof 环境。

  • 目录 bug 修复。

  • 改善其他细节等。
