马云谈与腾讯竞争：只是一个伴侣，不是要打败的对手

据快科技消息，4月23日，在2019中国绿公司年会上，马云发表演讲时谈到与腾讯的竞争时说，腾讯是阿里巴巴发展过程中的一个伴侣，但不是最终要打败的对手。

马云在演讲时表示，没有腾讯，阿里巴巴就不会不断地调整自己，不断地完善自己。

当然，腾讯也是在阿里巴巴发展过程中一个伴侣而已，因为他们不是我们要最终打败或取胜（的对手），而是为社会创造持续价值。“你身边需要有这样的对手来提醒你”，马云说。

早在2017年，马云就曾评价过与腾讯的关系。他曾表示，竞争是正常的，和马化腾关系很好，经常在一起做公益，并把他拉进了大自然保护协会。

“我们有我们的生态，上千万家卖家是我们的生态，消费者是我们的生态，我们没必要团结谁跟谁干，不是我们的风格，没有对错。阿里是经济体，不是一家公司。”

Ruby 2.4.6 发布，终止 Ruby 2.4 正常维护

Ruby 2.4.6 已发布，此次发布修复了近 20 个漏洞，包括数个安全漏洞。请点击以下链接了解详情：

  • https://github.com/ruby/ruby/compare/v2_4_5...v2_4_6

此版本后，Ruby 2.4 的正常维护将被终止，并启动安全性维护阶段。这意味着 Ruby 2.4.6 版本后除了安全性修复将不会被移植任何漏洞修复。安全性维护阶段计划持续 1 年，在此阶段后，Ruby 2.4 的维护会彻底结束。因此，建议用户升级至 Ruby 2.6 或 2.5。
