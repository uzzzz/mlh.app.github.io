华为发布全球首款5G汽车通讯硬件，下半年逐步商用

据AI财经社消息，4月23日，在上海车展，华为推出了全球汽车行业的首款5G通讯硬件。

华为表示，该款名为MH5000的模块基于今年1月推出的Banlong 5000 5G芯片。

此外，华为方面称，该模块将帮助其在今年下半年逐步实现5G网络技术在汽车领域的商业化。

自华为推出第一款车载模组ME909，为汽车提供了无线网络连接功能以来，华为已与国内十多家汽车厂商开展了合作。

另一方面，华为近年来在上海、深圳、无锡等地测试智能联网汽车技术，并与一汽、东风和长安等汽车厂商签订了合作协议。

腾讯昨日发布保护未成年人健康上网新尝试：将启动“16+”试点

据报道，近日，腾讯将在未来一款新游戏上启动“16+”试点。腾讯称把健康系统与游戏研发进行了更加深度的结合，使其在新游戏上线之时即能开始运转。

具体规则为：1. 在登录游戏时：根据健康系统的公安实名校验结果，年满16周岁的用户才可以获得系统授权、直接登录游戏体验；

2. 在游戏中：16周岁及以上的未成年用户仍会受到健康系统防沉迷规则的管理，每天限玩2小时。

今日头条教育产品gogokid大规模裁员

据中国企业家杂志报道，今日头条母公司字节跳动旗下教育类产品gogokid正大规模裁员。

gogokid于2018年5月正式推出，定位在线少儿英语产品，对标VIPKID，采用纯北美外教，在线1对1直播教学。

此外，字节跳动旗下另一个教育类产品aiKID，如今也已停止运营4个月。

滴滴：网约车平均抽成19%，低于成本支出导致亏损

据36氪报道，滴滴“有问必答”于近日上线试运营，在第一期中，滴滴网约车执行总裁陈熙表示，在滴滴平台上，受不同城市、订单距离长短、时间长短、拼车与否等因素影响，每笔订单会收取不同比例的平台服务费。

比如2018年第4季度，滴滴国内收取的平均平台服务费率约为乘客实际支付车费的19%。其中乘客支付的远程调度费、动态调价、感谢红包等几类费用是全额给到司机的，平台不收费。

中国人民大学成立高瓴人工智能学院

据中新网报道，中国人民大学22日成立高瓴人工智能学院。该学院在加强人工智能基础研究的同时，将人工智能与该校优势的人文社会科学进行全面融合合作，为中国输送一流的人工智能科学家、工程师、创业者及人文社科领域的人工智能开拓者。

中国人民大学校长刘伟介绍说，两年前该校80周年校庆时，该校90级校友张磊向母校捐赠3亿元人民币，用于支持学校创新型交叉学科的探索和发展。

经过严谨论证和耐心筹备，今天成立高瓴人工智能学院，中国人民大学将在各个方面给予全力支持，把该院建设成为“文理兼通”“中西合璧”“产学交融”的人工智能学院。

李国庆：希望当当独立发展，而不是被出售

据同花顺财经消息，4月22日下午，“王峰十问智库群”迎来“王峰十问”第三十期，对话嘉宾之一是当当网创始人、早晚读书创办人李国庆。

王峰问李国庆，是否想过出售当当？

对此李国庆表示，他当然是希望当当独立发展，而不是被出售。李国庆说，虽然当当上市后这10年老有被出售的消息，这说明资本市场看好当当或者有些巨头看好当当，但是我希望独立发展。

李国庆说，图书是非常独特的品类，当当在图书，横向、纵向扩张都还有很大的机会，“我过去几年管当当新业务时候，做了很多布局和尝试，我是非常满意的”。

三星：Galaxy Fold将推迟上市做进一步改进

4月23日凌晨消息，三星首款折叠屏手机Galaxy Fold原本定于本周在美国上市，4月24日在国内发布，但目前三星已经将发布会延期，计划在未来几周内公布发布日期。

同时，三星取消了原定于周二在香港和周三在上海举行的媒体发布会。　

三星表示，初步的调查结果显示，Galaxy Fold屏幕破裂可能与顶部和底部暴露的铰链区域影响有关。

还有一个问题是，在设备内部发现的“物质”影响了显示性能。Galaxy Fold“需要进一步改进”才能向消费者发布。

苹果每个月在亚马逊云上支出超过3000万美元

据外媒报道，苹果每个月在亚马逊云上支出超过3000万美元，是亚马逊云服务的最大客户之一。然而，苹果仍投入大量资源建设自己的云服务。

谷歌罢工组织者称其遭到报复：被强迫离岗和降职

据新浪科技报道，谷歌的两名员工活动人士表示，她们已因去年11月帮助组织了一次有数千名谷歌员工参加的罢工活动而受到报复，并计划在周五举行一次“市政厅”会议，让其他员工就其声称的报复事件展开讨论。

谷歌发言人发表声明称：“我们禁止在工作场所进行报复，会对所有指控进行调查。为了跟上不断变化的业务需求，员工和团队被分配新的任务或重组是很常见的，此处并无报复行为。”
