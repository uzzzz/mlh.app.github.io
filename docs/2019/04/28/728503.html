特斯拉回应上海特斯拉自燃起火：正在核实情况

36氪讯，针对有报道称上海一小区车库内特斯拉轿车突然自燃起火，特斯拉官方微博回应称：“在得知这起发生在上海的事故后，昨晚我们第一时间派出团队赶往现场。我们正在积极联络相关部门并配合核实情况。根据目前的信息显示，没有人员伤亡。”

任正非接受德国媒体采访：西方必须接受其他国家的崛起

4月17日，任正非接受德国《商报》和《经济周刊》采访，再次针对美国对华为5G的打压表态。

“很不幸，美国将5G技术视为一种战略武器。对他们来说，这就像一颗原子弹。”

据德媒报道，任正非警告说这可能会出现“新冷战”，正是美国挑起的贸易摩擦，导致世界变得激进和不那么平衡。他认为，如果西方不希望发生这样的冲突，“它必须保持开放，并接受其他国家的崛起。”

在谈及“间谍指控”时，任正非提出与德国政府达成一项“无间谍协议（No-Spy-Agreement）”，以平息柏林方面的安全担忧。

“上个月，我们与德国内政部进行了交谈，表示我们准备与德国政府签署一项不从事间谍活动的协议，并承诺华为不会在网络中安装任何后门。”任正非说。

荣耀赵明：今年将推出5G手机，发力笔记本品类

荣耀总裁赵明近日在接受采访时表示，荣耀已经在准备规模商用的5G手机，将在今年推出。

此外，赵明透露，2019年荣耀很快会有不同尺寸和系列的笔记本推向市场，他坦言，2018年荣耀进入笔记本行业还有一定的探索性，是在用做手机的理念、方式和研发思路做笔记本，但每一款产品都会带来一些全新的理念和设计上改进。

三大运营商加速自有携号转网：年底前全国正式提供服务

4月21日，三大运营商再次明确了今年工作的一个重要目标，即12月31日前在全国实现对所有手机用户的自由携号装网工作。为了监督到位，工信部敦促三大运营商强化自身优势，拨快携号转网时钟。

来电显示利益链：你的手机号码被平台任意标注了吗

据Techweb消息，手机的"来电显示”是一项基础通讯服务功能，基本上智能手机都会有标注信息，方便用户是否选择接听。

但是近期手机号码标注功能被一些平台滥用，甚至将至作为牟利的手段。

有网友称自己的手机号码被标注为教育机构，为此呼出的电话许多人都选择不接听，给生活造成了许多影响，走途径取消还需要额外缴费。

在这里向大家普及一条知识：我国《网络安全法》第22条规定：“网络产品、服务具有收集用户信息功能的，其提供者应当向用户明示并取得同意“。

因此，如果你一旦发现自己的手机号码被任性标注，应当依据法律维护自身权益，而不是花钱取消了事，只有用法律途径维护自己的正当利益，一些不法平台才不敢如此猖獗。

郭明錤：2020年iPhone将支持5G，苹果或同时采用高通、三星5G芯片方案

天风国际分析师郭明錤认为，苹果与高通专利和解意味着2020年下半年发布的新款iPhone将支持5G。

此外，为求降低供应风险、降低成本与提高议价力，预期苹果可能会同时采用高通（针对mmWave市场）与三星（针对Sub-6 GHz市场）的5G基频芯片方案。

郭明錤预计，2019与2020年的总iPhone出货量分别约1.88~1.92亿与1.95~2亿部。

三星折叠屏手机临时取消中国区发布会，推迟时间待定

三星中国方面公关人士临时通知媒体，取消了三星Galaxy Fold折叠屏手机下周三在上海的发布会。解释为场地原因所推迟，并且表示推迟时间待定。此前，有试用用户反映，该手机出现屏幕隆起、闪屏、黑屏等问题，甚至手机完全不能使用。

苹果和斯沃琪的官司又输了 这次因为one more thing

4月21日上午消息，据《悉尼先驱晨报》报道，苹果公司输掉了一起和瑞士手表制造商斯沃琪（Swatch）的官司，这次他们争夺的是“one more thing”在澳大利亚的使用权。

众所周知，“one more thing”是乔布斯的经典名言，其经常在发布会结尾使用这句话来引出一个最大的惊喜。这句话几乎是苹果发布会上的名言，即便是苹果现任CEO库克也会经常使用。

如今这句名言被斯沃琪用在自己的商标上。苹果公司曾与澳大利亚商标局争论过此事，苹果认为斯沃琪对这项商标的申请存在问题，他们不应该在商标中使用“one more thing”这一短语，而应该申请自己的商标。

然而斯沃琪表示，这句话的灵感来自于电影《神探可伦坡》（Columbo），后者经常会说“just one more thing”。斯沃琪将这一商标用于一款以黑色电影为灵感的手表上。

最终，听证官阿德里安·理查兹（Adrian Richards）站在了斯沃琪这边，命令苹果支付这家手表制造商法律费用。理查兹解释说，苹果公司从来没在任何“特定”产品或服务中使用“one more thing”这一短语。
