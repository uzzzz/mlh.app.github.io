京东泄露5千万用户数据？官方紧急回应：谣言！

针对近日传闻的“京东泄露的5千万用户数据”一事，4月26日，京东官方微博@京东发言人 回应称，确认这些数据不实京东账号数据，请勿造谣传谣。

京东称，经过示例数据查验，确认这些数据不是京东账号数据，另外还提醒：买卖公民个人隐私数据涉嫌触犯刑法。

以下为京东发言人声明全文：

针对外界谣传“京东泄露的5千万用户数据”一事，京东经过示例数据查验，确认这些数据不是京东账号数据，请勿造谣传谣！另外提醒：买卖公民个人隐私数据涉嫌触犯刑法！

百度起诉今日头条大量窃取搜索内容，索赔九千万并道歉30天

中证网4月26日消息，当天，据百度相关人士透露，今日头条（北京字节跳动科技有限公司）因大量窃取百度“TOP1” 搜索产品结果，被百度以不正当竞争为由起诉到北京市海淀区人民法院，要求北京字节跳动科技有限公司立即停止侵权，赔偿相关经济损失及合理支出共计人民币9000万元，并连续30天在其APP及网站首页道歉。除民事起诉外，百度也同时向法院提交了行为禁止保全申请书。

B站抽奖“有猫腻”被约谈

据北晚新视觉报道，近期，哔哩哔哩网站的源代码被泄露，开源项目平台Github上疑似出现了Bilibili网站后台工程，而且暴露出该网站在“节奏风暴抽奖活动”中“有猫腻”，引起网友关注。

上海市消保委为此两度约谈但企业始终未能正面回应，相关负责人敦促企业诚信对待消费者，尽快就相关问题作出令人信服的解释。

对于此事，B站回应称，设计此功能是为了活跃弹幕氛围，并不涉及现金交易。哔哩哔哩承认功能设计存在争议点，表示已将功能下线整改。

腾讯全球专利申请数量超3万件，仅次于谷歌

据腾讯方面表示，截至目前，该公司在全球主要国家的专利申请数量已超过30000件，已授权专利数量超过10000件。

专利申请数量在国内互联网公司中排名第一，在全球互联网公司中排名第二，仅次于谷歌。

腾讯专利范围覆盖了即时通信、邮箱、安全、视频、广告、支付、浏览器、游戏、音乐、地图、搜索、云、智能硬件、AI、VR、区块链等领域。

华为轮值董事长徐直军：5G在中国市场有两大刚需

据证券时报消息，华为轮值董事长徐直军在中国电信5G创新合作大会上表示，展望5G，中国市场有两大刚需，一是eMBB，即未来智能手机的速度会越来越快，二是行业数字化，通过5G把设备和管理拉近，可以将生产效率提高10倍以上。

美国一公司指控苹果iPhone中“个人热点”技术侵犯其专利

据外媒报道，美国一家名为WiNet Labs的公司正在起诉苹果公司，指控苹果iPhone和iPad中的“个人热点”功能侵犯了它的一项网络专利。

该公司表示，曾在2014年通过代理商提出将该专利出售给苹果，但遭到了忽视。WiNet Labs正在寻求陪审团审判，最终目标是索求“公平合理的专利授权费”、“三倍的损害赔偿”以及审判前后按照“法律允许的最高利率”计算的利息。

谷歌再胜一局：11亿欧元补缴税款案，法国法院驳回对其逃税指控

据外媒报道，在与法国税务当局就11亿欧元（约合12亿美元）补缴税款的纠纷中，谷歌又赢得了一轮胜利。

此前，另一家法院驳回了针对谷歌滥用法国税法漏洞以逃避支付相应税款的指控。

周四，法国巴黎行政上诉法院维持了一家下级法庭在2017年的一项裁决结果。该裁决推翻了法国税务当局的主张，称其审理过程无误。

瑞士联邦主席：瑞士不反对华为参与相关5G网络建设

据第一财经消息，瑞士联邦主席于利·毛雷25日晚在瑞士驻华大使馆举办的记者会上表示，瑞士不反对华为参加相关5G的网络建设。

针对5G问题，毛雷尔坦言：“ 我觉得，我们其实现在能够看到，就华为5G建设而言，欧洲和西方（其他一些国家）的态度并不是很一致，有一些人有疑虑，但瑞士是不反对华为参加相关5G网络建设的。”

谷歌开设内部性骚扰报告网站

据TheVerge消息，4月25日，谷歌表示，将改变员工报告骚扰和歧视指控的方式。

谷歌专门开设了一个网站，供员工报告性骚扰问题，并在年度内部不当行为报告中增加了性骚扰调查的信息。

此外谷歌还公开发布了关于骚扰和其他工作场所问题的政策文件，这份13页的文件详细说明了什么是骚扰和歧视，以及谷歌如何调查这些问题。



NetBeans成为Apache软件基金会顶级项目

近日，Apache软件基金会宣布Apache NetBeans成为顶级项目（TLP，Top-Level Project ）。

Apache NetBeans是一个功能齐全的开源IDE。作为一个可扩展的开发平台，它可以用于Java，C/C++，PHP等语言的开发。 

该项目原本是学生项目，后在2000年被Sun Microsystems收购并开源。Oracle在 2010年收购了Sun，并在2016年10月将NetBeans交给Apache基金会孵化。

目前，该项目最新的版本是Apache NetBeans 11.0，于2019年4月4日发布，这个版本也是项目进入Apache孵化器后的第三个主要版本。
