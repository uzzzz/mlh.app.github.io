超简单，四行命令安装输入法

安装了linux之后，发现不能输入中文，故想安装一个输入法。
首先想到的是百度输入法，但是百度输入法无linux版
在linux上的输入法有：
1. 搜狗输入法
2. 谷歌拼音输入法
ubuntu 18.04 LTS

考虑到烦人的广告或其他杂七杂八的事情，我选择了安装google输入法。
安装方法如下：

1.安装Fcitx

sudo apt install fcitx

(base) li@li-System-Product-Name:~$ sudo apt install fcitx
[sudo] password for li: 
Reading package lists... Done
Building dependency tree       
Reading state information... Done
fcitx is already the newest version (1:4.2.9.6-1). 
0 upgraded, 0 newly installed, 0 to remove and 17 not upgraded.
//这个是我已经安装好后，显示的内容。

Fcitx (Flexible Input Method Framework).是以GPL方式发布的输入法平台。短小精湛，是linux系统常用的中文输入法。

2.配置Fcitx

im-config

输入法设置：Input Method Configuration
输入只有会弹出窗口，在窗口里面设置：前两个点击ok即可。配置好之后重启一下电脑。

3.安装google输入法

sudo apt install fcitx-googlepinyin

4.配置google输入法

fcitx-config-gtk3

**按加号 + **，在弹出的画面当中，选中Google Pinyin即可。如果找不到，取消{Only Show Current Lauguage}的对勾。还找不到，重启一下电脑即可。
