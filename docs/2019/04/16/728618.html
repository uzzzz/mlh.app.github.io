noscript标签，一个容易被忽视却重要的标签！

有时候打开一个web页面或者是手机端的页面，会出现大面积的空白现象，你可能以为是 CSS 的问题，折腾了半天发现不是这样？那么问题在哪里？

后来才发现是我们之前因为某些网站的安全问题把浏览器的 Javascript 给禁用了。

大家都知道Javascript 使我们的页面变得丰富多彩，交互效果越来越强大，功能也越来越炫。但是有的网页效果完全依赖 Javascript 来实现，一旦离开了 Javascript 的支持，可能连基本的内容都显示不完全，所以出现了大面积空白的现象。
这时候，想要解决这个问题，让用户体验更加友好的话，我们老当益壮的noscript标签就该出场拯救了！

关于noscript标签

noscript标签是一个相当古老的标签，因为早期的浏览器不能够支持 JavaScript。noscript 标签在不支持JavaScript 的浏览器中用来显示要替代的内容。这个nocript元素可以包含任何的 HTML 元素。它的用法很简单，如下：

<noscript>
  	<p>You need to enable JavaScript to run this app./（本页面需要浏览器启用或者说支持JavaScript）</p>
</noscript>

现在看来JavaScript如此强大，浏览器基本支持 Javascript，但是用户还是会因为一些原因而禁用JavaScript。比如说为了：

  • 节省自己越发稀少的流量
  • 延长手机电池使用的时间
  • 特别不希望自己的隐私被各类统计/追踪脚本泄露出去

noscript标签的运用

1.浏览器不支持脚本的时候
2.浏览器支持脚本，但是脚本被无情的禁用了
【注】在这两种情况下，nocript标签就派上用场啦，该标签中的内容会被显示出来，提示用户的浏览器已经禁用脚本，来达到更好的用户体验。是不是很棒呀~~~
