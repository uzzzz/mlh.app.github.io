一、requests库的安装

cmd提示框中直接敲入pip install requests 即可，但有时会出现retry........然后暗转不成功的情况，在提示安装失败的同时会提醒你可能是pip版本过低，此时你需要升级pip的版本，升级的办法也很简单，只需要将提示中告诉你的那串字符再敲进去即可成功，也就是下图中的python -m pip install --upgrade pip即可大功告成。

 

 

二、bs4库的安装 

cmd直接输入pip install bs4即可成功，基本不会出错。

三、lxml库的安装 

lxml库的安装是比较麻烦的，不像上面的两个库一样，步骤如下：

（1）查看自己的pip版本，cmd输入python --version查看自己的版本。

（2）cmd输入pip3 install wheel，可看到安装成功提示.。

（3）到

https://www.lfd.uci.edu/~gohlke/pythonlibs/

这个网站找到和自己的pip版本匹配的whl版本进行下载，37即3.7版本，

下载好了以后，把下载的文件放到和你的python在同一个磁盘下，不然后面安装时找不到。 

（3）cmd输入pip install lxml-4.3.3-cp37-cp37m-win_amd64.whl ，pip install后面要根据自己下载的whl版本自行选择。

 

到这里你已经安装成功了，你可以检验一下，cmd输入python，进入程序，输入import lxml，没有报错及代表恭喜你成功了。
