前言：2019的开始，是坏的一年，也是更好的一年，夕叹匆匆又是一年，不过我们更要珍惜现在，正是重整旗鼓，努力拼搏的新年。笔主目前大三，即将踏入社会，希望和大家共勉，完成的自己的一个个小目标。

一、学习感言

既入java，便不撞南墙不回头了 。

  1. 学校提供的知识不会包揽你的人生全程，只是给你一个学习的平台，学校的专业课程不能落下，《java实用教程》，《数据结构》（精通这本书中的算法，以后都不愁HR怎么花里胡哨的考你），虽然不能给你的知识质的提升，但对于初学者，新手十分重要。有一定的java基础的话，我推荐《java编程思想4》，我觉得这本书之所以厉害，因为它在无形中整合了设计模式，这本书之所以难读，也恰恰在于他对设计模式的整合是无形的。
  2. 编程思想很重要，我们首先要学习的就是如何看懂代码，如何把这段代码理解透彻，其次就是功多技熟，多看多练多想。
  3. 接下来我分享大学三年我是如何学习java，给初学者，小白一些建议，对于编程过程遇到的各种问题，我会认真梳理并将完美解决的方法呈现给大家，希望各位大神也可以指点一二，让我也可以更上一层。

二、学习流程

前期以打好基础为主，就像盖房子，打好地基最为关键，因为地基的稳固程度，决定房子的高度。

1.认真学好《java实用教程》、《数据结构》，java的语法，java的面向对象非常重要。
2.JSP：JSP 语法和标签，自定义标签， EL , JSTL 库了解以及 MVC 三层架构的设计模式理念。
3.数据库SQL：增删改查的操作以及多表查询，学好了SQL，再去接触Oracle数据库。
4. Servlet 开发：从此开始踏入JavaWeb开发的重要一步，包括XML， Tomcat 服务器的安装使用操作， HTTP 协议简单理解，Servlet API 等， Java Web 开发的基础。(一开始我使用的是eclipse，因为老师是用eclipse教学，大三就去用了idea，个人觉得idea比eclipse实用。)
5. 用eclipse或者idea搭建SSH（struts2+spring+hibernate）框架。（我是先学的SSH，后面自学SSM（spring+springMVC+mybatis））。
6. 深入了解spring。（重点，重点，重点，spring太重要）。
7.下面的博客我会给大家看一下我用eclipse和idea用SSH框架和SSM框架做的一些小项目，其中经常遇到的一些问题，给大家做个详细解答，或者刚入门的小白正在学习框架遇到的一些问题，私信给我，我一定知而尽言，言而无尽。
