2019天梯赛+第一次面试总结

  • 柚子社区
开始前

因为是在东北大学比，离学校有点远，故前一天就火车到沈阳酒店去了。（都大三了，还被老师拉去写天梯赛。。还有项目要写。。还有笔试面试要准备。。精力不足啊，这也就是为什么拖了一个星期才写总结。。哎
因为比赛结束后一个小时就有面试，所以头一天晚上临时抱佛脚再看了看面经（雾）。说实话天梯赛我没怎么准备…也许做了一两次笔试+复习之前写过的一些题目就叫准备了（嗯？？）
比赛当天早早起床了，集合去东大校内。穿的本来就少，还雨夹雪，巨冷无比的在计算中心合照了几张，然后就提前去了食堂（开幕式偷偷地没有去。。），因为提前去了食堂，食堂大姐问我们吃啥，我们也不知道啊，于是拿出餐票。食堂大姐：？？？什么，比赛是今天？东大食堂搞错了时间，以为是明天。。所以匆忙召集人员去准备饭菜QAQ。（如果我们没有提前来估计所有人都没饭吃 哈哈哈）饭菜是真的不错，吃完后顶着雨夹雪跑去比赛地点。
有点激动的喝了瓶红牛提个神，在比赛门口等了半个小时顺便和队员们提醒注意事项，不要卡题啊之类的…
进去之后发现左边一个大工右边一个东大，我的妈耶，这谁顶得住啊，两边都是大佬。

比赛开始

比赛开始。18年的天梯赛第一题就有点复杂，当时也是第一次出去比赛，卡了将近半个小时！所以这次还是仔细准备好如果一开始有点难的话绝对不要卡题！先写简单题。第一题输出题，写几行代码当是ready。第二题字符串操作题，脑子有点晕沉沉的，头天没睡好，花了差不多二十分钟才A掉。。哎。。真的是不练题就不会写了。第三题也是字符串操作题，不过有了第二题的练手，思路清晰很多，快速写掉。第四题数学公式题，应该是最简单的题，不过脑子有点混乱？？公式写错了，0分，不管了赶紧下一题，待会回来看。后面接连三个C语言上机题。快速A掉。L1最后一题是个字符串模拟题，乍一看可以写啊，应该就是复杂一点，写到半途老师进来说因为L1最后一题太难了，把进阶分从800调到600。什么？这么难吗，难到要调分了，大概写完后测试一下，样例过了，提交1分。wtf??一看过的人数只有几十个人，好吧，果断放弃不卡题直接看L2了。这次学聪明了，看了一眼L2过的最多的是第三题，看题目是个BFS求树的高度，赶紧敲掉，然后写L2第四题，第四题是个栈模拟，不过因为把题意中的拿最上一个箱子的条件看成了拿所有堆在一起的箱子，卡了很久，也不知道为什么不对，心很慌，决定出去上个厕所洗把脸！回来后发现题意读错赶紧改改。提交只有23分，哎不管了，时间浪费了好多，直接看L2第一题，发现是个SET+模拟，好吧赶紧写掉，提交24分，后来发现少掉的一分是没写else cout << “SAD” << endl; 不过当时比赛也是心急浪费很多时间，没有去仔细看，时间还有十几分钟，赶紧回去看L1那个数学公式题，心里超级乱，各种乱交，一直错一直不爽，深呼两口气！仔细读了一遍题意，发现写错个变量？？哎。。改了交了还剩五分钟，也没时间写题了便看了眼提交榜，发现好像整体都很爆炸。最后153分，比18年那次天梯赛少了20分，心里很不是滋味= =

比赛结束-面试经过

比赛结束后问老师可不可以待会占用一下机房，因为没地方面试ovo。老师说不行，但是开车带着我和同学去了食堂找了个包间给我们用（东大老师和食堂大姐是真的好人啊。。哭）。刚比完赛就要开始人生第一次面试。脑子很沉很沉。因为第一次就是大厂，而且自己确实没准备好，面试前就知道不行，不过还是想试试面试的流程，一开始面试官让我自我介绍，我心急说了一堆。我都不知道我在说什么，然后面试官问我问题，大部分都是操作系统、计算机网络、数据结构、数据库，问的学校很多都没有教过或者是一些平时没有注意到的细节，有些概念我也是忘记了非常的模糊，也不知道怎么说，就和面试官说这块我不会。。想必面试官也是非常无语，怎么面到一个什么都不会的学生= =。最后用操作系统的背景引出一个n条线段最大重和数，我花了好久时间才理解面试官问的是啥。。因为这个题目背景我有点迷？这题大一的时候好像做过类似的，用贪心，不过忘了具体怎么想，便说了个哈希遍历。。哎，太蠢了。面试官问我你确定要开这个哈希空间吗？我…应该要开吧。。好吧，我不知道怎么写。最后我问了一个很愚蠢但是我非常想问的问题：我应该在之后的面试前要怎么去准备呢？或者说之后我应该去怎么学习来应对面试。面试官的意思就是无可奉告。。哭辽，于是乎结束了这次面试。主要是太菜了，不然也不会这么悲惨，不过在之前也就抱着看看是什么流程的心态，至少没有哭出来！坚强！
