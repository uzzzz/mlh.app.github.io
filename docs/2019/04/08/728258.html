活动日程


14:00-14:10    签到


14:10-15:00    Oracle基于x86服务器上那些不为人知的故障案例解析

随着信息化进程的推进，更多企业关键系统已经运行在了X86服务器上，构建科学、合理、规范的企业系统架构，打造安全、稳定的数据架构，势在必行，是企业系统建设的重要目标，本次通过实战案例分享在X86服务器上遇到的问题解析以及如何建立安全稳健的数据架构。


15:00-15:50    智能网络赋能新一代云计算数据中心

在摩尔定律放缓的时期，网络软件堆栈必须尽可能转移到架构上；借助Mellanox高速智能网络领先的技术有效释放CPU资源，如RoCE 、DPDK 、NVMeoF 、SNAP、ASAP²，提升云计算数据中心效率！并与Oracle Cloud Infrastructure深度集成，共同打造业内更高性能的AI和深度学习平台！


15:50-16:10    茶歇


16:10-17:00    Low Code App Development with Oracle APEX - 用 Oracle APEX 快速开发应用

介绍 Oracle APEX — 快速开发应用：无需编写复杂的代码和配置，在半小时内把概念变为现实。 介绍定义数据结构和导入样本数据到 Oracle 数据库，然后创建完全可用的应用程序。 最后，大家将可以在手机或者平板设备上访问这个只用了极少代码量创建的应用。


17:00-18:00    动手实验室：APEX 零基础入门体验

学习如何从电子表格导入数据创建你的第一个程序，然后我们会改进报表，添加日历组件。 第二个程序我们会定义数据结构，然后在新建表上创建应用。 虽然听起来很有挑战，但是你会发现基于一个电子表格 (或者现有表) 构建一个完全可用，并且可以适应移动设备端的的漂亮网页应用是多么快速和容易！ 我们将会使用 https://apex.oracle.com 免费体验这一切。

关于ACOUG


作为全国 Oracle 用户及技术爱好者的共享平台，ACOUG 自成立之日就一直积极致力于线上线下 Oracle 技术的分享传播。


从最初 ACOUG Tech Day，ACOUG Campus Tour，到 ACOUG China Tour，到近几年的 ACOUG Asia Tour、“数据技术嘉年华”，日益发展的不仅仅是规模和战线，如今，ACOUG 已经成长为涵盖云架构、数据技术，智能运维与行业实践等各方面技术的分享社区。如果你热爱技术，同时又乐于与大家分享心得体会，欢迎加入我们！
