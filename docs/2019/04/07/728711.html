VirtualBox安装Linux虚拟机

  • 柚子社区

本人最近想要搭建ES集群玩玩，于是开始尝试安装虚拟机，才有了这篇博客，过程都是跟网上大神们请教学习的，只是为了记录一下学习的过程以便以后能够回顾。非常感谢这篇文章给与的指导：https://www.veryarm.com/1258.html

  1. 下载ubuntu镜像文件，注意下载的是32位还是64位
    http://releases.ubuntu.com/
    2.打开VitualBox，按以下步骤操作
    点击新建
    给虚拟机命名（本人要建集群，所以以01，02，03…区分）






    无脑下一步，视个人电脑情况定内存和磁盘空间。
    3.选中新建的虚拟机，点击设置：

    选中存储 -->控制器：IDE下面的"没有盘片" -->分配光驱，选择step1下载好的镜像文件，点击ok保存设置

    4.回到VirtualBox界面，选中虚拟机然后启动
    选择 Install Ubuntu







    安装完成后移除虚拟盘然后restart，linux系统就装好啦(这个过程有时候快有时候慢，看心情，为什么我知道，因为我装了四遍（微笑脸）)

    5.这个时候屏幕大小还不能自动适配，需要进行以下操作：
    设备 -->安装增强功能 -->Run

    按回车结束，重新启动，然后就可以全屏幕啦~
