你正在毁掉一个数学天才

  • 柚子社区


牛顿他舅舅救了一个数学天才，

大多数人却正在毁掉一个数学天才。




/ 01 /

神童的早慧期


当他在还在妈妈肚子里的时候，你无聊地贴近肚皮问他 ：“1+1等于几？”


他灵慧地冲着妈妈肚皮上踢了两脚，一时，你欣喜万分，如获至宝，并将这段经历当作骄傲的饭后谈资。

 

更令你骄傲的是，你惊奇地发现他在数学上似乎真的天赋异禀。他2岁不到，就能一口气数到100，并能快速记住你的电话号码。3周岁时，就能进行1-10以内的加减法运算。电话号码，更是你给一串，他听后就能直接背下来复述一遍。这一切，都让你在客人来时，总不忘炫耀一把，让他表演一场精彩的电话绝技。

 

但骄傲之余，你也感觉到了压力，你深深地觉得这孩子不能等闲视之，望子成龙的心态开始发酵，就像牛顿他舅觉得侄子不能真在农田里挤一辈子牛奶一样。你绝不能耽误他，他应该尽早读书。于是，你购进各种数学书籍，专门潜心栽培他，着手幼儿智力培训计划。





/ 02 /

小学的聪敏期


上小学前，他已经熟练地背会了九九乘法表，并能够轻松自如地进行心算。上小学后，每每满分的数学成绩更是让他成为了他人眼中的别人家的孩子。在他身上，你引以为傲的喜悦实在无法掩藏，老师、同事、邻居们的吹捧，都让你飘飘欲仙。你越来越觉得你家的孩子是个天才，对他期望甚高。

 

以至于有次他数学考了90分，你觉得不可思议，不可原谅，猜疑对他太过松懈了，最近他做的习题不够了。

 

分数像个衡量他勤奋度、能力和智力的标准尺标。

 

毫无疑问，你给他进购了更多练习册，让他开始题海战术的操练。这一切，都使得他快马加鞭，数学知识的储备远超同龄人。但你却没注意到，他对数学的由衷热爱开始慢慢变质，他被培养得开始一点一滴去深究那卷面上写好的赤红的分数，想学好数学与要考得高分开始融汇交杂。

 

他害怕低分，他害怕低分后你眼神里的失望，他已经开始对数学有点畏惧了。





/ 03 /

初中的自症期


初中，学校难得举行一次春游。大家一块到动物园里玩，看着恐龙蛋，同学们纷纷讨论起了动画片里的恐龙世界，那些在蓝猫带领下回到的神奇的侏罗纪时代。而他，却久久出神，在想着如何计算恐龙蛋的体积，他思索着恐龙蛋的形状像椭圆，但又有所区别，那是否能用体积公式来求解？

 

沉浸在这样的思考中，就像老师在演说数学时一样，让他感觉到心灵一片平静和纯真的喜悦。但他所不知的是，那时他思索的问题，是在大学微分课程和非欧几何中才能得以解答。

 

不过，在类似的场景下，他却被赤裸裸地取笑了。

 

那天，你约了和朋友家一块去看车。你们问孩子们想要哪一部车时，朋友家的孩子对车可谓是如数家珍，立志长大后赚大钱买宝马奔驰孝敬父母。而他竟指着架子上的轮胎，说它的曲线真的非常美，结果，大家听了后都哈哈大笑。你顿时也觉脸上无光，开始暗暗担心自家的孩子是否像别人那样常喊地，是个书呆子。





/ 04 /

高中的迷茫期


转眼到了高中，考取名校的关键时期，你自然更不会放松对他的管制，他的思维也在长期高强度的训练下逐渐固化。

 

生活的单一，每天像被用鞭子“赶着走”式的学习状态，让他开始麻木。不管是在学校，还是在家里，日复一日地高压重复状态，和高强度的知识灌输，都成功地扼杀了他对知识，甚至是数学的兴趣。

 

在这种盲目的竞争机制下，追逐高分最终替代求知，顺便废掉了他创造的天性和对数学的灵感。


好不容易，他终于随着千军万马一起走过了高考这座独木桥，面对着如何择校，如何填报志愿，他的脑子，一片空白。


但这空白，却在深夜，被一只笔无意识的涂涂画画，竟然写满了数学公式。心之所向，终有所往。次日早晨，他兴奋地与你畅谈他对数学不灭的热爱，他在数学上超人的优势，以及他对梦想的美好憧憬。

 

他的激情澎湃在你看来，却不谙世事，幼稚可笑。你开始用你年龄搭起来的经验，亲朋好友组装起来的忠告，告诉他，学数学没多大前途，它就像哲学一样，不管用，虚而不实。大学毕业后，找不到好的工作，挣不了高薪，买不起房，买不起车，整个人生都是失败的。

 

这是来自底层基本的生存危机和害怕失败未来的深层恐惧。恭喜你！成功地用你的实用主义和残酷现实，抹杀了他对数学的赤诚憧憬。





/ 05 /

大学的毁灭期


如你所愿，你数学无用论的循循诱导发挥了功效，他跨入到著名学府，进入到了炙手可热的金融专业，从此将数学家不切实际的梦想抛之脑后，立志当上银行行长。

 

眼看他将起高楼，眼看他将宴宾客，你不禁泪眼婆娑。语重心长地嘱咐他上大学后，别恋爱多实习，力求毕业时能找个跨国公司。

 

但他却未如你所愿，悠闲的日子里，唯一让他觉得有点滋味的就是那本在图书馆借的纳什《博弈论》。


那晚，社团举办联谊，偌大的广场上，男同志们个个嗷嗷待哺，目不转睛地盯着广场中央那个唱着歌的女该。而他却静静地缩在一旁，分析着眼前的战局，舞台中央的美貌姑娘和被忽视的姿色平平的女孩，被他用数字在脑海里标记区分。而男性追求者则被他另外圈画，在脑海中迅速演算，最终而得：

 

“现在，在有限次博弈的情况下，如果我们都去追舞台中央的美女，那么，我们只会勾心斗角，大家的互相争夺，会使美女自视甚高，最终没人可以得到她。然后，追不到美女，如果我们接着就去追那些貌不惊人的女孩，她们定会冷淡我们，因为没人愿意做代替品。但是如果我们都不去追美女，而是在一开始，就大家协商好各自去追美女身旁的其他女孩，那我们就不会彼此阻挠，也不会给其他女孩造成自尊伤害，最终赢得男孩间的大胜利。”

 

周末，回到家后，他兴奋地和高中同学谈到受《博弈论》启发而得的男性泡妞共赢法则，畅谈假设在无限次博弈的情况下要怎样追女时，你敲开门愤怒地打断了他，会计从业资格证、证券从业资格证、注册会计师、特许金融分析师……这些证都考了吗？ 


你根本就不在意，他的无限次重复博弈模型的数学推导，正常恐怕要有专修博弈论的博士研究水平才能弄出来。

 

他被你义正言辞地教训，弄得羞愧难当，从此忙于各种考证，各种实习，以便能早日完成当上银行行长的目标，有关数学的一切从此烟消云散。





/ 结语 /

你完成了对一个天才的毁灭


王安石在《伤仲永》中，痛悯怜惜。仲永生而不凡，五岁便能书诗四句，并自为其名。旁人若是指物作诗，他当即立就，且文理可观。但其父见钱眼开，日日拉仲永环谒于乡人，卖弄才学，谋取钱币，令其不使学，最终废其才。

 

仲永不幸，毁于其父。而其父迟钝，浑然不知。历史的转盘扭动千回，相似的故事也这般反复上演。其形式不一，却本质相同。

 

自他初露聪慧，你便以爱的名义咄咄逼人，以不输在起跑线上为由毁他对知识纯真的兴趣，以追逐高分的功名欲毁他数学天性中的创造力，以残酷的实用主义毁他异想天开的灵光碎片。

 

试问，你，这样与仲永之父，又有何异？

 

你，从未静下来，认真思索，多年的教育，你是在帮他，还是毁他？你使劲推着他往所有人都认同的一条路走，却忘记停下问问他，想往哪走？

 

你，作为数学天才的亲生父母，占天然优势，常借口众多，理由正当，毁他所想于无形之中，并心安理得，逍遥法外。让他尽管儿时天赋异禀，长大后却泯然众人矣。

 

好的，最后，恭喜你，你完成了对一个天才的毁灭。
