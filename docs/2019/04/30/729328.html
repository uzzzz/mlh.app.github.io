微信是不是“蓝色药丸”？

  • 柚子社区



张小龙很困惑：

我们并不知道自己是否处在一个真实世界，

但微信不是蓝色药丸……



/ 01 /

微信，8岁巨人


黑客文化第一理论家、开源软件旗手Eric S. Raymond说过：“乔布斯比盖茨更加威胁自由，因为人们害怕微软，却热爱苹果。”


我们在朋友圈里看到了几乎同样的场景，张小龙像乔布斯一样在构建它的“封闭帝国”，将我们圈养在微信这座“虚拟空间”，而人们却更乐意手持长枪短炮去找百度开火。


2019年1月9日，张小龙出席微信公开课PRO版，长达4个小时的演讲内容平实无奇，但却有无数人硬生生找出“善良比聪明更重要”这样的陈词滥调来刷屏。


2019年1月15日，聊天宝、MT马桶、多闪三大社交软件围攻微信逐鹿社交时，朋友圈里出现各种为微信“保驾护航”的有趣段子。


可见，尽管对腾讯的抱怨长期存在，但朋友圈对张小龙的热爱发自内心，而不仅仅是因为“斯德哥尔摩综合症”。



2019年1月21日微信8周年，这个APP已经不能再简单用“日活数量”“用户数量”等维度来诠释，它悄悄地融入、改变，甚至在操控10亿人的生活。


因为很多人已经食下了微信这颗“蓝色药丸”。




/ 02 /

什么是“蓝色药丸”？


蓝色药丸出经典科幻《黑客帝国》，吃了之后你会忘记现实世界，享受虚拟世界的“幸福生活”。


故事的背景是22世纪：人工智能已经统治世界，它虚拟了一个全息的人类文明世界，在这个世界里，人工智能开发了一个程序母体Matrix，把人脑和虚拟世界Matrix连接起来，构建一个让人类大脑在虚拟世界中生存的场景。


此时的人类已经沦为机器的奴隶，成为Matrix系统的生化能源，大部分人并没有意识到自己已是机器控制的“缸中大脑”，每个人都在享受虚拟世界的美好生活。



有一部分人从虚拟世界中醒来，这些背叛者找到了拯救人类的救世主尼奥，告诉尼奥虚拟世界的真相，并在尼奥面前摆了两颗药丸：一颗蓝色药丸，一颗红色药丸。


吃了蓝色药丸，会忘记现实真相，继续在虚拟世界中生存；

吃了红色药丸，就能走出虚拟世界，回到真实世界。


主角尼奥当然勇敢地选择了回到真实世界，吃下了“红色药丸”，开始自己的“救世主”旅程。当然日子也不好过，流亡生涯启动，历遭机器人追杀，爱人最终逝去……


故事的结局在这里不再探讨，现在只谈蓝色药丸和红色药丸的选择问题。如果真有这样两粒药丸，对于现实世界的人来说，有几个能像主角一样选择“红色药丸”？


因为这是一条“反人性”的道路。沉溺于虚拟世界太久之后，可能大多数人会选择“蓝色药丸”，也只能选择“蓝色药丸”。




/ 03 /

虚拟世界>现实世界


可能有人会说，这只是一个科幻故事，现实世界就是现实世界。美丽的日落，疼爱的孩子，舌尖上的美食，这些不可能用“蓝色药丸”替代，永远永远也不可能，我们还是可以拥有现实世界的。


可事实真的是这样吗？


❶对很多人来说，在朋友圈里晒照片的快感可能已经远大于实景感受。

❷对有些人来说，展示与孩子在一起的“幸福生活”时，收获点赞的快乐可能超过与孩子在现实世界中游戏的快乐。

❸对部分人来说，面对网红美味并不能感受它到底好吃在哪里，更多的仪式感在于拍个照片分享给全世界。

…………


我们越来越被虚拟世界的感受所左右，也越来越在意虚拟世界的反馈。


“现实世界”你所采取的那些行动，不过是为了给“虚拟世界”添彩。现实成为手段，虚拟成为目的。


随着生产力的进化，对于人类这个社会性超越动物性的物种来说，虚拟世界的精神追求成为最大的进化动力。



互联网诞生之前，人类的社会性在于日常交往。今天，人的社会性已经被逐渐迁移到微信上。10亿用户，熟人圈子，这就是现实世界的完整镜像。


你是不是已经在做这样的事？你的同事就在旁边，但你情愿在微信上跟她沟通。


对于互联网的原住民来说，虚拟世界>现实世界，这不是“盗梦空间”，也不是“西部世界”，而是正在发生的现实。


2016年，高晓松还能对这个世界喊一声：人生不止眼前的苟且，还有诗和远方。


但在未来，可能所有的“诗和远方”，也只能发生在你的“朋友圈”。




/ 04 /

互联网时代的“瑞士军刀”


互联网的巨头有很多，苹果、脸书、微软、阿里、百度……

为什么要重点思考微信，为什么说微信才是“蓝色药丸”？


《纽约时报》做过一个视频短片，向不了解中国网络现状的美国人介绍了微信这个APP，非常形象地把它比喻为一把瑞士军刀，寓意是什么事情都可以解决。



是的，这是微信正在做的事情。毫无疑问，这是一个伟大的软件，同时，也是一个值得我们反思的产品。


与中国不一样，在国外人们用Facebook、Twitter、Instagram、Snapchat进行社交，在中国，朋友圈几乎包揽了所有的社交需求，同时有2000多万公众号给你提供资讯和服务，无数的小程序给你搭建应用场景。只要你不过于挑剔，它基本上可以满足你的所有要求。


虽然有抖音这样的移动互联网产品异军突起，但它仍然只是一个短视频资讯平台，谈不上真正的社交。


在欧美，人们用WhatsApp、Skype进行图文语音通讯，在中国则直接可以聊微信，或直接语音或视频通话，你甚至连电话都不想打了。


微信支付现在已经发展成为中国第一支付终端，过个高速路可以发红包、坐个地铁可以刷小程序。微信包含各种便捷的生活及城市服务，你的生活已经和它密不可分了。


微信，就是一把瑞士军刀。


但刀总是有两面的，它可能为你解决问题，也可能伤到自己。




/ 05 /

张小龙的“黑客帝国”


《纽约时报》提到的是2016年之前的微信，用户在“摇一摇”、“打飞机”、“抢红包”等功能中狂欢，世界万物以二维码重新连接。


2016年之后，微信向认证服务号开放支付接口，通过小程序重构现实世界的应用场景。没有哪个软件有微信这样的深度，它完整地构建了自己的商业闭环。



未来呢，微信又会是什么样子的？


这里要说一下张小龙，这个49岁的技术男，射手座，性格闷骚，内心敏锐。他欣赏凯文·凯利的一句话：“没有一个人可以成就人的所有潜能，没有一项技术可以满足所有的承诺。这需要所有的生命、所有的心灵、所有的技术共同憧憬一个现实。这需要所有的Technium，包括我们自己，去探索可以给世界惊喜的发明……”


所以他给小程序定下的一条基本规则是：用完即走。


从某种程度上来讲，张小龙具有先知气质，或者说，张小龙是一种先知人设。我们在《头号玩家》里也可以看到这样的影子，就是“绿洲”的创立者哈利迪，言语温和，性格内敛，不求名利，略带羞涩。


越是这样的人，越能够建立长期信仰。所以，微信到底能走到哪一步，可能比我们想象的更长久。


也许十年后，更多人会生活在微信构架的更多场景之中：吃饭、理发、上学、医疗、坐车、买房、看电影，谈恋爱……衣食住行什么都有，这是一个能满足人所有需求的世界。


我们唯一不知道的是，自己生活在一个APP里面。也根本不知道，APP之外还有APP，局域网之外还有互联网。


微信的社交占据了你的时间，支付在左右你的生活，公众号还在输出思想。


这不就是传说中的“黑客帝国”吗？




/ 06 /

三英战“微信”


微信真的就没有破绽吗？它真的不可战胜？它必将成为“黑客帝国”？


其实希望曾经是有的。


2018年8月20号，罗永浩宣布子弹短信诞生，子弹短信迅速成为App Store排行榜总榜第1名，单日下载量达44万。连续霸榜9天，时隔一个月拥有700多万名用户。


也许正是因为子弹短信的威力，大家以为微信并非不可战胜，才有了1月15日三英战“微信”这样的事件发生。字节跳动的多闪，前快播的马桶MT、锤子科技的聊天宝显然密谋已久，三家同时上阵，但战局远没有当年子弹短信的威力。



罗永浩对人性了解得更深刻了，直接用钱来刺激用户也是对的，只是现在没有多少人相信老罗有这样的财力。相反，他活成了自己最讨厌的样子，都是创业害的。


多闪则主打年青人口号，吓唬老年人去下载多闪以证明自己老当益壮。但不是年青人就可以做出年青人喜欢的产品的，最重要的还是对人性的洞察，心态年青最重要。


王欣则延续了自己的暗黑风格，想游走于当年的灰色地带，可惜互联网的蛮荒时代已经过去了，现在得正大光明地用产品说话。


直至今日，用一个段子来总结当前的战局：


张小龙：今天有三款社交App上线！

马化腾：今天一天？疯了吧？！

张小龙：嗯 都封了…


老实说，一款社交软件在三天内没有形成爆款，后面要再来蚕食微信地盘实在太难。


我们没有办法确定这三款产品的未来，现在下结论为时尚早，只是从产品气质的本身来讲，没有任何一款软件有“先知”气质。张小龙提前一周4个小时的演讲，有点过虑了。




/ 07 /

微信是不是“蓝色药丸”?


微信无疑是顶尖的互联网产品，为什么要对它这样苛责，这是不是心理有问题？

对一个为我们带来如此好处的APP，为什么会这样“杞人忧天”呢？

原因就在这里，正因为它是顶尖的互联网产品，才更让人担忧。

它的确在给予我们生活最大的自由，但同时也在将人类抛入囚笼。



乔布斯说过：我们希望产品是“技术与人文的结合点”，但实际上更多的产品在异化人类。


技术怎么在异化人类，张小龙也有困惑：“《黑客帝国》我看了好多遍，我觉得它挺有意味的。我们怎么知道我们处在一个真实的世界？确实我们不知道。但微信不是蓝色药丸……”

 

不过，微信是不是蓝色药丸不是张小龙说了算，而是正在“异化”的每一个用户。
