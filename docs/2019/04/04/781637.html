win10设置WiFi热点最有效的方法，不需要在cmd中输入netsh wlan set hostnetwork等命令，以及无法启动承接网络的原因

  • 柚子社区

问题描述：
对于拨号上网的用户，在打算使用win10自带的热点功能时，常常发现移动热点无法设置如下图所示。
对于百度教程：https://jingyan.baidu.com/article/d45ad1489bf2d269542b807b.html
我本人花费一个下午的时间也未设置成功，原因很简单，我们无法在网络和共享中心——更改适配器设置中找到Microsoft Virtual WIFI Miniport Adapter这个WiFi虚拟网卡，然后导致我们输入netsh wlan start hostednetwork时显示无法启动承接网络。我本人觉得这个网卡应该是需要笔记本本身连接WIAN时才能被开启，然后用来设置WiFi热点，下面的解决方法就是顺着这个思路来的。


对于拨号上网的用户解决设置热点的方法：
1、用自己的手机打开一个热点，然后让笔记本连接该热点
2、接下来你会发现在win10电脑的右下角，WiFi热点功能可以开启了，你只需打开即可
3、最后关闭你手机的热点，让手机来连接笔记本刚打开的热点即可

补充：如果你发现笔记本热点每隔一段时间掉一次的话，你把上述步骤走一遍即可，或者你嫌麻烦，我建议还是下载个猎豹wifi吧，比较好用，关键还稳定。猎豹WiFi链接下载：wifi.liebao.cn/ 。
对于连接无线校园网的同学设置热点的方法就更简单了，直接在右下角开启热点即可。让手机享受无限冲浪带来的乐趣，此方法已经让我和我整个寝室的人都很享受呢，关键无限上网还不花钱，哈哈哈哈。
原理：不管是软件开热点还是利用本地WIAN开热点，都是WIFI虚拟网卡（Microsoft Virtual WIFI Miniport Adapter共享宽带的网络或者校园网的网络。下图是WIFI虚拟网卡共享无线校园网的网络：
