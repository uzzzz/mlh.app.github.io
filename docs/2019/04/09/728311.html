97 后程序媛有什么特点？

  • 柚子社区

综合整理：程序员的那些事（id：IProgrammer）


国外开发者招聘平台 HackerRank 在调查 12211 名女程序员后，发布了《2019 Women In Tech Report | 2019 年科技行业女性报告》。


报告描绘了一幅令人鼓舞的画面，「Z世代」的女程序员拥有雇主迫切需要的技能。


一般来说，在美国及欧洲流行的「Z世代」，是指在上世纪 90 年代中叶到 2010 年前出生的人，也就是的 95 后。但 HR 报告将「Z世代」定义为 1997 年以后出生的人（97后）。程序员的那些事在下文中统一用「97后」，这更符合国内的说法。


下面一起来看看 97 后程序媛的情况。


0、97 后程序媛开始编程的年龄段


（橙色为97后程序媛，红色为97前）


97后程序媛接触编程的年龄，比97前程序媛更早。将近有 30% 的 97后程序媛在 16 岁之前就学了编程，相对来说，97前程序媛不到 18%。


学编程的年龄越来越年轻化。关于这点，可以从教育机会来解释。由于对程序员需求的增加，学校课程开始引入编程。比如：在英国和澳大利亚的小学，编程已作为必修课了。最近媒体报道日本也将从 2020 起把编程列入小学必修课。中国教育部也重新强调要在中小学引入编程课。



1、大多数97后程序媛精通 Python 和 Java


根据 HackerRank 的调查，在「最雇主欢迎的编程语言」中，JS 第一，Java 第二，Python第三。



随着「97后」女程序员步入职场，她们中大多数人至少精通前三种最受雇主欢迎的编程语言中的两种 —— Python 和 Java。这两种语言都被广泛用于计算机科学教育，所以也不足为奇。


（橙色为97后程序媛，红色为97后程序猿）


从上图中也可以看到，在97后程序员熟知的语言中，C 为第一， C++ 是第三。这可能是因为 C 和 C++ 是在编程入门时教给学生的第一种语言。



2、程序媛想迅速缩小未知语言之间的差距


调查还发现，59% 的招聘经理希望应聘者精通 JavaScript。虽然只有 50% 的 97后程序媛知道 JS，但剩下的 35% 的 97后程序媛计划今年学习。


目前，97后程序媛也无法满足招聘经理对 C# 和 Go 的需求，但她们正努力填补这一缺口。42% 的人将学习 C#， 34% 的人今年将学习 Go。


(浅粉色为要学的语言，另一种是已经会的语言)



3、97后程序媛对知名公司品牌的

重视程度是前几代人的两倍


在程序媛求职时对公司的考量中，「职业发展和学习」排第1，「工作/生活平衡」排第2，「工作有趣/Interesting problems to solve」排第3。


在考量「公司品牌」方面，97后程序媛的占比 14.16%，97前程序媛只有 6.7%。


(浅紫色为97后程序媛，紫色是97前）


97后程序媛对竞争性薪酬的重视程度也不同于前几代人。26% 的 97前程序媛认为薪酬是工作中的一个重要因素，而只有 16% 的 97后程序媛认为它很重要。


也许这可以用这样一个事实来解释：目前，大多数97后女性没有和97前女性一样的财务义务。因此，她们的优先考虑事项中，薪酬排在更靠后的位置，而更看重公司声誉等其他因素。



4、缺少面试准备，是面试最大障碍



和所有开发人员一样，对于97后程序员来说，在面试实习和工作时，缺乏对职位的明确界定是最大的障碍。


不过 97后是职场新人，面试经验有限，这很可能解释了为什么 97后女性(47%)和男性(45%)尤其会被雇主拒之门外，因为雇主在面试过程中没有从缺少准备的求职者身上看到所期待的东西。


事实上，大多数应届毕业生都觉得自己还没有为面试做好准备：66%的大学生说他们还没有为面试做好准备。



5、97后对硅谷的未来缺乏信心


在过去 5 年里，除亚太地区之外，硅谷是所有地区程序媛首选的全球科技中心。然而，与前几代人相比，所有地区97后程序媛都不大相信硅谷在未来仍将是科技中心。对硅谷兴趣下降，可能是因为生活成本上升，迫使企业家和公司总部搬迁。




EMEA 为 Europe、Middle East 和 Africa 的字母缩写，为欧洲、中东、非洲三地区的合称，通常是用作政府行政或商业上的区域划分方式，这种用法较常见于北美洲的企业。


APAC 这个大家应该熟悉，就是指「亚太地区」。


值得注意的是，所有地区 97 后程序媛来说，上海是下一个大型全球科技中心的首选地之一。


最近，中国一直在大力推动上海成为全球创新中心，这很可能是许多人开始将中国视为下一个大型科技中心所在地的原因。事实上，2018年中国在初创企业上的风险投资超过了美国。



亚太地区 97 后程序媛也认为，印度的班加罗尔（Bengaluru）在 5 年内很有可能成为科技中心。班加罗尔之所以具有吸引力，很可能是因为印度软件开发人员的数量正以快速增长，预计到 2023 年，印度将拥有世界上最多的开发人员。班加罗尔是印度科技初创企业数量最多的城市，也是全球第三大科技初创企业的所在地。
