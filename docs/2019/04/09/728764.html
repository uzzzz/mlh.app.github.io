功能：
1.自动关闭声音
2.自动1.5倍速
3.大题小弹框自动选择A选项（由于不算入分数之内，所以懒得选择正确答案）并关闭
4.自动在播放100%时，跳到下一集。

setInterval(function(){
  var times = $(".passTime").css("width"); 
  if (times == "100%") {
    $(".nextButton").click();
  }
  $(".speedTab15").click();
  if($(".volumeIcon").parent().attr("class")!='volumeBox volumeNone'){
    $(".volumeIcon").click();
  }
  if($("#popbox_title").text()){
	 $('.answerOption>label').children('input').click()
     $(".popbtn_cancel").click();
  }
  console.log("success");
},5000);

使用方式：

  1. 登陆后知到打开视频播放的页面
  2. 打开开发者管理工具（F12）
  3. 点到console标签
  4. 输入上面这段脚本
  5. 回车运行。
