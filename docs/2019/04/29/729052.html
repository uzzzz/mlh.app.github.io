【重磅要闻】


近日，Stack Overflow 发布 2019 年开发者调查报告，今年早些时候有将近 9 万名开发者参加了这份平均用时 20 分钟的调查，报告涉及开发者基本情况、技术、工作、社区、方法论等多个维度。


此次调查结果反映出如下主要趋势：


1、增长速度最快的主流编程语言 Python，在今年的编程语言排行榜上超越 Java，位居第二受喜爱的语言（仅次于 Rust）。


2、过半参与者在 16 岁时写下了自己的第一行代码。


3、运维专家和网站可靠性工程师薪资最高，经验最丰富的开发者对自己的工作满意程度最高，找新工作的比例也最低。



图片来源网络


4、调查涉及的国家中，中国开发者最乐观，他们认为现在的人们要比父辈拥有更好的生活；而位于法国、德国等西欧国家的开发者对未来最不乐观。


5、关于影响生产力的因素，不同类型的开发者提及不同的挑战。男性开发者更倾向于认为承担非开发工作是较大的问题，而性别少数群体更倾向于认为，恶劣的工作环境是较大问题。

360公司转让奇安信股权 进入政企安全市场铺开“大安全”战略


在A股上市的360公司近日发布公告称，将对外转让其所持北京奇安信科技有限公司（以下简称“奇安信”）的全部股权。360公司此前是奇安信第二大股东，所持奇安信股权占其总股权的22.59%。

本次股权转让完成后，360公司与奇安信之间将不再存在股权关系，双方“投资与被投资”、“授权与被授权”的关系宣告结束。


谷歌成功发布 Cloud Code


为履行让开发者更轻松创建云原生（cloud-native）应用的承诺，谷歌推出了一系列适用于 IntelliJ 和 Visual Studio Code 开发环境的插件，而本次推出的性能插件就是帮助开发者从他们的 IDE 中创建、调试和部署代码至 Kubernetes 集群中。


据了解，开发者可以像以往一样编写代码，但是谷歌的 Cloud Code 就会处理将其移植成为云原生应用的所有工作。这些工具目前已经整合到 Cloud Bulid 和 Stackdriver 等谷歌 DevOps 工具中。


Cloud Code 结合了许多现有的开源工具，包括处理 Kubernetes API 的命令行工具集 Kubectl，为 Java 应用创建容器的 Jib 以及为 Kubernetes 应用设置连续部署通道的 Skaffold等。可喜的是，该服务适用于几乎所有流行的编程语言，包括 Microsoft 的.NET Core。


开源创新


腾讯宣布开源三大开发工具


不久之前，腾讯在 OSCAR 开源先锋日上宣布开源三大开发工具：腾讯蓝鲸智云社区版“PaaS 平台”、调度编排 SaaS“标准运维”和基于 TARS 框架开发的分布式 NoSQL 存储系统 DCache。


据腾讯开源运营负责人王春雨介绍，早在你2019 年 1 月 24 日，腾讯在 GitHub 就累计获得 Star 数突破 20 万颗。目前腾讯蓝鲸 PaaS 平台、标准运维已在 GitHub 上开源，后续将会有更多产品开源。


Facebook 开源图神经网络库 PBG，无需 GPU 搞定数十亿节点图嵌入


早前DeepMind开源了自己的图神经网络库Graph Nets library，在业界掀起了一股图神经网络的热潮。最近，Facebook开源了自己的图神经网络库PyTorch-BigGraph（简称PBG），再大的图都能快速生成图嵌入，并且完全不需要GPU，对拥有数十亿节点的大规模图嵌入、电子商务中的推荐系统、社交媒体中的链接预测等任务都提供了极大便利。


研究人员在最近的SysML会议上发表了一篇关于PBG的论文，文中公开可用的社交图数据集实验结果表明“PBG优于竞争方法”。

台积电五纳米芯片将在明年投入量产


日前，积电宣布已经完成了设计 5 纳米芯片所需工具的研发，部分 alpha 客户已经开始 5 纳米芯片的风险量产，这意味着这项技术将在 2020 年投入量产。


台积电的 5 纳米工艺是该公司第二代使用深紫外光刻 (DUV) 和极紫外光刻 (EUV) 的制造技术；相比使用深紫外光刻的第一代 7 纳米工艺，5 纳米芯片面积能缩小 45%，晶体管密度能达到 1.8 倍，功耗减少 20%。


华为推出方舟编译器 称可提升安卓系统效率


之前华为 P30 系列国内发布会上，华为消费者业务 CEO 余承东宣布推出华为方舟编译器，称可解决安卓程序“边解释边执行”的低效。


他表示，方舟编译器可实现架构级优化，显著提升性能。所公布的数据显示，方舟编译器可让操作系统流畅度提升 24%，系统响应提升 44%，第三方应用操作流畅度提升 60%。如今方舟编译器将面向业界开源，希望众； App 开发厂商尽快使用。此外余承东还宣布对 GPU Tubro 进行升级，新增支持了 50 多款游戏。


关于5G


国内首个5G电话由中国移动在北京打通


近日中国移动对外宣布：在北京5G试验网上，北京市域成功接通了第一个5G电话。，据悉本次通话使用了4G现网升级的NSA核心网以及IMS网络，不仅成功做到了不换卡、不换号场景下5G手机间的通话，并且5G语音通话声音清晰。


图片来源网络


其实之前，中国移动北京公司就已经完成首个国内5G自动驾驶示范区的工作任务，承担“5G自动驾驶峰会”业务演示任务，助力了全球首场5G+真4K体育赛事直播（咪咕CBA），完成了基于5G网络无人机视频演示业务等。


对于本次接通5G通话，中国移动表示：在北京通过2.6GHz 5G网络环境，实现端到端的5G手机通话，标志着中国移动在北京的5G业务应用水平迈上了一个新台阶。


隐私安全


微软部分Outlook网页邮箱遭黑客入侵 隐私被泄露


据外媒最新消息，近日微软向一些用户发送了通知电子邮件，告知Outlook的网页邮箱服务遭到了一次黑客入侵，这可能直接影响到Outlook用户。


据国外媒体报道，微软称从2019年1月1日至2019年3月29日，一名黑客或一群黑客入侵了一家微软技术支持代理人的账户，这名代理人代表微软负责处理客户的技术投诉。


同时，这家操作系统开发商表示，在发现未经授权的入侵行为之后，该公司迅速禁用受影响的技术支持代理人的账号密码；然而却表示，黑客可能访问并查看了一些Outlook用户帐户的内容。


另外，一位微软前工程师通过加密聊天告诉科技媒体：“技术支持代理人可以看到你有多少封电子邮件，数据库在哪里，电子邮件的内容，你最后发送邮件的人等信息。”
