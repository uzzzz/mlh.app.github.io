Java一降再降

不再是最火语言


（各语言使用人数分布）

Java的份额一降再降，与此同时，今年，Python 在总排名中险胜 Java，而它去年和前年分别超过 C# 和 PHP。

现在，Python 是增长最快的主要编程语言。更让人唏嘘的是，Java还被开发者选为“最令人害怕的语言”第一名，让人不忍直视。下图是关于语言的喜爱度和害怕程度排名：

经常在网上看到程序员说，“今天又开始‘造轮子’了！”有人以为程序员要转行轮胎厂了。

其实“造轮子”是指——业界已经有公认的软件或者库了，程序员却还坚持要自己开发。这重复的工作量，不仅是一种时间的负担，更让人很难有成就感。

为什么现在公司和开发者都越来越爱用类似Python这样简短代码的语言了？现在不是大鱼吃小鱼，而是快鱼吃慢鱼的时代。人的时间比机器的时间值钱，而机器可以堆。

从公司角度而言：

国内基于Python创业成功的案例不在少数，豆瓣、知乎、果壳，全栈都是 Python，大家对Python自然有信心。并且从这几家公司出来的程序员与 CTO，创业的话一般都会选择Python，雪球定会越滚越大。

从开发者个人角度而言：

计算机语言只是用来达成目（偷懒）的工具，各种强大的第三方库，拿来就能用才是王道，让程序替代我们执行一些枯燥繁琐的工作。至于句式是否优美、能否反映出使用者的智力逼格，Who cares？

比如：同样是显示Hello World，Python更近似于自然语言（英语）的表述。

所以Bruce大神才会说“ 人生苦短，我用Python ”。

Java 再垫底

成开发行业收入最低群体


在Stack Overflow 的调查中，我们还观察到一组数据，那就是：Java 语言开发者竟成为收入最低的群体。

Java真的很低吗？我们爬取了全国的Python职位做对比，最后发现python薪资最高的是一线城市，竟然平均月薪高达20k。


关于赚钱，我特别喜欢的心理学专家李松蔚，曾写过这样一段话：

很多人都以赚钱为目标。

但多少人有勇气直视这个目标呢？我们的精力有多少真正花在「怎么赚钱」上，又有多少是在隐隐自证「我这样的人，反正也不可能赚大钱了」？

我没有积蓄，没有背景，工资不高，孤身无靠，我也不懂赚钱那些门道……反正找理由的话，理由多的是。可怜的哦。但如果你看到这里，已经不想再继续可怜自己了，你把注意力放回到目标本身，你才真的开始为赚钱这件事本身而改变。

这是一种思考方式的蜕变。

我们常说，有钱不只是一种状态，而是一种思维习惯。这种思维的第一步，就是放下跟「自己」有关的执念，把焦点投放到目标本身。看似困难的目标，也可以拆解成具体的步骤，一步一步实现，而不是用各种形式证明自己做不到，自我哀怜。

Python 连续三年成为最想使用的编程语言，所以你准备好学习Python、挣钱通往财富自由了吗？你打算怎么开始，怎么发展呢？
