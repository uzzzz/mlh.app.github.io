大疆前员工泄露公司源码被罚20万，获刑半年

据《南方都市报》报道，深圳法院近日对大疆源代码泄露案做出一审判决，综合考虑犯罪情节以及自愿认罪、有悔罪表现，以侵犯商业秘密罪判处大疆前员工有期徒刑六个月，并处罚金20万人民币。

这些泄露出去的代码，已用于该公司农业无人机产品，具有实用性。尽管大疆公司采取了合理的保密措施，但该次事件依然给大疆造成经济损失116.4万元人民币。

“阿里巴巴普惠体”品牌定制字体发布

据报道，汉仪字库为阿里巴巴设计的品牌定制字体“阿里巴巴普惠体”正式发布。

共有Light / Regular / Medium / Bold / Heavy 5个字重。西文部分汉仪字库西文组扩展了日文假名、汉语拼音、台湾注音符号等，最终共计124145个字符。

阿里巴巴官微表示，其“花10个月设计了这套新字体，免费开放给书写梦想的你。”

赵明宣布荣耀启动“二级火箭”战略：快速突进中国第二

近日，云南昆明首家荣耀潮玩店荣耀Life正式开业，荣耀总裁赵明首次披露了荣耀手机“二级火箭”全新战略，其中包括快速突进中国市场第二、全球第四的全新战略目标。

此外，云南昆明荣耀Life也是荣耀2019年成立的第100家Shopping Mall门店、第100家连锁品牌形象门店。

截至2019年4月，荣耀线下门店累计已建成近2000家。

张勇卸任天猫法定代表人，85后蒋凡接任

4月24日，全天候科技通过企查查发现，近日浙江天猫网络有限公司发生多项工商变更，阿里巴巴集团首席执行官、阿里巴巴集团董事局董事张勇卸任该公司法定代表人以及董事长兼总经理职务，改由蒋凡接任；

另外，阿里巴巴集团首席风险官郑俊芳（监事）退出，新增张彧（监事）；同时，该公司经营范围新增演出及经纪业务。

HTC：第二款区块链手机今年下半年发布

据台湾媒体消息，HTC区块链首席运营官陈信生表示，第2代区块链手机会在今年下半年推出。

HTC区块链智能手机“EXODUS 1”于去年第四季度正式上市，报道称其表现“符合销售预期”。

除了新一代的区块链手机，陈信生还预告称，会将数字资产管理范围扩展到浏览、信息等领域。

特效示意图，CSDN截图

当下，漫威大作《复仇者联盟4》正在全球各地热映。据9to5Google报道，谷歌在搜索服务中加入了一个来自《复仇者联盟》人物灭霸（Thanos）的彩蛋。

现在在谷歌搜索上搜索“灭霸”，谷歌会在页面右侧的图片搜索结果附近“混”进去一个无限手套；

苹果再次因涉嫌滥用其应用商店控制权而遭到指控

据外媒报道，苹果将再次遭到审查，因为有屏幕时间管理应用程序开发商向欧盟竞争办公室投诉它滥用其对应用商店的控制权来打击竞争对手的产品。

《纽约时报》报道称，苹果去年删除或限制了17个屏幕限时App和至少11个家长控制App。

苹果的一位女发言人称，这些应用程序之所以被撤销或限制，是因为它们从用户的设备上收集了太多数据，与苹果发布屏幕时间和家长控制功能无关。

波音737 MAX最快5月底复飞：软件更新已完成

据美国媒体CNBC称，国联邦航空管理局（FAA）或将会在5月底或6月初批准波音737 MAX机型的复飞。

目前波音公司还没有上述消息回应。此前，波音公司首席执行官丹尼斯·穆伦堡（Dennis Muilenburg）在西雅图的波音机场发表讲话称，该公司在过去几周进行了127次Max 测试飞行，累积飞行时间为203小时。

微软发布.NET for Apache Spark首个预览版

Spark是一种流行的开源分布式处理引擎，适用于大型数据集的分析。Spark可用于处理批量数据，实时流，机器学习和即席查询。

.NET for Apache Spark旨在使.NET开发人员可以跨所有Spark API访问Apache®Spark™。

到目前为止，Spark已经可以通过Scala，Java，Python和R访问，但不能访问.NET。

.NET for Apache Spark的后期更新如下：

  • 简化入门体验，文档和示例。

  • .NET支持用户定义的聚合函数。

  • 与Visual Studio，Visual Studio Code，Jupyter笔记本等开发人员工具进行本机集成。
