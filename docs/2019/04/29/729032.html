新华三首发数字大脑计划


由紫光集团和旗下新华三集团主办的2019 Navigate领航者峰会近日在重庆正式召开，紫光集团联席总裁兼新华三首席执行官于英涛现场发布了数字大脑计划，这也是数字大脑计划的首次亮相，他着重强调了“计划”二字，表明这并不是一个简单的产品。


数字大脑”是数字化时代让组织高效运行与成长的能力中心，其构成可以概括为“4+N”模式，主要包含两个重要部分：“智能数字平台”（即“4+N”模式中的4）以及与合作伙伴共同打造的智慧应用生态（即“4+N”模式中的N）。



在数字大脑“4+N”的模式中，新华三的智能数字平台是核心基石，也是支撑智慧应用的丰富土壤，其中包括：向内生智能演进的数字基础设施、作为数字化转型核心引擎的业务能力平台、实现由智慧驱动信息防护的主动安全，以及为IT环境提供全栈式智能保障的统一运维等。



微软收购物联网系统公司Express Logic 以帮助其进入物联网领域


据国外媒体报道，近日微软表示，已收购物联网系统公司Express Logic，但目前交易条款尚未披露，此举反映出公司继续专注于围绕所谓的“物联网”创造营收。


物联网描述了各种各样的简单设备如何通过连接互联网来共享数据、接收网络更新等， 其他云提供商，如亚马逊和谷歌也提供了一些关于物联网的功能。


三星折叠屏手机临时取消中国区发布会，推迟时间待定


日前，三星中国方面公关人士临时通知媒体，取消了三星Galaxy Fold折叠屏手机在上海的发布会，解释为场地原因所推迟并且表示推迟时间待定。


但此前有试用用户反映，该手机出现屏幕隆起、闪屏、黑屏等问题，甚至手机完全不能使用。


图片来源网络


对于临时取消发布会，一位媒体人在朋友圈表示，“感叹一句，折叠屏手机真不是那么好玩的。”


如今，折叠屏手机被认为是智能手机的下一重大趋势，引起了强烈的关注。不过关于折叠屏手机耐用性问题、寿命问题等，外界一直表现出担忧；在今年的WMC上，华为与三星均展示了折叠屏手机，但均不让用户触摸体验，仅仅隔着玻璃罩展示。


Google 发布小型芯片 Jacquard，想把所有物体都变成交互界面


在今年的 TED 大会上，Google 先进科技与计划部门（ATAP）负责人 Ivan Poupyrev 将演讲变成了半个产品发布会。但在演讲的最后几分钟，Ivan Poupyrev 第一次对外展示了微型计算芯片 Jacquard，作为连接实体物品与云计算的统一载体。


图片来源网络


据了解，Jacquard 是一块相当小的芯片，大小与手指指节相近。这块芯片以背部的铜片作为接触点，连接塞入了可导电且防水的特殊织物的实体物品，云计算用于控制其功能，包括识别手势操作等。


缔盟云正式加入云安全联盟CSA，助力云安全产业快速发展


近日，缔盟云（杭州云缔盟科技有限公司）正式加入云安全联盟（Cloud Security Alliance，CSA）成为CSA会员单位。


据悉，云安全联盟CSA发起于2008年，是中立的非盈利世界性行业组织；目前已协助美国、欧盟、日本、澳大利亚、新加坡等多国政府开展国家网络安全战略、国家云计算战略、国家云安全标准等工作，并为业界客户提供安全标准认证和教育培训。缔盟云是一家技术驱动的云安全创新型企业，是全栈云化智能防御的安全厂商。


传华芯通本月底关闭，根源于Arm服务器的困局


近日，The Information援引高通与贵州省的合资企业华芯通半导体（HXT）员工的消息称，公司高管在内部会议上表示，公司将于4月30日关闭。报道还称，截至2018年8月，高通和贵州省对华芯通半导体一共投资5.7亿美元。对此消息，高通和华芯通暂未官方回应。


不过有报道称，华芯通CEO汪凯已经离职，关于华芯通关闭的的传闻表示“一言难尽”。华芯通关闭的传言迅速引发的巨大的讨论，关键点主要是Arm服务器的成败以及对中国芯发展的影响。


腾讯获准在中国销售Switch游戏机 任天堂股价应声飙升逾14%


据美国财经网站CNBC报道，在腾讯获准开始在中国销售任天堂的Switch游戏机后，这家日本游戏巨头的股价飙升逾14%，创下自2018年10月18日以来的单日新高。


图片来源网络


Niko Partners分析师丹尼尔•艾哈迈德（Daniel Ahmad）表示，长期以来，投资者一直在问任天堂何时进入中国这个全球最大的游戏市场，而与腾讯结盟将是一个受欢迎的举措。但他警告称该市场不应过于兴奋，因为任天堂和腾讯仍需获得中国更多监管机构的批准。


马斯克欲在美国东海岸挖隧道 1小时车程将缩至15分钟


据国外媒体报道，本周伊隆·马斯克(Elon Musk)领导的隧道挖掘企业Boring发布了一份长达505页的环境评价报告，描述了关于其通过高速交通系统连接华盛顿特区和巴尔的摩的提议。


如果计划成行，乘客从华盛顿特区抵达巴尔的摩市仅需要15分钟的时间，这是马斯克在东海岸实现超级高铁梦想的一小步，但却是必要一步，其曾在两年前的一条推文上提到过此事。


图片来源网络


据悉，这份名为《华盛顿特区到巴尔的摩环线项目》(Washington, DC, to Baltimore Loop Project)的文件，详尽列出了马斯克和他的公司要完成高速地下交通系统实际上需要做的所有事情。
