自从春运开始后，GitHub 上又火了好几款用 Python 写的开源的抢票软件。


上面几个项目，可以从这里看到：https://github.com/search?q=12306


据 1 月 27 日央视财经消息，为了买到回家的火车票，一些人求助抢票软件，那么抢票软件都是怎么运作的呢？据了解，第三方软件抢票就是不断刷 12306 网站的余票信息，从而为消费者抢票，但第三方抢票软件存在安全隐患，个人隐私信息得不到保障。鉴于此，中国铁路总局也是采取了相关措施。



据中国铁路总局介绍，这些第三方软件的相关机器特征已经被识别并被实施限制措施。也就是说，即使用户花钱购买了加速服务，购票的成功率也绝不会像各个抢票软件显示的一样。为出行安全，最好不要通过第三方代购网站和手机客户端购票。


各位已买到火车票的童鞋，是在 12306 官方渠道买到的，还是第三方渠道呢？
