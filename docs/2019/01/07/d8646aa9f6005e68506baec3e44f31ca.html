Tensorflow——jupyter notebook调用某个库时，出现找不到这个库情况的解决方案

  • 柚子社区

1、激活tensorflow环境

终端下输入：

source activate tensorflow

2、进入jupyter notebook

出现如下问题：没有找到matplotlib库

3、解决方法

  • 在tensorflow环境下，输入conda list 查看这个版本的python依赖的anaconda里面有哪些已经安装的库目录，显示如下

  • 可以发现没有我们要调用的matplotlib库，因此，我们执行conda install matplotlib进行这个库的安装，安装是安装在anaconda里

  • 再次查看conda的库目录，可以发现此时已经有matplotlib库目录

  • 再次进入jupyter notebook，调用matplotlib，发现已经可以正常调用
