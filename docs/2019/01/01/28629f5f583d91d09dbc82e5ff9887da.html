2018 年总结 —— 二本保研 985

  • 柚子社区

现在已经 2019 年了，过去的 2018 年是我人生中的一个转折点了。写这篇文章，一是为了记录一下自己这一年的经历，二是给那些跟我一样本科出身不太好的保研同学参考。

Why 读研？

2018年的一开始我就已经决定要读研究生，主要原因是以后想做 ML 相关工作，但在本科阶段并没有系统地学习过，希望通过研究生阶段好好学习一下，但是暂时没有读博打算。

基本背景

本科学校是沈阳不知名的学校，在本省有的专业招一本有的招二本，但在我们省都是二本。由于学的不是学校优势专业，所以各种比赛机会比较少、经费也少，简历就比较单薄。本科学校才有推免资格两年，名额比较少，分到学院就更少了，所以在年初的时候我还在准备统考，当时先复习了数学、英语，暑假的时候主要看了专业课。我比较菜，并没有多少奖项，两个国三，其他的都是一些不入流的奖，四六级 500+。

夏令营

到了五六月份，我抱着试一试的心态去报名了一些 985 学校的夏令营，基本都没什么音讯，现在其他专业转计算机的学生太多了，有不少 985、211 的学生都在转计算机，因此我们这种本科不是很好的学生就不占优势，机会比较少（大佬当我没说）。大家不要把主要精力放在夏令营了，有点困难，九推还是有不少机会的。

九推

正式九推之前，我参加了中科院上海高等研究院的一个交流活动，其实也就是推免面试加上两天的活动，但是时间安排的比较靠前，大概在九月十几号，这个时候很多学校正式的推免面试还没有开始。我就是冲着保底 offer 去参加这次活动和面试，简历上并没有太多的项目，然后老师拿着成绩单开始问我的专业课。因为各个老师的方向不太一样，所以老师们都喜欢问一下比较偏向基础理论的问题，但是会问你拓展到实际的问题。中科院的老师比较喜欢实践能力比较强的学生，因为中科院是要接项目的。除此之外，由于我的专业是比较偏向数学的，我还被问了几个数学问题，我没怎么准备数学的问题，所以回答的不是很好。但是很幸运的是我拿到了 offer，也结识了几位大佬，高研院给我的印象基本都就是 money 无限多。

九推结识的几个小伙伴互相推荐学校，然后我报了不少，但是很多并不是很上心，就是抱着随缘的心态投的简历。同时，我联系了很多学校的老师，有一些回应了，有一些没有回应，多尝试才会有更多的机会嘛。虽然我报了很多学校，但是我也就着重关注了几所学校：华东师范大学数据学院和软件学院，浙江大学软件学院等等。

离开上海高等研究院之后，我去了南京理工大学，上午机试可以说是比较简单的，下午我的面试时间比较晚要到下班时间了，老师们问的比较快也没有问什么实质性的问题，简单的自我介绍，英文介绍一下自己的专业，然后竟然跟我聊起了我的高中，说我们高中还不错，对于这样的面试我也是一脸懵13，offer 跟白送一样。面试完本该去苏州大学面试，但是我不想去了就回家准备浙大软院的面试了。

在准备浙大软院面试期间我一直在等华东师范的面试通知，然而等我在浙大面试结束了我才发现 Gmail 把华师的面试通知的邮件当作 spam 了，当时差点就要骂人了，冷静了一下想了想可能命中无华师吧。其间，我还联系了大连理工大学的老师，后来联系的老师名额估计满了把我推荐给同系的其他老师了，我也收到了几个老师的面试邀请，当时我并没有一口答应，并且如实说了自己还在等浙大的面试结果。

浙大面试的过程是我最紧张的吧，我是下午第一组第二个，面试的过程先是英文自我介绍，然后老师看了成绩单问了一个代数问题，回答的不太好（唉，复习基本没怎么看数学），接着老师问了我的项目，因为我没有什么大型项目，所以我就介绍了一下自己的小项目的技术原理，结果感觉被老师嫌弃了反问我大学一个项目最多写过多少代码，回答的比较尴尬就略过了吧，最后问了几个简单的 JAVA 问题和政治问题就结束了。出来之后感觉自己就凉了，果然就凉了，浙大软院老师的面试很看重实践项目，自己实践项目实在是太少了，我看很多录取的人简历写的都是爬虫之类，我觉得完全可以做一做。我最后的成绩好像在两百多个人的中间位置，离最后的录取人次差了十名不到，自己实力不济也没什么可以抱怨的。

录取

最终我被大连理工大学软件学院录取，老师的方向还算是我想学的方向，两年毕业后我还是条好汉。

总结

  1. 二本推免到 985，不要有心理负担，尽管去尝试，我觉得只要优秀完全有可能，大学同学不少都去了哈工大、天大、华科等等很好的 985 高校。

  2. 计算机相关专业的同学，我觉得还是需要提升自己的项目经历，多做一些实践类型的项目，当然如果能跟着老师发一些论文也是极好的，相当有优势。

  3. 可以广撒网，现在大佬都广撒网，再不撒广点漏都捡不到了。

  4. 很多学校都是导师说了算的，大家要注意提前联系导师。

大家一定可以保研到自己理想的学校，加油！
