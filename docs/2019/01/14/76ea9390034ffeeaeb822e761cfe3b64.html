1. Stack overflow

Stack overflow.com 或者是编程界中最流行的网址了 , 是一个与程序相关的 it互联网 技术性问答平台，用户还可以在网站免费上传问题，浏览问题，数据库索引有关内容，在创建主页的那时候运用简易的 HTML 。部分的程序员学习 API 或是编程语言 , 一般有赖于代码示例 ,stackoverflow 就可以提供很多的代码片段。

Stackoverflow 的另外优势在于它的社交性。你可以在某些特殊的标签下查询问题 , 例如“ Java ”、“ regex ” , 你就会见到什么是高频率难题。这针对学习 , 以及 Java 博主书写热门话题全是一种很好的资源。

2. DZone

它是1个十分有趣的网站 , 有相当多的开发人员在这一网站上共享他们博客文章。如同一场探险一样 , 在这 , 你永远不知道下一秒会见到哪些内容。

3. LeetCode

假如有 Java 这方面的面试问题 , 比如“在 Java 中 , 内存中的数组是什么的” , 你可以从许多 Java 实例教程中搜到参考答案。但是 , 碰到类似“如何将 SortedArray 转化成一个 BalancedTree ”那样的问题 , 你就应当去求助 LeetCode 了。 LeetCode 包括一连串算法问题 , 它能为关于现代信息技术的招聘面试提供1个社会性的网络平台。这个网站最好的地方也是 , 它还可以根据大小不一样的数据在线查验你的代码是不是无误。许多人认为 , 招聘面试取得成功的关键所在反复这三个关键方法步骤 : 编码 -> 阅读 -> 讨论。

4. Java SE 技术文档

这一网站包括了在运用 Java SE API 时候采用的全部文档 , 即便你也是 Java 开发高手 , 我敢肯定这里你仍能看到一些实用的东西。例如 , 你能阅读理解一些像“ Java 类的本质”、“部署”这样的基础教程。

5. Github

你或许晓得可以免费的将你的项目代管在哪儿 , 但你不一定了解它是不是1个学习现如今流行的 Java库和框架的优良资源。例如 , 你要学 Spring MVC Framework, 你可以搜寻而且搜到某些开源的项目。遵循“照葫芦画瓢”的基本原则 , 你可以根据案例迅速的学习框架结构 , 特别是在是你早已具备运用类似框架的经历。

6. Coursera

这是关于视频讲座的最合适网站 , 这里你能搜到很多顶级大学的着名教授的出色计算机科学课程。那些教授中 , 很多以至于是一些计算机科学领域的发明者。

7. Java World

这一网站含有了很多的 Java 教程视频 , 涉及到各式各样的话题。其中许多文章都写的非常好 , 而且有图片、插图进行详细说明。它还可以被用于作为深度学习的教材。

8. DevStore

这个网站是开发者服务商店，这个网站集成了诸多的开发人员服务，以及有关的服务评测，另外还给开发者提供许多免费的源码供我们学习。主要同类第三方服务的对比功能很强大，可以根据自己的需要找出适合自己的第三方服务。

9. Wikipedia 维基百科

这个网站是查询或学习基本上全部定义的最好的资源之一。比如 , 做为一个杰出的 Java 开发人员, 你可能仅仅想掌握某些定义并不愿深度研究。这儿是一个可以全免费查寻最新消息的好去处。比如, 什么叫面向服务的编程 ? 另外 , 它还能为你的写作提供大量素材。例如 , 同一个词在不同的领域可能会有完全不同的含义。有时 , 研究这些词在不同领域的不同含义是非常有意思的。

10. Program Creek

与以上几个网站相比 ,Programcreek.com 在规模上要小许多。可是规模小也有优势 , 它在系统维护这方面做的好些 , 读起来更趣味。这里 , 你能找出别的其他网站都未曾涉及的话题 , 并且基本上每篇文章都包括好看的插图或是示例编码。它包括了出自不一样领域的人发布的稿子 ( 科研、工业 )，而且常常为 Java 开发人员提供高质量材料。期待有一天 , 它也会出现在一些人的网站前十名排名名册中。
