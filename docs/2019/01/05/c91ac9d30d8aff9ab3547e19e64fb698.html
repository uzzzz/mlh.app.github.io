ubuntu16.04中安装qq国际版教程

  • 柚子社区

从windows系统转到ubuntu，还是习惯性的需要安装qq，原来用的网页版qq已经停用了（而且没有更新的qq体验真的很差！），这个是ubuntu16.04下安装的qq国际版，下面是我的安装过程，分享给大家～

step1：下载qq的安装包（资源审核后就可以下载了）：

https://download.csdn.net/download/weixin_42056625/10898263

 

step2：进入下载qq的文件夹，右键进行解压，选择解压到此处

 

step3：安装需要的32位运行库

 

sudo apt-get install libgtk2.0-0:i386  lib32ncurses5

 

step4：解压并安装wineqqintl

sudo dpkg -i wine-qqintl_0.1.3-2_i386.deb

如果运行失败，则先输入

sudo apt-get install -f

再次进行安装即可成功

sudo dpkg -i wine-qqintl_0.1.3-2_i386.deb

接着安装

sudo dpkg -i ttf-wqy-microhei_0.2.0-beta-2_all.deb  fonts-wqy-microhei_0.2.0-beta-2_all.deb

 

step5：到这里我们已经成功安装qq了，我们打开qq，登录，会提示“您的QQ号不适用与此版本，请下载最新版QQ尝试”的问题。不用急，只需要在手机端（账号、设备安全）里临时关闭设备锁即可以成功登录qq了！

 

step6：登录成功后，我们会发现qq好不能显示成中文的问题，我们需要做一点调整即可，终端输入

sudo gedit /usr/share/deepinwine/qqintl/wine-qqintl

在export LANG=zh_CN.utf8下面加入

export XMODIFIERS="@im=fcitx"

export GTK_IM_MODULE="fcitx"

export QT_IM_MODULE="fcitx"

Final：最后，我们只需要重启qq就可以尽情的使用qq了，打开qq后在图标处右键锁定到启动项就可以固定图标了。～美滋滋
