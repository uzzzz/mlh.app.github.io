一入链圈深似海

任何一项技术出来，先被高估是惯例，而只有在其经历了泡沫破碎的低谷之后，才能迎来被市场所接受的成熟期。想必，区块链技术的发展亦是如此。而如今，则以区块链技术为支撑的虚拟货币最为世人关注。目前的市面上存在着上百种代币，但就市值而言，比特币、瑞波币和以太坊雄踞前三。学习资料也可以加下Python扣扣裙：3零4零5零799自己下载学习下

科技大佬研究加密货币——近日，硅谷科技巨头之一的 Facebook 社交网 CEO 马克·扎克伯格在自己的 Facebook 主页公布了自己 2018 年的目标——整顿 Facebook，研究加密货币。

比特币得道，狗币升天——狗币（又叫多吉币、狗狗币或旺旺币）的吉祥物是网络上非常火的柴犬表情包，区块链和网络代码源自莱特币（Litecoin）。截止1月7日，狗币总市值突破 20 亿美元关口，而仅仅约在两周前的圣诞节当天，它才触及 10 亿美元高点。

以太坊价格一年猛涨130倍——据跟踪数字货币价格的网站CoinDesk称，这种数字货币的价格一度达到1417.38美元的新高。在过去一年中，以太坊的价格增长了大约130倍。而以太坊创始人 Vitalik却是个年少有成的90后科技天才。

挖矿网站频遭黑客袭击——12月上旬，加密货币挖矿网站NiceHash曝出了黑客入侵事件，结果导致当时价值将近6400万美元的比特币失窃、以及网站关停24小时。

人人网因区块链项目被约谈——1月8日，人人网（NYSE：RENN）股价就下跌1.02美元至11.91美元，跌幅为7.85%。而此前人人网曾发布RRCoin白皮书，提供开源区块链平台——人人坊，并会成立RRCoin基金会。随后，人人网遭监管约谈，被网友评为“给区块链抹黑”。

1月9日下午，真格基金创始人徐小平的一句“拥抱区块链技术革命，学习区块链技术”被网友曝光，受到各路网友的抨击。

1月10日开始，国内区块链概念股集体疯长。截止1月11日收盘，编入通达信区块链概念的个股出现8只涨停，16只区块链概念股盘中触及涨停。

企业动态

国内

阿里程序员相亲，竟一双国产运动鞋被拒

最近，在杭州萧山，一个小伙因为在相亲时穿了一双特步牌运动鞋，遭女方拒绝了。原因是：27岁的男生穿特步鞋约会不合适；自己喜欢精致一点的男生。1月8日下午，认证为阿里巴巴官方文化账号的@阿里味儿官方微博表示，这个被拒绝的小伙是阿里巴巴的高级研发工程师。

百度App涉嫌“用户监听”，百度说没能力，小米认为就是你的

1月8日，因涉嫌用户监听电话被江苏消保委起诉的“百度”再次上了热搜。百度方面表示，手机百度有一个骚扰电话拦截的功能，小米手机把这个功能翻译成了“监听电话”，造成了误解，旗下App不会、也没有能力监听电话。第二日，小米方面回应，打脸百度称：“调用智能手机系统内的"READ PHONE STATE" 权限可获得手机号码、IMEI、MSI；同时具备监听电话状态能力以获取电话呼入、呼出、通话中的状态。”

传下半年IPO 小米能撑起2000亿美元估值吗？

1月8日，小米准备在今年下半年进行IPO。接近小米高层人士称，雷军在去年11月的确与投行进行了接触，并向对方同步了2000亿美元估值的目标，受到了投行的认可。目前，官方对此表示不予置评。

雷军曾在2016年夏季达沃斯论坛上明确表示小米IPO的时间点大约在2025年，但现在的种种迹象显示日期要提前了。

华为进军美国受挫，竟被美运营商巨头临时放鸽子

1月9日，据国外媒体《华尔街日报》报道，华为与美国第二大移动运营商美国电话电报公司 AT&T 的合作计划已经告吹，后者不会在美国销售华为手机。

苹果iCloud中国服务器将转由云上贵州运营

1月10日，据苹果公司透露，从今年2月28日起，中国内地的iCloud服务将转由云上贵州公司负责运营。也就是说，从当天起，云上贵州公司将负责iCloud在中国内地的运营，并将拥有与iCloud 中国内地用户的法律和财务关系。有网友调侃道：这是换了一种监听方式？

国外

“我将毁灭地球”的网红机器人索菲亚为什么挨骂？

1月4日，AI大神、Facebook AI实验室负责人YannLeCun，在接受科技媒体 Business Insider时对一直以来善于制造舆论话题的机器人索菲亚，谈到了自己的看法：“索菲亚之于 AI，就像魔术之于真正的魔法。也许我们应该把这称为“对AI的货物崇拜”、“AI 界的波将金村”或者是“AI 版的《绿野仙踪》”。换句话说，这根本就是扯淡。至于 Tech Insider，你也是这起骗局的帮凶之一。”

苹果“降速门”继续发酵，集体诉讼已增加到26起

1月7日，据国外媒体报道，苹果因为故意延缓老款iPhone性能或未通知消费者在iOS 10.2.1版本系统进行电源能管理行为引发的“降速门”已经在全球范围内面临超过26起集体诉讼。

百度自动驾驶平台Apollo2.0版上线，砸2亿美元全球挖人

美国当地时间1月8日，百度在拉斯维加斯举办了主题为“AI is Changing the World, China Speed”的发布会。在会上，百度宣布自动驾驶平台阿波罗2.0正式上线，并公布了在自动驾驶领域的人才计划。

苹果 macOS 再曝漏洞，输任意密码可进入 App Store 首选项

1月11日，据外媒 MacRumors 报道，苹果当前版本的 macOS High Sierra 再曝安全漏洞，它允许输入任意密码解锁系统设置中的 App Store。

英特尔承认CPU漏洞对性能产生影响：幅度为2%至14%

1月9日消息，英特尔承认，公司SYSmark测试显示，打补丁之后的CPU性能降幅大约为2%至14%。不过，英特尔依然坚持Meltdown和Spectre两个CPU漏洞不会对电脑性能产生太大影响。他们表示，常规的消费级和企业级应用都不会因为打补丁而影响任何性能：“第8代酷睿平台配合固态存储受到的性能影响只有6%，甚至更低。”

甲骨文被曝服务器漏洞，攻击者窃获 22.6 万美元加密货币

1月10日，据外媒 Ars Technica 报道，研究人员 Renato Marinho 发布报告称，攻击者利用甲骨文去年年底修补的 Web 应用程序服务器漏洞，针对 PeopleSoft 和 WebLogic 服务器开展了多轮攻击。攻击者并没有利用这个漏洞来窃取数据，而是将 PeopleSoft 和 WebLogic 服务器变成了挖矿工具，使用它来进行加密货币挖矿——有一名攻击者利用这个漏洞，获得了至少 611 个门罗币（XMR），总价值高达 22.6 万美元。

苹果开放 iOS 系统降级通道，结果不到 2 小时就关闭了

据外媒 9to5Mac 报道，昨天下午，苹果意外开放了 iOS 系统降级通道，用户通过 iOS 固件下载网站 IPSW.me，可以将系统从 iOS 11 降级到 iOS 10.2 和 10.3。然而不到 2 小时，降级通道又迅速关闭了。

DB-Engines 2017 年度数据库：PostgreSQL 实至名归

1月10日，DB-Engines 表示，PostgreSQL 在 2017 年的数据库排名中，比其他监测到的 341 个数据库管理系统都更受欢迎。因此，决定宣布 PostgreSQL 为 2017 年的年度 DBMS。计算结果基于当前（2018 年 1 月）分数和 2017 年 1 月差值获得。最终的结果代表了一个产品在 2017 年获得的人气。

PHP 霸主地位被动摇，JIT 是穷途末路后的绝地反击？

TIOBE 2017 年度编程语言榜单已出炉，世界上最好的语言 PHP 再度无缘年度编程语言。距离其上次（2004 年）获得年度编程语言，已有 13 年之久。而从历年 TIOBE 编程排行榜趋势图也可以看到，自 2014 年以来，PHP 总体处于持续下滑趋势。

PHP 在服务端编程语言领域依旧占据主导地位，同时，PHP 社区组持续不断地做版本迭代更新，性能提升。下一个大版本将引入 JIT 特性，这个被奉为点石成金的技术会给 PHP 带来更好的性能，更大的发展吗？

2017年哪些机器学习项目最受人关注？

最近，Mybridge发布了一篇文章，对比了过去一年中机器学习领域大约8800个开源项目后，选出30个2017年度优秀的开源项目，包含机器学习开源库、数据库以及其他应用程序，这些项目多数都是在2017年1-12月发布。Mybridge AI 通过流行度、参与度和新鲜程度来对它们进行评级，数据显示它们的 GitHub 平均 stars 是 3558 个。
