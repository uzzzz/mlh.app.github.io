总结 | 2018 年终总结

最近罗胖的「时间的朋友」跨年演讲刷遍了票圈，20 年的承诺，这是第四次，也是倒数第十七次演讲，而我却是第一次听，实则惭愧。刚刚算是把四个小时的演讲完全听下来，学到了小趋势、事情真相、非共识、百年人生、人的逻辑和长期主义等关键词，里边涉及到很多颠覆个人认知的例子，建议没看过的朋友抽时间静下来感受一下。

整个演讲的核心思想即是在告诉我们，世界已经由「大河模型」变成了「多米洛骨牌模型」，2018 年开始，一张通票吃遍天的大趋势已经结束，取而代之的是不断出现和被身边人所发现并利用的「小趋势」。小趋势是影响趋势的趋势，带来改变的改变，它没法跟，也不能跟，我们所能做的就是随时感知它，然后激发那些你早已准备好的力量。

所以想了想还是觉得自己应该开始把 2018 年复盘一下，至少为 2019 年定下一些小目标，为身边的小趋势准备一些属于自己的力量，以待感知到这个世界的小趋势时能够有激发驱动力。下面主要从工作与学习、读书、理财和公众号四个方面进行反思。

先说工作与学习。说实话，自己还是一名硕士应届毕业生，2018 年 10 月定了自己的工作，base 地为武汉的 Synopsys。在找工作前的准备阶段确实规划的时间太短，所以 2019 要做的第一件事是为自己的未来做好准备。这个不太好量化，加上目前工作所涉及到的具体内容还不是特别清楚，就先列出脑海中所能想到的几件必做事情。

  • 实习 | 春节前联系公司确定实习时间，在入职前实习三个月以上

  • 算法 | 再学《剑指 Offer》，记录思路并产出公众号文章

  • 算法 | LeetCode刷题100道以上，产出公众号文章

  • 算法 | 学完《程序员代码面试指南—IT名企算法与数据结构题目最优解》

  • 英语 | 懂你英语课程完成一期

  • 论文 | 1 月份完成毕业论文

  • 2019 年 7 月中期复盘以上内容，并添加工作所涉及且需要学习的具体内容

再说说读书。2018 年主要阅读了《好好学习——个人知识管理精进指南》、《你凭什么做好互联网》、《半小时漫画中国史》、《番茄工作法图解》和《刺》等书籍，真的太少了。2019 年，更新自己的读书笔记到公众号，目标 30 篇以上，并提供思维导图，望大家监督。

关于理财。2018 年学习并参与了 A 股、港股打新、基金投资，想必大家都知道 2018 所流行的一句话，「今年什么都不做你就赢了」，手动捂脸，具体没细算，加起来应该亏有 10000 元左右。2019 年，主要完成以下几个任务：

  • 坚持两个港股账户的打新，希望有肉吃

  • 学习《股市进阶之道——一个散户的自我修养》，产出思维导图

  • 不碰 A 股，只做指数基金定投。

再就是公众号。2018 年先后更新了一部分 Java 面试题，后期一直在思考要怎么样去做，耽搁了接近快两个月的时间，目前粉丝数量只有 154 人，非常感谢你们的支持。2019年，公众号将涵盖 Java 面试题、算法详解、读书笔记。主要目标如下：

  • 粉丝数量达到 10000

  • 保证隔日更，保证质量

以上。

 

2018，对我而言是一个平凡的一年，但 2019，我将让其成为我不平凡的一年，我们一起吧？
