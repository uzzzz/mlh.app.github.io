64 除非有特别标志，负责在城市镇、乡村或发展区以外的公路，最高的驾驶速度是：

1.    看交通情形，打灯号及立即驶离路边

3.    看清楚交通情形，打灯号，当安全的时候才从路边驶出

4.    打灯号，然后驶出路边

4.    解脱汽车离合器，使车溜下

1.    接近十字路口

4.    遇到迎面车辆的车头灯，使你看不清楚
