lync2013移动客户端外部用户登录的条件

  • 柚子社区

    最近在生产环境中部署了一套Lync Server 2013，实现了PC端内外部用户登录，但发现移动客户端外部无论如何也登录不了，本来以为是没有部TMG的问题（按照微软官方的要求是需要部署TMG才能实现移动客户端外部用户登录的），但是后来发现其实不需要部TMG也能实现移动客户端外网登录。下面列出实际部署过程中Lync 2013移动客户端外部用户登录的条件：


1、打CU1补丁：


下载地址：http://yun.baidu.com/s/17mMPg

安装好补丁后，按顺序执行以下命令：

  • 更新数据库：

      标准版：Install-CsDatabase -Update -LocalDatabases

      企业版：Install-CsDatabase -ConfiguredDatabases -SqlServerFqdn <SQL Server FQDN> -Verbose

  • 更新中央管理存储：

     Install-CsDatabase -CentralManagementDatabase -SqlServerFqdn sqlbe.contoso.net -SqlInstanceName RTC -Verbose
     如果不确定SqlInstanceName是什么，可以运行此命令查看：Get-CsManagementConnection
   
    
    
  • 更新拓扑：Enable-CsTopology
    • 
   
    
    
     
  • 启用统一的通信 Web API：
    • 
    
 CD 'C:\Program Files\Microsoft Lync Server 2013\Deployment'
      .\Bootstrapper.exe

添加A记录：lyncdiscover.contoso.com


或CNAME记录：别名：lyncdiscover       目标：lyncfe.contoso.com


3、端口发布：
