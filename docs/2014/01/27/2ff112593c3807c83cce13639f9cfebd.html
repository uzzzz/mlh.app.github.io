ubuntu 12.04 Qt4的安装与使用

  • 柚子社区

Qt Creator的官网上有各类Qt5 的安装包   但目前  个人仍旧比较喜欢qt4   故在这里介绍下


这里从装ubuntu开始说吧    首先   到ubuntu的下载网站里  下载ubuntu12.04 http://releases.ubuntu.com/precise/   并安装好    


ubuntu安装完成后 首先要下载好qt  第一个要确保网速    要改变ubuntu的源   ubuntu默认的更新源是国外的  速度实在让人无法忍受    

更改方式  用权限更改/etc/apt/sources.list 

这里贴出网易的更新源  速度非常快  我用大概400K左右    直接更换sources.list里面的代码即可

deb http://mirrors.163.com/ubuntu/ precise main restricted
deb-src http://mirrors.163.com/ubuntu/ precise main restricted
deb http://mirrors.163.com/ubuntu/ precise-updates main restricted
deb-src http://mirrors.163.com/ubuntu/ precise-updates main restricted
deb http://mirrors.163.com/ubuntu/ precise universe
deb-src http://mirrors.163.com/ubuntu/ precise universe
deb http://mirrors.163.com/ubuntu/ precise-updates universe
deb-src http://mirrors.163.com/ubuntu/ precise-updates universe
deb http://mirrors.163.com/ubuntu/ precise multiverse
deb-src http://mirrors.163.com/ubuntu/ precise multiverse
deb http://mirrors.163.com/ubuntu/ precise-updates multiverse
deb-src http://mirrors.163.com/ubuntu/ precise-updates multiverse
deb http://mirrors.163.com/ubuntu/ precise-backports main restricted universe multiverse
deb-src http://mirrors.163.com/ubuntu/ precise-backports main restricted universe multiverse
deb http://mirrors.163.com/ubuntu/ precise-security main restricted
deb-src http://mirrors.163.com/ubuntu/ precise-security main restricted
deb http://mirrors.163.com/ubuntu/ precise-security universe
deb-src http://mirrors.163.com/ubuntu/ precise-security universe
deb http://mirrors.163.com/ubuntu/ precise-security multiverse
deb-src http://mirrors.163.com/ubuntu/ precise-security multiverse
deb http://extras.ubuntu.com/ubuntu precise main
deb-src http://extras.ubuntu.com/ubuntu precise main

然后apt-get update 一下  更新下源信息   

安装g++  

sudo apt-get install g++

然后开始用命令apt-get装qt4 

 sudo apt-get install libqt4-dev libqt4-debug libqt4-gui libqt4-sql qt4-dev-tools qt4-doc qt4-designer qt4-qtconfig qtcreator
 安装包可以供需要选择  但建议这些全选    

有可能会报错  找不到libqt4-debug 

因为libqt4-debug 被libqt4-dbg代替了  故改成libqt4-dbg即可     

之后就是等待安装结果了  

安装完成后   在命令行敲入qtcreator 即可打开qtcreator

完成~~
