3、测试：

用VS2010新建一个工程，将上面解压后的include文件里的json文件夹放到工程目录下，并将生成的也放到工程目录下。如下：



#include <iostream>
#include "json//json.h"

#pragma comment(lib, "json_vc71_libmtd.lib")
int main()
{
	Json::Value val;
	Json::StyledWriter style_write;

	val["name"] = Json::Value("xiaoli");

	std::cout << style_write.write(val) << std::endl;
	return 0;
}
