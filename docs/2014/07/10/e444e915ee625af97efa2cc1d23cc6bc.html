sql server Windows账户被删并且sa账户被禁用解决办法

  • 柚子社区

作者：曲军昌

步骤：

1.      http://technet.microsoft.com/en-us/sysinternals/bb896649去这个网址下载这个文件。


2.      下载之后放到任意盘下即可，比如我放在D盘。


3.      打开CMD命令框（必须以管理员身份运行），然后进入到D盘下的PSTools文件夹下。



4.      进入该文件夹后输入命令psexec –s –I ssms,然后回车。


回车之后会弹出


然后输入数据库实例名，点击连接就可以了。你就可以随便修改账户了。
