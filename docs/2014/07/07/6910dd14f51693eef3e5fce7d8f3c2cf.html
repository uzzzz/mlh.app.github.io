win 下g++ 安装、环境配置等

  • 柚子社区

工具：eclipse for c++；

因为eclipse没有集成c++的编译器及运行环境，所以需要自己额外安装g++等；

方法：

使用MinGW来下载和安装需要的工具；

下载地址：http://www.mingw.org/download.shtml

1、安装与下载：

需要安装的东西，（网上截图）

安装完：MinGW后， 会弹出如下界面：


之后，右键需要安装的文件， 然后“Make for installation”， 之后选中左上角的“Installation”——>"Apply Changes", 然后在弹出的界面中点击“Apply”即可~


2、环境配置：

变量名                            变量值
        PATH                             C:/MinGW/bin;
        LIBRARY_PATH                C:/MinGW/lib
        C_INCLUDEDE_PATH        C:/MinGW/include
        CPLUS_INCLUDE_PATH      C:/MinGW/include/c++/3.4.2;C:/MinGW/include/c++/3.4.2/mingw32;C:/MinGW/include/c++/3.4.2/backward;C:/MinGW/include；

说明：CPLUS_INCLUDE_PATH 不用配置也可以，因为我安装的WinGWin目录下就没有那些东西， 其他几个应该都需要；


3、创建项目：

选中红色框中的选项，否则可能不能执行等其他错误；

创建好之后，编译、执行，OK~
