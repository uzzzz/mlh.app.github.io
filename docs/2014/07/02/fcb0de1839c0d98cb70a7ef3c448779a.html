Eclipse+mingw+Qt绿色配置方法（Windows下）

  • 柚子社区

本配置需使用的文件Eclipse+MinGW+qt-win-opensource-4.8.0-mingw+qt-eclipse-integration-win32-1.6.1。

下载地址：http://pan.baidu.com/s/1i3vFqlf

从以上下载的文件可以保证不会出现版本不匹配。

配置步骤如下：

1、安装Eclipse。解压就行，不需安装。

2、安装MinGW。解压就行，不需安装。将\mingw\bin目录下的mingw32-make.exe改为make.exe。

3、安装qt-win-opensource-4.8.0-mingw。直接安装就行。

4、安装qt-eclipse-integration-win32-1.6.1。Eclipse与MinGW的路径分别选择第1步与第2步的路径。

5、配置环境变量。

新建MINGW_PATH=D:\Qt\mingw

PATH添加%MINGW_PATH%\bin;D:\Qt\qt-win-opensource-4.8.0-mingw\4.8.0\bin;（后者为第3步的路径，可以在windows下直接使用指令qmake等，以及相关的动态链接库都在这）

新建：

C_INCLUDE_PATH=%MINGW_PATH%\include

CPLUS_INCLUDE_PATH=%MINGW_PATH%\lib\gcc\mingw32\4.4.0\include\c++;%MINGW_PATH%\lib\gcc\mingw32\4.4.0\include\c++\mingw32;%MINGW_PATH%\lib\gcc\mingw32\4.4.0\include\c++\backward;%MINGW_PATH%\lib\gcc\mingw32\4.4.0\include

LIBRARY_PATH=%MINGW_PATH%\lib

6、配置Eclipse。

1）在开始菜单里用QtEclipse Integration v1.6.1----Start Eclipse with MinGW打开Eclipse；

2）在Eclipse的Window---Preference，选择左边的Qt，然后单击右边的add，添加第3步安装的位置与版本号，如下图：


7、新建Qt project测试

1）在Eclipse中File--New---Project...

2）选择Qt目录下的一种方式(如Qt Gui Project)

3）在.ui上添加插件。window--show view---other...---Qt C++ Widget Box，拖入想要的插件。

4）编译，运行。


备注：第7步中可能会出现Eclipse error - method setupUi could not be resolved错误，解决方法：

1）选择window--preferrence；

2）选择C/C++---Indexer
