smartfusion的引脚分成三类，可以用作普通用户io但是，可外接电平不一样：

MSIO 最高电平3.3V

MSIOD 最高电平2.5V

DDRIO 最高电平2.5V


电源引脚：

VDD 核心供电引脚

VPP 电荷泵引脚

VDDIx 为blank x的引脚供电引脚

VREF0 MDDR的参考电平，当不使用MDDR时，此引脚可以悬空或者接地（MDDR时cortex-M3外接DDR的接口）

VREF5 FDDR的参考电平，当不使用FDDR时，此引脚可以悬空或者接地（FDDR时fpga外接DDR的接口）
