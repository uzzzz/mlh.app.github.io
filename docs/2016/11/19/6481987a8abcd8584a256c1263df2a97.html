Github下载安装教程

  • 柚子社区

1. 注册Github账号。

Github官网网址：https://github.com/ 

首先去Github官网上注册一个账号。点击上面的网站进去，点击sign up注册账号。


注册过程就不细讲了。

2. 下载安装Github。下面提供下载方法

a> 去百度搜索Github for Windows。选择Github for Windows - Download


b> 选择Free Download，然后选择Download Github desktop进行下载。




附下载地址：https://desktop.github.com/

3. 点击GitHubSetup进入下载界面，下载完成后，就可以开始安装了。



这种下载方法我下载到一般不能继续下载，大家可以尝试一下。给一个网上的离线下载分享网站GitHub for Windows


4. 安装完成后会出现两个图标GitHub和Git Shell，然后就可以正常使用了。
