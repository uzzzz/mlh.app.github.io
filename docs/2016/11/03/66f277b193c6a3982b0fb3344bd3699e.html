关于苹果开发者账号申请付费流程

  • 柚子社区

昨天自己为了学习开了个人个账号，准备上上自己写的项目然后就进行了付费开通，苹果改版了跟去年不一样了记录下。

1.点击网址免费申请：https://developer.apple.com  登录以后

1.点击加入


2.点击右上方Enroll （报名）

3.然后

4.我选择的是

individual/sole proprietor/sing person business（个人业务）


内容填写对查表：

Legal Name（姓名）

Phone （联系电话）

Enter your home or business address in your local language.（输入个人家庭地址或者工作地址）

Address Line 1 （地址行1）

Address Line 2（地址行2）选填

Town / City  (镇/市)

State / Province  (州/省)

Postal Code (邮政编码) 选填

Country （国）

Romanized Contact Information （罗马联系信息）

Enter your home or business address using the English alphabet. Do not use special characters.

（用英文字母输入你的家庭或商业地址。不要使用特殊字符。）

Given Name （名字）

Family Name （家族名字/姓）

Address Line 1（地址行1）

Address Line 2（地址行2）选填

Town / City (镇/市)

State / Province  (州/省)

Postal Code (邮政编码) 选填

5.

同意协议后提交

同意协议后提交提交后转到付款银行卡信息填写页面，填写付款信息。这里要填写具有美元支付功能的银行卡

填写付款信息后 提交，付款成功就出现这个页面啦

https://secure1.store.apple.com/cn/shop/sign_in?c=aHR0cHM6Ly9kZXZlbG9wZXIuYXBwbGUuY29tL2xvZ291dC98MWFvczdjMWQ0ZDlhYTRmZmE5NTA2OGIzM2E0MGY0MzU3NzNkMjhjZjE0OGY&o=O0Q0NTIx&r=SXYD4UDAPXU7P7KXF&s=b0b581bc3ef166515f847759ad03a776&t=SJ4242KXUTFDC4JHH9FYD7APATD7P9DA9&up=t

会调到这个页面进行登录 然后付款 用visa卡  付款成功之后10分钟 账号就激活了 没有之前等待的时间很长的情况
