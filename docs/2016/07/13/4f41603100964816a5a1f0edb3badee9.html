《Qt5.5.1+VS2013开发环境配置》

  • 柚子社区
版权声明：本文为博主原创文章，未经博主允许不得转载。 https://blog.csdn.net/guoxudong2012/article/details/51899836
（一）下载安装包
1.Qt安装包
Qt5.5.1下载地址： http://qt-project.org/downloads.
2.Qt插件（Visual Studio Add-in）
Qt插件下载地址： Visual Studio Add-in 1.2.3 for Qt5   (supports versions 2008, 2010, 2012 and  2013 , does not work with the Express edition)
3.VC2010运行库
VC2010运行库下载地址： http://www.pc6.com/softview/softview_55017.html
（二）安装VS2013（如果已安装，跳到下一步）
（三）安装VC2010运行库（如果已安装，跳到下一步）
（四）安装Qt插件及配置
1.打开qt-vs-addin-1.2.3-opensource.exe，默认下一步直到安装完成。
2.安装Qt插件成功后打开VS2013，菜单栏多一览“Qt5”。

3.设置Qt环境信息，点击“Qt Options”。

4.在“Qt Options”中设置Qt版本名以及Qt安装路径。

（五）设置编译器选项
1.打开项目文件，“右击项目文件”->“properties”->“Linker”->“Advanced”->
将“Image Has Safe Exception Handles”设置为“NO（/SAFESEH:NO）”->点击应用，确 定。
（完）
