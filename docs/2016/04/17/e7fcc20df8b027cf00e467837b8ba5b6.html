1.Node Js 安装（windows环境下）

载node js官网下载安装包，官网网址：https://nodejs.org/en/ 或者在作者上传在CSDN的资源也可用：http://download.csdn.net/detail/u010989191/9493005 。推荐去官网下载。
nodejs有一些热心网址整理出不错的用户手册，可以在下面的链接中获取：http://download.csdn.net/detail/u010989191/9492469。
如果使用npm安装，安装过程中出错而转向下载exe文件的形式安装，可能需要用到数据源。使用国外的源经常会卡死或者挂掉，推荐使用国内淘宝源：http://registry.npm.taobao.org/。

安装后进入控制台输入node -v即可查看node js的版本，能正常查看则表示安装成功。如下图所示：

2. sublime text 3安装及配置nodejs环境

作者已经在CSDN上传了最新的sublime text 3版本和相应的注册码。可以去官网下载或者直接在以下地址下载：http://download.csdn.net/detail/u010989191/9492465。

安装好sublime text 3后需要配置Package Control。安装方式如下：
1）打开sublime text 3。
2）按Ctrl+’`’打开控制台。
3）输入以下内容进行安装。

import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

4）在preferences->Package Control打开package control。
5）输入 install package。
6）输入node js，选择安装JavaScript & NodeJS Snippets.sublime-package。

安装成功后，新建一个js文件，输入concole.log(“hello”)；然后使用Ctrl+b运行测试。如果在sublime的控制台输出hello则表示环境安装成功。如下图所示：

如果没有成功，则表示刚才的插件安装失败，这时需要自己手动安装。在以下网址可以下载到sublim text3 node js的插件：https://github.com/tanepiper/SublimeText-Nodejs。
使用方式如下：
1）解压zip文件， 务必重命名文件夹为“Nodejs”。
2）在sublime的preferences->browse packages目录下放置刚才改名后的Nodejs文件夹。
3）打开Nodejs文件夹，找到文件“Nodejs.sublime-build“，修改”encoding”: “cp1252”,为”encoding”: “utf-8”。
4）要用sublime打开文件“Nodejs.sublime-settings” 或者 设置preference ->package settings ->Nodejs ->setting-default 打开文件并 更改成以下代码：

{
  // save before running commands
  "save_first": true,
  // if present, use this command instead of plain "node"
  // e.g. "/usr/bin/node" or "C:\bin\node.exe"
  "node_command": "D:\\nodejs\\node.exe",
  // Same for NPM command
  "npm_command": "D:\\nodejs\\npm.cmd",
  // as 'NODE_PATH' environment variable for node runtime
  "node_path": true,
  "expert_mode": false,
  "ouput_to_new_tab": false
}

关闭重启sublime text 3即可正常编辑和运行nodejs。

2. npm 安装expresee

建议修改默认的源。采用以下方式：npm config set 。
即在终端输入：npm config set registry http://registry.npm.taobao.org/即可修改默认的源为淘宝源。
然后输入npm install express（相对路径）。这样的安装速度会快很多。也可以使用npm install express -g（绝对路径的安装方式）。
