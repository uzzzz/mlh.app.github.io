二：LiteIDE 安装（  LiteIDE 具备代码补全提示、断点跟踪调试，但没有代码编译期出错提醒功能）


1、MinGW 下载安装

下载地址：http://sourceforge.net/projects/mingw/ 

                 下载后直接安装，默认路径C:\MinGW

              直接运行 MinGW Installation Manager → Basic Setup → 选中 mingw-developer-toolkit 和 mingw32-base 两个包 → Installation → Apply Changes

    安装成功之后把 MinGW 安装目录的 bin 目录的路径追加到 Path 环境变量里。

最后，打开 CMD 窗口，执行 gcc -v 命令，看到以下界面证明安装成功：


2、 LiteIDE安装

下载 liteidex30.2.windows-qt5.zip

解压后得到 liteide目录，将其拷贝到 C:\go 目录下，打开 C:\go\liteide\bin,为liteide.exe 创建桌面快捷方式。

       3、 LiteIDE 编译环境设置

双击桌面上的liteide.exe 快捷方式打开LiteIDE,打开工具栏中的system下拉条，选中win64。

查看 ----> 编辑当前环境，默认配置如下：



  4:、设置编译选项并编译

编译 ---> 编译配置， BUILDARGS 设置为 -gcflags "-N -l" ，以去掉编译优化，方便调试。

                

ctrl + B 构建项目，生成一个exe文件。

                按F5启动调试，程序在main函数处停止
