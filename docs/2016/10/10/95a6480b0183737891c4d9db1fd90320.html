sublime中go插件的安装

  • 柚子社区

1. 安装windows版本的go，golang中国有各个版本的国内镜像，下载很快，这个网站也不赖，可以作为学习go的资源。 

下载 go1.7.windows-amd64.msi 依次下一步即可

安装完毕后，go的环境变量会默认添加到系统环境变量中


2. 配置环境变量 
(1). 新建 变量名：GOBIN 变量值 ：c:\go\bin 
(2). 新建 变量名：GOPATH 变量值：d:\gopath 
(3). 新建 变量名： GOROOT 变量值：c:\go 

在 gopath下建立目录 src bin pkg


3. sublime里面使用ctrl shift p，命令install进入模块安装模式，需要安装Go Sublime 
4. 安装gocode：首先确保安装好了go和msysgit的windows命令行版本，然后命令： 
　　go get -u github.com/nsf/gocode 
　　go install github.com/nsf/gocode 
5. 修改GoSublime配置：在 Preferences菜单下，找到Package Settings，然后找到 GoSublime，再往下找到 Settings - Default。再打开的文件中，添加如下配置，并保存：
