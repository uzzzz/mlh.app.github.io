Android dumpsys 使用之分析网络使用情况

  • 柚子社区

1. netstats

netstats 是 Android 提供的一个分析网络使用情况的工具，可以提供启动以来的网络使用率的统计数据。

2. 输入

需要查看网络的统计信息，使用以下命令：

adb shell dumpsys netstats

又一个没有使用说明的命令

3. 输出

输出的具体信息，根据 android 版本的不同而有所差异，主要包含以下几类信息：

  • Active interfaces
  • Active UID interfaces
  • Dev statistics
  • Xt statistics
  • UID statistics (sometimes also called “Detailed UID statistics”)
  • UID tag statistics

Active interfaces/Active UID interfaces

一下为一个输出示例

Active interfaces:
  iface=tun0 ident=[{type=VPN, subType=COMBINED}] 
  iface=wlan0 ident=[{type=WIFI, subType=COMBINED, networkId="DEZNG"}] 
Active UID interfaces:
  iface=tun0 ident=[{type=VPN, subType=COMBINED}] 
  iface=wlan0 ident=[{type=WIFI, subType=COMBINED, networkId="DEZNG"}]

官方文档中介绍说，大多情况下这两个是一样的，但是我不知道这两个的含义是什么。
