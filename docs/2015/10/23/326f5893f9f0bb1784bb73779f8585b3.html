ETF基金是指交易型开放式指数基金，下面将分享LOF基金接口，并演示返回示例，包括基金代码、基金简称、最新净值、累计收益率、累计净值、日增长值、日增长率、周增长率、月增长率、最新净值时间等。

接口名称：LOF基金接口

接口平台：web api

接口地址：http://web.juhe.cn:8080/fund/netdata/lof

支持格式：JSON

请求方式：GET

请求示例：http://web.juhe.cn:8080/fund/netdata/lof?key=您申请的APPKEY
