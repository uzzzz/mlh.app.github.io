莱特币（litecoin）矿池p2pool架设之Windows篇

随着币价的不断下跌，矿池偷算力和手续费，所以本人写了此教程，希望能帮助到有需的人。下一篇应该是LINUX环境下的架设。本教程只是演示如何架设，具体实施看个人情况。
打个小广告：本人现有多台自用的DELL服务器出售详情可联系QQ：4094567

环境和工具：

  • Windows2003(我用的是一台阿里云的服务器，配置1核1G硬盘默认40G）
  • 莱特币钱包
  • p2pool
  • python2.7.9
  • Twisted 14.0.2 for Python 2.7
  • zope.interface-4.1.1.win32-py2.7
  • WMI-1.4.9.win32
  • pywin32-218.win32-py2.7
  • gcc-mingw-4.3.3-setup

下面开始安装

首先要下载莱特币钱包并同步完成，点击32bit下载。

在同步过程中我们将其它所需的环境下载并安装好。
这里我已下载好，开始安装，全部默认下一步直到完成即可。

安装顺序：
1、解压p2pool(这里我解压到C盘根目录)
2、python-2.7.9
3、Twisted-14.0.2.win32-py2.7
4、zope.interface-4.1.1.win32-py2.7
5、WMI-1.4.9.win32
6、pywin32-218.win32-py2.7
7、gcc-mingw-4.3.3-setup
注意在安装GCC时必须选择“Link with MSVCR90.DLL”

8、在命令提示符下进入P2POOL目录下的litecoin_scrypt文件夹内，安装scrytp算法支持。

cd c:\p2pool\litecoin_scrypt
setup.py build --compile=mingw32 install

9、在开始>运行中输入%appdata%\litecoin进入litecoin配置文件夹。

%Appdata%\litecoin

创建一个文本文件，将名字另存为litecoin.conf，并编辑以下内容

rpcuser=secureusernamelikerT3K8kBr98butchangethis!
rpcpassword=securepasswordlike0R3RgGK8XIbutchangethis!
server=1
listen=1
discover=1
bind=0.0.0.0
daemon=1
rpcport=10332
port=10333
mininput=0.00000001
blockprioritysize=1
blockmaxsize=750000
mintxfee=0.00001000
minrelaytxfee=0.00001000
gen=0

rpcuser=用户名；rpcpassword=密码
建议设置一个复杂的密码包含大小写字母和数字。

等待钱包更新完毕后关闭钱包然后重新打开钱包

此处有一个绿色√为更新完毕。

运行矿池

C:\p2pool\run_p2pool.py 用户名 密码 --net litecoin --give-author 0 -f 0 -a LcjmUck1PB2dZun9My9N2W8hwFpAfeEkfP --bitcoind-rpc-port=10332 --bitcoind-p2p-port=10333


运行后如果出现3个…success!即表示成功运行。

如果是比特币矿池只需要把钱包换成比特币钱包（比特币钱包配置文件目录在%Appdata%\bitcoin）并且将上面命令中的–net litecoin改为–net bitcoin即可。

如果觉得本教程对你有帮助请给点打赏
BITCOIN=1BrV3UKnh4ATbQtQymRPGRsdBYs5imbcDZ

OK，到这里本次教程就结束了，大家可以将上面的命令建立一个批处理文件方便运行。
