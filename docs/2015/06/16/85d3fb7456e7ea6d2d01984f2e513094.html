VS2010/VS2012/VS2013+QT5编程之开发环境配置

  • 柚子社区

用VS+Qt5开发并不是什么难事，下面详细讲一下如何配置编译环境：

第一步：

              安装VS和其对应的sp补丁包，网上很容易就能下载到安装包，傻瓜式安装，这里不多介绍。

第二步：

              点击打开链接

          分别在vsaddin里面选择下载最新的qt-vs-addin，在qt里面选择要下载的Qt版本，例如我用的是vs2010+Qt5.4.2，那么我需要下载qt-opensource-windows-x86-msvc2010_opengl-5.4.2.exe。

          先安装QT再安装qt-vs-addin。

第三步：

              以VS2010为例


打开VS，如图所示，选择Qt5->Qt Options。


如上图所示，依次选择完成后如图

点击OK按钮即可完成配置。


常见问题：

编译环境配置后，打开以前的项目后出现Qt Visual Stdudio Add-in问题，

The following error occured:

There's no Qt version assigned to this project for platform Win32.Please use the 'change Qt version' feature and choose a valid Qt version for this platform.


解决方法：


右键左侧项目->选择Qt Project Settings

选择Version版本后点击OK即可。
