树莓派--安装chromium

  • 柚子社区

在线安装

有网络支持的童鞋，可以直接运行

sudo apt-get install chromium-browser chromium-browser-l10n

离线安装

希望离线安装的童鞋，可以前往http://mirrordirector.raspbian.org/raspbian/pool/main/c/chromium-browser/ 下载

  • chromium-browser_22.0.1229.94~r161065+dfsg-0.1+rpi1_all.deb
  • chromium-browser-inspector_22.0.1229.94~r161065+dfsg-0.1+rpi1_all.deb
  • chromium-browser-l10n_22.0.1229.94~r161065+dfsg-0.1+rpi1_all.deb
  • chromium-inspector_22.0.1229.94~r161065+dfsg-0.1+rpi1_all.deb
  • chromium-l10n_22.0.1229.94~r161065+dfsg-0.1+rpi1_all.deb
  • chromium_22.0.1229.94~r161065+dfsg-0.1+rpi1_armhf.deb

之后执行

sudo dpkg -i ./*.deb

chromium便安装完成，可以在浏览器菜单中找到

其实，raspbian镜像已经给出了chromium-browser_43.0.2357.65的源码包，只是小弟才学疏浅，暂时也没有精力在小派上进行编译，故，就将就着先拿版本22玩儿吧，坐等官网出43版本的deb
