目前公司在做App转移操作，在网上搜索相关资料加上自己的亲自操作，整理成一个文档，希望能给你提供帮助。


如转载请添加出处。

文档下载地址

此文章只是为了记录一个Apple ID下的APP，转移到另外一个Apple ID 账户下。为了说的清楚下面用A账户（有App，要转出去）B账户（接收A账户App，接收者），来说明。

1、      登录APPLE 开发者中心（http://developer.apple.com），使用A账户登录。

2、      MemberCenter


3、      ManageYour Apps

4、      选择你要转移的App


5、      看协议，同意转移，建议直接点击Continue（因为你不同意就不能转移，就这么简单）

6、      填写转移账号


Recipient's Team Agent Apple ID 填写接收者Apple ID

Team ID  登录接收者（B账户）Apple ID→ Member Center→ Your Account→Team iD (类似于：ABDC8DEDOE) 如下图所示：



7、      确认你要转的APP的信息和同意条款，点击右下角 Request Transfer 按钮前请确定你把 同意此协议的复选框给打钩了，你也可以下载此协议。

8、      点击申请转移：


9、      登录A账户→Contracts,Tax,and Banking 在最下面你会找到你要转移的App 状态，如下图：


接收者（B账号）有60天期限，就是必须在60天之内回复，否则将会强制取消转移。

10、  所以赶紧去登录B账号。同样的网站，同样的地点，不同的账号，同样进到Contracts,Tax,and Banking后拉到最下面，你就可以看到转移App：


11、  点击ReView，开始填写转移到B账号下的信息：


还有下面的App Review Information 最后勾选协议→提交→Done→往下拉：


苹果说两个工作日内会有答案，但是我好像只用了5min不到登录B账号就有了，但这里仅仅是说B账号有转移的App，但是这时点击View in App Store你会发现显示的仍然是A账户的信息，信息更新会花费2-3个工作日时间。

登录B账号→iTunes Connect→Manager You App：



2-3个工作日后：登录B账号→iTune connect→Manager You App→View in App Store 就可以看到B账号的信息了。

至此，App转移的工作就全部完成，B账号就拥有了App的各项权利，收费，升级，限免等。希望此文档能为你提供帮助，谢谢支持。
