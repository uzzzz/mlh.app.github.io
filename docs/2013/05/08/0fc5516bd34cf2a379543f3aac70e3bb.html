Windows下Qt4.7.4安装的大致过程

  • 柚子社区

1、获取Qt开源版源代码压缩包、MinGW编译器并安装Qt：

         1）先从地址http://en.sourceforge.jp/projects/sfnet_nasame2013/downloads/sources/MinGW-gcc440_1.zip/处下载MinGW-gcc440_1.zip压缩包，可以解压到C盘（即C:\mingw）；

         2）从官网地址http://download.qt-project.org/archive/qt/4.7/处下载Qt的源代码文件qt-win-opensource-4.7.4-mingw.exe；

         3）直接双击exe文件开始安装，安装文件存放在C盘（即C:\Qt\4.7.4），其中要选择安装的MinGW目录，直接选择目录C:\mingw即可。

2、修改设置环境变量以配置Windows下的Qt开发环境：

         右键点击我的电脑-->属性-->高级-->环境变量，把PATH变量改为%PATH%;C:\mingw\bin;C:\Qt\4.7.4\bin;                  

 QMAKESPEC变量改为win32-g++即可，然后点击确定退出。

3、写一段小程序测试：

         在目录C:\Qt\4.7.4下新建一project文件夹，在目录project目录下新建一hello文件夹，然后在hello目录下创建名为hello.cpp的文件，输入如下代码：

#include <QApplication>
#include <QLabel>

int main(int argc, char *argv[])
{
	QApplication app(argc, argv);

	QLabel *label = new QLabel("Hello Qt!");
	label->show();

	return app.exec();
}

         然后在开始菜单找到Qt，打开Qt 4.7.4 Command Prompt：

                           

         然后依次输入qmake -project、qmake、make，并由此产生的可执行文件被保存在应用程序所在目录的debug文件夹中（如C:\Qt\4.7.4\project\hello\debug\hello.exe）：

                           

         最后进入当前目录的debug目录中，执行hello，会出现标签窗口，即全部安装完成。
