VC6.0 代码字体 设置 format font 添加自己喜欢的字体

  • 柚子社区

提供2套个人比较喜欢的编码字体，Courier New  或者Lucida Console 字体 。
现在将VC6.0 的编码 字体改为上面两种，从此不用在忍受vc的单调字体，示例：

①改成Courier New 字体：

代码字体示例：

Windows Registry Editor Version 5.00
