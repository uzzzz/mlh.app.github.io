简介：近期，般若链（ProChain）是区块链创新领域最火的项目之一。关于它的优势，更多的报道集中在其背后优秀的核心团队，面向的巨大市场，鲜明的行业针对性，以及超高的技术壁垒。然而，本文作者从一个一般人不太注意的方面来阐述了般若链设计当中独居匠心的一面——通证经济（Token Economy）系统的设计。对于区块链技术爱好者以及创业者来说，这不是一个第一眼就会很关注的方面，但真正的区块链行家知道，经济系统的设计，就是，区块链项目的游戏规则和协作制度设计，是区块链项目的根本大法，也是决定区块链项目长期成功最重要的因素之一。本文作者在国内较早提出和研究了区块链通证经济系统的设计问题，并与般若链核心团队深入交流，对其设计有较深刻的认识。我们特别邀请他撰写文章，介绍般若链经济系统设计方面的深层考虑。


关联性披露声明：本文作者是般若链ProChain 的联合创始人及高级顾问，但般若链的经济系统设计是由核心团队独立完成的，并非出于本文作者。此外，本文与作者所在机构无关，仅代表个人观点。

 

1.  通证经济系统（token economic system）是什么？


Token Economy是随着区块链发展而新出现的一个概念。其中 token 这个词，一直以来被不恰当的翻译成为“代币”，因此很多人将 token economy 翻译为“代币经济”。

我在不久前的一篇文章中提出将 token 翻译为“通证”，即“可流通的加密数字权益证明”。如此一来，token economy 自然就应该翻译为“通证经济”。

区块链的威力在于创造一种新的协作关系。在中心化系统里，处于权威地位的中心，一方面与普通用户之间相互依存，另一方面又在很多地方是博弈的双方。因此中心权威往往利用自己的权力和技术优势，施加各种限制、监管、执法手段，动用各种先进技术对他们不喜欢的行为实施围追堵截，猫鼠游戏延续数千年，至今我们依然能够在那些强大的互联网公司的商业模式中清楚的看到这出戏码。而在区块链的体系，没有中心化权威，没有人举着鞭子到处吆喝监管，但整个生态圈居然能够按照一个行为规范相互协作，秩序井然，欣欣向荣。何以能如此呢？


奥秘就在于通证（token）。


每一个发币的区块链项目，都是试图以其所发行的币（通证）作为一种经济激励的工具，促进生态圈内各个角色的协作。你的贡献越大，你得到的币越多。大家协作得越好，币价越高。因此，每一个发币的区块链项目都在试图设计一个通证经济系统（token economic system）。

 

我们不必过多纠结于概念，实际的来说，通证经济系统的设计，就是要用经济激励的手段，让你整个生态圈的每一个人、每一个角色尽可能行善事，不作恶。为此，你在设计时要回答下面这些问题：


  • 你认为使用这个通证的用户分成哪几类角色？他们的利益诉求各是什么？

  • 什么是对你这个生态系统有益的行为？什么是有害的行为？

  • 你怎样激励每一个用户做好事？

  • 你怎样说服每一个用户不做恶？


区块链主要是通过发行有价值的通证（token，俗称代币），并制定相应的通证流转分配制度，来惩恶扬善，让好人有好报，坏人穷三代。因此区块链项目的通证经济系统设计，就是整个产业生态的根本制度设计。

 

更具体来说，做好这个根本制度设计，需要回答一系列的细节问题：

 

  • 你发的通证总量是多少？总量封顶吗？

  • 如果总量封顶，那么发行的速率和节奏是怎样的？

  • 如果总量不封顶，那么通货膨胀率设定在多少？为什么这么设定？

  • 你的通证有预挖量吗？预挖比例占多少？谁是预挖出的代币的最初拥有者？预挖通证如何进行初次分配？

  • 你的通证在那些场景下应用？有什么使用价值？

  • 你的通证有价值锚定吗？如果没有价值机制，如何防止通证价值无底线下跌？如果有价值锚定，如何防止价格上涨受限？

  • 通证按照什么原则进行流转？如何进行二次分配？

  • 如何让通证更多的流向做好事的用户，而抽离作恶的用户？

  • 整个系统的总体发展方向由谁决策？按照怎样的方式决策？有人不服决策将如何处理？

    …

 

喜欢炒币的人往往不喜欢关注这些问题，但本质上来讲，正是这些设计决定了一个区块链系统的中长期价值走向。当前很多人抱着短期快入快出的心态，对这些较深的问题缺乏思考，也很少责问。结果，目前国内外多数区块链项目在通证经济系统设计方面非常草率，通证想发多少发多少，想怎么发怎么发，说什么便是什么。更有甚者，为了迎合短期炒作的心态，故意设计一些有利于短期操纵币价的经济系统，炒上去再说，套现之后哪管洪水滔天。这种设计非常不可取，也必将付出惨痛的代价。


2.  比特币的通证经济系统设计


我们拿比特币做例子来介绍下通证经济系统的设计。比特币的经济系统设计本身是非常出色的。比特币今天强悍的表现，不能不说跟比特币经济系统的设计有莫大的关系。

 

我们先说比特币经济系统中设计的巧妙的地方。

 

评价一个通证经济系统的设计好坏，首先要看这个通证的目的是什么。同一个设计，放到这个目的下，可能是好的设计，而放到另一个目的下，就可能是糟糕的设计。比特币的核心是支付，也就是汇款。为了实现汇款，需要有多个记账节点来记账。因此，比特币系统的本质就是一个大帐本，而所有的矿工节点都是记账的会计。

作为一个大帐本，比特币最希望这些矿工节点做什么事？当然就是作真账，不作假账。为此中本聪设计了一个挖矿奖励的机制。只要正义的节点在网络中占大多数，那么做真账的节点可以得到新挖出的比特币作为奖赏。而如果一个节点做假账，那么整个网络稍后会发现，并且抹掉假账，使恶意行为徒劳无功。仅仅这样还不够，还要狠狠的惩罚它。所以随着假账被抹除，恶意节点辛辛苦苦挖出来的比特币，在口袋里还没捂热，马上就会被注销掉，损失不可谓不惨痛。正是因为这样爱憎分明的设计，使得比特币中几乎全部的矿工节点都在“行善积德”。

当然比特币里是可以进行所谓 的51%攻击的。那么如果作恶节点占据整个网络的多数，网络确实可以做出假账，把黑的说成白的，假的记成真的。但是其结果是什么呢？比特币信用将崩溃，价值归零，而这些作恶节点作为比特币的拥有者，也将成为损失最大的玩家之一。

这套巧妙的经济系统设计，对于维护比特币的安全性来说，其重要意义甚至比比特币的密码学应用和多项技术创新来得更大。


当然，比特币的经济系统设计也不是没有值得商榷之处。


比如总量封顶，是一个剧烈的通缩模型，会导致币价长期上涨。作为一个打破旧世界的开天辟地的数字货币，如果不能够长期剧烈升值，也很难获得足够的关注和能量。因此总量封顶这个设计对比特币来说是一大得分项。如果比特币从一开始就作为一个保值增值金融资产来设计，这个模型是完美无缺的。但另一方面，比特币是作为一个支付通货来设计的，而一个剧烈通缩的货币，是不会有人愿意把它拿出来花掉的，其流动性非常差，偏离了比特币作为支付通货的初衷。这是当前比特币应用当中一个比较突出的问题，其根源就在于经济系统设计中的取舍抉择。

其次，比特币价格基本上是无锚的，也就是说，比特币没有任何作为价值支撑的抵押资产，也没有任何必须应用比特币的场景可以用来作对价基准，完全基于人们的信心，因此比特币的价格上不封顶，下不保底。作为第一种成功的加密数字或货币，这一点无可指责。但我们还是要知道，加密数字通证是可以设计价格锚的。锚设计是一个专门的话题，限于篇幅本文不能开，未来将专文阐述。

此外，比特币赋予算力拥有者过高的权力，在设计上缺少制衡的安排，加上 SHA256 算法易于硬件加速的缺陷，使得比特币成为算力拥有者按照丛林法则竞争的场所。不过同样，这对于开天辟地的比特币来说，并不是致命的问题，但对未来的区块链经济系统设计，就是必须深思的方面了。


3. 般若链的通证经济系统设计


般若链本身的价值是多方面的，包括流量反欺诈、去中介化、自动结算、精准营销等，特别是将大数据、机器学习、计算广告学与区块链结合起来形成的高技术壁垒，使之在今天面向广告产业的区块链项目中脱颖而出。但是般若链毕竟是区块链系统，它要达成自己的目标，技术很重要，但正向经济刺激更重要。如何促进互联网广告产业生态中广告商、内容提供商和读者三者之间的良性互动，通力合作，是般若链的重要课题。

般若链全平台统一通证称为PRO币，PRO无疑是般若链整个生态体系中核心的激励因素，因此核心团队围绕以上目标对PRO经济系统进行了精心的设计。他们期望让参与者通过做正确的事，能够获得更多的PRO，而人人都做正确的事，则会有效的推动PRO币价的提升，使得所有人获益。

PRO在实际应用场景中的作用，般若链白皮书中有详细的交代，甚至还有案例分析，因此我在这里就不赘述了。本文只是关注通证经济系统的设计。当然，一个区块链项目的经济系统设计巨细靡遗，不可能在一段文字里讲清楚。下面只是把其中几个有特色的设计介绍一下。

 

总量封顶、柔性价值锚定的长期通缩模型


PRO的发行总量为一亿枚，是一个总量封顶的通证。由于PRO面对着全球数千亿美元、而且还在快速增长的互联网广告市场，因此如果它的应用不断扩大，则将处于长期通缩，其币价上涨空间是巨大的，甚至可以说是上不封顶。这一点与当前大多数区块链通证项目的设计类似，虽然漂亮但并不突出。

PRO币值系统的设计中，真正可圈可点的是其价值柔性锚定的部分。PRO在整个系统中，既可以作为媒体资源采购，也可以用于内容服务平台（媒体）和用户的奖励，并且通过二级市场自由定价。这里的核心在于，在未来相当长的一段时间内，广告主的广告采购行为主要仍会通过法币进行，你不可能要求宝洁、麦当劳这样的大广告主很快采纳PRO 币作为广告投放通证，这是 2B 业务合规要求所决定的。而一项广告资源，如果通过法币采购有一个价格，而通过PRO 采购有另一个价格，那么就在这个场景下，确定了法币与 PRO 价格的一个汇率，从而形成了锚定关系。

这种锚定关系对于PRO 来说，实际上是一个价格托底机制和稳定机制。何为托底机制？就是一层价格保护网。设想如果在二级市场上，PRO 的价格跌破应用场景中确定的汇率，那么对于广告主或者代理商来说，他们可以用法币到二级市场上大量兑换 PRO，然后用于购买广告资源，从而实现套利。这种行为反过来会引起二级市场上 PRO 价格的上涨，重新回到平衡汇率上。这对于PRO 币价格的稳定及其有利。

同时，这个锚定关系是柔性的，因为法币与 PRO 之间的汇率是浮动的，随着 PRO 应用的场景越来越多，其稀缺性会不断放大，因此 PRO 的托底价格可以长期稳定上涨，甚至可以说上不封顶。

这也就是说，PRO 的价格锚设计是一个带有托底保护、稳定机制的不封顶通缩模型，这是一个非常漂亮的设计，是充分利用了互联网广告行业特点的一个设计，别的区块链项目很难抄袭。


通过二次分配对良好行为实施激励


PRO 总量一亿枚，一次性全部预挖。初次分配的方案，对于核心团队和关键利益方，分配了较大的份额，并且限定了较长的锁定期，分阶段解锁。对“区块链原教旨主义者”来说，PRO 的初次分配方案显得中心化了。但从实践来看，任何通证系统在其发展早期都是有一个中心化阶段的，关键是看早期的中心是否有足够的动力去维护一个健康的生态环境。很显然，在比特币发展早期，如果中本聪愿意，他可以通过自己的恶行把比特币的声誉彻底毁掉，但是他当然不会有这个动机，他希望比特币福寿永昌、万寿无疆，所以会小心翼翼的呵护比特币度过最初的幼年期。同样，在早期中心化的时代，中心化权力掌握在核心团队和关键利益方当中是比较可靠的设计，因为他们手中的通证存量最大，而他们最大的利益所在，也就是把生态做好，提升币价。PRO 显然符合这样一个逻辑。


关键问题在于二次分配。在一次性预挖全部通证的系统中，对于用户来说，二次分配的游戏规则其实就是挖矿规则。在这方面，般若链核心团队结合互联网广告行业的特点，设定了一套精致的机制。

由以上描述可以看出 PRO 币在整个应用场景中作为支付、抵押和奖励的游戏规则。实际上这个列表中还应该加入两个角色：二级市场交易者和般若链关键利益方，而其中二级市场交易者其实是一个集合身份，其中可能有广告主，有用户，有媒体，也可能有很多纯粹的数字通证投资者和投机者，把所有这些人的动机、诉求、利益偏好放在一起考虑，才能形成完整的视图。不过这样一来，要解释整个系统的复杂性就上升了，不是这篇文章能够讲清楚的。简化一下核心结论，在目前般若链的二次分配规则之下，我们虽然不能排除个别价格炒作和作恶行为的动机空间，但总体上来讲，这套规则鼓励各方做正确的事情。

 

为了理解这种激励机制，我这里特别从媒体的角度来分析一下，其他角色的角度读者可以自行推导分析。


传统的互联网广告生态圈中，媒体为了获得更多的广告分成，有很大的动机来作弊，比如利用自动化程序制造假点击，或者购买假流量的服务。为了防范这种行为，现在的优秀的广告发布和监控系统都使用了最先进的AI 技术来判断欺诈，而媒体就与这样的系统之间形成了零和甚至负和博弈关系。一个想骗，一个想管，相互较劲。

我们看看ProChain 体系之下会发生怎样的变化。由于媒体在广告点击当中获得的是 PRO币，其真正价值不仅取决于数量，还取决于 PRO 币的二级市场价格。ProChain 也有强大的反欺诈系统，但比这个系统更强大的是内在经济机制：如果媒体大规模的骗流量、骗点击，使得 ProChain 的声誉受损，那么 PRO 币价下跌，可能会导致媒体的实际损失大于其收获。更重要的是，由于更多的个体被绑进这个利益机制，如果有一方试图实施流量欺诈，他很可能会发现有无数双眼睛盯着他，准备举报他来领赏，使他的欺诈风险和欺诈成本大大上升。

我想通过这个简单的分析，可以帮助大家理解ProChain 经济系统设计对于广告产业的正面激励。

不过故事不止于此，未来可以围绕 PRO 出台期货合约，为广告主、媒体和用户提供套期保值等衍生价值，则其对于互联网广告产业的影响将更为巨大。

 

中心与去中心结合的设计和跨链通证


详细阅读过 ProChain 白皮书的读者可能会发现，ProChain 与其他区块链项目相比，有一个重大的不同，就是它的主要业务逻辑和技术设施并不是放在链上的，而是传统的中心化系统。比如它的广告交易平台、DSP、SSP、大数据分析系统、反欺诈系统等等，都是用传统的、成熟的中心化技术设施实现的。

 

且慢，我们不是在谈区块链、去中心吗？这么多的中心化系统放在这里，是不是不够先进呢？不够去中心呢？


不，这恰恰是般若链设计当中的一个重大优势。

 

般若链的设计者很好的理解了通证的本质，理解了区块链作为通证基础设施的优势和不足。他们把高性能、重业务逻辑的适合由中心化系统解决的事情放在传统架构上，而只是把通证的发行和交换这一件事情放在区块链上。这种设计立刻将两个世界中最好的部分结合起来：既有传统中心化系统的成熟高效，对复杂业务逻辑的支持，又有区块链的安全和可信。

反过来，现在有很多项目宣传要把所有的任务都去中心化，说每一个传统中心化应用都将有一个更好的去中心化版本，这种想法是过于极端了。难道我们的报表系统、搜索引擎、大数据分析、机器学习、交易撮合这些天然中心化的重型业务逻辑也要放在区块链上来做吗？难道因为有了去中心化的区块链，传统的中心化应用就都一文不值了吗？当然不会是这样。中心化与去中心化结合才是王道，般若链在这件事情上做了一个很好的平衡和示范。

此外，这种方案还有一个很多人没有意识到的优点，就是通证跨链。由于复杂的业务逻辑都转移到了传统业务架构上，通证身上的业务逻辑负荷很低，所以对链的要求就很低。这样一来，无需借助诸如 Polkdot 之类的前沿跨链技术，仅仅通过一些精致的设计，PRO 币就可以同时运行在多个链上。这当然是一个过程。当新的、有特色的基础链推出之后，般若链就可以把其 PRO 币部署到新链上的，整个系统仍然可以畅行无碍，并且享受新一代区块链技术带来的优势。比如说，如果未来 EOS 平台成熟，PRO 完全可以保持发行总量不变（仍为1亿枚）的前提下，跨以太坊和 EOS 运行。而对那些重度绑定以太坊智能合约的通证项目，实现这种跨链通证就机会渺茫了。

跨链通证可以大大提升系统的实用性、适用范围，但更重要的是，每一次跨链，都是搞制度升级、制度创新的好时机。制度也是要与时俱进的，也是需要改革的，我们今天想得再完美，随着形势的发展变化，制度也会老化过时，也需要调整。这时就需要有一个升级的办法。但升级是有风险的，尤其制度升级，触动大量既得利益，阻力和风险很高。跨链代币为升级改革提供了很好的机制，比如我们可以搞老链老办法，新链新办法，靠增量带动制度变革，给整个般若链平滑升级机会。这就相当于开新特区，用新特区带动新改革。这其中的妙处，我们这一代中国人都不陌生。

 

4. 总结


我在一些场合经常说，区块链项目的厉害之处，三分在技术，七分在制度。所谓制度，就是经济系统设计。但当前大多数区块链项目并不太重视这个问题，而数字货币的投资者也不太理解和关注这个方面。在这种情况下，教育市场的工作要做，同时也要对一些设计比较优秀的区块链项目予以介绍和分析。

般若链的经济系统设计，结合了互联网广告市场的特点，大战略对头，很多小细节的设计也非常漂亮，值得拿出来分析一番，也值得其他行业的区块链项目借鉴。

当然，般若链的经济系统设计不会是尽善尽美，比如在当持有大量PRO 的人处于某种目的试图对般若链的信用实施破坏的时候，应当如何应对？如何确保 PRO 的长期分布充分合理？社区的议事、决策和治理规则如何确立？这些问题还需要进一步的思考。我本人也会与核心团队一起在这些方面继续努力，让般若链的通证经济系统始终保持领先。
