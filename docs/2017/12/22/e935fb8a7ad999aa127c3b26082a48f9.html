001 Go语言安装、卸载

  • 柚子社区

最近安装Go环境，看了一些网上的帖子写的不是很全，看了一下官网的教程，还是比较清晰，对于不能翻墙的朋友是个麻烦，下面简单写一下。

Go官网地址：https://golang.org/doc/install

一、Go语言安装

如果有历史版本的Go，需要在安装前卸载历史版本，请参考Go语言卸载。

1、Windows上安装

 1.1 、下载

下载地址：https://golang.org/dl/

  1.2、安装

安装MSI文件或者zip压缩包版本解压，需要安装或者解压到C:\go目录，否则需要配置环境变量。

1.3、配置

      1.3.1配置说明

 a、增加环境变量 GOROOT=C:\Go\  b、zip版本配置：将以下目录加入到系统环境变量PATH后  c:\Go\bin

2、Ubuntu上安装

 2.1 、下载

下载地址：https://golang.org/dl/

下载当前最新版本文件：go1.9.2.linux-amd64.tar.gz

  2.2、安装

sudo tar -C /usr/local -xzf go1.9.2.linux-amd64.tar.gz  

  2.3、配置

  2.3.1默认安装路径（/usr/local）配置文件配置 
        sudo vi /root/.bashrc         将这句话放入最后并保存         export PATH=$PATH:/usr/local/go/bin

  2.3.2非默认安装路径配置文件配置

     sudo vi /etc/profile      打开 /etc/profile后将Go语言实际解压路径配置好,既增加GOROOT环境变量。         （例如 sudo tar -C /home/zhouxs/go  -xzf go1.9.2.linux-amd64.tar.gz  ）    export   GOROOT=/home/zhouxs/go   export   PATH=$PATH:$GOROOT/bin

3、Mac上安装

 安装后，将/usr/local/go/bin放到环境变量中

二、Go语言卸载

1、Windows卸载

1.1、删除环境变量GOROOT，并去掉PATH中c:\Go\bin

1.2、删除c:\Go\文件夹

2、Ubuntu卸载

2.1、删除/usr/local/go下的文件（Linux and FreeBSD）

2.2、删除环境变量信息

   sudo vi /etc/profile

3、Mac卸载

3.1、删除/usr/local/go下的文件
3.2、删除环境变量信息/usr/local/go/bin

三、Go语言安装测试

1、命令行测试

        执行  go version         返回  go version go1.9.2 windows/amd64

2、运行代码

2.1、windows下编译

        将以下代码保存成hello.go并放到C:\Go\src\hello目录下，运行以下命令进行编译，生成hello.exe文件

        C:\Users\Gopher\go\src\hello> go build

package main import "fmt" func main() { fmt.Printf("hello, world\n") }

2.2、Ubuntu下编译


         将代码保存成hello.go并放到/home/zhouxs/gosrc/hello目录下，运行  go build 命令进行编译，生成hello文件
