第一次使用liteIDE开发Go程序

  • 柚子社区

开发Go程序的IDE有很多选择，
（1）、elipse安装GoClipse插件，
（2）、Vim/Emacs，万能开发工具
（3）、文本编辑工具gedit(linux)/Notepad++(Windows)/Fraise(Mac OS X)
（4）、liteIDE，一款专业为GO语言开发的集成开发环境
（5）、jetbrains新出的Go IDE，GoglandIDE
使用LiteIDE是一个不错的选择，软件体积小，不怎么耗内存，这个是我的优先选择。第二，它是一个专业GO编程工具。

第一步、下载
下载地址
https://www.golangtc.com/download/liteide
选择自己对应系统的版本
解压后就可以用，bin目录下的启动文件Windows系统是.exe,Linux是.sh文件。
第二步、新建
文件——>新建——>Go1 Command Project
第三步、修改你的Go环境路径
一般默认在C：\go

点击进去修改 

# native compiler windows amd64

GOROOT=D:\Go
#GOBIN=
GOARCH=amd64
GOOS=windows
CGO_ENABLED=1

PATH=c:\mingw64\bin;%GOROOT%\bin;%PATH%

LITEIDE_GDB=gdb64
LITEIDE_MAKE=mingw32-make
LITEIDE_TERM=%COMSPEC%
LITEIDE_TERMARGS=
LITEIDE_EXEC=%COMSPEC%
LITEIDE_EXECOPT=/C

第四步、运行程序
点击中间靠右边的蓝色BR（BuildAndRun的简称）按钮即可
运行结果：

D:/Go/bin/go.exe build -i [D:/GO_WorkSpace]
成功: 进程退出代码 0.
D:/GO_WorkSpace/GO_WorkSpace.exe [D:/GO_WorkSpace]
Hello World!
成功: 进程退出代码 0.

同时还得注意，同一个项目下边只能有一个main（）方法的文件，否则会报错，程序的入口，是唯一的。
