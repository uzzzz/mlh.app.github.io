基础 CPU 挖矿教程

如何挖矿？

Step1:获得一个钱包地址

钱包分为两个部分讲,一个是在线钱包,一个是本地钱包.(按需选择，目前还有轻钱包，可以看群文件教程)

1.本地钱包

在 XMR 的官方网站下载钱包: Monero: Downloads(点击downloads这几个字就是下载地址,群里也有,下不下来就去官网下)

建议下载 GUI 版本的，比较便于操作．由于我演示用的是 Mac,我就下这个 mac 版的,其他系统的 GUI都是基本相同的，直接下载就行了．

客户端支持中文，选择简体中文。

然后点击创建一个新的钱包(如果之前有钱包的可以用从密钥或者种子码恢复钱包,具体的进阶再讲吧.)

按照要求记录下必要的信息 。

给钱包加个密码

点击侧边栏中的收款，红框中的就是你的地址了。

2.在线钱包

在线钱包的创建

Step1:获得一个钱包地址
在 MYMINERO 的官方网站 myMonero

Step2:点击Create an Account(创建一个账户)

Step3:保存备份好Login Key

Step4:确认Login Key

Step5:钱包已经建好了,红框内为钱包地址.

Step2:寻找一个靠谱的矿池

这里使用的是小一点的矿池，以防说广告：快蛙

矿池其实主要考量因素，主要是两个－－稳定性和抽成比例.

自己可以根据这两个条件，自己选择合适的矿池。

Step3:挖矿设置

由于是CPU挖矿教程,这里显卡驱动的安装步骤跳过[以后再讲].

下载挖矿软件并设置,这里分为两个软件介绍.

XMRIG

在这里下载挖矿软件：xmrig(点击downloads这几个字就是下载地址,群里也有,下不下来就去官网下)

下载后解压，应该得到三个文件

打开config文件，把文件中的url、user、pass修改成下面的样子，记得修改user为你的钱包地址。

“url” : “pool.xmr.kwaiwa.com:10001”,
“user” : “这里填写刚才你在Step1中得到的钱包地址”,
“pass” : “邮箱地址”,
然后双击xmrig，如果一切顺利，你将看到类似的界面

还是解释一下这个文本里其他意思吧，懒人翻译官．

简单将两个比较主要的设置

这个软件默认的捐赠是 5 %,最低为 1 %
“donate-level”: 5, // donate level, mininum 1%

算力打印时间,默认是 60 秒.
“print-time”: 60, // print hashrate report every N seconds

xmr-stak

xmr-stake-cpu下载地址
解压

2、文件目录要在英文目录下，否则会报错。
3、如果出现缺少dll，请安装VC_redist.x64.exe，下载地址：https://go.microsoft.com/fwlink/?LinkId=746572（不要问在这个问题上纠结了多久。）

一、下载后解压，应该得到5个文件

二、首次运行xmr-stak的时候，需要配置你的钱包和矿池地址。双击打开软件，你会看到这些提示

其实在这前面还有一张图，没有截图，意思就是如果输入0就退出，其他任意数字字母回车继续
输入monero7

Step4:查看收益

挖矿的过程中，可以按h键查看挖矿的速度。也可以直接在浏览器中输入矿池的官方网址，在官方网址指定的查询栏中，输入你的钱包地址查看你的挖矿和收益情况。
点击矿池官网kwaiwa

很多时候，本地的速度和矿池显示的速度不同步，因为在本地软件，速度是以秒为单位计算的，而在矿池方，速度可能按小时或者半小时进行计算，会存在误差！

需要说明的是，挖矿，只有在矿池整体爆块的时候，才会有收益，在爆块之前，你将不会看到任何收益。
