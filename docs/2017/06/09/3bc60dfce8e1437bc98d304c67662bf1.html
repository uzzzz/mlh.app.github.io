IBM Bluemix 是一个基于开放标准和云的平台，可以用于应用的快速构建、运行及管理。

Bluemix中国版推荐的明星产品有Watson 物联网平台，Cloudant NoSQL 数据库，区块链，移动消息连接服务，国际版上提供超过100种服务，有些功能在中国版上还没有，不过会陆续的上线。

Bluemix以Buildpacks的形式来支持各种语言的运行时，支持几乎所有类型的应用，并且允许我们根据其标准去做一些定制化的拓展。

基于样板或Cloud Foundry 应用程序可以快速的创建一个java应用程序，并可通过Bluemix控制台（一个基于浏览器的 UI 界面）很方便的管理应用程序，你启动、停止、增减实例及内存等等。
