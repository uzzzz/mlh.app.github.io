如何将自己写的项目上传到github上？

  • 柚子社区

首先，你要先确定自己在Github上有自己的账户名，账户邮箱和密码。如果没有可以先去https://github.com注册，如果有了可以点击Sign In登录。之后，我们开始进行命令行操作（当然你也可以下载客户端进行操作）

首先：

打开终端，输入：

       cd 直接将项目文件夹拖进来；

然后：

第一步：git init

第二步：git config --global user.name “你的账户名”

第三步：git config --global user.email "你的账户邮箱"

第四步：git add 直接将项目文件夹拖进来

第五步：git commit -m "对项目文件进行注释"

这时候点击回车键，会生成一些描述，就是将项目中的各种文件都进行了创建，并创建相关的描述文件，我们可以对README.md文件进行编辑（下面会提到在哪里进行项目说明）

第六步：在登录的Github界面中创建一个New repository

进入下面的界面：


第七步：git remote add origin https://URL，如图：


这时候如果中断中提示：fatal: remote origin already exists.，已经存在了，不用管，执行下一步

第八步：git push -u origin master

这个时候点击回车键后，可能会提示让你输入：Github上你的账户名和账户密码


之后就是等待上传的结果（可能有点慢，等终端上传完成即可）

然后刷新你在Github上刚才新建时的那个页面就会出现上传的结果
