前言

  • windows10上开发并安装了VS2015和Qt5.7.1，在Qt官网下载了qt-vs-addin-1.2.5 插件，但是在安装时出现了bug，错误为：The installer could not find an installation of Visual Studio 200x。译文：这是因为您的电脑上没有安装微软公司的开发工具Visual Studio 200x。Qt5插件安装程序说它不支持Visual C++ Express安装程序找不到安装的Visual Studio 200x。这明显是不合理的。后来在网上查了很多，原来很多人都出现了这个问题，那应该不是安装的错误或者系统的问题，接着我又仔细查了插件的相关信息，原来由于VS2015不再支持add-in，所以要用其他手段。

一、安装VS2015

首先安转visual studio 2015，C++相关的一定要选上，别的视需要而定。具体问题可以留言挥着自行百度，google。

二、安装Qt5.6

进入http://www.qt.io/download-open-source/，选择“Qt Offline Installers”，下载“Qt 5.7.1 for Windows 64-bit (VS 2013, 922 MB) ”，下载后的文件名为：qt-opensource-windows-x86-msvc2015_64-5.7.1.exe，运行它，一键式安装。

四、安装VS015的Qt插件Qt5Package

由于VS2015不再支持addin，所以Qt以前Qt VS Addin也就没用了，不过，根据Qt官方的说法，他们也在开发能让VS2015使用Qt的工具。在他们开发完毕之前，我们先用VS的一个叫做Qt5Package的插件代替addin。
地址：https://visualstudiogallery.msdn.microsoft.com/c89ff880-8509-47a4-a262-e4fa07168408
