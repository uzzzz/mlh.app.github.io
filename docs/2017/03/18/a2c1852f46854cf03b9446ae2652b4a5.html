九年来，Black Duck开源年度奖一直致力于发现过去一年中出现的最具创新性与影响力的开源项目。尽管开源项目阵营一直在快速变化，但年度新人奖一直在为行业趋势提供重要参考。
过去一年中，多个技术领域表现极为活跃，其中最值得关注的无疑是区块链技术——它已经超越虚拟货币开始在更多层面发挥作用。机器学习——包括深度学习与神经网络——亦在智能范畴之外进军金融服务甚至设计与制造行业。
大数据、软件定义网络(简称SDN)、容器管理以及安全亦属于热门领域。下面，我们将了解这一年中的各位获奖新人!

1.Sawtooth Lake

Sawtooth Lake为英特尔公司为Hyperledger项目开发的新型分布式分类帐平台，旨在解决现有区块链技术中的可扩展性与安全性问题。它提供的模块化解决方案与防篡改一致性算法可用于随机选择领导者、基于时间证明机制(所需计算资源远低于比特币的工作证明)声明新块并可显著提升交易安全性。Sawtooth Lake能够针对特定领域建立事务序列以满足各类用命，并利用其中的域定义规则约束相关事务。这正是将区块链技术引入广泛业务应用中的一大重要进步。

2.Carbon Data

Apache CarbonData是一套具备完全索引性的列式Hadoop本地数据存储库，其可与Spark相集成以实现查询优化。它采取独特的方式以实现数据组织、多级索引以及优化方法，从而带来更快的数据过滤速度、压缩效果以及更为强大的搜索与查询处理能力，同时更为高效地利用计算资源。字典编码机制允许用户更快完成聚合，并在聚合完成后再进行“延迟解码”。CarbonData还通过单一文件格式支持多种不同的数据访问模式，包括OLAP查询、顺序访问(大扫描)以及随机访问(窄扫描)。

3.Deep Scalable Sparse Tensor Network Engine (简称DSSTNE)

Amazon推出的DSSTNE——音为“destiny”——对数据稀疏性及可扩展性进行了优化，同时专注于优化多GPU配置，从而彻底改变了神经网络运行环境。受到Amazon产品对大规模与低延迟计算能力需求的启发，DSSTNE专门用于支持大型网络、稀疏数据集以及并行训练。Amazon宣称DSSTNE在处理稀疏数据集时，速度可达谷歌TensorFlow的二到十五倍。目前，DSSTNE仍专注于扮演一套规模化稀疏数据深度学习框架，并成为电子商务及企业用户的理想引擎选项。

4.OpenCORD

开放网络实验室——一家负责发掘软件定义网络全部潜能的非营利性组织——开发的OpenCORD是一套端到端解决方案，用于将SDN、NFV与云同商用基础设施加以结合，共同为服务供应方网络提供数据中心级别的规模与敏捷性能力。OpenCORD适用于电信中央办公与接入、家庭乃至企业环境，采用开放性构建组件的公共基础设施以降低资本投入与运营费用，并利用网络的可编程性与灵活性缩短产品上市周期。目前，OpenCORD的用户包括AT&T、韩国电信、中国联通以及NTT，它同时与开放网络基金会合作以通过开源软件及软件定义标准进一步推动网络解决方案发展。

5.Poseidon

今年，两大组织正利用尖端机器学习与软件定义网络功能努力提升网络安全水平。凭借着In-Q-Tel(简称IQT)技术加速机构的支持并与美国政府、学术界以及美国情报界的密切合作，Lab41与CyberReboot成功打造出Poseidon项目。Poseidon的存在旨在解决两大关键性问题——您的网络中承载着什么?其又在做些什么?Poseidon能够为网络中添加或删除的项目以及流量提供情境感知能力。它利用机器学习技术检查网络上的交互活动，并通过学习追踪并阻止各类与恶意活动相关的线索。在初步测试当中，Poseidon获得了84%的恶意活动检测率，而误报比例仅为2.2%。

6.Trireme

Trireme负责回答一个亟待解决的问题：如何强化Linux环境以确保容器能够高效安全地运行?通过对SDN的持续探索并将安全策略、规模化与补救性举措加以融合，云安全厂商Aporeto推出了Trireme项目——这是一套针对分布式应用的云原生安全解决方案。它能够通过端到端身份验证与授权机制建立大规模安全策略。Aporeto公司还选择与Kubernetes开展紧密合作，并借助后者灵活的网络策略框架、快速发展的技术社区以及对云原生应用与规模性议题的关注为Trireme项目的发展提供助力。

7.Ansible Container

Ansible Container是Ansible开发团队面向Docker文件打造的一套替代性方案。它能够自动实现容器构建、部署以及管理，且整个过程只需要依托于Ansible Playbooks。Ansible Containers是一套平台中立性方案，适用于大多数常见容器编排引擎，具体包括Kubernetes与OpenShift。它的模块化特性允许大家在开发与部署过程中分别面向Docker与Kubernetes，且配置过程非常简单。自于2016年夏季在DockerCon大会上披露以来，该项目获得了可观的关注与社区发展反响。2017年，Ansible Container即将推出1.0版本，而关注重点则放在将相关服务扩展至更多用例与角色范围身上。

8.Kolibri

Learning Equality一直致力于利用它的Kolibri应用以非营利性方式为全球各资源水平较低的地区提供在线教育服务。尽管目前尚未公开下载，但Kolibri确实已经开始向教育资源有限地区的学生与教师提供学习资源。Kolibri提供交互式练习方案、自定进度机制以及协作式学习工具，能够为学生及教师提供实时反馈与指导。Kolibri的“种子”终端设备能够通过离线本地网络实现不同设备间的内容共享与更新。另外，Kolibri以最小质量损失对内容进行压缩，因此能够在小型、低成本设备上顺畅交付大量内容。
