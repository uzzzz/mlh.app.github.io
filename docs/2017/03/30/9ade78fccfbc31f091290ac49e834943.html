从0到1搭建React Native开发环境

  • 柚子社区

RN开发之前要配置一系列的开发环境，这里刚配置好，这里做一个总结记录，希望对大家有帮助。

一、首先，我的安装流程大体上是nvm->nodejs->npm

1.nvm是什么呢？全称node version manage。看到全称应该知道了吧，其实nvm就是用来管理nodejs的一个管理器，他可以安装多个版本的nodejs和npm版本。

NodeJs是什么？ nodejs是React Native开发中所需要的代码库。

那npm又是什么呢？NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题，常见的使用场景有以下几种：

a.允许用户从NPM服务器下载第三方包到本地使用。

b.允许用户从NPM服务器下载并安装别人编写的命令行程序，到本地使用。

c.允许用户将自己编写的包或命令行程序上传到NPM服务器供他人使用。

由于新版本的nodejs已经集成了npm，所以我们安装nodejs的时候，也会一并把npm安装好。

好，下面我们进行安装。

二、nvm的安装

因为我自己用的是windows 64位（我相信现在很少有32位的机子了），这里就讲一下自己的安装过程。

1.首先，在github上找到项目，我安装的是V1.1.2版本，下载相应的nvm-setup.zip,解压后安装。

安装完成后，我们在命令行输入nvm 可以查看安卓版本及相关命令help

2.添加国内镜像

由于nvm有些下载地址被墙了，所以这里强烈推荐加入国内镜像，我这里用的是淘宝的，速度很快。

输入nvm root查看nvm路径信息，我的是C:\User\Administrator\AppData\Roaming\nvm，用资源管理器打开相应目录，找到里面的settings.txt，并打开。

在最后一行加入两行代码

node_mirror:  https://npm.taobao.org/mirrors/node/
npm_mirror:  https://npm.taobao.org/mirrors/npm/
然后保存。保存后关掉cmd命令框，重新打开cmd。

3.安装nodejs和npm

在新打开的命令框中输入nvm install 6.9.4 64,就会启动淘宝镜像自动下载安装对应的node和npm版本。

4.npm注册淘宝镜像

命令输入 npm config set registry=https://registry.npm.taobao.org
