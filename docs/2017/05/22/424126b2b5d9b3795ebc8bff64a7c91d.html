微软正式发布XAML Standard与.NET Standard 2.0：现已提供下载

  • 柚子社区

微软在本月早些时候召开的 Build 2017 开发者大会上的披露的 XAML Standard 和 .NET Standard 2.0，现已正式发布。新工具旨在为开发者们带来“基于同一标准的跨平台 XAML 语言结构”（基于 UWP 和 Xamarin.Forms），以及基于社区反馈带来基于 .NET 的精选大集，用户们有望在“今年晚些时候”接触到 Xamarin.Forms 和 UWP 加持的 XAML 标准应用。

.NET Standard 2.0

　　如果你对 .NET 2.0 的更感兴趣，应该不用等太长的时间，不过微软当前也没有给出更多细节。

　　视频地址：http://v.youku.com/v_show/id_XMjc3NjgzMjYyOA==.html

　　传送门（.NET Standard 2.0）：

　　https://github.com/dotnet/standard/blob/master/docs/netstandard-20/README.md

XAML Standard 1.0

　　开发者们可以留意 .NET Standard 官方 GitHub 页面的最新编译并积极提问。

　　视频地址：http://v.youku.com/v_show/id_XMjc3NjgzNTE5Mg==.html

　　传送门（XAML Standard 1.0）：

　　https://github.com/Microsoft/xaml-standard/milestone/1
