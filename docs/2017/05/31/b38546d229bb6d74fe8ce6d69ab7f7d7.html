GOLANG项目：基于命令行的音乐播放器

  • 柚子社区

项目需求：

1、设计一款音乐播放器，可以查看、添加、删除音乐

2、基于命令行方式


项目环境：

1、Go version go1.7.5 windows/amd64

2、开发环境：LiteIDE


1、项目目录结构

src目录讲解：
main.go : 主程序文件
library : 音乐库，负责音乐的查找、删除、增加
musicplay : 音乐播放模块，负责各种格式音乐的播放
