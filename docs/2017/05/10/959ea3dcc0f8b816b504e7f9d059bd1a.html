Qt5 简单使用webview 和openssl报错处理

  • 柚子社区

使用QWebView类是Qt里面的一个浏览器小控件。QWebView的主要功能是用于浏览网页，QT5这边已经添加到QtWebKitWidgets里面，所以头文件和Qt4不一样。每个QWebView都包含着一个QWebPage，而QWebPage是用于存储和编辑网页的类，应用需求挺广，浏览网页，应用于项目就很方便，比如工作人员浏览服务器修改一些配置文件

pro文件添加   

QT       += core gui network webkitwidgets
Cpp文件添加

  
#include <QApplication>

  
#include <QtWebKitWidgets/QWebView>

  
using namespace std;

  
int main(int argc, char *argv[])

  
{

  
    QApplication a(argc, argv);

  
        QWebView *pWebView = new QWebView();

  
       //加载一个网页地址，这里使用QT官网，也可以改成其他的网址

  
    //   pWebView->load(QUrl("http://qt-project.org/"));

  
        pWebView->load(QUrl("https://hao.360.cn/?wd_xp1"));

  
       //显示网页

  
       pWebView->show();

  
        return a.exec();

  
}

这两个文件可以到点击打开链接，百度网盘里面进行下载。
