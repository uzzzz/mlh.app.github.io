硅谷Live / 实地探访 / 热点探秘 / 深度探讨


在 2014 年的采访中，投资了特斯拉、SpaceX 等公司的硅谷著名投资人 Tim Draper 大胆预测， “到 2018 年，比特币会涨到 1 万美元一个！”



万万没想到，比特币真的要破一万美元了！



今天最高价格已经接近$9800美元。



比特币也从“一币一宏光”进入到“一币一捷达”时代，当然，离”一币一别墅”还有一定距离。



美国韭菜上车了！


为什么比特币最近突然大涨？最重要的原因是美国的“韭菜”们上车了。


“韭菜”啥意思呢？币圈“割韭菜”的意思是一部分革命先烈亏本离场，新生力量又加入币市，就像韭菜一样，割一茬很快又长一茬。让我们对“韭”当“割”，唱出心中喜悦，轰轰烈烈，把握青春年华！


据说在刚刚过去的感恩节期间，美国家庭们在餐桌上除了吐槽火鸡难吃，讨论最多的话题就是比特币了。


这个并不是空穴来风，美国购买比特币的APP Coinbase 在11月22号宣布，他们的注册用户超过1300万，已经超越了Charles Schwab（嘉信理财）的开户数，美国最大的金融服务公司之一。



感恩节期间短短两三天时间，Coinbase 新增了10万注册用户，而从感恩节开始以来，比特币的价格已经上涨了1600美元。


Coinbase 也成为苹果App Store上搜索上最多的应用，说明什么？普通的美国人也开始买入比特币了。



供求关系决定比特币的短期交易价格


从短期来看，市场供求是决定市场价格的直接因素，它使价格围绕着市场价值或生产价格上下波动。比特币这波暴涨，可能就是来自西方国家的旺盛需求。


自从11月2号，全球最大期货交易所芝加哥商品交易所（CME）宣布，计划年底前推出比特币期货合约，当时比特币价格才6300美元左右。比特币越来越受传统金融机构的青睐，越来越多的资金进入币市，受此推动，比特币价格一路飙升。


根据CNBC 报导，受此消息影响，第二天大约10万人注册了Coinbase，比特币也短短一个月内飙升了3600多美元。



比特币价格波动非常剧烈，一天暴涨10%或者暴跌10%，再次也向各位投资者提醒风险。


到底什么是比特币？


比特币诞生源于神秘人物中本聪2008年的白皮书《比特币：一种点对点的电子货币系统》。


比特币总数固定在2100万个，没有发行方，去中心化，这些特性让比特币成为一种革命性货币。基于区块链技术(想了解区块链？点击阅读原文)，比特币的每个交易都经过全网广播，人人可见，可以查到任何交易历史，无法伪造、篡改交易记录，数字加密货币解决了网络去中心化、无主权货币、避免双重支付、可靠高效、廉价匿名等一系列过去觉得不可能解决的问题。



在过去的7年间，比特币价格翻了68万倍。随着比特币启动隔离验证和区块容量扩容的消息带来的新一波暴涨行情，再加上席卷全球的勒索病毒点名只接受比特币“赎金”，比特币又一次进入大众视野。比特币也引发了投资的狂潮，资金涌向了交易所、数字货币钱包、数字货币矿场、和区块链应用等领域。


比特币价格由于投机等因素波动非常大，但其实际价值需要时间的考验。


区块链的1994年


如今的比特币和区块链的发展程度，就如同1994年的互联网。


互联网的第一个杀手级应用是邮件，而比特币可以看成区块链的第一个杀手级应用。邮件最早也只是工程师的玩具，但后来成为互联网的重要基础设施。比特币早期也只是在黑市中使用，但如今已经慢慢凸显其实用价值。



人类正处于一场从物理世界向虚拟世界迁徙的历史性运动中。


传统的互联网不是为传递价值而生，互联网上信息的传输，本质是信息的拷贝。而现实中的货币流通要依靠中心化的组织做背书来维护运行，比如微信支付、支付宝、银联等。但现在有哪家公司能活1000年以上的吗？所以，依靠中心化的方式实现价值传递，弊病很多。


而区块链是第一个能够实现价值传递的网络，区块链技术有望带领人类从信息互联网过渡到了价值互联网的伟大时代。


这个意义有多大呢？


在人类社会中，价值传递的重要性与信息传播不相上下。


互联网的出现，使信息传播手段实现了飞跃，信息实现了高效流动，但互联网价值传递的效率依然很慢。当前互联网上的电子货币本质上依然是传统的纸币，跨国支付也依然是个大问题。而区块链的诞生正是人类构建价值传输网络的开始。它将使人们能够在网上像传递信息一样方便、低成本地传递价值，这些价值可以表现为资金、资产或其他形式。


这个虽然看起来简单，但意义可能是巨大。我们这么来看，微信、支付宝小额移动支付的便利激活了一个万亿级别的知识付费行业（方便地打赏和购买），这是支付的便利带来的行业变革，而区块链带来的价值流动的便利性会对全球带来更巨大的影响。


对于价值传递，价值流动越快，社会就越有活动。因为价值互联网，人类社会也将迎来一场更完美的革命。


该不该投资


区块链长期是一个非常有颠覆性的行业，当然也面临了很多发展的问题，新“韭菜”们也会遇到很多坑，比特币交易所不断爆出被偷、被黑，短期来看很多投资比特币的人其实是在亏钱。


而这么多年来，比特币价格起起伏伏，但 Tim Draper 始终坚信：


“未来，比特币和区块链将给我们的生活带来深刻的变化。人们将很快发现，用了几个世纪的银行系统，将被比特币和区块链取代。 而人们需要做的，就是接受这些变化，拥抱这些变化!”
