(二) Hyperledger Fabric 启动你的第一个网络

  • 柚子社区

一、准备工作

1.      通过教程——《(一) Hyperledger Fabric在CentOS7.2 64位系统的开发环境搭建》 确定已经下载完成Hyperledger Fabric官方Sample codes。

 

二、生成网络

1. 生成网络标识

cd <你的目录>/fabric-samples/first-network
chmod –R +x byfn.sh
./byfn.sh –m generate


执行成功后的结果：

 

2. 启动网络

./byfn.sh –m up

启动成功：


中间输出结果略

 


Go是chaincode的默认语言，此处已经将镜像编译至容器，也有对Node.js chaincode的支持。

可将开发语言修改为nodejs，使用以下命令：

./byfn.sh -m up -l node

 

3. 关闭网络

./byfn.sh –m down

关闭成功：

 

三、工具说明

1. Cryptogen

生成密码（x509证书和签名密钥）提供给各种网络实体使用。这些证书是身份的代表，允许在实体通信和处理时进行签名、验证身份中使用。
