雷电网络

雷电网络是一种链外扩展解决方案，可实现近乎即时，低成本和可扩展的支付。 它是以太坊区块链的补充，可与任何兼容ERC20的代币一起使用。 雷电项目正在进行中。 其目标是研究状态通道技术，定义协议和开发引用实现。

技术

Raiden网络是一个在以太网络之上的基础设施层。 虽然基本思想很简单，但是底层的协议是相当复杂的和实现是不平凡的。 尽管如此，技术可以被抽象出来，使开发人员可以与一个相当简单的API接口来构建基于Raiden的可扩展的去中心化应用程序。

Raiden Network 101

区块链不能很好地扩展，因为所有转移的顺序和结果需要全球共识。 每个参与者需要了解共享分类帐的所有更新。 硬件和带宽限制对分布式网络中可以共享的每秒更新数量设置了限制。 雷电网络的基本思想是避免区块链共识的瓶颈。 这是通过利用支付渠道网络完成的，该网络允许链外安全地转移价值，即不需要每次转账都涉及区块链。

要了解更多信息，请阅读Raiden Network 101，查看当前规范或查看代码。

用例

Raiden网络的明显用例是支付。 目前，全球支付行业每年产生大约2万亿美元的收入 - 而且还在增长。

零售付款

有很多高调的以太坊代币项目竞相向大众提供区块链支付。 特别是在发展中国家，这些努力可能会改善数百万人的生活。 雷电网络可能是一个至关重要的组成部分，因为重视采用依赖于可扩展的技术和有竞争力的费用。

P2P现金

现金，因为我们知道它是在倒退，因为有无现金社会的趋势。 基于可扩展区块链的支付可以帮助保持现金的私有性和分散性，同时将用户体验更新为新一代的期望。

小额支付

区块链是成为即将到来的机器对机器经济的支付基础设施的热门候选。 物联网可能将商业交易的数量增加一个数量级 - 转移成本越低，用例就越多。

微支付可用于获取API，带宽，计算能力，存储，电力，基本上任何基础设施的细小访问。 内容或娱乐（如网页，游戏，视频或音频流）也是如此。

现在已经有许多人提议的DApp依靠网络参与者之间的小额支付来激励合作行为。


即时代币交换
除了付款之外，代币交换可能是第二个最突出的区块链使用案例。 如果当前的代币化趋势继续下去，情况更是如此。 基于雷电网络的原子代币互换功能构建的去中心化交易所可以以低成本即时交换代币。
