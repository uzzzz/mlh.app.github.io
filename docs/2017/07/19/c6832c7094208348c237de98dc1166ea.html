看了你就知道为什么iTunes要对电脑授权了

  • 柚子社区

很多人都不明白为什么iTunes要对电脑授权才可以进行同步等操作。其实这是苹果公司为了保护IOS设备的措施，我们在使用iPhone时，手机中会有很多隐私的东西，如果没有这项措施，我们就可以随意在其他电脑上登陆iTunes同步手机数据了。而这项措施相当于防线的存在，让我们对可以信任的电脑授权后，再进行手机数据的同步。


如何在iTunes中对电脑授权

（1）首先在可信任的电脑上下载最新的iTunes，登陆Apple ID


（2）打开在iTunes的“账户”菜单栏中选择“授权—对这台电脑授权”，在弹出的窗口中输入密码点击授权即可。


在同一台电脑多次授权并不会占据授权额度。

如何查看授权额度

苹果公司既然想到来了电脑授权措施，就不会是无限制的授权。保护数据的安全，当然是越少设备同步越好，所以在iTunes中我们只能对5台电脑进行授权。

那怎么知道授权额度是否用完呢？

（1）与授权步骤相同，我们在iTunes的“账户”菜单栏中点击“查看我的账户”，进入“APP Store”。


（2）在“APP Store”界面中，可以看到登陆ID的账户信息，在账户信息下，我们可以看到电脑授权的状况。


额度用完怎么办

当我们使用多台电脑时，就会对多台电脑进行授权，超过5台电脑，我们的授权额度就会用完。这时若要在第六台电脑上进行授权，取消最不常使用的电脑授权，我们就又获得授权一次额度了。

那如何取消对电脑的授权呢？

对电脑授权和取消对电脑的授权是在同一个菜单栏中的，打开iTunes“账户”菜单栏中的“授权”，点击“取消对这台电脑的授权”即可。


若电脑离得太远，也可以通过“账户—查看我的账户—APP Store”中选择全部解除授权即可。


以上就是关于在iTunes中对电脑授权的相关资料。
