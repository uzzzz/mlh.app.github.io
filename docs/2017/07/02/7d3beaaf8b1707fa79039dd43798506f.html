区块链的最显著的创新是采用了混合挖矿的方式来维护区块链的网络安全，即工作量证明挖矿PoW，及权益证明PoS挖矿。

PoS挖矿让持币者（任何在其区块链钱包里有余额的人），可以用通过持有区块链的证明来进行PoS挖矿。以下是利用区块链客户端进行PoS采矿的教程。


前置条件：你的区块链钱包内必须有余额，也就是不能为零。而且这些余额在你钱包里最少已经停留了30天。否则是无法进行PoS挖矿的。
风险提示：在线运行热钱包具有一定的风险，请确保你的个人电脑运行环境安全。本文不对因此操作而导致的损失负责。(Sunny 称将在0.4的版本中进一步提升PoS挖矿的安全性。)
一、软硬件要求
挖矿软件：区块链客户端即可。
（请选择最新版本下载，目前是0.3，下载次数是32，268次，美国居首，中国次之）如果你是windows系统，请选择win32系统；Linux请选择Linux系统。
挖矿硬件：能够运行区块链客户端的个人电脑PC即可。
第一步:
所有电子货币的钱包都需要用密码保护，记得使用强密码！（经本人试验，部分符号无法被识别，所以如果出现无法挖矿的话，请考虑更换钱包密码。）
钱包加密后，会在客户端右下角出现一个上锁的标志。并且会提示，钱包已加密，无法进行PoS挖矿。（如果钱包没有加密，默认只要一运行客户端，就会自动进行PoS挖矿。）

2. PoS的重要概念


币龄：我从某人收到了10个币，并持有30天，那么这10个币的总币龄就是 300 币天.
一旦你的钱包解密用于PoS采矿后，就会消耗你的币龄用于生成一个POS区块。币龄累积越多，生成POS区块的速度就越快。如果你几年都没有上线，可能一上线就会挖到PoS区块，但这样的话，你也有可能错失复利产生的效益。
当向其它人发送区块链时：所发送的币的币龄就会被销毁，只有呆在钱包的币才会累积币龄。


----------------------------------------------------分割线----------------------------------------
个人理解: PoS挖矿时，是通过比较各个客户端的币龄，并且将区块给予币龄最大的客户端。所以你的币龄越大，获得区块的速度也就越快。我个人经历是等待4个月后，不到1小时就有PoS进帐。
个人经验: 每个月定期运行几天基本上就能够挖到1个PoS区块。每次运行4-5个小时。客户端就能挖到PoS区块。如果你在1个月后运行好几天都没有收获，那么请耐心多给区块链一些时间。
