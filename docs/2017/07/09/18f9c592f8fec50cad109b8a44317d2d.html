GOlang和LiteIDE在Win10环境64位中安装与配置教程（附链接）

  • 柚子社区

            GoLang一般指GO语言。Go语言是谷歌推出的一种全新的编程语言。LiteIDE是一款专门为Go语言开发的跨平台轻量级集成开发环境（IDE），本文讲一下在WIn10系统 64位环境中，下载安装Golang1.8安装版过程，以及安装LiteIDE的过程，边学边用，加深自己的印象，仅供参考，进入正题：

第一步下载安装包：go语言安装包更新很快。最新的1.9的测试版已经出来，本文提供的是go1.8.windows-amd64.msi标准版本，这是安装版，一路点next即可完成安装。 支持win10/WIN8/WIN7环境，64位安装。由于是安装版，只需要下载后解压，选择安装位置即可。

go1.8版本下载链接:链接:http://pan.baidu.com/s/1eSDx8yu

第二步测试安装包是否安装成功，运行cmd命令，输入go,看到如图字样说明安装成功。


下载编译器：LiteIDE 是 Google Go 语言的一个开发工具。本文提供的是最新版下载liteidex32-2.windows-qt5.zip支持win10/win8/win7。下载地址  http://download.csdn.net/detail/u014125025/9893314。下载后放置一个盘中，比如D盘。

第三步运行上述LiteIDE解压后的exe文件，可有如下界面。


接着配置环境分小三步：（1）在该界面上找到system,选择64位（根据自己的电脑选择），如下所示


（2）查看 →管理GOPATH...，创建新目录D:\Golang，并添加到“自定义目录”。作为存放项目工作空间。

（3）找到查看 → 编辑当前环境，修改 GOROOT、GOARCH、GOOS、CGO_ENABLED、PATH、LITEIDE_GDB 。

如何修改，可以运行cmd输入 go env 可得到go安装版的配置环境，按照上面的信息在LiteIde编辑当前环境中修改添加。如图所示：


第四步：运行Hello World。点文件——新建——模板选择Go1  Command  Project，给项目命名，点击OK。点击导航栏上的BR，下方输出Hello world。

至此golang安装包及开发工具安装完成，可以开启开发之旅。
