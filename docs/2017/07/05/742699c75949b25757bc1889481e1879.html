其实，小编有点犹豫要不要写这一篇教程，其实数字货币的挖矿方法和LTC几乎一样，只需要将挖矿端口修改为8336就可以了，但考虑到还是有不少新手，最终还是忍不住写了，如果你已经掌握了LTC挖矿，那么你可以不用看本教程，直接将挖矿端口改为8336，然后和LTC一样地挖就可以了。



如果您对新星币数字货币还不了解，可以点击学习什么是新星币数字货币，如果您对数字货币已经有所理解，那么就可以马上开始挖矿之旅了。


首先，你需要 下载数字货币钱包客户端，并安装好它。


第二步：点此下载挖矿软件CGMiner


第三步：找到一个好的数字货币矿池，在这里小编推荐大家使用比特时代提供的完全免费的数字货币矿池，所以如果你不打算寻找其它数字货币矿池的话，这一步就可以省略了，小编在后面的步骤中已经把矿池的地址和端口都已经配置好了


第四步：如果您的显卡是NVIDIA系列显卡，请跳过此步，如果你的显卡是ATI系列显卡，请把你的显卡驱动程序更新到最新版本，然后 点此下载安装一个AMD SDK 2.8


第五步：将前面下载到的CGMiner解压到某个目录下，然后进入cgminer文件夹，在cgminer的文件夹里面，新建一个文本文件(TXT)，内容写为
cgminer --scrypt -o
将上面的XXXXXX改为你的数字货币钱包收款地址，然后保存。你可以从数字货币钱包如下图所示的位置，找到你的数字货币收款地址，这是一个全球唯一的地址


数字货币收款地址


第六步：将你刚才新建立的TXT文本文件的.txt后缀改为.bat后缀（如果你比较菜，文本文件后面不显示后缀名而不会改文件名后缀，请点这里


第七步：双击运行前面你做好的bat文件，此时cgminer开始正式运行，运行界面如下所示，这意味着你已经开始挖矿了，尝试等待几个小时，然后再看看你数字货币钱包里的数字货币余额增加了多少（由于挖矿难度比较高，挖矿的人数很多，所以如果你个人的显卡性能如果不太好，可能要等上10个小时才会产生收益）


挖矿界面演示


你的显卡性能越好，挖矿速度就越快，获得的新星币数字货币就会越多。
