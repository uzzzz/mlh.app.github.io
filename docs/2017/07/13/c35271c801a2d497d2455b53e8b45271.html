qt5.6.2和Visual Studio 2013的开发环境的整合，手把手详细图文

  • 柚子社区

首先需要下载qt5.6.2和Visual Studio 2013的整合插件;

打开qt开源社区：http://www.qter.org/portal.php;

依次点开以下页面：







选择下载 qt-vs-addin-1.2.5.exe;

下载完成后进行插件的安装：





在接下来的页面中把用不到的其他版本的选项的勾选全部取消，只久勾选Visual Studio 2013选项，不然可能会在开发中出问题
















至此就完成了安装，安装过程中可能会有一个错误窗口弹出，不用管它，等会会在Visual Studio 2013中配置它；


接下来打开Visual Studio 2013：


在Visual Studio 2013菜单栏上会看到qt5的菜单项，这说明qt插件安装成功了；


如果现在就新建qt项目会报出如下错误：



这是因为，我们不没有配置qt插件;接下来配置qt插件：

在菜单栏依次选择 qt5——>Qt Options;











连续点击ok就配置好了，就可以新建qt项目了；


新建qt项目：











点击finish就可以创建成功了

项目结构如下图示：



双击Form Files目录下的demo.ui文件可以召唤出qt视图编辑器；


可以向主界面拖拽控件进行界面编辑：



至此Qt和Visual Studio 2013的开发环境就整合完成了
