关于一个量化交易Python库easytrader的安装

  • 柚子社区

最近在找一个轻量级的，可以在云端运行的，python库。对比vnpy这种期货的ctp框架。esaytrader更适合股票交易。废话不多说贴出，官网链接。点击打开链接

安装其实很简单：

1、安装Python 3.5，必须要在安装的时候点击pip和添加环境变量path。


2、接下来在windows中运行cmd,使用了环境变量后，无需到相应目录中了。直接运行pip install easytrader,直接会从pip在网上的库中下载安装，但是会安装报错。 安装时会提示 No module named xxx, 请使用 pip install xxx 安装对应缺失的module, 然后再重新 pip install easytrader


3、这是我的报的错误，缺少“requests”和"six"两个库。接着直接再来pip install requesets 和 pip install six ,然后再次pip install easytrader搞定


4、最后pip list 库的列表有了easytrader就可以了。
