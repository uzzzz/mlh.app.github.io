区块链风暴来袭，这五大行业将受到强烈冲击

  • 柚子社区

自从第一枚比特币诞生以来，加密货币就代表了经济实现重大飞跃的一次机会。尽管比特币出现了，没有中心化的机构，并且成为了传统货币的数字化选择，驱动加密货币运作的底层计算机协议则是从之后才被认为具备大量其他的应用场景。从分布式账本的不可更改性到加密学带来的安全性，区块链的大多数特点使其成为了解决各个行业问题的方案。自区块链的概念出现以来，下列五个行业受到了最强烈的冲击。



1. 金融服务业


鉴于区块链技术的第一个应用场景就是数字货币，金融服务行业因此面临重大挑战也就没什么好奇怪的了。搭建在区块链上的加密货币可以由一支小规模的开发者团队来研发，也就是说银行不再需要存在。由于区块链账本中的交易是公开的、可验证的以及分布式的，发起支付也就不涉及信任问题，也不用担心对方不发起支付。这就导致大部分从事金融交易的企业变得毫无意义。


2. 外汇交易


海外侨汇（remittances）就是把从某个国家赚到美元转给位于另一个国家的亲友。很多国家从这类汇款中取得了大量的收益。在墨西哥，通过汇款进入该国的资金甚至高于出售石油所赚取的资金。据统计，每年全球各国之间收发跨境汇款的金额达到了近5千亿美元，驱动这类交易的网络能够通过汇率以及交易手续费轻松赚取数十亿美元的利润。


就连其他形式的国际交易，比如说海淘，都深受货币兑换费用的困扰。在国外旅游的时候，机场或者市内提供的货币汇率通常并不优惠。基于区块链技术的无国界加密货币几乎能够实时完成资金的跨境转移，没有第三方的存在也就减少了不必要的花费。


3. 供应链管理


国际贸易是世界上最复杂的行业之一，需要通过数千个不同的渠道协调无数件事。航运以及物流业的重要参与者都在研究区块链技术在供应链管理中的应用。通过区块链平台，系统能够轻松且自动追踪独立的产品、零部件或者商品从供应商到消费者手中的一系列过程，全面且安全地记录序列号、运输历史以及所有人信息。在这里，区块链能够带来更高的透明度以及运作效率，甚至能够加强国际海关的监管。


4. 零售以及电商广告


随着市场选择的增加，零售以及电商行业的竞争力不断提高。通常这类行业的公司都将广告视为吸引新客源的方式。但是通过广告开拓新业务需要依赖可信任的第三方广告网络作为中间人。这样一来，这些中间人就成了企业和潜在客户之间的信息传递者。


区块链技术的基本特点之一就是公开、可验证账本安全的去中心化。区块链能够将用户数据的控制权交到每个人手上，并且消除复杂且通常较为昂贵的中间人体系。举个例子，BitClave正在用区块链移除广告服务中间人，建立企业和客户之间的直接联系。Brave和Presearch则是另外两个致力于将数据管理权交给用户的区块链项目。通过让消费者有选择性且安全地共享他们的身份数据，区块链领域的公司能够用智能合约激励项目以及智能筛选等功能帮助用户更好的与广告建立联系。


5. 身份管理


广告技术的快速发展归功于区块链技术提供的创新方式，用户可以因此安全且密切地监控他们的个人数据使用方式。从出境游到医疗记录或者财务贷款，个人信息在很多人的日常生活中占据了重要的位置。在区块链上存储经过验证的身份信息无需经过第三方的验证也能实现快速高效的身份证明。


在线身份验证可能造成交易的延迟，不同的资料来源甚至会导致数日的延迟。由于区块链代表了不可更改的、经过公开验证的数据库，人们可以整合所有线上数据。身份认证技术能够影响所有交易，彻底改变企业收集并使用用户数据的方式。
