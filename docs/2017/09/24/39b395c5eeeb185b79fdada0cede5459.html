Bat脚本实现定时演示PPT后自动关闭PPT

  • 柚子社区

前几天有个小姐姐找到我说要实现一个定时演示PPT的小程序，想让我帮帮忙，我是各种难为她

之前随便写一个一版脚本：整体思路是定时器倒计时，倒计时结束以后kill到这个进程的名字。

代码如下：

@echo off  
echo  @author qiyulin  
echo  @date 2017.09.21
set /p id=Enter Seconds: 
set /p name=Enter Process Name: 
for /L %%a in (
 %id%,-1,0
) do (
 echo %%a
 if %%a LSS 1 (
	echo kill %name%
	taskkill /F /IM %name%
 )
 ping -n 2 localhost 1>nul 2>nul
 cls
)
pause
popd
先输入秒数，然后输入进程名。挺二的啊，哈哈哈。

好了言归正传，今天终于改成了小姐姐想要的样式，大体思路是这样的。

1. 输入倒计时秒数
2. 选择是WPS还是OFFice的PPT

3. bat脚本3s倒计时后打开PPT
4. bat脚本调用vbs脚本执行F5按键（实现ppt全屏演示）
5. 演示同时启动定时器来倒计时

start.bat 代码如下：

@echo off  
echo  @author qiyulin  
echo  @date 2017.09.21
set /p id=Enter Seconds: 
set /p name=WPS 1,OFFICE 2: 
rem 3s 打开ppt
for /L %%r in (
 2,-1,0
) do (
 echo Opening ppt %%r
 if %%r LSS 1 (
	start "" "e:\timer\timer.ppt"
 )
 ping -n 2 localhost 1>nul 2>nul
 cls
)
rem 启动vbs脚本执行f5
start "" "e:\timer\keyf5.vbs"
rem 开始倒计时
for /L %%a in (
 %id%,-1,0
) do (
 echo PPT demo %%a
 if %%a LSS 1 (
	if %name% EQU 1 (
		echo kill WPS
		taskkill /F /IM wpp.exe
	)
	if %name% EQU 2 (
		echo kill OFFICE
		taskkill /F /IM POWERPNT.EXE
	)
 )
 ping -n 2 localhost 1>nul 2>nul
 cls
)

popd

keyf5.vbs 代码如下:
 
Set objShell = CreateObject("WScript.Shell")
objShell.SendKeys "{F5}"
