Cryptonex——任何国家的货币与任何加密货币或代币能够兑换

Cryptonex(CNX)是在其自身区块链平台上开发的一种全球化的去中心加密货币。主要目的是使世界上任何国家的货币与任何加密货币或代币能够兑换，通过手机和支付卡能买到理想的的商品和服务。快速，并且佣金最低。

1.转换

9月份推出网页钱包后，CNX，BTC和Ether的存储将可实现。我们计划添加最流行的加密货币。只使用一个网页钱包，用户便可以访问所有流行的加密货币。
由于在后台即时转换，用户将避免长时间的等待，因为它发生在许多交易所上，并且将具有执行交易的担保。如果平衡至少一种加密货币是积极的，则用户能够使用支付卡或移动应用程序。该应用程序将显示法定货币和加密货币的价格，所以用户可以选择使用哪种钱包来支付。硬件钱包（闪存驱动器）将发布以确保大量的加密货币的安全。

2.交易

Cryptonex计划于2017年10月至11月在交易所上市。目前，我们已经达成了加密货币交换协议，具有eth / btc，cnx / btc，cnx / eth，以及类型btc / usd，cnx / usd等。未来，计划收购交易所，可以利用其所有功能：存款，提款，购买付款等。

3.加密货币

我们将增加收购“长期”，动态开发的和最受欢迎的加密货币。

4.采矿池

在连接第三方采矿池后，用户将能够在网络钱包中获得奖励，并采用所有功能：在商店购物等。

5. API

通过连接到高速采集API，可以根据目的使用区块链和其他互联网服务。

6.银行

我们计划在新加坡或另一个属于友好管辖的国家建立或购买现成的银行。所有许可证的文件的费用为40万至900 000美元。注册时间需要1-4个月（取决于具体任务——收购或建立）。这也包括与Visa /Mastercard支付系统的连接以及发卡给有意愿的客户。具有非接触式付款和虚拟卡的移动应用将是最优选的。同时我们计划增加第三方银行收购。

7.卡、虚拟和现实

由于在后台的瞬时交换，用户可以即时获取虚拟卡或实卡上的资金，并可以在账户之间进行支付和交易。

8.移动应用

它将包括网页钱包和非接触式付款的功能。它将适用于iOS和Android。

9.保护

Cryptonex使用加密算法Scrypt。它保护所有交易和钱包免受外部黑客攻击。由于我们的技术解决方案，木马和其他恶意程序无法找到网络钱包的重要文件。访问是被加密的，强烈建议将密码保存在安全的地方。更重要的是，采矿程序提供了安全性，因为所有节点都对区块进行验证，如果信息失去同步，则可疑交易被拒绝。
Cryptonex的一些特殊功能在其他加密货币中已实现，例如Dash借记卡和Cryptopay。然而，没有将所有的资金流都聚集在一个功能完整的区块链中。Cryptonex团队将所有当前的技术整合到了平台和后台。这允许只用在一个地方执行主要的操作。我们保证交易的速度和安全性。

因此，Cryptonex可总结如下：
1.积累来自当地比特币网络，加密货币交易所和法定货币交易所的流量。
2.方便了存款/取款和交换手续。
3.整合了区块链资金流动，为块链社区增长提供新的机遇。
