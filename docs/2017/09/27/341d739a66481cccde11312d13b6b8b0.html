记者：铅笔盒

简单了解区块链

区块链本质上讲是一种基于哈希密码学的分布式数据库技术，作为一种底层协议，将在改造传统业务的很多痛点和未来业务的应用上，发挥很大的作用。利用哈希密码学原理搭建的区块链网络具有信息不可篡改、各节点集体维护、去中心化、可编程性、开放性等特点，因此使得智能合约、数字资产的确权等等成为可能。


区块链可以应用的领域

因为区块链具备这些特性，我们不仅可以加强互联网金融行业，诸如众筹、互联网保险、互联网衍生品交易的安全，还能对传统金融监管提供强大技术支撑。区块链在非金融行业的应用场景也在逐步形成。区块链技术可以和教育、社会管理、文物鉴定、电子票据、医疗健康、慈善公益等很好的结合起来，也可以为物联网、人工智能、云计算、大数据、下一代通信等前沿技术的快速发展提供解决思路。


当前区块链的发展情况

目前，区块链行业在世界范围内还处于初创阶段，但是未来前景无限，这一点是毋庸置疑的。当前，国内的布局主要是三种类型的企业：

一、做区块链应用平台相关的企业，如小蚁；

二、做企业级应用场景的企业，如我司，即指旺金科；

三、做媒体资讯类，如区块链铅笔等；


区块链未来可能发展的方向

在币圈的光华随着国家的监管禁令褪去之后，区块链技术企业应该深耕细作，专注做好底层技术、场景设计、接口技术等的研发，并配合国家相关部门搭建起严格、完善行业监管框架，为区块链行业更健康的发展和实现其“下一代互联网”的功能性应用做出努力。随着区块链技术的成熟和知名度的上升，到被公众完全接受这一过程并不漫长，因此目前将是区块链应用技术企业布局的大好时机，我司将针对区块链未来应用场景加强这方面的人才培养，只有靠团队合作才能赢得未来。也将会投入更多的人力物力去支持区块链的发展。

10月15日，指望金科将举办一场针对区块链人才的专场招聘会。

招聘岗位如下：

区块链社区运营

岗位要求：

1. 具备良好的网络营销技巧，对网络销售的系统管理有一定经验，有论坛运营经验优先；

2. 3年以上服务平台运营经验，熟悉平台操作规范和流程，熟悉平台商务合作模式，并能积极促进合作；

3. 具备良好的沟通能力、学习能力及逻辑思维能力；

4. 抗压性强，具有快速准确的反应能力和分析能力。


区块链产品经理

岗位要求：
1、 本科及以上学历，互联网或金融行业背景，熟悉区块链金融业务者优先；
2、 5年以上工作经验，其中2年以上产品设计经验；
3、 优秀的沟通、理解、学习、分析能力，执行力强，善于组织协调并推动项目进度；
4、 有优秀的交互原型设计能力，熟练使用Axure等原型设计工具；
5、 敏锐的行业洞察力，熟悉互联网用户行为习惯，具有产品创新思考能力；
6、 具备优秀的沟通、协调、组织能力，有较强责任心与抗压性。


区块链研究员

岗位要求：

1、2年以上金融应用开发经验，或相关的工作经历；1年以上架构工作经验；

2、熟悉多种数据结构和算法，开发语言不限；

3、有相关金融分析软件、交易软件开发经验者优先；

4、熟悉区块链/加密算法者优先；

5、对技术研究有兴趣有热情，欢迎各类极客和潜在极客。


区块链售前经理

岗位要求：
1.本科及以上学历，5年以上IT行业工作经验，3年以上软件售前工作经验；
2.有独立完成大型项目售前工作的成功案例，具备优秀的客户方案演讲能力；
3.具备优秀的产品和方案撰写能力，熟练使用OFFICE办公软件，能制作优秀的PPT方案；
4.具备行业市场分析能力、逻辑思维能力、分析问题、人际沟通协调及解决问题的能力；
5.对区块链技术有一定了解，以及区块链技术发展趋势，了解常见区块链的技术架构。
