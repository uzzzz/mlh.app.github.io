比特股Bitshares Web Wallet 创建账号并修改Memo Key

  • 柚子社区
  1. 打开Bitshares Web Wallet网站 https://wallet.bitshares.eu/，以下示例使用Testnet Wallet实现，点击“创建账号”
  2. 选择“使用钱包模式”
  3. 输入账号名称和钱包密码，点击“创建账号”按钮

  4. 账号创建成功后，建议备份账号信息

    4.1 升级账号为“终身会员”（可选）

    4.2 以刚刚创建的“终身会员”为推荐人，创建新账号

  5. 账号创建完成后，检查账号配置，会发现“活跃权限密钥”和“备注密钥”相同，需要修改备注密钥，推荐使用cli_wallet生成密钥对

    5.1 通过命令行打开cli_wallet，连接至witness node （以Testnet witness node为例）

    programs/cli_wallet/cli_wallet -s wss://node.testnet.bitshares.eu

    如果报错：

    Remote server gave us an unexpected chain_id
    {"remote_chain_id":"39f5e2ede1f8bc1a3a54a7914414e3779e33193f1f5693510e73cb7a87617447","chain_id":"fd8f04f922054673fb92c154cbf473202529a188a87aac080f7451e8cb55a8b5"}

    复制remote_chain_id中的内容粘贴到下面的–chain-id后面：

    programs/cli_wallet/cli_wallet -s wss://node.testnet.bitshares.eu --chain-id 39f5e2ede1f8bc1a3a54a7914414e3779e33193f1f5693510e73cb7a87617447

    5.2 设置钱包密码

    >>> set_password 密码

    5.3 解锁钱包

    >>> unlock 上一步设置的密码

    5.4 生成钥匙对

    >>> suggest_brain_key
{
  "brain_priv_key": "EXCESS XXXX XXXX XXXX XXXX XXXX XXXX XXXX CLAMPER",
  "wif_priv_key": "5Ka3s*******************************************rPLNg",
  "pub_key": "TEST8A******************************6p91VQcm"
}

    5.5 复制wif_priv_key中的内容，导入到Web Wallet中，并修改“备注密钥”的公钥为pub_key中的内容（每个账号的备注公钥都应执行一次7.4, 7.5的操作进行修改）

  6. 全部修改完成后，一定要备份！下载备份文件并妥善保存。
