第一次获取github上go语言编写的一个项目遇到问题的处理

  • 柚子社区

我安装的是最新的go语音版本1.9，下载的是go1.9.windows-amd64.msi 安装文件。文件需要到https://golang.org/ 去下载。由于众所周知的原因，需要翻墙，否则网站打不开。

安装很顺利。安装完后打开一个cmd窗口，输入：

go version


说明已经安装成功。

想着到github上找一个go语言编写的项目弄到本地查看编译下。找到一个https://github.com/json-iterator/go ，在如何获取地方看到：

go get github.com/json-iterator/go

这是要在cmd 窗口直接输入这行命令就能获取。我于是试了下。可是，输入命令后看着是在运行，但是运行结束后，在我的go语言安装目录的，src ,pkg等下面都没找到，这难道没安上？还是哪里有问题。各种查找。最后终于找到一个解决办法。需要自己创建一个工作区目录。为什么不使用安装目录，是因为要保证安装目录的纯洁性。

我在F盘创建了一个工作目录：F:\web code\golangWorkspace ，目录必须是英文，不能有汉字。在这个目录中还要创建3个目录。分别是src，pkg，bin三个目录。并且要在windows系统变量中增加GOPATH，值设置为：F:\web code\golangWorkspace



然后再打开一个cmd窗口，输入：

go get github.com/json-iterator/go

执行成功后，可以看到在工作区的src目录下已经有了相应的项目文件。在github.com目录下。



至此问题得以解决
