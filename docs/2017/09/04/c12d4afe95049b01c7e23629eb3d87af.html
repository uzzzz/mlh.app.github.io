3.安装JSON插件（直接安装或在线插件安装均可）

3-1.直接安装
下载安装包https://github.com/dzhibas/SublimePrettyJson，解压缩到Packages目录（菜单/Sublime Text->Preferences->Browser Packages…),然后重启即可

3-2.在线插件安装Package Control
按Ctrl+ `(此符号为tab按键上面的按键) 调出console（注：避免热键冲突）
粘贴以下代码到命令行并回车：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

安装完成后重启。

3-3.安装Pretty JSON
菜单/Sublime Text->Preferences->Package Control，选择Package Control:Install Package，输入Pretty JSON，按下回车完成安裝，最后重启。

4.使用：
windows : ctrl + alt + j 组合键格式化json字符串
Mac : ctrl + command +j 组合键格式化json字符串

end
