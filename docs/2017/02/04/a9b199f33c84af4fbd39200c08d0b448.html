南宁市二手房直接交易流程(一)

  • 柚子社区

       2017年2月4日上午办理了房屋过户，现在写一下流程，为有需要的朋友提供指导。

       大部分人买房还是通过房屋中介进行，这样的话资金有所保障，但是也要花费一大笔的中介费。我这里就遇到了这样的问题，经过和卖方协商我们决定不通过中介双方进行交易（这种方式还是不推荐大家使用，毕竟使用中介的话会安全很多且会省去很多麻烦事）。废话少说，这里直接说办理过户流程。我这篇博客只写到完成过户手续，只是针对我个人的交易过程，大概分为3个过程。

       一、房管局领取房屋合同并查询个人名下房屋情况（房管局地址：南宁市园湖北路35号房产大厦C座一楼、三楼）

1.1 买卖双方到房管局一楼查询夫妻双方名下房产情况（5~8号窗口都可办理），会拿到一个房产情况证明。这一步主要针对首套房的买方或者卖方，有了这个证明在交税的时候会有优惠。买方的首套房证明会少交一部分契税，卖方的首套房证明会少交一部分的个人所得税。如果不是首套房，则不需要这个证明。

1.2 到房管局3楼8号窗口购买3份房屋交易合同（2块钱一份），这里办理房屋过户必须使用房管局提供的合同。3楼有问询处，如果有不懂的问题可以到那里询问，她们会给你一张办事指南，会告诉你办理过户需要提供哪些材料。这里对于房屋交易合同进行一下简单的说明，房屋交易合同中有关于中介以及首付款交给第三方托管的条文。如果买卖双方没有通过中介，则中介那里可以不填写。首付第三方托管信息那里也可以不填写，之前房管局有第三方托管平台，不过现在没有了，现在房管局才不管你们之间是如果交易首付款的。听说银行有第三方托管，我也没有弄个，我把首付直接交给了卖方，卖方给我写了一个收条（这里不推荐大家使用这种方式，这种方式并不安全，还是要看卖方是不是诚实可信）。

1.3到房管局1楼税务问询处查询交税情况，她们会给你一个交税指南，告诉你到哪里交税以及交税需要携带什么材料。房管局1楼只能交纳一手房的税，不能交纳二手房的税。

      二、地税局交税

办理房屋过户前需要先到地税局交税。我家住在青秀区，就去了离我家最近的地税交税点（南宁市青秀区民族大道89号金禄大厦3楼）交纳的地税。这里需要携带的材料包括：买卖双方的身份证原件以及复印件、房产证原件以及复印件、房屋交易合同原件以及复印件、买卖双方的房屋查询证明（1.1步中得到的证明）。虽然交税的那里免费复印，但是为了节省时间还是建议大家去办理的时候就直接把需要的材料都复印好。交税成功后会拿到完税证明以及交税申请表。

      三、房管局办理过户

  交税完成后，带着交税证明以及其它的材料（包括买卖双方身份证原件以及复印件、房产证、购房合同等）到房管局3楼办理过户手续。过户手续办理完成后会得到一个房产证领取凭条，等房产证办理好后房管局会通过短信的方式通知买方到房管局领取不动产证，买方带着这张凭条以及身份证就可以到房管局领取不动产证了。


        我这里交税以及办理过户只花费了不到两个小时的时间，只要带齐了所需要的材料，办理还是非常快的。接下来等房产证办下来之后，我要带着房产证去办理贷款给卖方钱。办理公积金贷款的步骤等我办理完成后更新博客以供大家参考。
