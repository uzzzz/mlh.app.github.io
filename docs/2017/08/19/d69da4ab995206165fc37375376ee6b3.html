【Docker】将RancherOS安装到硬盘

  • 柚子社区

RancherOS镜像默认使用内存来加载，如果需要安装到硬盘，方法如下

下载IOS镜像

  • 官网地址：https://github.com/rancher/os

用puttygen生成RSA的公钥和私钥

  • 公钥取名为：rsa_public
  • 私钥取名为：rsa_private

编辑配置文件

  • 在内存中启动RancherOS镜像
  • 执行：vi cloud-config.yml 编辑配置文件
#cloud-config

rancher:
    network:
        interfaces:
            eth0:
                address: 192.168.88.105/24
                gateway: 192.168.88.254
                mtu: 1500
                dhcp: false

ssh_authorized_keys:
    - ssh-rsa 公钥内容粘贴写在这里

用命令安装RancherOS到硬盘中

  • 执行：sudo ros install -c cloud-config.yml -d /dev/sda

用私钥登陆并修改用户密码

  • 刚装好的系统的rancher用户不知道密码，所以必须用刚刚创建的私钥登陆并修改
  • 打开putty，在auth设置中载入刚刚生成的私钥，用户为rancher

  • 登陆后，执行命令：sudo passwd rancher
  • 输入两次新的密码
  • 安装完成
