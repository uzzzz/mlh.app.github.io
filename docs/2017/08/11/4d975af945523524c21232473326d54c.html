从接收端ccs导出解调前的14652点数据（64QAM），数据见下方，然后在MATLAB中画出星座图：

f = ccs2matlab('demodIn_IQversion_TxTest1F_shgao.dat');
plot(f,'.r')

其中ccs2matlab函数将ccs中导出的16进制的32 bit数据转换为复数，首先将ccs导出数据中前面6个头去掉，然后将实部和虚部取出，并且将其中的补码转换为负数，最后组成复数，程序如下： 

function[r] = ccs2matlab(filename)

fid = fopen(filename, 'r');
t  = fscanf(fid,'%x');
fclose(fid);
r = t(7:end);
Q_data = floor(r/65536);
I_data = r-Q_data*65536;
Q_data(find(Q_data > (2^15-1))) = -2^16 + Q_data(find(Q_data > (2^15 -1 )));
I_data(find(I_data > (2^15-1))) = -2^16 + I_data(find(I_data > (2^15 -1 )));
r = complex(I_data, Q_data).';
