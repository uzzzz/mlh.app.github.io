区块链赚钱攻略：最新干货 区块链人士必读！

  • 柚子社区

 区块链已经成为金融领域的“网红”选手，无论是金融机构还是技术公司，甚至是普通的网民，他们迫切的关注着这个新兴技术，希望凭借这个新兴的风口，快速的积累起财富。虽说风口上猪都能飞上天，但怎么飞？这也是问题。笔者作为区块链从业人员，提供几个思路，帮助大家打开脑洞。

 

1、炒币

 

数字货币的涨幅，这个就不用说，4月份的时候数字货币的总市值才250亿美元，6月份的时候就破千亿。如果有位投资者足够聪明或者足够幸运，在2010年的时候向比特币投入1000美元，现在，他就可以成为千万富翁。




当然，币圈存在一夜暴富的神话，也同样存在币价大幅跳水，腰斩再腰斩的“惨案”。So，投资有风险，理财需谨慎！

 

2、Ico

如果说炒币类似于炒股，那ico就类似于IPO，团队在GitHub上上传一些源码，整个几页白皮书，差不过就可以对外发行代币，币民按照一定比率拿比特币或者eth换取项目代币。

 

在过去的一个月里，参与ico几乎就是躺着挣钱的代名词，每个ico项目代币出后之后，都是5倍、十倍甚至百倍的利润。当然，随着募集到比特币或者eth的团队的“砸盘”行为，和获利资金的退出，逐渐出现代币发行之后无人接盘导致的“破发”现象。

而且ico因为缺乏相关政策监管和行业准则，ico项目鱼龙混杂，建议投资之前多做功课。

 

3、创业

没错，扎扎实实学习技术，尽职尽责做市场调研，努力找投资，做产品，做推广……虽然区块链行业ico很容易圈到一笔钱，但投ico的人又大多只是普通投资者，在专业程度和人脉资源上，不如投资机构。区块链毕竟是个有潜力的技术，一些政府部门对其很支持，比如贵阳市政府，鼓励区块链企业上市，最高奖励1千万元。

 

如果你不是只想圈一波钱就走，想切实做一个长远的事业，那你可以选择这条目测很艰难险阻、但很有意义的道路。国内布萌区块链就是这样做的，拜访了几百家企业，依据场景需求来对区块进行封装，帮助企业快速建立区块链应用，目前有超过千万的用户。像布萌这种公司，在风口上找到一条合适的路，融资也不是那么难的事情。

 

4、辅助

玩过游戏的，都懂辅助的概念，那就是，当别人要在风口上飞的时候，你在旁边提供翅膀，那绝对是为赚不赔的买卖。炒币赚钱，那来做个交易所吧，手续费还是可以收收的；现在ico比较火？那众筹网站可以做做的，跟项目团队谈个分成也是有可能的；创业？那肯定需要曝光度吧，办办行业展会、培训什么的，相信进账也不少……

 

区块链不仅仅只是一项技术，它更大的意义在于其所蕴含的自由、民主的理念，这种理念可以充分调动人们的积极主动性，从而创造更大的价值。因此，上面几种方式完全不能覆盖区块链领域赚钱的方式，主要给大家抛砖引玉。赚钱方式千千万，实际操作才最难，最后，祝愿大家都能在这个领域吃到新的“蛋糕”。
