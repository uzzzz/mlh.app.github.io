一、远程调用与本地事务区分开(别人挂, 我们不挂)

悲观锁

乐观锁
select version from xxxx where id = xxx;
update xxx set xxx = xxx where id = xxx and version = 'version';

基于状态机的乐观锁
int i = update xxx set status = 4 where id = xxx and status = 1;
i = 1 锁成功, i = 0 锁失败

三、交易申请

方案 1
@事务{
    结果=调用()
    保存(结果)
}

方案2
@事务{
    锁=锁(订单)
}
if(锁){
    结果=调用();

    @事务{
        保存(结果)
    }
}

方案3 (重点)
@事务{
    锁=锁(订单)
    if(锁){
      申请 = 插入(申请)
    }
}

//============================ 可重复执行, 例如定时任务
if(申请){
    @事务{
        锁 = 锁(申请);
    }
    if(锁){
      结果=调用();
    }
    @事务{
      保存(结果)
    }
}
