Go言学习系列--开发工具（二）

  • 柚子社区

       前面的一节简单的介绍了Go语言的开发工具之一LiteIDE，现在用eclipse配置Go语言的开发环境，因为eclipse也是一个比较流行的开发工具，所以这章简单的介绍一下eclipse怎么配置Go语言的开发环境。

       首先从https://www.eclipse.org/downloads/下载eclipse，这个找相关的文章很容易就配置成功啦，我本地一直使用，所以本身就已经安装配置好了，现在简单介绍怎么配置Go语言的开发环境。

       双击打开eclipse，然后点击菜单栏上的 help -> install new software，如下图所示：

然后点击弹窗中的 Add 按钮，在小弹创中Name输入 goclipse，location为：http://goclipse.github.io/releases/，点击ok

然后会加载出一下的插件（要是没有出来可能换地址了或者网址被屏蔽了，那只能找新的地址啦），选择需要的插件，点击 next 按钮


最后点击同意，这个没办法，用别人的东西，不同意用不了，然后点击 finish


然后等待下载安装，这个过程可能有点长，耐心等待。


安装完成之后会提示重新启动eclipse，启动之后点击 window->preferences，打开的新弹窗中可以看到一个Go的菜单，如下图所示：



到这里就是配置一些其他的了，所以还得下载其他的插件

下载gocode，用于go的代码补全提示 下载地址：https://sourceforge.net/projects/mingw/files/MinGW/Base/stats/timeline

下载msysgit，用于把gocode生成gocode.exe文件，这个需要配置环境变量

GOROOT  C:\Go //安装的目录

GOOS  windows //操作系统

GOARCH  amd64

GOPATH  D:\go\test  //编译成的gocode.exe的目录

PATH 增加C:\Go\bin //安装执行的目录

我的除了GOPATH 以外，其他的自动生成，msysgit安装完成之后找到msysgit的安装目录下的git-cmd.exe文件，双击打开当前的文件，运行下面的代码

go get -u github.com/nsf/gocode
GOPATH 目录下的bin目录中生成gocode.exe文件，然后把它考到go安装目录的bin目录下。
然后再Go菜单下配置各种文件对应的EXE的地址，基本就可以啦，就可以创建一个第一个Go语言的开发程序啦。
