  •    LiteIDE

                下载地址：  https://sourceforge.net/projects/liteide/

                简介：LiteIDE is a simple, open source, cross-platform Go IDE（LiteIDE是一个简单，开源，跨平台的 go 语言 IDE），IDE(Integrated Development Environment) 为 集成开发环境。

                点击 download 按钮它会自动下载，我下载的之后的文件为

             解压完成之后，双击bin文件夹里面的 liteide.exe（八卦图 ( ⊙ o ⊙ )！） 文件就可以运行了。打开liteide工具之后，点击菜单上的 文件 -> 新建 -> 选择对应的模板、项目或文件名称、保存的位置（如下图所示）就可以了。我选择的模板为 “Go Source File”

           保存完成之后，点击工具栏上的 “R” 按钮就可以运行啦，点击最下面的一排按钮中的 “3：编译输出”，就可以看出输出的结果。

          默认的代码：

// helloword
package main

import (
	"fmt"
)

func main() {
	fmt.Println("Hello World!")
}
