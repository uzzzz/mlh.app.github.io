Qt for Mac：配置/搭建开发环境

  • 柚子社区

经常看到有人在问，所以干脆整理下，其实步骤很简单。
我用的环境，在写稿时都是最新的，分别是：macOS 10.12.4、Xcode8.3.2和Qt5.8.0，都是在虚拟机里弄的。

准备

  • 首先要准备一个macOS系统，这个是真机或者虚拟机都可。我这里用了虚拟机，安装步骤我就跳过不讲了。

PS：我制作macOS系统安装U盘的命令是

sudo /Applications/Install\ macOS\ Sierra.app/Contents/Resources/createinstallmedia --volume /Volumes/macOS --applicationpath /Applications/Install\ macOS\ Sierra.app --nointeraction

  • 其次要准备一个AppleID，用于到AppStore下载Xcode
    如果你不想到AppStore下载，可以前往官网下载xip包，解压后直接用即可：
    https://developer.apple.com/downloads/

  • 然后下载Qt的安装包，下载地址为：
    http://download.qt.io/official_releases/qt/5.8/5.8.0/qt-opensource-mac-x64-clang-5.8.0.dmg
    如果你不想要这个安装包，可以前往官网找其他的：
    https://www.qt.io/download-open-source/#section-3

环境配置

  • 打开AppStore，搜索Xcode

  • 点击左上角的Get，登录AppleID后进行下载和安装

  • 启动一次Xcode，点击Agree，安装剩余的组件

如果看到了这个主界面，说明Xcode配置好了

  • 安装Qt，下一步下一步即可

  • 打开QtCreator，编译工程并运行，完成！
